# 致大学生与初入职场者的一封信

> 写给那些站在起点、充满困惑又满怀期待的你

---

亲爱的你，

如果你正在读这封信，也许你刚刚拿到毕业证，或者刚刚开始第一份工作，或者正站在某个十字路口，不知道该往哪里走。

我想告诉你一些没有人在课堂上教过你的事。

---

## 关于"选择什么专业/职业"这件事

**别人的答案是别人的，你的答案要自己找。**

这个时代充满了职业排名、薪资榜单、"最有前途的10个行业"。但请记住：所有这些榜单，描述的都是**过去**，而你要活的是**未来**。

真正重要的问题不是"哪个行业赚钱"，而是：

```
你愿意在哪件事上连续失败五年，还不想放弃？
```

不是"你擅长什么"，也不是"父母觉得什么好"，而是你的**抗挫折韧性**在哪里最强。那里，往往藏着你真正的方向。

**实践建议**：
- 毕业前尝试3种不同类型的实习（大公司/小公司/创业公司各一次）
- 找5个你真心佩服的人，问他们"你最后悔没做什么"
- 在正式工作前，用1-2个月做一件让自己骄傲的个人项目

## 关于AI时代的求职现实

坦白说：你们这一代面对的职场，是历史上变化最快的职场之一。

AI正在接管大量"初级工作"：
- 写基础文案 → AI能做
- 分析数据表格 → AI能做
- 翻译文件 → AI能做
- 写模板代码 → AI能做

**这不是要吓你，而是要告诉你：你必须从第一天起就思考"AI做不了什么"。**

你的护城河在这里：

| AI很强的地方 | 你应该建立的能力 |
|-------------|----------------|
| 批量执行已知任务 | 定义"该做什么任务"的判断力 |
| 生成标准内容 | 独特的个人视角和审美 |
| 处理结构化信息 | 读懂人心的情感智慧 |
| 模拟已有答案 | 提出真正新问题的好奇心 |
| 不知疲倦地工作 | 真实关系和信任积累 |

**最重要的一句话**：不要做"会用AI的人"，要做"驾驭AI的人"。前者是工具操作员，后者是方向决策者。

## 关于第一份工作

第一份工作几乎所有人都会后悔——要么觉得选错了，要么觉得薪水低，要么觉得没有成长。

这很正常。

**第一份工作的真正价值不在薪水，在于：**

```
你能不能在18个月内，积累一个"让你讲起来眼睛发光"的成果？
```

一个真实的项目经历、一次带团队解决难题、一次说服别人的成功——这些会跟随你10年。薪水会随着这些翻倍。

**入职第一个月最重要的三件事**：
1. **认识关键的人**：找到公司里最厉害的同事，想尽办法和他们共事
2. **完成一件小事**：快速交付一个虽然小但质量高的成果，建立信任
3. **理解公司真正关心什么**：不是写在PPT上的使命，而是老板每天担心什么

## 关于学历与技能

学历是门票，技能是通行证，判断力是护照。

```
学历让你进入面试房间。
技能让你通过面试。
判断力决定你最终走多远。
```

很多顶尖公司已经悄悄降低对学历的要求——Google、Meta、Apple都已不强制要求本科学位。他们更想要的是：**你能解决什么真实问题的证明**。

所以，无论你的学校是985还是普通院校：

- 做出一个真实作品（代码/设计/报告/视频）比100行简历更有力
- 在某个细分领域建立认知度（写文章、参与开源、做播客）
- 英语+AI工具是这个时代最低成本的"杠杆"，请认真对待

## 关于迷茫这件事

你会迷茫。这不是你的问题，这是正常的。

25岁前没有人真正知道自己该做什么。大多数成功的人，都是在"错误"的路上积累了足够多的经验，才找到了正确的方向。

**迷茫时问自己这三个问题**：

```
1. 我现在做的事，90岁回头看会后悔吗？
2. 如果钱不是问题，我明天会去做什么？
3. 我最近一次忘记时间在做什么？
```

不要急着找到答案。但每隔3个月，认真问一次自己。

## 关于人脉这件事

很多人理解的"人脉"是错的——他们以为人脉是"认识有用的人"。

真正的人脉是：**你对别人有没有真实价值，而不是别人对你有没有用**。

建立人脉的正确方式：
- 帮助比你弱的人（你永远不知道5年后他会成为什么）
- 在比你强的人面前展示真实的你（不是表演，是真诚）
- 维护关系不靠节假日祝福，靠"我看到一篇文章想到了你"

最好的人脉，来自共同做成过一件事。

## 关于钱这件事

请从第一份薪水开始，每月存20%，无论多少。

不是因为你现在需要钱，而是因为：**存钱的能力，本质上是延迟满足的能力**，这个能力会决定你20年后的人生轨迹。

同时：在职业早期，为成长付费是值得的——买好课、去好城市、参加好的社群，这些回报率远超任何理财产品。

## 关于这个AI时代给你的礼物

最后，我想告诉你一件可能出乎你意料的事：

**你们这一代，是历史上拥有最强工具的一代人。**

一个大学生，今天可以用AI：
- 在48小时内做出一个可用的产品原型
- 用1周自学一门新技能到入门水平
- 用1个月完成过去需要一个小组3个月做的研究

这意味着什么？

这意味着：**个人的力量从未像今天这么大。**

过去，你需要进大公司才能做大事。今天，一个人+AI可以创造真实的价值，触达真实的用户，产生真实的影响。

你不需要等待任何人给你机会。你可以从今天开始，**自己给自己制造机会**。

## 写在最后

没有人会告诉你，这条路很容易走。

但我可以告诉你：那些看起来"很成功"的人，几乎没有一个是按计划走到今天的。他们只是：

```
比别人多试了几次，
在该坚持的时候没有放弃，
在该转弯的时候没有死撑，
然后——运气来了的时候，他们恰好在场。
```

保持好奇，保持善良，保持行动。

**你比你想象的更有可能。**

加油。

## 附：给不同阶段的你的行动清单

### 如果你还在大学（距离毕业1-3年）

- [ ] 现在就开始做实习，不要等到大三大四
- [ ] 找到一个你真正感兴趣的细分领域，深挖6个月
- [ ] 开始记录自己的学习和思考（写作/视频/开源项目均可）
- [ ] 学会用AI工具（ChatGPT/Cursor/Midjourney至少掌握1-2个）
- [ ] 建立第一个"证明你能力"的作品集

### 如果你刚入职（0-6个月）

- [ ] 找到公司里最厉害的人，主动请教
- [ ] 在前3个月完成一件让老板印象深刻的小事
- [ ] 理解公司的商业模式：钱从哪来，花在哪里
- [ ] 参加一个跨部门项目，扩大你的内部人脉
- [ ] 建立自己的"每周学习记录"习惯

### 如果你正在迷茫（不知道该干什么）

- [ ] 列出你做过的所有让自己"有成就感"的事
- [ ] 和10个不同行业/职业的人聊天，问他们一天怎么过
- [ ] 报名一个短期课程或黑客松，强迫自己行动
- [ ] 给自己设定一个"90天小实验"：尝试一个新方向
- [ ] 停止"研究"，开始"做"——哪怕做错了也比不做强
