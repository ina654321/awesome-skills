### AI时代的通用生存法则

```
法则1: 用AI做"做了"，用脑子做"想清楚了"
  → AI处理信息，你负责判断和决策

法则2: 选择让AI"更有用"的工作方向
  → AI越难做的事，你越安全（创意/判断/关系/现场）

法则3: 建立"人+AI"的复合竞争力
  → 最好的竞争力：行业深度 × AI工具熟练度

法则4: 持续学习，缩短认知差距
  → AI更新速度极快，停止学习 = 快速落后

法则5: 拥抱变化，不要抵制
  → 抵制AI工具的人，会被使用AI工具的人替代
```

## 通用成长框架

每条职业路径都遵循 **5级成长模型**：

```
🌱 第1级 · 入门 (0-1年)
    ↓ 掌握核心基础，完成首个实际项目
🌿 第2级 · 初级 (1-3年)
    ↓ 独立完成任务，形成专业习惯
🌳 第3级 · 中级 (3-6年)
    ↓ 主导项目，开始带人，形成方法论
🦅 第4级 · 高级 (6-10年)
    ↓ 跨团队影响力，系统化思维，定义标准
👑 第5级 · 精英 (10年+)
    行业影响力，战略视野，培养下一代精英
```

## 如何使用本指南

1. **找到你的职业** — 在上表中选择对应路径
2. **定位当前阶段** — 对照每个级别的能力描述自测
3. **制定90天计划** — 专注攻克当前阶段的关键技能
4. **结合技能库** — 使用 `skills/` 目录中对应的 SKILL.md 加速学习
5. **定期复盘** — 每季度重新对照路径评估进展
