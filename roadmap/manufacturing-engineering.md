# 制造与工程职业路径 Manufacturing & Engineering Roadmap

> 覆盖方向：机械工程师 · 质量工程师 · 生产管理 · 供应链 · 工业工程

---

## 职业全景

```
制造与工程
├── 工程研发
│   ├── 工程师 → 高级工程师 → 主任工程师 → 总工程师 → 首席工程师
│   ├── 机械/电气/化工/材料等方向
│   └── 研发工程师 → 技术专家 → 技术委员
├── 生产制造
│   ├── 操作工 → 班组长 → 车间主任 → 生产经理 → 生产总监 → VP制造
│   └── 工艺工程师 → 高级工艺 → 工艺总工
├── 质量管理
│   ├── 质量检验 → QC工程师 → 质量经理 → 质量总监 → VP质量
│   └── 六西格玛黑带 → 质量顾问
├── 供应链
│   ├── 采购专员 → 采购经理 → 采购总监 → CSO(首席采购官)
│   └── 物流专员 → 供应链经理 → 供应链总监 → CSCO
└── 工业工程(IE)
    └── IE工程师 → 精益专家 → 运营卓越总监
```

## 🌱 第1级 · 入门 (0-12个月)

### 工程师入门（机械为例）
- [ ] CAD软件：AutoCAD / SolidWorks / CATIA基础
- [ ] 工程图纸：读图、标注规范（GB/ISO标准）
- [ ] 材料基础：常用金属/塑料/复合材料的性能
- [ ] 制造工艺：车铣刨磨、冲压、注塑基础
- [ ] 公差与配合：几何公差（GD&T）基础

### 质量工程师入门
- [ ] 质量基础：质量七大工具（鱼骨图、控制图、帕累托等）
- [ ] 检验标准：AQL抽样方案、量具使用
- [ ] 品质文件：SOP、检验规程、FMEA入门
- [ ] 不合格品处理：MRB流程（材料评审委员会）

### 生产管理入门
- [ ] 生产计划：排产逻辑、产能计算
- [ ] 班组管理：日常看板、交接班管理
- [ ] 安全管理：5S现场管理、安全操作规程
- [ ] 精益基础：7大浪费的识别

### 关键里程碑
- ✅ 完成工厂实习或校企合作项目
- ✅ 独立使用CAD完成一个零件设计
- ✅ 参与一次工艺改善提案

### 常见陷阱 ⚠️
- 只重理论不重实操：制造业必须去现场
- 不懂成本：工程师必须有成本意识
- 忽视安全：安全永远是第一位的

## 🌿 第2级 · 初级工程师 (1-4年)

### 机械工程深化
- [ ] CAE仿真：ANSYS / ABAQUS有限元分析基础
- [ ] DFM/DFA：面向制造/装配的设计
- [ ] 产品开发流程：APQP（先期产品质量策划）
- [ ] 样品管理：PPAP（生产件批准程序）
- [ ] GD&T进阶：复杂公差分析、公差链计算

### 电气/电子工程
- [ ] 电路设计：原理图、PCB Layout基础
- [ ] 嵌入式基础：MCU、传感器接口
- [ ] EMC/EMI：电磁兼容基础
- [ ] 安全认证：CE、UL、CCC认证理解

### 质量工程进阶
- [ ] FMEA（失效模式与影响分析）
- [ ] MSA（测量系统分析）：量具R&R研究
- [ ] SPC（统计过程控制）：控制图应用
- [ ] 8D问题解决方法
- [ ] IATF 16949 / ISO 9001体系理解

### 供应链初级
- [ ] 采购流程：供应商选择、询价、谈判、下单
- [ ] 库存管理：EOQ模型、安全库存计算
- [ ] ERP系统：SAP / Oracle MRP模块基础
- [ ] 进出口基础：Incoterms贸易条款

### 关键里程碑
- ✅ 独立主导一个零件/组件的开发（从概念到量产）
- ✅ 通过ISO内部审核员资格
- ✅ 完成一次不良品根因分析并制定改善措施

---

## 🌳 第3级 · 中级 (4-8年)

### 工程专家能力

**技术深化**
- [ ] 专业方向选定：精密制造/新能源/航空/汽车/医疗器械
- [ ] 系统工程：MBSE（基于模型的系统工程）
- [ ] 仿真能力：多物理场仿真（流体/结构/热）
- [ ] 知识产权：专利挖掘、技术申请

**项目领导**
- [ ] 跨部门项目管理：研发/质量/采购/生产协调
- [ ] 成本控制：材料成本降低（Kaizen提案）
- [ ] 供应商开发：SQE能力，帮助供应商提升

### 精益/六西格玛方向

**精益生产**
- [ ] 价值流图（VSM）绘制与分析
- [ ] 快速换模（SMED）：减少停机时间
- [ ] Poka-Yoke（防错设计）
- [ ] TPM（全员生产维护）
- [ ] 拉动式生产：看板系统设计

**六西格玛**
- [ ] 绿带（Green Belt）：DMAIC项目执行
- [ ] 黑带（Black Belt）：复杂问题解决，培训绿带
- [ ] DFSS（面向六西格玛设计）

### 供应链中级
- [ ] S&OP（销售与运营计划）流程
- [ ] 需求预测模型
- [ ] 多级库存优化
- [ ] 风险管理：供应中断应急预案
- [ ] 全球采购策略

### 关键里程碑
- ✅ 获得六西格玛黑带或精益专家认证
- ✅ 主导完成一条新产线的建设（生产方向）
- ✅ 负责一个全球供应商的管理（供应链方向）

---

## 🦅 第4级 · 高级管理 (8-15年)

### 工程总监/制造总监

**组织管理**
- [ ] 管理50-200人工程/生产组织
- [ ] 年度预算制定与管控
- [ ] 技术路线图：未来3年制造技术方向
- [ ] 工厂规划：产能扩张、自动化投资决策

**智能制造**
- [ ] 工业4.0：IoT传感器、数字孪生、AI质检
- [ ] MES系统：制造执行系统设计与实施
- [ ]自动化：机器人、协作机器人应用
- [ ] 数据驱动制造：实时监控、预测维护

**卓越运营**
- [ ] OEE（设备总体有效性）持续提升
- [ ] 质量成本（COPQ）优化
- [ ] 碳中和制造：减排路径制定

### 供应链总监/CSCO

- [ ] 端到端供应链可视化系统建立
- [ ] 供应链韧性策略：多源采购、近岸制造
- [ ] 供应链金融：应收账款、动态折扣

---

## 👑 第5级 · 制造精英

### VP/SVP Manufacturing
- 🏆 管理多个工厂/事业部的制造运营
- 🏆 主导工厂自动化转型（降低人工成本30%+）
- 🏆 将工厂OEE提升至85%以上（世界级水平）
- 🏆 建立卓越制造中心（CoE），向全公司推广

### 首席供应链官（CSCO）
- 🏆 重塑全球供应链网络布局
- 🏆 在重大危机（疫情/地缘政治）中保障供应稳定
- 🏆 将供应链能力转化为公司竞争优势

## 🤖 AI时代的制造与工程规划

### 工业4.0与AI制造的现状

```
已在工厂落地：
  ✅ 机器视觉质检：AI视觉检测替代人工目检（速度10x，准确率更高）
  ✅ 预测性维护：传感器+AI预测设备故障（减少计划外停机80%）
  ✅ AI排产优化：考虑多约束的智能排产（比人工排产效率高15-30%）
  ✅ AI质量控制：实时SPC+AI异常预警
  ✅ 数字孪生：AI驱动的工厂数字镜像，实时优化生产参数

即将普及：
  - 机器人自动化：协作机器人（Cobot）+ AI视觉，完成更复杂任务
  - AI工艺优化：材料/参数自动调优（无需人工试验）
  - 自动化质检报告：AI自动生成每批次质量报告
  - 智能仓储：AGV+AI路径规划+语音拣货
```

### 各角色AI融合建议

**工程师 — AI是设计和分析的加速器**
```
设计阶段：
  - AI辅助CAD：Autodesk Fusion AI / SOLIDWORKS AI 拓扑优化
  - 生成式设计：输入约束条件，AI生成最优结构
  - 仿真加速：AI Surrogate Model（替代模型）代替耗时的CFD/FEA
  - 文档生成：AI自动生成设计说明书、BOM清单

分析阶段：
  - 故障分析：Claude帮你分析故障树，生成初步RCA报告
  - 数据分析：Python + Claude 快速处理测试数据
  - 专利检索：AI快速扫描相关专利，识别侵权风险

差异化能力（AI不能替代）：
  ✅ 工程直觉：多年经验积累的"感觉对不对"
  ✅ 现场解决问题：突发故障的临机应变
  ✅ 跨专业协调：机械/电气/软件的综合协调
```

**质量工程师 — AI提升质检精度和效率**
```
AI质量应用：
  - 机器视觉：外观缺陷（划痕/气泡/变形）的AI自动检测
  - 过程异常检测：SPC控制图的AI智能告警
  - 预测性质量：生产参数→产品质量的AI预测模型
  - 供应商质量：AI分析历史来料数据，预测供应商风险

质量人员的新技能要求：
  → 理解AI质检系统的原理（知道AI的局限和误报原因）
  → 会标注训练数据（为AI质检模型提供高质量标注）
  → 建立AI质检的验证规范（如何测试AI准确率）
```

**生产管理 — 智能工厂的调度优化**
```
AI在生产运营的应用：
  - 智能排产：基于需求预测+产能约束的AI动态排产
  - 能耗优化：AI调节设备运行参数，降低单位能耗
  - 人员调度：AI根据订单、人员技能、出勤预测优化排班
  - OEE分析：AI自动分析停机原因，提供改善优先级建议

管理者的转型：
  ❌ 旧角色：手工收集数据、人工分析报表
  ✅ 新角色：监督AI系统输出，做高层次改善决策
  → 学习数字化工厂管理系统（MES/ERP的AI模块）
```

**供应链 — AI驱动需求感知和风险管控**
```
AI供应链应用：
  - 需求预测：AI结合市场信号+历史数据，精度提升20-40%
  - 库存优化：动态安全库存计算，减少积压同时防缺货
  - 供应商风险：AI监控供应商财务/舆情/交付数据
  - 物流优化：AI路径规划+实时调度（司机App集成）

新兴机会：
  🔥 供应链数字化顾问：帮中小企业部署AI供应链工具
  🔥 供应链AI产品经理：制造业专用供应链AI产品
```

### 行动建议

1. **立即**: 了解你工厂/公司已有的AI质检或预测维护系统，主动学习原理
2. **1个月**: 学习 Python 基础数据分析（pandas/matplotlib），处理你的生产数据
3. **3个月**: 用 AI 工具完成一个质量分析或生产优化的小型项目
4. **6个月**: 主导一个数字化改善项目（哪怕是一个工位的数据采集可视化）
