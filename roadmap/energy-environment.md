# 能源与环境职业路径 Energy & Environment Roadmap

> 覆盖方向：新能源工程师 · 环境顾问 · 碳管理 · 可持续发展 · 传统能源

---

## 职业全景

```
能源与环境
├── 新能源
│   ├── 太阳能：光伏工程师 → 系统设计师 → 项目总监 → 新能源事业部总裁
│   ├── 风能：风机工程师 → 风场设计 → 运营总监
│   ├── 储能：电池工程师 → 储能系统架构师 → 储能事业部负责人
│   └── 氢能：氢能工程师 → 制氢/储氢/用氢技术专家
├── 传统能源
│   ├── 石油天然气：勘探/开采/炼化工程师 → 油田总工程师
│   └── 电力：电力工程师 → 电网规划师 → 电力系统专家
├── 环境保护
│   ├── 环境工程师 → 高级环境工程师 → 环境总监 → CEnO
│   ├── 环境顾问 → 环境合规专家 → 环境战略顾问
│   └── 生态学家 → 生态修复专家
├── 可持续发展/ESG
│   ├── ESG专员 → ESG经理 → ESG总监 → CSO(首席可持续官)
│   ├── 碳资产管理师 → 碳市场交易员 → 碳战略总监
│   └── 可持续发展顾问 → 绿色金融分析师
└── 核能
    └── 核工程师 → 反应堆工程师 → 核安全专家
```

## 🌱 第1级 · 入门 (0-12个月)

### 新能源工程师入门

**光伏方向**
- [ ] 光伏基础：太阳能电池原理、IV特性曲线
- [ ] 系统组成：组件/逆变器/支架/电缆/汇流箱
- [ ] 电气基础：直流/交流系统、并网规范
- [ ] 设计软件：PVsyst基础使用（光伏模拟）
- [ ] 标准规范：NEC / IEC 61215 / 62446

**储能方向**
- [ ] 电池化学：锂离子电池工作原理、SOC/SOH
- [ ] BMS（电池管理系统）基础
- [ ] 储能系统拓扑：集中式/模块化储能
- [ ] 安全规范：热失控机制和防护

### 环境工程入门
- [ ] 环境科学基础：大气/水/土壤污染原理
- [ ] 环境法规：环保法、排放标准、EIA制度
- [ ] 监测技术：水质分析、大气采样
- [ ] 环评基础：建设项目环境影响评价流程

### ESG/可持续发展入门
- [ ] ESG框架：GRI标准、SASB、TCFD
- [ ] 碳核算：GHG Protocol（温室气体核算体系）
- [ ] 碳市场基础：EU ETS、中国碳交易市场
- [ ] 可持续发展报告：ESG报告阅读和分析

### 关键里程碑
- ✅ 参与一个太阳能/储能/环评项目实习
- ✅ 通过初级环境评价师/碳资产管理师考试
- ✅ 能独立完成一个小型光伏系统设计

---

## 🌿 第2级 · 初级 (1-4年)

### 光伏/新能源工程师

**技术深化**
- [ ] 大型地面电站设计：百兆瓦级项目设计经验
- [ ] 屋顶分布式方案：工商业/户用系统方案
- [ ] 电网接入：并网申请、继电保护、无功补偿
- [ ] 设备选型：组件/逆变器/跟踪支架选型对比
- [ ] 施工监督：电气施工规范、安全措施

**项目开发**
- [ ] 资源评估：光照数据分析（NASA POWER / 气象局）
- [ ] 项目前期：土地可行性、电网接入条件
- [ ] 财务模型：光伏项目IRR/NPV计算
- [ ] 补贴政策：各省市新能源政策解读

### 储能工程师进阶
- [ ] 储能电站系统集成：PCS/BMS/EMS协调控制
- [ ] 调频/调峰应用场景设计
- [ ] BESS安全标准：GB/T 36276、UL 9540
- [ ] 电池Cycling测试与寿命预测

### 环境工程进阶
- [ ] 水处理技术：物化处理/生化处理工艺
- [ ] 大气污染控制：除尘/脱硫/脱硝技术
- [ ] 固废管理：危废处置、垃圾填埋/焚烧
- [ ] 土壤修复：污染场地调查与修复技术
- [ ] 环保验收：竣工环保验收流程

### ESG/碳管理进阶
- [ ] 企业碳盘查：范围1/2/3排放核算
- [ ] 碳减排路径：SBTi目标设定方法论
- [ ] ESG尽职调查：投资方ESG评估
- [ ] 绿色建筑：LEED / 中国绿建标准

### 关键里程碑
- ✅ 独立设计并推动一个50MW+光伏项目落地
- ✅ 取得注册环保工程师/环境评价工程师
- ✅ 完成企业碳盘查项目并出具报告

## 🌳 第3级 · 中级 (4-8年)

### 新能源项目总监

**技术领导**
- [ ] 多技术方案比选：光伏/储能/风电组合优化
- [ ] 技术标准制定：公司内部设计规范
- [ ] 关键设备谈判：组件/逆变器采购谈判
- [ ] O&M（运营维护）策略：预测性维护、数字化运维

**商务与融资**
- [ ] 电力交易：现货市场、中长期合同、绿证
- [ ] 项目融资：银行贷款、绿色债券、基础设施基金
- [ ] EPC合同：关键条款谈判（性能保证、违约条款）
- [ ] 政府关系：地方政府沟通，政策争取

### 环境总监

- [ ] 建立公司环境管理体系（ISO 14001）
- [ ] 环境风险管理：事故预防和应急预案
- [ ] 政府监管关系：环保局、生态环境部的沟通协调
- [ ] 绿色供应链：供应商环保要求制定

### ESG总监

- [ ] 董事会ESG报告：TCFD信息披露
- [ ] ESG评级提升：MSCI/Sustainalytics/CDP
- [ ] 气候风险分析：物理风险/转型风险量化
- [ ] 绿色金融：绿色债券发行、可持续发展挂钩贷款

### 关键里程碑
- ✅ 主导100MW+新能源项目开发到并网全流程
- ✅ 带领团队完成公司级碳中和路径规划
- ✅ ESG评级提升至行业前25%

## 🤖 AI时代的能源与环境规划

### AI对能源环保行业的变革

```
新能源+AI的融合：
  ✅ 光伏发电预测：AI气象预测+发电量预测，精度提升30%
  ✅ 储能调度：AI优化充放电策略，延长电池寿命+提升收益
  ✅ 风功率预测：AI超短期功率预测，优化电网调度
  ✅ 设备故障预警：光伏/风机的AI健康监测
  ✅ 电力现货交易：AI辅助电价预测和交易策略

环保+AI：
  ✅ 碳排放监测：卫星图像+AI计算全球碳排放
  ✅ 环境合规：AI实时监控排污数据，自动报告
  ✅ 新材料发现：AI加速电池/光伏材料的研发
  ✅ 气候模型：AI提升气候预测精度和分辨率
```

### 各角色AI融合建议

**新能源工程师 — AI优化系统性能**
```
光伏方向的AI应用：
  - 智能运维平台：AI识别组件级故障（阴影/热斑/失配）
  - 发电量预测：LSTM/Transformer模型的光伏出力预测
  - AI清洗调度：根据灰尘量预测和收益分析决定清洗时机
  - 虚拟电厂：AI聚合分布式光伏参与电网调峰

必须学会的新技能：
  → Python时序数据分析（pandas / statsmodels）
  → 能量管理系统（EMS）的AI模块配置
  → 电力现货市场：AI辅助交易策略
```

**储能工程师 — AI是最强的电池管理大脑**
```
AI在储能中的核心应用：
  - BMS + AI：SOC/SOH的高精度估计
  - 充放电策略：套利优化（预测电价，最优充放时机）
  - 寿命预测：根据使用模式预测电池剩余寿命
  - 安全预警：热失控前的早期预警（多传感器融合+AI）

新兴热点：
  🔥 储能EMS工程师：AI驱动的储能能量管理系统
  🔥 电池数字孪生：AI构建电芯数字孪生用于寿命管理
```

**ESG/碳管理 — AI重塑可持续发展工作**
```
AI在ESG中的爆发性应用：
  - 碳盘查自动化：AI+ERP数据自动计算Scope 1/2/3排放
  - ESG报告生成：AI自动生成初稿，大幅降低报告编制成本
  - 气候风险量化：AI建模物理风险（洪水/干旱）和转型风险
  - 供应链碳追踪：AI追踪产品从原材料到消费者的全链碳足迹

新职业机会：
  🔥 气候科技（Climate Tech）分析师：
     评估气候科技投资的技术可行性和商业前景

  🔥 碳数据平台工程师：
     建立企业碳数据管理平台（AI+ERP集成）

  🔥 AI气候政策分析师：
     用AI分析政策变化对碳市场和企业合规的影响
```

**环境工程师 — AI赋能环境监测与修复**
```
AI环保应用：
  - 空气质量预测：机器学习模型+气象数据
  - 污染源溯源：AI分析多点监测数据，反推排放来源
  - 生态系统监测：卫星+AI识别植被变化、物种分布
  - 环评AI辅助：AI快速检索相关法规、生成合规清单

环保行业的数字化趋势：
  → 在线监测（CEMS）数据AI分析
  → 无人机+AI的环境巡查
  → 数字化环评报告（减少重复性工作）
```

### 能源+AI的差异化职业定位

```
最具潜力的3个交叉方向：

1. 新能源+AI+金融 = 绿色金融AI分析师
   核心：用AI评估新能源资产价值、建立绿色投资模型

2. 储能+AI+电力市场 = 储能交易工程师
   核心：AI策略 × 电力现货市场 = 套利收益最大化
   稀缺度：极高（懂储能+懂AI+懂电力市场的人很少）

3. 碳管理+AI+合规 = 数字碳资产经理
   核心：碳数据平台 × CCER开发 × 碳交易策略
```
