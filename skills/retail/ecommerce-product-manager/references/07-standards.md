# Standards & Reference

## 7.1 E-Commerce Platforms & Documentation

- [Shopify Developer Docs](https://shopify.dev/docs) - Storefront API, Admin API, checkout extensions
- [Magento Developer Documentation](https://developer.adobe.com/commerce/docs) - Adobe Commerce API reference, module development
- [Google Analytics 4 Ecommerce Tracking](https://developers.google.com/analytics/devguides/collection/ga4/ecommerce) - Enhanced ecommerce events
- [Google Tag Manager Ecommerce Guide](https://support.google.com/tagmanager/topic/7185184) - Container implementation
- [Salesforce Commerce Cloud Docs](https://developer.salesforce.com/docs/commerce/commerce-api/overview) - OCAPI and SCAPI references

## 7.2 Conversion Optimization Standards

- [Google HEART Framework](https://research.google/pubs/pub36299/) - User experience measurement framework (Happiness, Engagement, Adoption, Retention, Task Success)
- [Baymard Institute E-Commerce Checkout Research](https://baymard.com/ecommerce-checkout-usability-benchmark) - Industry benchmarks for cart abandonment and checkout optimization
- [CXL Institute Research](https://cxl.com/research/) - Conversion rate optimization studies and A/B testing methodology
- [Nielsen Norman Group - E-Commerce UX](https://nngroup.com/topic/e-commerce/) - Usability heuristics for online retail

## 7.3 Product Information Management

- [PIM (Product Information Management) Standards](https://www.gs1.org/standards/other/identification) - GS1 product identification (GTIN, GPC)
- [Google Merchant Center Requirements](https://merchant.center/) - Product data specification for Shopping ads
- [Schema.org Product Markup](https://schema.org/Product) - Structured data for rich snippets
- [GS1 GPC (Global Product Classification)](https://www.gs1.org/standards/gpc) - Category hierarchy standard

## 7.4 A/B Testing & Experimentation

- [Google Optimize Sunset / GA4 Experiments](https://support.google.com/analytics/answer/12751709) - Server-side experiment API
- [Optimizely Stats Engine Documentation](https://docs.developers.optimizely.com/experimentation/v10.0-core-api) - Statistical methodology for experiments
- [IETF HTTP Batched A/B Testing RFC](https://datatracker.ietf.org/doc/html/rfc9230) - Privacy-preserving batched experiments

## 7.5 Omnichannel & Payment Standards

- [PCI DSS (Payment Card Industry Data Security Standard)](https://www.pcisecuritystandards.org/) - Payment security compliance
- [3D Secure 2.0 (EMVCo)](https://www.emvco.com/emv-technologies/3d-secure/) - Authentication protocol for card-not-present transactions
- [PSD2 (Payment Services Directive 2)](https://ec.europa.eu/info/law/payment-services-psd-2-sca/regulation-eu-2018189_en) - European open banking standard
- [Stripe API Reference](https://stripe.com/docs/api) - Payment processing integration standards
