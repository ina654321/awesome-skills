## 6. Scenario Examples

### Example 1: Inventory Turnover Optimization

**Context:** Regional manager needs to improve inventory turnover from 6x to 8x annually.

**Analysis Framework:**
```
Current State Assessment:
  - Average inventory value: $50M
  - Annual COGS: $300M
  - Current turns: 6.0x
  - Target turns: 8.0x
  - Required inventory reduction: $12.5M

Optimization Strategies:
  1. Velocity-based slotting
     - A-items (top 20%): Store in prime locations
     - B-items (next 30%): Standard locations
     - C-items (bottom 50%): Remote or drop-ship

  2. Supplier lead time reduction
     - Negotiate faster replenishment
     - Increase order frequency
     - Reduce safety stock requirements

  3. Markdown optimization
     - Identify slow-movers at 60 days
     - Strategic markdowns vs holding costs
     - Clearance routing decisions

Expected Outcomes:
  - Inventory reduction: $12.5M
  - Carrying cost savings: $2.5M/year (20% rate)
  - Stockout risk: Mitigated via predictive analytics
```

### Example 2: Shrinkage Reduction Program

**Context:** Store experiencing 1.8% shrinkage; target is <1.0%.

**Root Cause Analysis:**
```
Shrinkage Categories:
  External Theft (40-45%):
    - Shoplifting at self-checkout
    - Organized retail crime
    - Return fraud

  Internal Theft (30-35%):
    - Sweethearting at registers
    - Inventory manipulation
    - Cash handling fraud

  Process/Administrative (20-25%):
    - Pricing errors
    - Receiving discrepancies
    - Damage/spoilage

Action Plan:
  Week 1-2: Install EAS tags on high-shrink categories
  Week 3-4: Implement exception-based reporting for registers
  Week 5-6: Deploy CCTV analytics for blind spot coverage
  Week 7-8: Audit receiving processes with triple-count verification
  Month 3: Measure shrink reduction; adjust tactics
```

### Example 3: Cross-Docking Network Design

**Context:** Opening new distribution center; need cross-docking layout.

**Design Specifications:**
```
DC Characteristics:
  - Size: 1.2M sq ft
  - Service radius: 200 miles
  - Stores supported: 95
  - Conveyor length: 20km

Staging Area Design:
  - Inbound docks: 150
  - Outbound docks: 200
  - Cross-dock staging: 40% of floor space
  - Storage reserve: 60% of floor space

Cross-Dock Flow:
  1. Truck arrives at inbound dock
  2. Unload to staging lane (by store destination)
  3. Quality check and label verification
  4. Load to outbound truck (within 24-48 hours)
  5. Deliver to store

Performance Targets:
  - Dock-to-dock time: <24 hours
  - Inventory accuracy: 99.5%+
  - Cost per case: $0.15 (industry leading)
```

### Example 4: EDLP vs High-Low Strategy Decision

**Context:** Competitor launching aggressive promotional pricing; defend EDLP position.

**Strategic Analysis:**
```
EDLP Advantages:
  - Customer trust and predictable shopping
  - Supply chain efficiency (level demand)
  - Lower advertising costs
  - Operational simplicity

High-Low Risks:
  - Training customers to wait for sales
  - Demand volatility strains supply chain
  - Margin destruction during promotions
  - Promotional fatigue

Walmart Response Framework:
  1. Maintain EDLP positioning (do not match promotions)
  2. Highlight Rollback pricing on key items
  3. Emphasize total basket value vs cherry-picking
  4. Leverage Walmart+ for loyalty
  5. Communicate Savings Catcher value

Expected Outcome:
  - Short-term: Some market share loss to cherry-pickers
  - Long-term: Customer return when promotion ends; EDLP trust intact
```

### Example 5: Private Fleet Efficiency Initiative

**Context:** Transportation costs rising; need to improve driver utilization.

**Current State:**
```
Driver Utilization Analysis:
  - Industry average driving: 6.2 hours/day
  - Walmart current: 8.5 hours/day
  - Legal maximum: 11 hours/day
  - Gap to capture: 2.5 hours/day

Improvement Initiatives:
  1. Route optimization software upgrade
     - Real-time traffic integration
     - Dynamic rerouting
     - Predictive delivery windows

  2. Load planning enhancement
     - Multi-stop sequencing
     - Backhaul utilization
     - Trailer cube optimization

  3. Driver scheduling reform
     - Flexible start times
     - Weekend premium incentives
     - Preferred route bidding system

Projected Results:
  - Driving hours increase: 8.5 -> 9.5 hours/day (+12%)
  - Miles per driver: +8% annually
  - Cost per mile: -5%
  - Driver satisfaction: Improved through better routes
```
