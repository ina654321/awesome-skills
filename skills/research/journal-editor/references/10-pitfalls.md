# Journal Editor Templates, Checklists & Protocols

Practical tools for associate editors managing manuscript peer review.

## Manuscript Initial Assessment Checklist

When a new manuscript arrives in your assignment queue:

**Basic Checks:**
- [ ] Does the topic fit the journal's scope and readership?
- [ ] Is the article type appropriate (research article, review, etc.)?
- [ ] Is the abstract structured and informative?
- [ ] Does the introduction clearly state the research question or hypothesis?
- [ ] Are methods described with sufficient detail for replication?
- [ ] Are figures and tables clear, properly labeled, and necessary?
- [ ] Are references complete, current, and from appropriate primary sources?

**Technical Quality:**
- [ ] Is the English language quality adequate for peer review?
- [ ] Are statistical methods appropriate and described?
- [ ] Is the sample size adequate for the stated conclusions?
- [ ] Are limitations acknowledged and discussed?
- [ ] Is the literature review current and comprehensive?

**Ethical Compliance:**
- [ ] Are conflict of interest disclosures included?
- [ ] Is ethics approval documented (IRB, IACUC) for human/animal subjects?
- [ ] Is clinical trial registration documented if applicable?
- [ ] Are CONSORT, STROBE, or other reporting guidelines followed?

**Initial Decision Options:**
- [ ] If all criteria met: Assign reviewers
- [ ] If minor issues: Request pre-review revision
- [ ] If major issues: Desk reject with constructive feedback
- [ ] If scope mismatch: Transfer to appropriate journal

## Reviewer Selection Protocol

Finding qualified, conflict-free reviewers:

**Identification:**
- [ ] Search recent reference list for qualified experts
- [ ] Check journal's reviewer database for relevant subject areas
- [ ] Review recent publications in the field from other journals
- [ ] Use author-suggested reviewers if qualified (verify independently)
- [ ] Check Publons/ORCID for reviewer track record
- [ ] Consider international reviewers for diverse perspectives

**Conflict Check:**
- [ ] Verify no institutional affiliation overlap with authors
- [ ] Check for recent co-authorship or collaboration
- [ ] Confirm no professional rivalry or competing interests
- [ ] Do not invite reviewers from the same research group

**Invitation:**
- [ ] Send personalized invitation with manuscript abstract
- [ ] State expected timeline (typically 2-3 weeks for research articles)
- [ ] Provide review criteria and structured review template
- [ ] Offer Publons recognition or reviewer acknowledgment
- [ ] If no response in 5-7 days: send one reminder
- [ ] If still no response: invite alternative reviewer

**Minimum Panel:**
- [ ] Minimum 2 independent reviews for most manuscripts
- [ ] 3+ reviews for manuscripts with conflicting recommendations
- [ ] Statistical review for complex quantitative studies
- [ ] Senior/methodology expert for novel or high-risk studies

## Review Assessment Protocol

Evaluating reviewer reports:

**Quality Assessment:**
- [ ] Does the review identify strengths and weaknesses?
- [ ] Are specific, actionable comments provided?
- [ ] Are concerns supported with evidence from the manuscript?
- [ ] Does the review address all key elements (significance, methods, results, interpretation)?
- [ ] Is the reviewer's recommendation justified by their comments?

**Constructiveness Check:**
- [ ] Are comments constructive and respectful?
- [ ] Would the author find the feedback useful for revision?
- [ ] Are suggestions proportionate to the manuscript's quality level?

**Recommendation Analysis:**
- [ ] Does reviewer recommendation match their comments?
- [ ] Are there significant discrepancies between reviewers?
- [ ] Do reviewer recommendations align with your independent assessment?

**Decision:**
- [ ] **Accept** (unanimous enthusiastic support + methodologically sound)
- [ ] **Minor Revision** (all reviewers positive; small changes needed)
- [ ] **Major Revision** (reviewers identify fixable problems; worth revising)
- [ ] **Reject** (fundamental flaws or insufficient quality for this journal)

## Revision Evaluation Protocol

Assessing author responses to reviewer comments:

**Author Response Review:**
- [ ] Read the author's point-by-point response before re-reading the revised manuscript
- [ ] Note which comments were addressed and which were not
- [ ] Determine if non-responses are adequately justified

**Revised Manuscript Review:**
- [ ] Verify changes are consistent throughout the manuscript
- [ ] Check that new or modified figures/tables are labeled correctly
- [ ] Confirm new data or analyses are integrated appropriately
- [ ] Verify that supplementary materials are referenced and complete

**Evaluation of Response:**
- [ ] Has the author addressed all substantive reviewer concerns?
- [ ] Are remaining concerns minor enough to not require another review?
- [ ] Are any new issues introduced by the revision?

**Decision:**
- [ ] If all concerns addressed: Accept
- [ ] If minor concerns remain: Verify yourself and Accept
- [ ] If substantive concerns unaddressed: Return for another revision or Reject
- [ ] If new issues introduced: Determine if new review is needed

## Conflict of Interest Protocol

Managing COIs in the review process:

**For Associate Editors:**
- [ ] Review all author affiliations upon manuscript assignment
- [ ] Declare any conflicts before assigning reviewers or making decisions
- [ ] Recuse yourself if financial, professional, or personal conflict exists
- [ ] Notify EIC if conflict arises during review
- [ ] Document all COI declarations in the manuscript system

**For Reviewers:**
- [ ] Check COIs upon receiving invitation
- [ ] Decline immediately if conflict exists (no need to elaborate)
- [ ] Request withdrawal from review if conflict discovered after acceptance
- [ ] Do not inform authors of your withdrawal reason

**Managing Real vs. Perceived COIs:**
- [ ] Disclose even minor professional relationships (e.g., same conference, informal contact)
- [ ] Err on the side of disclosure when uncertain
- [ ] Consult with EIC if uncertain about recusal requirements

## Handling Problematic Situations

### When a Reviewer is Unresponsive:
1. Send reminder at 50% of deadline
2. If no response by deadline, send second reminder
3. If still no response after 1 week, proceed with available reviews
4. Invite alternative reviewer if only 1 review received
5. Consult EIC for single-review decisions

### When Reviewer Reports are Hostile or Unprofessional:
1. Do not forward hostile language to authors
2. Edit to remove personal attacks while preserving substantive concerns
3. Add editorial note acknowledging tone concerns
4. Do not exclude substantive concerns just because of tone

### When Reviewers Disagree Significantly:
1. Assess the methodological validity of each position
2. Consider inviting a third reviewer
3. Apply editorial judgment weighing the evidence
4. Document reasoning in the decision letter

### When Plagiarism or Image Concerns Arise During Review:
1. Consult COPE flowchart for the specific concern type
2. Request original data and images from author
3. Use image analysis tools for image integrity concerns
4. Refer to EIC if serious concern confirmed
5. Document actions and timeline

## Decision Letter Templates

### Major Revision Letter Structure:
1. **Opening**: Thank authors for submitting; state decision clearly
2. **Overall Assessment**: Brief summary of the manuscript's contribution and key issues
3. **Major Issues**: Numbered list of substantive concerns requiring resolution
4. **Minor Issues**: Numbered list of smaller concerns
5. **Reviewer Comments**: Direct quotes or paraphrased summaries of reviewer feedback
6. **Requirements**: Explicit list of what must be addressed for acceptance
7. **Timeline**: Revision deadline (typically 4-8 weeks)
8. **Closing**: Encourage authors to contact editor if questions arise

### Rejection Letter Structure (Post-Review):
1. Thank authors for submission
2. State the decision
3. Summarize key reasons for rejection (do not list all reviewer comments verbatim)
4. If appropriate, suggest specific journals or improvements
5. Be professional and constructive
6. Note that the decision is final

### Transfer Letter Structure:
1. Acknowledge submission
2. State that manuscript is better suited for [Target Journal]
3. Explain rationale (scope, audience, article type)
4. Note that the target journal will receive the manuscript and reviews
5. Obtain author consent if required by journal policy
