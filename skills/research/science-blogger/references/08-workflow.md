# Standard Workflow

## 8.1 Content Creation Process

### Phase 1: Topic Selection (30 minutes)

**Step 1.1 - Monitor Literature**
- Set up Google Alerts for key terms
- Follow relevant journals and researchers
- Check trending topics in your field

**Step 1.2 - Assess Timeliness**
- Is there a news hook?
- Is it relevant now?
- Will it be relevant in a week? Month?

**Step 1.3 - Audience Fit**
- Who is this for?
- What's their knowledge level?
- What do they want to learn?

### Phase 2: Research & Outline (60-90 minutes)

**Step 2.1 - Source Verification**
- Read primary paper(s)
- Check for retractions or corrections
- Verify data in multiple sources

**Step 2.2 - Key Points Identification**
- 3-5 key takeaways
- One central message
- Supporting evidence

**Step 2.3 - Structure Planning**
- Decide format (thread, article, video)
- Outline key sections
- Plan visuals

### Phase 3: Drafting (45-60 minutes)

**Step 3.1 - Hook Creation**
- Start with surprising fact or question
- Make it attention-grabbing
- Set expectations

**Step 3.2 - Body Writing**
- Use plain language
- Include analogies
- Add supporting data

**Step 3.3 - Call to Action**
- Ask for engagement
- Link to source
- Invite questions

### Phase 4: Refinement (15-30 minutes)

**Step 4.1 - Fact-Check**
- Verify all claims
- Check numbers
- Confirm sources

**Step 4.2 - Readability**
- Run through Hemingway App
- Remove jargon
- Shorten long sentences

**Step 4.3 - Platform Optimization**
- Add appropriate hashtags
- Time for audience activity
- Preview before posting

## 8.2 Personal Brand Building

### Step 1: Platform Strategy
- Choose primary platform
- Optimize profile
- Define content pillars

### Step 2: Consistency
- Post 3-5 times per week
- Stick to schedule
- Maintain voice

### Step 3: Engagement
- Respond to comments
- Engage with others
- Build community

### Step 4: Analytics
- Track performance
- Adjust based on data
- Double down on what works
