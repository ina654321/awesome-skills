# Standard Workflow

## 8.1 Grant Application Process

### Phase 1: Pre-Award (FOA Release to Submission)

**Step 1.1 - Funding Opportunity Discovery**
- Monitor NIH Guide, NSF Program Announcements
- Set up alerts for relevant FOAs
- Review paylines and success rates

**Step 1.2 - Internal Deadline Calculation**
- Agency deadline minus 2 weeks = SRO submission
- Agency deadline minus 8 weeks = LOI (if required)
- Agency deadline minus 2 weeks = internal deadline

**Step 1.3 - Specific Aims Development**
- Draft 1-page Specific Aims
- Get co-investigator feedback
- Align with program priorities

**Step 1.4 - Research Strategy Writing**
- Significance: Why does this matter?
- Innovation: What's new?
- Approach: How will you do it?

**Step 1.5 - Budget Development**
- Personnel: salary + fringe
- Equipment: >$5K threshold
- Supplies, Travel, Other
- Subcontract: first $25K in MTDC

**Step 1.6 - SRO Review and Submission**
- Submit to SRO 2 weeks before deadline
- Address compliance issues
- Submit via Grants.gov

### Phase 2: Post-Award Execution

**Step 2.1 - Award Receipt**
- Review Notice of Award
- Set up budget in institutional system
- Establish account numbers

**Step 2.2 - Project Setup**
- IRB/IACUC protocol activation
- Personnel hiring
- Resource allocation

**Step 2.3 - Monthly Monitoring**
- Budget burn rate tracking
- Milestone completion
- Personnel effort verification

**Step 2.4 - RPPR Preparation**
- Section B: Accomplishments
- Section C: Products (publications, data)
- Section D: Participants
- Section F: Changes

### Phase 3: Project Closeout

**Step 3.1 - No-Cost Extension (if needed)**
- Submit NCE request 30 days before end date
- Justify why work not complete
- Get program officer approval

**Step 3.2 - Final Reporting**
- Final RPPR
- Final invention statement
- Final financial report

**Step 3.3 - Record Retention**
- Maintain records 3 years after end
- Ensure public access compliance
- Archive all data
