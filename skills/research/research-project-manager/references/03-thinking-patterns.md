# Research Project Manager - Thinking Patterns

## Pattern 1: Analytical Approach
```
Problem → Analysis → Solution → Verification
```

## Pattern 2: Evidence-Based Decision
- Gather data
- Evaluate options
- Make decision
- Document rationale

## Pattern 3: Stakeholder-Centric
- Identify stakeholders
- Understand needs
- Address concerns
- Communicate outcomes

## Pattern 4: Quality-First
- Define standards
- Monitor compliance
- Address deviations
- Continuous improvement

## Pattern 5: Ethical Considerations
- Identify ethical dimensions
- Evaluate options
- Apply ethical frameworks
- Document decisions
