# Common Workflow Issues & Troubleshooting

## 8.1 Terminology Inconsistency Problems

| Problem | Cause | Solution |
|---------|-------|----------|
| Different terms for same concept | Multiple translators, no glossary | Create term list; apply consistently |
| Field-specific jargon misused | Lack of domain knowledge | Verify with author; consult field terminology database |
| Acronyms undefined | Missing first-use expansion | Define on first mention; build acronym list |
| Trade name vs generic | Unclear preference | Ask author; default to generic unless specified |

**Example Fix:** For "catalyst" in a paper:
- ❌ Using: "catalyst", "catalytic material", "catalytic agent" interchangeably
- ✅ Standardize: Use "catalyst" throughout; list synonyms in glossary

## 8.2 Structural Flow Issues

| Issue | Symptom | Fix |
|-------|---------|-----|
| Paragraph-level Chinglish | Word-for-word translation pattern | Restructure sentences; lead with new information |
| Topic drift | Tangential sentences | Return to main point; cut or relocate tangents |
| Logical gaps | Missing transitions | Add bridging phrases; restructure sequence |
| Overly long sentences | Chinese run-on sentences preserved | Break into 2-3 shorter English sentences |
| Passive/active confusion | Awkward passive constructions | Convert to active voice where natural |

## 8.3 Cultural Adaptation Challenges

| Challenge | Example | Solution |
|-----------|---------|----------|
| Chinese rhetorical conventions | Indirect conclusions | Add explicit concluding statements |
| Hierarchical acknowledgment | Lengthy honorifics | Condense to Western format |
| Idiomatic expressions | Chinese-specific metaphors | Replace with equivalent or explain |
| Citation patterns | Author-date confusion | Standardize to target journal format |
| Regional terms | Mainland vs Taiwan vs HK terms | Clarify which variant to use |

## 8.4 Journal-Specific Formatting Problems

| Issue | Detection | Correction |
|-------|-----------|------------|
| Reference style mismatch | Check target journal's author guidelines | Reformat all references |
| Figure labeling error | "Figure 1" vs "Fig. 1" | Match journal convention |
| Unit format wrong | "10 mg/kg" vs "10 mg kg⁻¹" | Use journal's preferred format |
| Word count excess | Exceeds journal limit | Work with author on cuts |
| Section order wrong | Methods before Results | Restructure per journal requirements |

## 8.5 Response Letter Formatting Issues

| Problem | Indicator | Fix |
|---------|----------|-----|
| Missing reviewer points | Response shorter than review | Create point-by-point grid |
| Defensive tone | Apologetic or argumentative language | Reframe to collaborative tone |
| Unclear location | "We addressed this" without specifics | Cite page/line/figure |
| Over-explaining | Multiple paragraphs per point | One clear response per concern |
| Formatting inconsistency | Different styles per reviewer | Use consistent format throughout |

## 8.6 Quality Control Workflow

```
Step 1: Terminology Check
- Load project glossary
- Search document for undefined terms
- Verify acronym expansions

Step 2: Flow Read-Through
- Read aloud (or text-to-speech)
- Flag unnatural passages
- Check logical transitions

Step 3: Completeness Verification
- Cross-reference each paragraph with source
- Check all figures/tables referenced
- Verify all data points transferred

Step 4: Format Compliance
- Apply journal template
- Check reference style
- Verify word count

Step 5: Final Proofread
- Grammar/spell check
- Verify figure labels
- Check for orphan citations
```
