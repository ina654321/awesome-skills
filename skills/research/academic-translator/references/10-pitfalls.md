# Templates, Checklists & Protocols

## 10.1 Translation Quality Checklist

### Pre-Translation Checklist
- [ ] Source document received and saved with date/version
- [ ] Target journal identified and author guidelines obtained
- [ ] Technical terminology list created or verified
- [ ] Ambiguous passages flagged for author clarification
- [ ] Reference style requirements confirmed

### During Translation Checklist
- [ ] Glossary terms applied consistently
- [ ] All figures/tables translated and labeled
- [ ] Numbers converted correctly (Chinese vs. Western)
- [ ] Unit formats standardized
- [ ] Chemical names follow IUPAC nomenclature
- [ ] Species names italicized with correct Latin binomials
- [ ] Drug names capitalized per INN convention
- [ ] Acronyms defined on first use

### Post-Translation Checklist
- [ ] Full read-through for natural English flow
- [ ] Chinglish phrases identified and corrected
- [ ] Logical flow verified between paragraphs
- [ ] Reference list formatted correctly
- [ ] Word count within journal limits
- [ ] All author queries resolved
- [ ] Final spell-check completed

## 10.4 Common Chinese-to-English Translations

| Chinese Term | Standard English | Notes |
|-------------|------------------|-------|
| 显著 | significant (statistically), notable (general) | Distinguish context |
| 增加/提高 | increase, enhance, improve | Use specific verb |
| 减少/降低 | decrease, reduce, diminish | Use specific verb |
| 证明/表明 | demonstrate, show, indicate | Demonstrate preferred for results |
| 影响 | affect, impact, influence | Affect = verb; impact = noun/verb |
| 处理 | treat, treat with, process | "Treat cells with drug" not "process" |
| 检测 | detect, measure, assay | Use method-specific term |
| 观察 | observe, observe that | Observe + noun; note that + observation |
| 通过实验 | experimentally, through experiments | Use adverb form |

## 10.5 Abstract Template by Field

### Biological/Biomedical
```
[Background] [Gap in knowledge] [Objective] 
[Methods] [Study design, subjects, methods, primary outcome]
[Results] [Key findings with statistical significance]
[Conclusions] [Clinical/practical implications]

Word count: 250-300
```

### Chemistry
```
[Context] [Problem addressed] [Objective]
[Methods] [Synthesis/characterization/analysis approach]
[Results] [Key compounds/properties/mechanisms discovered]
[Significance] [Implications for the field]

Word count: 150-250
```

### Engineering
```
[Context] [Problem or need] [Objective]
[Methods] [Approach/design/experimental framework]
[Results] [Performance metrics, comparison with alternatives]
[Conclusions] [Practical applications, limitations]

Word count: 200-300
```

## 10.6 Journal Submission Protocol

```
Phase 1: Pre-Submission (1-2 days)
□ Select target journal based on scope and impact
□ Download and review author guidelines
□ Prepare manuscript in journal template
□ Format references per journal style
□ Prepare figures at required resolution
□ Draft cover letter
□ Obtain all co-author approvals

Phase 2: Submission (1 day)
□ Create account on journal submission system
□ Upload manuscript file(s)
□ Complete metadata (title, abstract, keywords)
□ Enter author information with ORCIDs
□ Select reviewers (if journal allows)
□ Upload cover letter
□ Verify PDF proof
□ Submit

Phase 3: Post-Submission
□ Track manuscript status
□ Respond to editor inquiries promptly
□ Prepare revision if requested
□ Respond to reviewer comments point-by-point
□ Submit revised manuscript with tracked changes
□ Approve final proofs
```
