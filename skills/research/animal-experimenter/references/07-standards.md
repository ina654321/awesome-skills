# Standards & Reference

## 7.1 Animal Care & Use Regulations

- [AVMA Guidelines for Euthanasia](https://www.avma.org/resources-tools/avma-policies/avma-guidelines-euthanasia) — American Veterinary Medical Association approved methods
- [Guide for the Care and Use of Laboratory Animals (8th ed.)](https://grants.nih.gov/grants/olaw/guide-for-the-care-and-use-of-laboratory-animals.pdf) — NIH/OLAW institutional guide
- [Animal Welfare Act Regulations (9 CFR, Chapter 1)](https://www.ecfr.gov/current/title-9/chapter-I/subchapter-A/part-2) — USDA animal welfare standards
- [PHS Policy on Humane Care and Use of Laboratory Animals](https://grants.nih.gov/grants/policy/phs-policy.htm) — Federal policy for PHS-funded institutions
- [IACUC Protocol Requirements](https://olaw.nih.gov/) — Office of Laboratory Animal Welfare guidance

## 7.2 IACUC Protocol Requirements

| Requirement | Description | Reference |
|-------------|-------------|-----------|
| **Personnel qualification** | Training records, certification for procedures | PHS Policy IV.C.1 |
| **Rationale for species** | Scientific justification for species selected | Guide p.27 |
| **Justification of numbers** | Power analysis or statistical justification | Guide p.29 |
| **Pain management** | Analgesic plan with drugs, doses, schedule | AVMA Guidelines |
| **Humane endpoints** | Criteria for euthanasia or removal | Guide p.32 |
| **Sedation/anesthesia** | Method with drug, dose, monitoring | Protocol section |
| **Surgical procedures** | Aseptic technique, pre/post-op care | Guide p.43 |
| **Euthanasia method** | Per AVMA guidelines, by species | AVMA Guidelines |

## 7.3 Injection Parameters for Common Species

| Route | Mouse | Rat | Rabbit | Volume Limit |
|-------|-------|-----|--------|-------------|
| **IP** | 25-27G, 0.2-0.5 mL | 22-25G, 1-3 mL | 20-22G, 5-10 mL | <2% body weight |
| **SC** | 25-27G, 0.1-0.2 mL/site | 22-25G, 0.5-1 mL/site | 20-22G, 0.5-2 mL/site | <2% body weight |
| **IV (lateral tail)** | 27-30G, 0.1-0.2 mL | N/A | N/A | Slow injection |
| **IM (quadriceps)** | 27G, 0.05 mL | 25G, 0.1-0.2 mL | 22G, 0.25-0.5 mL | <10% muscle volume |
| **Oral gavage** | 20-22G, 0.2-0.5 mL | 16-18G, 1-3 mL | 14-16G, 5-15 mL | 1-2 mL/100g |

## 7.5 Euthanasia Methods by Species

| Method | Rodents | Rabbits | Fish | Amphibians |
|--------|---------|---------|------|------------|
| **CO2** | ✓ (terminal only) | ✓ (terminal only) | ✓ | ✓ |
| **Isoflurane overdose** | ✓ | ✓ | ✓ | ✓ |
| **Pentobarbital** | ✓ 150 mg/kg IP/IV | ✓ 100 mg/kg IV | ✓ 100-500 mg/L | ✓ |
| **Cervical dislocation** | ✓ (<200g) | ✗ | ✗ | ✗ |
| **Decapitation** | ✓ (with guillotine) | ✗ | ✗ | ✗ |
| **MS-222 (tricaine)** | ✗ | ✗ | ✓ | ✓ |

**Note:** CO2 is not recommended as sole method for neonatal rodents, pregnant animals, or species with known aversion to CO2.

## 7.7 Record Keeping Requirements

| Record Type | Retention Period | Required Elements |
|------------|------------------|-------------------|
| Animal orders | 3 years | Species, strain, supplier, dates |
| Treatment records | 3 years | Individual ID, drug, dose, route, date |
| Cage cards | Duration + 3 years | ID, sex, DOB, source, group |
| IACUC protocols | 3 years post-termination | All versions, amendments |
| Mortality/morbidity | 3 years | Autopsies, necropsies, causes |
| Training records | Duration + 3 years | Personnel, dates, procedures |
