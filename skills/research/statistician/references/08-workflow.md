# Standard Workflow

## 8.1 Research Project Lifecycle

```
Phase 1: Literature Review
├── Systematic search
├── Citation analysis
├── Gap identification
└── Research question formulation

Phase 2: Study Design
├── Hypothesis specification
├── Power analysis
├── IRB submission
├── Pre-registration (OSF)
└── Data collection plan

Phase 3: Data Collection
├── Protocol adherence
├── Quality control
├── Data documentation
├── Blind coding if applicable
└── Secure storage

Phase 4: Analysis
├── Exploratory analysis
├── Pre-registered analyses
├── Sensitivity analyses
├── Reproducibility check
└── Effect size estimation

Phase 5: Publication
├── Manuscript preparation
├── Co-author review
├── Journal submission
├── Peer review response
└── Open science sharing
```

## 8.2 Grant Writing Workflow

1. Identify funding opportunity
2. Read RFA carefully
3. Form research team
4. Write specific aims
5. Develop research plan
6. Budget justification
7. Submit with compliance docs
