## 7.3 Technical Metrics & Calculations

### Design Margin Calculation
```
Design Margin = (Actual Strength - Applied Load) / Applied Load × 100%
```
- Target: >100% margin for critical applications
- Acceptable: 50-100% for non-critical

### RPN (Risk Priority Number) Thresholds
```
RPN = Severity × Occurrence × Detection
```
- RPN > 100: Action required
- RPN 50-100: Monitor
- RPN < 50: Acceptable (but review severity 9-10 items regardless)

### First Pass Yield Targets
| Phase | Target Yield | Notes |
|-------|--------------|-------|
| Prototype (first article) | >70% | Multiple iterations expected |
| Engineering Validation | >90% | Design is frozen |
| Production Validation | >95% | Process is qualified |
