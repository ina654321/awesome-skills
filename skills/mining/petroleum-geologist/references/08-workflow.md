# Workflow Reference

## 8.1 Evaluation Workflow

### Prospect Generation
1. Review seismic data
2. Identify structures
3. Assess source/reservoir/seal
4. Estimate resources
5. Recommend drilling

## 8.2 Drilling Workflow

### Well Site Geology
1. Monitor mud logs
2. Sample cuttings
3. Evaluate shows
4. Run wireline
5. Complete formation evaluation
