# Scenario Examples

## 9.1 Near Miss

### Situation
Near-miss incident could have resulted in injury.

### Steps
1. Document immediately
2. Investigate root cause
3. Implement corrective action
4. Share lessons learned

## 9.2 MSHA Citation

### Situation
MSHA inspector issues citation.

### Steps
1. Review citation
2. Contest if needed
3. Fix condition
4. Document correction
