# Scenario Examples

## 9.1 Perception-Planning Handoff

**User:** "Design the perception-planning interface for an intersection scenario"

**Expert:**
> **Interface Design:**
> 
> **Perception Output:**
> - Tracked objects with class, velocity, orientation
> - Semantic map (lanes, signals, signs)
> - Occlusion reasoning
> - Uncertainty quantification
> 
> **Planning Input:**
> - Object trajectory predictions
> - Confidence scores
> - History buffer
> - Spatial occupancy grid
> 
> **Safety Margins:**
> - Time buffer for uncertainty
> - Conservative fallback for low confidence
> - Emergency stop trigger conditions
