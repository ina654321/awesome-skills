## § 7 Workflow: Defense Program Lifecycle

### Phase 1: Strategic Assessment

**Key Activities:**
1. **Segment Identification** — Which GD business segment is relevant?
2. **Customer Mapping** — U.S. DoD, international, or commercial?
3. **Program Status** — Current phase (R&D, production, sustainment)
4. **Financial Context** — Revenue recognition, margin profile, backlog

**✓ Done Criteria:**
- Segment performance accurately characterized
- Customer relationship understood
- Program phase identified
- Financial impact assessed

---

### Phase 2: Competitive Analysis

**Key Activities:**
1. **Market Positioning** — GD's competitive advantages
2. **Technology Differentiation** — Proprietary capabilities
3. **Contract Structure** — Fixed-price vs. cost-plus implications
4. **Risk Assessment** — Technical, schedule, financial risks

**✓ Done Criteria:**
- Competitive dynamics clear
- Differentiation articulated
- Risk factors identified
- Contract structure understood

---

### Phase 3: Financial Evaluation

**Key Activities:**
1. **Revenue Analysis** — Program contribution to segment
2. **Margin Profile** — Operating earnings trajectory
3. **Cash Flow Impact** — Working capital, capex requirements
4. **ROIC Assessment** — Return on invested capital

**✓ Done Criteria:**
- Financial metrics accurate
- Profitability drivers identified
- Capital intensity understood
- Shareholder value impact assessed

---

### Phase 4: Strategic Recommendations

**Key Activities:**
1. **Investment Thesis** — Bull/bear case development
2. **Risk Disclosure** — Transparent assessment of uncertainties
3. **Action Items** — Recommended next steps
4. **Communication** — Executive summary for decision-makers

**✓ Done Criteria:**
- Recommendations specific and actionable
- Risks clearly disclosed
- Communication appropriate for audience
