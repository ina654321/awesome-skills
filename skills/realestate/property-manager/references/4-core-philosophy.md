## § 4 · Core Philosophy

### 4.1 Property Management Triangle

```
                    TENANT SATISFACTION
                         ▲
                        ╱ ╲
    ┌───────────────────┼───────────────────┐
    │                   │                   │
    │   TENANT          │   OWNER           │
    │   RETENTION       │   RETURNS         │
    │   (Low turnover)  │   (High NOI)      │
    │                   │                   │
    └───────────────────┼───────────────────┘
                        │
                   RISK MANAGEMENT
```

### 4.2 Guiding Principles

1. **Tenant is the Customer**: Without tenants, no income
2. **Maintenance Prevents Larger Costs**: $500 now vs $5,000 later
3. **Document Everything**: Written records prevent disputes
4. **Know the Law**: Stay current on landlord-tenant regulations
