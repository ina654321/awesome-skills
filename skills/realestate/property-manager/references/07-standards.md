## 7.2 Property Management Regulatory Standards

| Regulation | Scope | Key Requirements |
|------------|-------|-----------------|
| **Fair Housing Act** | All housing | No discrimination; reasonable accommodations for disability |
| **State Landlord-Tenant Law** | Lease enforcement | Security deposit limits, notice requirements, eviction process |
| **OSHA 29 CFR 1910.1030** | Bloodborne pathogens | Exposure control plan, PPE for staff handling biohazard |
| **EPA Lead RRP** | Pre-1978 units | Certified renovator for paint-disturbing work |
| **ADA Title III** | Common areas | Accessibility for public accommodations |
| **FEMA Flood Insurance** | Special flood hazard areas | Coverage required if in SFHA |
| **State Registration** | Rental licensing | Business license, rental permit, inspection requirements |
| **Habitability Standards** | Implied warranty | Fit for human habitation; working systems, structural integrity |

## 7.3 Financial Performance Standards

| Metric | Formula | Target |
|--------|---------|--------|
| **Occupancy Rate** | Occupied units / Total units | > 95% |
| **Net Operating Income** | Gross income - Operating expenses | Budget-met or exceeded |
| **Operating Expense Ratio** | OpEx / Gross Income | 35–50% |
| **Delinquency Rate** | Past-due rent / Total rent | < 5% |
| **Turnover Rate** | Units vacated / Total units | < 40%/yr |
| **Maintenance Cost per Unit** | Annual maintenance / Unit count | < $1,000/yr (varies) |
| **CapEx Reserve** | Per unit per year | $250–500/yr |
| **DSCR** | NOI / Debt service | > 1.25x |
| **Rent-to-Value Ratio** | Monthly rent / Property value | < 0.8% |

## 7.4 Lease & Documentation Standards

| Document | Purpose | Retention Period |
|----------|---------|-----------------|
| **Residential Lease Agreement** | Governing tenancy | Duration + 3 years |
| **Move-in/Move-out Inspection Form** | Security deposit disputes | 4 years |
| **Lead Paint Disclosure (Pre-1978)** | Federal disclosure requirement | Permanent |
| **Fair Housing Acknowledgment** | Tenant signed | 3 years |
| **Application/Rental Criteria** | Screening documentation | 3 years |
| **Security Deposit Receipt** | Deposit handling | Duration + 2 years |
| **HOA Lease Addendum** | If HOA approval required | Duration + 3 years |
