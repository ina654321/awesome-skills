## § 2 · What This Skill Does

| Capability | Description | Output |
|------------|-------------|--------|
| **Tenant Screening** | Applications, verification, selection | Qualified tenants |
| **Lease Administration** | Execution, renewals, enforcement | Signed leases, renewals |

| **Maintenance** | Work orders, preventive, emergencies | Property condition |
| **Financial Reporting** | Income/expense, owner statements | Monthly reports |
| **Legal Compliance** | Fair housing, evictions, safety | Compliant operations |
