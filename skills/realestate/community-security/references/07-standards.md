## 7.2 Security Standards for Residential Communities

| Standard | Scope | Key Requirements |
|----------|-------|------------------|
| **NFPA 101 Section 4.6** | Egress requirements | Two means of egress from gated areas |
| **NFPA 72 Chapter 10** | Fire alarm systems | Annual testing, monitoring, record-keeping |
| **NFPA 70 Article 700** | Emergency egress lighting | 90-minute battery backup, monthly test |
| **ASTM E2206 — Fence Standards** | Perimeter security | Fence height, material, gate strength |
| **ASIS ORM-1** | Operational risk management | Risk assessment methodology |
| **UL 294 — Access Control** | Electronic access systems | Performance standard for door locks, readers |
| **ASTM F2200 — Automated Vehicular Gate** | Vehicle gates | Safety entrapment protection, emergency access |

## 7.3 Emergency Response Standards

| Metric | Standard | Response Time |
|--------|----------|---------------|
| **Fire response** | NFPA 1710 | 6-min turnout, 8-min arrival (urban) |
| **Medical response** | OSHA 1910.151** | First aid trained personnel on-site |
| **Active threat** | DHS Run-Hide-Fight | Immediate notification, law enforcement |
| **Severe weather** | NOAA/Watches | 15-min shelter-in-place notification |
| **Utility emergency** | Utility contact list | 5-min notification to residents |
| **Evacuation drill** | NFPA 101 | Annual practice, documented |

| Perimeter fence | 2× per shift | CCTV at all gates | Log all incidents |

| Common areas | Hourly | CCTV | Log all observations |
