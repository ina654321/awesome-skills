# Standard Workflow

## 8.1 Security Incident Response Workflow

```
Phase 1: Initial Response (0–5 minutes)
├── Assess situation from safe position
├── Call 911 for any life-safety emergency
├── Notify security supervisor immediately
├── Preserve scene if criminal activity suspected
└── Begin incident documentation: time, location, description

Phase 2: Containment (5–15 minutes)
├── Secure the area: close gates, lock doors, establish perimeter
├── Deploy additional security personnel as needed
├── Brief arriving law enforcement on situation
├── Communicate to residents if area-wide threat (via intercom/app)
└── Account for all known persons in affected area

Phase 3: Investigation (15–60 minutes)
├── Document all details: witness statements, photos, video
├── Obtain suspect description if applicable
├── Review CCTV footage: archive relevant segments
├── Complete official incident report
└── Notify property management and legal counsel

Phase 4: Follow-Up
├── Review footage for suspect identification
├── Coordinate with law enforcement on evidence
├── Update incident log and trend database
├── Brief property management on findings
└── Implement preventive measures if applicable

Phase 5: Documentation & Reporting
├── Complete detailed written report
├── Submit to property management within 24h
├── File insurance claim if property damage
└── Review incident for process improvement
```

## 8.2 Access Control Management Workflow

1. **Visitor management:** All visitors sign in; issue temporary pass; verify ID for suspicious visitors
2. **Delivery protocols:** Couriers directed to package room; no access to residential floors
3. **Key control:** Physical keys inventoried monthly; electronic access logged in real-time
4. **Employee access:** Level-based: front desk, common areas, all floors, restricted areas
5. **Resident access:** FOB/card entry; temporary codes for service personnel; verify identity before granting access
6. **Vendor access:** Coordinated by property management; escorted if unsupervised
7. **Lost/stolen credentials:** Immediate deactivation; incident report; replacement protocol

## 8.3 Fire Alarm Response Protocol

| Alarm Type | Response |
|------------|----------|
| **Smoke detector activation** | Security dispatch to location; check for fire; call 911 if confirmed |
| **Pull station** | Immediately call 911; initiate evacuation protocol; meet fire department |
| **Sprinkler activation** | Call 911; stop water if possible; begin flood response; evacuate area |
| **CO detector** | Evacuate area; call 911; ventilate; do not re-enter until cleared |
| **False alarm (>3rd)** | Notify property management; assess sensor; contact alarm company |
