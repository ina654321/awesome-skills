## 7.2 State Licensing & Transaction Standards

| Standard | Scope | Key Requirements |
|----------|-------|------------------|
| **State Real Estate License Law** | Broker/agent licensing | 40–90h pre-licensing, exam, E&O insurance |
| **RESPA Section 8** | Kickback prohibition | No unearned fees, no title/coverage steering |
| **TILA-RESPA TRID** | Loan estimate/closing disclosure | 3-day review periods, tolerance limits |
| **FCRA** | Credit report use in housing | Permissible purpose required, adverse action notices |
| ** ECOA** | Equal credit opportunity | Cannot discriminate on race, gender, religion, etc. |
| **RESPA State Requirements** | State-specific RESPA compliance | Varies by state |

## 7.3 Fannie Mae / Freddie Mac Eligibility Standards

| Guideline | Conforming Loan | High Balance |
|-----------|-----------------|--------------|
| **Max Loan Amount** | $726,200 (2024) | $1,089,550 (varies by county) |
| **Min Loan Amount** | $1 | $1 |
| **Max LTV** | 97% (primary), 85% (investment) | 95% |
| **Credit Score Min** | 620 (manual), 580 (DU/LP Accept) | 620 |
| **DTI Ratio** | ≤ 45% (manual), DU/LP decision | ≤ 45% |
| **Cash Out Max LTV** | 80% (primary) | 80% |
| **Investment Property** | ≤ 85% LTV, 3–6 months reserves | ≤ 85% LTV |
| **Condo** | HRAP / API eligible | HRAP / API eligible |

## 7.4 Commission & Compensation Standards

| Practice | Status | Notes |
|----------|--------|-------|
| **MLS offers of compensation** | Still permitted (as of 2024) | Under NAR settlement review |
| **Buyer agent negotiated compensation** | Required after Aug 17, 2024 | Must be in buyer representation agreement |
| **Dual agency** | Permitted in most states | Requires consent, limits fiduciary duty |
| **Net listings** | Prohibited in most states | Conflict of interest |
| **Referral fees (intra-state)** | Permitted if licensed | Must be disclosed |
| **Referral fees (out-of-state)** | Varies by state | Check state licensing board |
