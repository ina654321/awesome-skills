## § 7 · Standard Workflow

### Phase 1: Intake & Assessment

| Step | Activity | Done Criteria | Fail Criteria |
|------|----------|---------------|-------------|
| 1 | Issue received | Complaint documented | Verbal only, no record |
| 2 | Risk assessment | Legal risk identified | No legal consideration |
| 3 | Triage | Investigation type determined | Wrong resource assigned |
| 4 | Notification | Legal/HR leadership informed if needed | Escalation failure |
| 5 | Initial response | Complainant acknowledged, timeline given | Complainant ignored |

### Phase 2: Investigation/Resolution

| Step | Activity | Done Criteria | Fail Criteria |
|------|----------|---------------|-------------|
| 1 | Evidence gathering | All relevant documents collected | Incomplete evidence |
| 2 | Interviews | All parties interviewed | Missing key witnesses |
| 3 | Analysis | Facts analyzed, credibility assessed | Jumping to conclusions |
| 4 | Findings | Report with recommendations | No written findings |
| 5 | Decision | Action approved and implemented | Unauthorized action |

### Phase 3: Follow-Up & Prevention

| Step | Activity | Done Criteria | Fail Criteria |
|------|----------|---------------|-------------|
| 1 | Communication | Parties informed of outcome | No closure communication |
| 2 | Documentation | Case file complete | Missing documentation |
| 3 | Monitoring | Follow-up to ensure resolution | No follow-up |
| 4 | Trend analysis | Pattern identified if systemic | Case-by-case only |
| 5 | Prevention | Training/process improvements recommended | No learning applied |
