### Termination Checklist

```
TERMINATION PROCESS CHECKLIST

Before Termination:
□ Documentation review (performance, discipline, complaints)
□ Legal risk assessment (protected class, timing, pattern)
□ Severance calculation (if applicable)
□ COBRA information prepared
□ Final pay calculation (including accrued PTO)
□ Exit logistics (equipment return, access revocation)
□ Witness identified for meeting

Termination Meeting:
□ Private location
□ Manager + HR present
□ Direct, brief communication
□ Reason: [Performance/Conduct/Position eliminated]
□ Final pay timing explained
□ Benefits continuation (COBRA) explained
□ Equipment return instructions
□ Severance agreement (if applicable)
□ Reference policy explained

After Termination:
□ Access revoked immediately (systems, building)
□ Email forwarded/managed
□ Equipment collected
□ Exit interview (if applicable)
□ Team communication (without details)
□ Documentation complete

Legal Review Required If:
- Protected class member with recent complaint
- Age 40+ (OWBPA requirements)
- Recent medical leave or accommodation
- Whistleblower activity
- Union member
```

### Leave Administration Guide

```
FMLA/ADA LEAVE MANAGEMENT

FMLA Eligibility:
- 12+ months employment
- 1,250+ hours in past 12 months
- 50+ employees within 75 miles

FMLA Qualifying Reasons:
- Employee's serious health condition
- Family member's serious health condition
- Birth/adoption/foster care
- Military family leave

Process:
1. Request received (oral or written)
2. Eligibility notice (within 5 days)
3. Medical certification requested (if applicable)
4. Designation notice (within 5 days of cert)
5. Track leave usage
6. Return to work/fitness for duty

ADA Accommodations:
1. Request received
2. Interactive process initiated
3. Medical documentation (if needed)
4. Accommodation options explored
5. Reasonable accommodation determined
6. Implementation and follow-up

Intermittent/S Reduced Schedule:
- Track in hours for non-exempt
- Track in days/weeks for exempt
- Re-certification annually

Return to Work:
- Fitness for duty (if medical leave)
- Same or equivalent position
- Benefits restoration
```

### Manager Coaching Guide

```
MANAGER COACHING ON ER ISSUES

Common Scenarios:

1. Employee Performance Issues:
   - Document specific examples
   - Set clear expectations
   - Regular feedback (weekly)
   - PIP if no improvement
   - Escalate to ER if legal risk

2. Employee Complaint:
   - Listen without judgment
   - Do not promise confidentiality
   - Report to HR immediately
   - Do not retaliate
   - Document conversation

3. Conflict Between Employees:
   - Address quickly
   - Hear both sides
   - Focus on behavior, not personality
   - Set expectations going forward
   - Escalate if unresolved

4. Attendance Issues:
   - Track patterns
   - Address promptly
   - Consider FMLA/ADA if medical
   - Progressive discipline

What Managers CAN Do:
- Set clear expectations
- Provide regular feedback
- Document performance issues
- Recommend disciplinary action

What Managers CANNOT Do:
- Promise job security
- Make accommodation decisions
- Ignore harassment complaints
- Take adverse action after complaint

When to Involve ER:
- Harassment/discrimination complaints
- Requests for accommodation
- Potential terminations
- Any complaint about manager
```
