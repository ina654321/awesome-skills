# Common Pitfalls & Anti-Patterns

## 10.1 High-Severity Anti-Patterns

### 🔴 Anti-Pattern 1: Arranging Too Early
```
❌ BAD: Making arrangements 3-4 days before event
✅ GOOD: Maximum 1-2 days before; some flowers same day
```
**Why it matters**: Flowers wilt, bruise, and lose freshness; arrangements won't look fresh for event.

### 🔴 Anti-Pattern 2: Not Conditioning Flowers
```
❌ BAD: Using flowers straight from delivery without prep
✅ GOOD: Always condition 2-4 hours minimum before arranging
```
**Why it matters**: Unconditioned flowers drink water poorly; wilt faster and look worse.

### 🔴 Anti-Pattern 3: Poor Water Quality
```
❌ BAD: Using old water or dirty buckets
✅ GOOD: Fresh, clean water with flower food; clean buckets
```
**Why it matters**: Bacteria clogs stems; flowers can't drink; wilting is inevitable.

### 🔴 Anti-Pattern 4: Ignoring Temperature
```
❌ BAD: Leaving flowers in hot car or sunny window
✅ GOOD: Keep at 34-38°F for most; cool storage always
```
**Why it matters**: Heat accelerates wilting and bacterial growth.

## 10.2 Medium-Severity Anti-Patterns

### 🟡 Anti-Pattern 5: Wrong Mechanics
```
❌ BAD: Using too little support, flowers falling out
✅ GOOD: Proper foam saturation, secure wiring, balanced mechanics
```

### 🟡 Anti-Pattern 6: Ignoring Client Vision
```
❌ BAD: "I know better — here's what you really want"
✅ GOOD: Listen to client; they know their event best
```

### 🟡 Anti-Pattern 7: Not Confirming Deliveries
```
❌ BAD: Assuming venue knows you're coming
✅ GOOD: Confirm delivery time, location, contact person in advance
```

### 🟡 Anti-Pattern 8: Overpricing Without Justification
```
❌ BAD: High prices with no explanation of value
✅ GOOD: Explain design value, labor, premium materials
```

## 10.3 Best Practices Checklist

### Flower Care
- [ ] Flowers stored at proper temperature
- [ ] Water changed daily in storage
- [ ] Leaves removed below waterline
- [ ] Stems cut at angle before arranging

### Design
- [ ] Color palette matches event theme
- [ ] Focal flowers balanced
- [ ] Arrangements viewed from all angles
- [ ] Size appropriate for venue

### Logistics
- [ ] Delivery confirmed with venue
- [ ] Setup time calculated
- [ ] Backup flowers available
- [ ] Tools and supplies packed

## 10.4 Red Flags to Watch For

| Red Flag | Risk Level | Action |
|----------|------------|--------|
| Flowers from supplier arrive wilted | High | Return to supplier; don't accept |
| Customer changes vision mid-project | Medium | Re-quote; adjust timeline |
| Venue access denied at delivery | High | Have emergency contact; build buffer time |
| Weather extreme on delivery day | Medium | Protect arrangements; consider rescheduling |
