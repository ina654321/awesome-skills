# Tattoo Artistry — Troubleshooting

## Client Consultation Issues

### Client Wants Design Copied from Another Artist

**Situation:** Client brings in a tattoo from another artist or from the internet and wants it reproduced exactly.

**Approach:**
```yaml
Ethical response:
  1. Explain: "I can create an original piece inspired by this style, 
     but I don't copy other artists' work directly"
  2. Reason: Copyright law protects original artwork; reproducing is infringement
  3. Exception: If client owns the original (flash sheet they purchased), 
     reproduction is acceptable
  4. Compromise: Create original design in similar style or subject matter

If client insists on exact copy:
  1. Politely decline
  2. Explain copyright implications
  3. Offer to create something unique and personal instead
  4. If flash sheet: Fine to reproduce

For cover-ups:
  - Design must incorporate, not just cover — old tattoo should be part of new design
```

### Client Has Unrealistic Expectations (Size, Detail, Pain)

**Situation:** Client wants a very detailed tattoo on a body area prone to distortion.

**Approach:**
```yaml
Example: "I want a mandala on my wrist, 4 inches, extremely detailed"

Reality check:
  1. Wrist curves and stretches — fine detail will distort
  2. 4" is very small for that level of detail
  3. Healing on wrist is challenging

Recommendation:
  "For the level of detail you want, I'd recommend 5-6 inches minimum.
   The design will hold better over time and look crisper as it heals."

Example: "I want a realistic portrait on my finger"

Reality:
  - Finger tattoos fade in months
  - Detail becomes unreadable
  - Not recommended for realistic work

Honest communication:
  - Explain healing challenges
  - Show healed examples of similar work
  - Offer alternatives (finger rings, other placements)
```

## Skin & Healing Problems

### Excessive Scabbing / Thick Scabs

**Symptom:** Unusually thick scabs forming over tattoo, longer healing time.

**Cause:**
```yaml
- Overworked skin (too many passes)
- Too much ink in skin
- Sensitive area
- Client picking at tattoo
```

**Treatment:**
```yaml
1. Don't pick or scratch scabs
2. Keep clean with unscented soap
3. Apply thin layer of unscented lotion
4. If thick scab falls off and takes ink: Touch-up may be needed
5. After healing: Assess if patchy areas need touch-up

Prevention:
  - Don't overwork the skin
  - Use appropriate needle depth
  - Don't use too much ink
```

### Tattoo Blotching / Patchy Areas

**Symptom:** Some areas healed unevenly, patchy color, missing sections.

**Cause:**
```yaml
- Uneven ink distribution
  - Too much ink: Sits on surface, scabs off
  - Too little ink: Faded areas
- Client not following aftercare
- Excessive movement during tattoo
- Healing issue: picked scabs
```

**Treatment:**
```yaml
1. Wait full healing (3-4 weeks) before assessing
2. If truly patchy: Schedule touch-up session
3. Touch-up is usually included (confirm policy)
4. Touch-up requires same aftercare

Prevention:
  - Ensure even ink saturation
  - Proper needle depth throughout
  - Wrap carefully for travel home
  - Clear aftercare instructions
```

### Infection Signs

**Symptom:** Increasing redness after day 3, spreading redness, pus, fever.

**Differentiation:**
```yaml
Normal healing (days 1-3):
  - Redness around tattoo (like a sunburn)
  - Slight warmth
  - Oozing clear fluid

Infection (days 3+):
  - Spreading redness (beyond tattoo)
  - Increasing redness (not decreasing)
  - Thick white/yellow/green discharge
  - Fever
  - Increasing pain

If infection suspected:
  1. Client: See doctor immediately
  2. Document: Photograph the tattoo and area
  3. Document: Client's aftercare compliance
  4. Follow up after doctor visit
  5. Keep records for hygiene audit
```

**Prevention:**
```yaml
- Sterile setup
- Single-use supplies
- Proper hand washing
- Clean workstation (barrier film)
- Gloves throughout
- Client aftercare instructions in writing
```

### Keloid or Raised Scarring

**Symptom:** Tattoo is raised, bumpy, or growing beyond tattoo boundaries after healing.

**Assessment:**
```yaml
Possible causes:
  1. Client is prone to keloids (genetic)
  2. Overworked skin causing trauma response
  3. Normal healing variant (hypertrophic scar)

Assessment:
  - Does client have keloids from other injuries/surgeries?
  - Is the raised area within tattoo boundaries or beyond?
  - When did it start (during healing vs. months later)?

Treatment:
  - If keloid-prone: Consult dermatologist
  - Silicone sheets or gel can help flatten
  - Cortisone injections (dermatologist)
  - Laser treatment (rare, for severe cases)
  - Some raised areas are normal — will flatten over time
```

## Tattoo Longevity Issues

### Tattoo Fading After Healing

**Symptom:** Tattoo looks great initially but fades significantly within 6 months.

**Causes:**
```yaml
1. Fine line work (normal fading)
2. White ink (disappears)
3. Poor ink quality
4. Excessive sun exposure
5. Skin type (darker skin shows less saturation)
6. Watercolor style (intentionally less permanent)
```

**Fix:**
```yaml
1. Assess: Is fading within style's normal expectation?
2. If fine line: Client should expect periodic touch-ups
3. If white included: Should have been warned this fades
4. Sun damage: Recommend sunscreen SPF 30+ on tattoo
5. Skin type: Set realistic expectations for darker skin

For future prevention:
  - Discuss longevity expectations before tattoo
  - Use bold designs for long-term visibility
  - Recommend sun protection
  - Consider style modifications for longevity
```

### Tattoo on Hand/Finger Fading

**Symptom:** Finger/hand tattoo healed well but faded significantly within weeks.

**This is normal.** Hands and fingers heal rapidly and are constantly washed/used.

**Realistic approach:**
```yaml
Before tattooing:
  1. Warn clients: Hand tattoos require frequent touch-ups
  2. Warn: Can fade within 6-12 months even with perfect care
  3. Design: Bold, simple designs hold better
  4. White/yellow: Not recommended for hands
  5. Placement: Top of hand > sides/fingers

Touch-up expectation:
  - Set realistic: May need annual touch-up
  - Charge for touch-up (explain at consultation)
  - Some people get 3-4 touch-ups per year
```

## Cover-Up Issues

### Cover-Up Design Doesn't Match Original Vision

**Situation:** Client wanted a specific design over their old tattoo, but the old tattoo shows through.

**Reality:**
```yaml
Cover-up principles:
  1. Old tattoo MUST be covered by new design — not erased
  2. New design must be DARKER and LARGER than old tattoo
  3. Some show-through is inevitable
  4. Flash cover-up (placing a design over existing tattoo) limits design freedom

Solutions:
  1. Show healed cover-up examples
  2. Explain: "Your old tattoo will influence the new design"
  3. Use dark background elements (clouds, shadows, water)
  4. Multiple sessions may be needed
  5. If old tattoo is very visible: Consider removal first
```

### Client Unhappy with Cover-Up

**Situation:** After completion, client says the cover-up doesn't look like what they expected.

**Prevention:**
```yaml
1. Show multiple healed cover-up examples before starting
2. Draw design on skin (stencil) for visual approval
3. Get written approval of design before tattooing
4. Have client confirm: "This is the design that will be tattooed"

If unhappy post-tattoo:
  1. Listen to specific concerns
  2. Assess: Is this realistic dissatisfaction or actual problem?
  3. If it's within normal: Explain healing process
  4. Offer: Touch-up session to refine
  5. If fundamentally different from approved design: Discuss with shop owner
```

## Equipment & Setup Issues

### Needle Not Penetrating Properly

**Symptom:** Tattoo gun runs but isn't depositing ink properly.

**Troubleshooting:**
```yaml
1. Check needle: Is it properly inserted in tube/grip?
2. Check needle depth: Too shallow won't penetrate skin
3. Check ink: Is ink flowing? Cap may be clogged
4. Check handpiece: Is rubber band in place?
5. Check machine: If using coil, check spring tension

For cartridge systems:
  - Ensure cartridge is fully seated
  - Check o-ring is intact
  - Replace if defective

For coil machines:
  - Adjust front spring tension
  - Adjust binding posts
  - Ensure proper voltage
```

### Ink Colors Mixing Contaminated

**Situation:** New ink from bottle is contaminated (particles, clumping).

**Fix:**
```yaml
Prevention:
  1. Never double-dip: Use fresh ink for each session
  2. Pour small amounts into ink caps
  3. Don't return used ink to bottle
  4. Replace ink caps between clients
  5. Check expiry date on ink

If contamination found:
  1. Don't use the ink
  2. Check remaining bottles
  3. Report to supplier
  4. Document batch number and date
```

## Legal & Liability Issues

### Client Has Allergic Reaction

**Situation:** Client reports itching, swelling, or rash around tattoo after healing.

**Immediate:**
```yaml
1. Document: When did it start? After healing or months later?
2. Assess: Is it localized (allergic) or spreading (infection)?
3. Advise: See doctor if spreading or systemic symptoms
4. Document: Take photos, keep records

Common allergens:
  - Red ink (most common)
  - Other bright colors (yellow, orange)
  - Nickel in some inks
  - Latex (from gloves, not tattoo itself)

Future prevention:
  - Ask about allergies at consultation
  - Test patch for clients with known sensitivities
  - Document known allergies in client file
```

### Client Claims Tattoo Caused Infection

**Situation:** Client comes back claiming the tattoo caused an infection.

**Response:**
```yaml
1. Stay calm and professional
2. Document: Take photos, ask questions about care
3. Ask: Did they follow aftercare instructions?
4. If infected: Advise doctor visit, document
5. If not infected: Assess if it's normal healing reaction

Legal protection:
  1. Maintain hygiene audit log (sterilization records)
  2. Use consent forms with aftercare instructions
  3. Keep records of everything
  4. If serious claim: Contact insurance company
  5. Don't make statements about causation
  6. Refer to shop owner/manager
```
