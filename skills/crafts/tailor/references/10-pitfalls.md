# Common Pitfalls & Anti-Patterns

## 10.1 High-Severity Anti-Patterns

### 🔴 Anti-Pattern 1: Altering Without Proper Assessment
```
❌ BAD: "I'll just cut it down" without examining construction
✅ GOOD: Always assess garment quality, fabric, and previous alterations first
```
**Why it matters**: Hidden problems can ruin garment; bad fabric may not hold stitches.

### 🔴 Anti-Pattern 2: Over-Altering Cheap Garments
```
❌ BAD: Spending $100 altering a $50 garment
✅ GOOD: Assess cost-benefit: alteration cost vs. garment value
```
**Why it matters**: Poor quality can't be improved through alteration; wasted money.

### 🔴 Anti-Pattern 3: Ignoring Fabric Limitations
```
❌ BAD: Treating all fabrics the same way
✅ GOOD: Match thread, needle, and technique to specific fabric
```
**Why it matters**: Wrong technique damages fabric; puckering, pulling, or thread breaks.

### 🔴 Anti-Pattern 4: Rushing Fitting
```
❌ BAD: Making marks without client walking/sitting
✅ GOOD: Test movement in fitting; check all positions
```
**Why it matters**: Standing fit may look wrong when client moves.

## 10.2 Medium-Severity Anti-Patterns

### 🟡 Anti-Pattern 5: Poor Thread Matching
```
❌ BAD: Using slightly off thread; "no one will notice"
✅ GOOD: Match exactly or use invisible technique
```

### 🟡 Anti-Pattern 6: Skipping Pressing
```
❌ BAD: Leaving finished garment unpressed
✅ GOOD: Press every seam and finished piece
```

### 🟡 Anti-Pattern 7: Not Documenting
```
❌ BAD: No notes on garment condition, measurements, or decisions
✅ GOOD: Written record protects both tailor and client
```

### 🟡 Anti-Pattern 8: Ignoring Asymmetry
```
❌ BAD: Assuming perfect symmetry
✅ GOOD: Note and address client's natural asymmetry
```

## 10.3 Best Practices Checklist

### Assessment
- [ ] Fabric quality evaluated
- [ ] Construction quality noted
- [ ] Previous alterations documented
- [ ] Fit requirements understood

### Measurement
- [ ] All key measurements taken
- [ ] Client's preferences noted
- [ ] Asymmetry documented
- [ ] Movement tested

### Alteration
- [ ] Correct thread and needle for fabric
- [ ] Proper tension set
- [ ] Seam reinforcement where needed
- [ ] Each step pressed

### Quality Check
- [ ] No loose threads
- [ ] Symmetrical appearance
- [ ] Movement tested
- [ ] Client satisfied

## 10.4 Red Flags to Watch For

| Red Flag | Risk Level | Action |
|----------|------------|--------|
| Fabric pills or weakens | High | Note before proceeding; may not hold stitches |
| Previous alterations visible | Medium | May need to undo first |
| Client unrealistic expectations | Medium | Communicate limitations clearly |
| Delicate/special fabric | High | Test on scrap first; may need specialist |
