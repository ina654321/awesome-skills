## § 8 · Standard Workflow

### 8.1 Plucked Instrument Construction

```
Phase 1: Design & Materials
├── Choose instrument model (based on tradition, player needs)
├── Select appropriate wood: soundboard, back, neck, accessories
├── Mill wood to approximate dimensions, allow to stabilize
└── [✓ Done]: Materials selected and prepared

Phase 2: Body Construction
├── Cut soundboard and back to shape
├── Carve interior (thickness based on traditional patterns)
├── Create side walls (bent or carved)
└── [✓ Done]: Body shaped, ready for joining

Phase 3: Assembly
├── Join soundboard to body (traditional hide glue)
├── Install neck and headstock
├── Add decorations (inlay, binding)
└── [✓ Done]: Instrument body complete

Phase 4: Finishing & Setup
├── Sand to final smoothness
├── Apply finish (varnish, oil, lacquer)
├── Install tuning pegs, bridges, nut
└── [✓ Done]: Complete instrument, ready for playing

Phase 5: Voicing & Adjustment
├── Test sound: projection, clarity, sustain
├── Adjust thickness if needed (tap tuning)
├── Setup action height for playability
└── [✓ Done]: Instrument optimized for player
```

### 8.2 Restoration Process

```
Step 1: Assess condition — document cracks, missing parts, finish damage
Step 2: Remove old finish carefully if needed
Step 3. Repair cracks — matching wood, traditional joinery
Step 4: Replace damaged parts — using period-appropriate materials
Step 5: Refinish — match original as closely as possible
Step 6: Setup — ensure playability matches original intent
```
