# Traditional Instrument Making — Troubleshooting

## Wood Selection & Preparation Issues

### Wood Warping After Cutting

**Symptom:** Prepared wood pieces warp (twist, bow, cup) during drying or after shaping.

**Fix:**
```yaml
Prevention:
  - Always plane both faces equally
  - Allow wood to acclimate 1 week per inch of thickness
  - Store on stickers (1" spacers) in controlled environment (45-55% RH)

If warp occurs:
  1. Lightly dampen the concave side with wet cloth
  2. Clamp to flat surface with shim between
  3. Allow to dry slowly (7-10 days)
  4. Re-check with straightedge

For critical pieces (soundboard):
  - Select quartersawn wood (more dimensionally stable)
  - Avoid flat-sawn with wide annual rings
  - Store in climate-controlled space
```

### Wood Too Green for Carving

**Symptom:** Freshly cut wood is too moist, causing tools to clog and carving to distort.

**Fix:**
```yaml
For soundboard wood (paulownia):
  1. Rough-carve to 20% oversized dimensions
  2. Air-dry slowly with stickers and weight
  3. Check MC monthly — target <10% for paulownia soundboard
  4. Complete final carving when MC is 6-8%

For body/thick pieces:
  - Kiln dry if time-pressured (slow kiln: 2-4 weeks)
  - Do not rush — accelerated drying causes checking
  - End-grain sealing: Paint/wax ends to prevent surface checking
```

## Sound Quality Issues

### Soundboard Produces Dull or Muted Tone

**Symptom:** Completed instrument has poor projection and dead tone.

**Diagnosis & Fix:**
```yaml
Common causes:
  1. Soundboard too thick (most common)
  2. Finish too thick or wrong type
  3. Bracing interference with top vibration
  4. Body not properly joined (air leak)
  5. String height too high

For thick soundboard:
  1. Measure thickness vs. template — should be within 0.5mm
  2. Use thickness sander or hand planing to reduce evenly
  3. Target: 3-4mm for guqin/pipa soundboard (varies by instrument)
  4. Tap test: Should produce clear, sustained tone

For finish damping:
  1. Remove excess finish: Sand to 0.1-0.2mm
  2. French polish for least damping
  3. Test after each coat — if tone dulls, remove and redo
```

### Tonal Imbalance (Hot/Cold Spots)

**Symptom:** Some notes project well while others are weak or harsh.

**Fix:**
```yaml
Step 1: Identify the problem notes
  - Play each note and compare volume/quality
  - Note which region has the problem

Step 2: Tonal carving
  - Hot spot (too loud): Lightly scrape the area (<0.1mm at a time)
  - Cold spot (too quiet): Check for thickness variation; scrape thinner

Step 3: Re-test after each adjustment
  - Play scale from low to high

Step 4: Bridge placement
  - Move bridge slightly toward tail for brighter tone
  - Move toward neck for darker tone
```

## Structural & Joinery Issues

### Cracks Appearing in Body

**Symptom:** Hairline cracks appear in body or soundboard after completion.

**Fix:**
```yaml
For hairline cracks (<0.5mm):
  1. Clean crack with compressed air
  2. Apply thin hide glue with syringe
  3. Clamp gently
  4. Let cure 24 hours
  5. Lightly sand and refinish

For wider cracks (>0.5mm):
  1. Clean crack thoroughly
  2. Make spline: cut matching wood strip to width of crack
  3. Glue spline into crack with hide glue
  4. Clamp and let cure
  5. Plane/sand flush

Prevention:
  - Maintain consistent humidity (40-55% RH)
  - Avoid rapid temperature changes
  - Never leave instrument in direct sunlight
```

## Finish Issues

### Finish Cracking or Peeling

**Symptom:** Surface finish develops cracks or separates from wood.

**Fix:**
```yaml
Complete finish failure:
  1. Strip finish entirely (denatured alcohol for shellac)
  2. Sand back to bare wood (320→400→600 grit)
  3. Check wood condition
  4. Re-apply finish from scratch

Local repair:
  1. Scrape/remove damaged area only
  2. Feather edges into surrounding finish
  3. Re-apply finish in thin coats

Prevention:
  - Never apply finish over contaminated surface
  - Allow proper drying time between coats
  - Apply thin coats
```

## String & Setup Issues

### Strings Buzzing Against Frets or Soundboard

**Symptom:** Unwanted buzzing sound when playing.

**Diagnosis:**
```bash
# Hold string at 1st and last fret
# Measure gap at 7th fret
# Should be 0.2-0.5mm
```

**Fix:**
```yaml
For nut slot too deep:
  1. Fill slot with bone/ebony dust + thin glue
  2. Re-cut to correct depth

For action too high:
  1. Lower bridge foot by planing bottom
  2. File nut slots deeper only if needed

For unlevel fret:
  1. Check fret heights with straightedge
  2. Level with file or stone
  3. Re-crown with fret file
```

## Restoration Issues

### Delaminated Plywood or Laminated Layers

**Symptom:** Layers of body wood are separating.

**Fix:**
```yaml
For minor delamination:
  1. Clean crack with compressed air
  2. Inject thin hide glue with syringe
  3. Clamp with padded cauls matching the curve
  4. Let cure 24-48 hours

Prevention:
  - Avoid exposing to high humidity
  - Never leave in cold car
  - Proper case humidification
```

### Inappropriate Previous Restoration

**Symptom:** Restored instrument has wrong materials (epoxy, modern wood fillers).

**Assessment:**
```yaml
When to remove old repairs:
  - Epoxy instead of traditional hide glue
  - Wood filler instead of matching wood
  - Lacquer over original French polish

When to preserve:
  - Old repairs that are stable
  - Repairs with historical significance
  - Repairs from respected makers
```

## Tool Issues

### Chisel/Carving Tool Dulling Quickly

**Fix:**
```yaml
For soft wood (paulownia):
  - Sharp is critical — dull tools tear grain
  - Use higher bevel angle (25-30°)
  - Strop frequently during carving

Sharpening routine:
  1. Flat grind on diamond plate (220-400 grit)
  2. Create secondary bevel on leather strop
  3. Strop before each session
  4. Test: Should slice paper cleanly
```
