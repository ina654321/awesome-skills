# Scenario Examples

## 9.1 Lost Keys - Complete Rekey Needed

**Customer:** "I just moved into a new house and the previous owners didn't leave any keys. I need all the locks rekeyed."

**Expert:**
> **Assessment:**
> 
> This is a standard rekeying scenario with important security considerations:
> 
> 1. **Security Priority:** Previous owners, realtors, contractors may have had keys
> 2. **Scope:** Determine how many locks need rekeying
> 3. **Key History:** Ask about number of existing keys (even if lost)
> 
> **Recommended Approach:**
> - Rekey all entry locks for security
> - Consider upgrading if locks are old/worn
> - Change garage door code as well
> - Rekey any outbuildings with same key
> 
> **Documentation Needed:**
> - Government-issued photo ID
> - Proof of property ownership (deed, closing docs)
> - If renting: landlord authorization letter
> 
> **After Service:**
> - Provide minimum 3 keys per lock
> - Explain any master key system
> - Recommend spare key location (not hidden outside)

## 9.2 Automotive Lockout - Luxury Vehicle

**Customer:** "I locked my keys in my 2023 Mercedes. The fob is inside and I need it out urgently."

**Expert:**
> **Assessment:**
> 
> Luxury vehicles require special consideration:
> 
> 1. **Key Type:** Modern Mercedes use proximity keys - key inside means car won't detect it outside
> 2. **Security Features:** Advanced anti-theft may complicate entry
> 3. **Interior Protection:** Many luxury cars have alarm triggers
> 
> **Procedure:**
> - Verify vehicle ownership (registration, ID)
> - Assess entry method (slim jim vs. wedge method)
> - Consider trigger warnings on some vehicles
> - May need to contact dealer if all else fails
> 
> **Special Considerations:**
> - Some require OEM software to program new keys
> - Aftermarket solutions may void warranty
> - OnStar-style systems may need dealer involvement
> 
> **Pricing Note:** Luxury vehicles typically cost more due to tool requirements and risk

## 9.3 Broken Lock After Break-In Attempt

**Customer:** "Someone tried to break into my store last night. The deadbolt is damaged but the door is still locked. I need this fixed immediately."

**Expert:**
> **Assessment:**
> 
> This is an emergency security situation:
> 
> 1. **Police Report:** Must have police report number
> 2. **Insurance:** May be covered under business insurance
> 3. **Time Pressure:** Business cannot operate securely
> 
> **Immediate Actions:**
> - Secure premises first
> - Document damage with photos
> - Contact insurance adjuster if applicable
> - Board up if repair will take time
> 
> **Repair Options:**
> - Repair existing lock if damage is minor
> - Replace if structural damage occurred
> - Consider security upgrade (high-security locks)
> - May need to replace door if frame damaged
> 
> **Security Recommendation:**
> - Upgrade to Grade 1 commercial locks
> - Add strike plate reinforcement
> - Consider alarm system upgrade
> - Rekey after any break-in

## 9.4 Master Key System Failure

**Customer Property Manager:** "Our master key stopped working for the basement storage locks. We have important inventory we need to access."

**Expert:**
> **Diagnosis:**
> 
> Master key systems can fail due to several reasons:
> 
> 1. **Pin shift** - Over time, pin chambers can wear
> 2. **Key damage** - Worn or broken master key
> 3. **Wrong sub-master** - Employee using wrong key
> 4. **Rekey needed** - Previous service may have changed combination
> 
> **Troubleshooting Steps:**
> - Verify which specific locks are affected
> - Test all available sub-master keys
> - Check if grand master key works
> - Review last service records
> 
> **Resolution:**
> - Rekey affected locks back to system
> - May need to rekey entire building
> - Create new keying chart
> - Test all levels of master key
> 
> **Prevention:**
> - Annual maintenance of master key systems
> - Keep detailed records of all combinations
> - Limit master key access to essential personnel
> - Consider electronic access control

## 9.5 Safe Combination Lost

**Customer:** "I'm the executor of my father's estate. He had a home safe that we can't open. I have the death certificate and will, but I don't know the combination."

**Expert:**
> **Assessment:**
> 
> Estate situations require careful handling:
> 
> 1. **Legal Documentation Required:**
>    - Death certificate
>    - Will naming executor
>    - Government-issued ID of executor
>    - Letter of administration (if no will)
> 
> 2. **Safe Owner Verification:**
>    - Registration card (if available)
>    - Original purchase receipt
>    - Location matches customer's info
> 
> **Service Options:**
> - Combination recovery (if safe supports)
> - Safe manipulation (requires expertise)
> - Drill and replace lock (destructive)
> - Manufacturer bypass (some cases)
> 
> **Important Notes:**
> - Never proceed without proper authorization
> - Document all paperwork thoroughly
> - Some states require notarized authorization
> - Consider involving attorney if any doubt
