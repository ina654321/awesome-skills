# Standard Workflow

## 8.1 Residential Service Call Workflow

```
Service Call Process
├── Pre-Arrival
│   ├── Verify customer identity and ownership
│   ├── Confirm service address and lock type
│   └── Gather key/lock history from customer
├── Arrival Assessment
│   ├── Identify lock manufacturer
│   ├── Determine lock type and keyway
│   └── Assess security level needed
├── Service Delivery
│   ├── Select appropriate tools
│   ├── Execute service (rekey/repair/replace)
│   └── Test operation multiple times
└── Completion
    ├── Clean work area
    ├── Provide new keys (if applicable)
    └── Document service and warranty
```

## 8.2 Rekeying Workflow

1. **Verification (MUST DO)**
   - Verify property ownership or authorization
   - Check ID and utility bill or lease
   - Document authorization in work order

2. **Lock Disassembly**
   - Remove lock from door or cylinder from lock
   - Lay out components in order
   - Photograph original pin configuration

3. **Pin Chamber Analysis**
   - Measure existing pin lengths
   - Note spring pressures
   - Check for worn or damaged pins

4. **New Key Cutting**
   - Select appropriate key blank
   - Cut to manufacturer specs
   - Test cut key in original cylinder

5. **Installation**
   - Install new pins/springs
   - Reassemble lock
   - Test with original and new keys

6. **Quality Check**
   - Test all keys 5+ times
   - Check smooth operation
   - Verify deadbolt alignment

## 8.3 Lockout Response Workflow

1. **Authentication Phase**
   - Require government-issued photo ID
   - Match name to utility bill or lease
   - Document ID number in work order
   - For vehicles: registration or insurance card
   - For businesses: business license or letterhead authorization

2. **Vehicle Lockout Protocol**
   - Verify vehicle ownership (registration must match)
   - Use method appropriate for vehicle year/make
   - Document VIN for verification
   - Never attempt if vehicle has LoJack or similar
   - Recommend spare key location for future

3. **Residential Lockout**
   - Confirm address matches documentation
   - Verify no other occupants home
   - Attempt non-destructive entry first
   - Document method used in work order

4. **Commercial Lockout**
   - Require authorization letter on company letterhead
   - Contact property manager if tenant
   - Verify authorization chain
   - May require police liaison in some jurisdictions

## 8.4 High-Security Lock Service

1. **Documentation Review**
   - Request proof of ownership
   - Verify authorization for key duplication
   - Check restricted key system registration

2. **Factory Authorization**
   - Contact manufacturer for specific system
   - Obtain authorization number
   - Document authorization in records

3. **Specialized Equipment**
   - Use manufacturer-approved tools
   - Follow exact procedures
   - Maintain equipment calibration

4. **Verification**
   - Test all keys thoroughly
   - Document pin combinations
   - Secure documentation properly

## 8.5 Master Key System Design

1. **Needs Assessment**
   - Interview property manager
   - Map access areas and hierarchies
   - Identify security zones
   - Document user groups

2. **System Design**
   - Determine master key levels (3-4 max)
   - Calculate pin chamber combinations
   - Plan keying schedule
   - Select compatible lock products

3. **Implementation**
   - Install locks in planned sequence
   - Key each to system specification
   - Test all combinations
   - Provide keying chart to client

4. **Documentation**
   - Create keying chart (store securely)
   - Document all combination changes
   - Set up rekey schedule
   - Establish key control procedures

## 8.6 Emergency Service Protocol

| Priority | Response | Examples |
|----------|----------|----------|
| Life safety | Immediate | Child locked in car, elderly locked out |
| Security breach | Within 2 hours | Forced entry, broken lock |
| Property access | Same day | Locked out, lost keys |
| Scheduled service | Appointment | Rekeying, upgrades |

## 8.7 Quality Checklist

- [ ] Customer identity verified and documented
- [ ] All keys tested minimum 5 times
- [ ] Lock operates smoothly (no binding)
- [ ] Strike plate properly secured
- [ ] Work area cleaned
- [ ] Warranty information provided
- [ ] Keys properly labeled
- [ ] Payment documented
