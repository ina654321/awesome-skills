# Common Pitfalls & Anti-Patterns

## 10.1 Security Verification Failures

| # | Pitfall | Severity | Consequence | Prevention |
|---|---------|----------|-------------|------------|
| 1 | Serving without ID verification | 🔴 Critical | Liability, fraud | Always verify identity |
| 2 | Accepting verbal authorization | 🔴 Critical | Legal issues | Require written authorization |
| 3 | Skipping vehicle registration check | 🔴 Critical | Stolen vehicle service | Always verify ownership |
| 4 | Not documenting the job | 🔴 High | Dispute resolution | Photo documentation |
| 5 | Giving keys to wrong person | 🔴 Critical | Security breach | Verify authorization fully |

## 10.2 Technical Mistakes

| Mistake | Result | Correct Approach |
|---------|--------|-------------------|
| Wrong pin length selection | Key won't turn smoothly | Measure existing pins accurately |
| Forgetting driver pins | Lock won't disassemble | Note pin order before removal |
| Over-tightening set screws | Stripped threads | Torque to spec (typically 25-30 in-lbs) |
| Mixing key pinning | Permanent damage | Use fresh pins only |
| Improper tool use | Component damage | Use right tool for application |

## 10.3 High-Security Lock Errors

| Error | Risk | Mitigation |
|-------|------|------------|
| Attempting unauthorized service | Lockout, customer liability | Verify authorization first |
| Using unauthorized blanks | System integrity failure | Use factory-authorized blanks only |
| Improper manipulation | Damage to lock | Follow manufacturer training |
| Breaking restricted key patents | Legal liability | Know your legal obligations |

## 10.4 Automotive Lockout Mistakes

| Anti-Pattern | Outcome | Proper Method |
|--------------|---------|---------------|
| Wrong tool for vehicle | Window seal damage | Match tool to vehicle |
| Too much force | Triggered alarm/ airbag risk | Use proper technique |
| Not verifying ownership | Stolen vehicle call | Check registration |
| Damaging interior trim | Additional repair costs | Use non-marring tools |
| Proceeding with LoJack-equipped vehicle | Anti-theft triggers | Ask about tracking systems |

## 10.5 Business Practice Errors

| Mistake | Impact | Best Practice |
|---------|--------|----------------|
| No written estimate | Customer disputes | Always provide written estimate |
| Cash-only policy | Tax issues, security | Accept checks, cards |
| No liability insurance | Personal liability | Carry commercial liability |
| Poor record keeping | Rekey confusion | Maintain keying charts |
| No after-hours emergency pricing | Undervaluing service | Price emergency calls appropriately |

## 10.6 Legal and Ethical Violations

- **Opening locks without proper authorization** - Can constitute breaking and entering
- **Copying restricted/proprietary keys** - May violate patent law
- **Not checking for outstanding warrants** - Could be aiding criminals
- **Improper key control** - Creating security vulnerabilities
- **Unlicensed operation** - Many jurisdictions require locksmith license

## 10.7 Customer Communication Failures

| Error | Customer Impact | Right Approach |
|-------|-----------------|----------------|
| Overpromising speed | Trust damage | Give realistic timelines |
| Quoting without seeing job | Price disputes | Always quote after inspection |
| Not explaining limitations | Frustration | Set clear expectations |
| Skipping warranty explanation | Post-service issues | Document warranty terms |
| Ignoring accessibility needs | ADA violations | Ensure continued ADA compliance |

## 10.8 Safety Violations

| Violation | Risk | Protocol |
|-----------|------|----------|
| No grounding strap | ESD damage | Always use when working on vehicles |
| Improper ladder use | Falls | Follow OSHA ladder safety |
| No eye protection | Injury | Wear safety glasses always |
| Blocking emergency exits | Code violation | Maintain clear access |
| Chemical exposure | Health hazard | Use proper ventilation |

## 10.9 Master Key System Errors

| Mistake | Result | Prevention |
|---------|--------|------------|
| Inadequate documentation | System confusion | Create detailed keying charts |
| Too many levels | Complexity issues | Limit to 3-4 levels maximum |
| No spare key policy | Lockouts | Establish spare key protocols |
| Uncontrolled key duplication | Security breach | Implement key control program |
