# Standard Workflow

## 8.1 Device Intake Workflow

```
Intake Process
├── Customer Interview
│   ├── Document symptoms and history
│   ├── Check for liquid damage indicators
│   └── Verify warranty/insurance status
├── Visual Inspection
│   ├── Check for physical damage
│   ├── Inspect connectors and ports
│   └── Document cosmetic condition
├── Functional Assessment
│   ├── Power on/off testing
│   ├── Display and touch response
│   ├── Camera, speakers, microphones
│   └── Connectivity tests (WiFi, BT, Cellular)
└── Repair Authorization
    ├── Generate repair estimate
    ├── Confirm parts availability
    └── Schedule repair slot
```

## 8.2 Screen Replacement Workflow

1. **Pre-repair Documentation**
   - Back up customer data (if possible)
   - Power down device completely
   - Document original condition with photos
   - Remove SIM tray and any case accessories

2. **Disassembly**
   - Apply heat to soften adhesive (40-50°C, 2-3 minutes)
   - Use suction cup and plastic pry tools
   - Work around perimeter, never insert tools deep
   - Disconnect battery first (MUST DO for OEM batteries)

3. **Screen Removal**
   - Remove screws securing display bracket
   - Disconnect display flex cables
   - Remove broken screen assembly
   - Clean frame of residual adhesive

4. **Installation**
   - Test new screen before full adhesion
   - Connect all flex cables securely
   - Apply new adhesive strips
   - Secure with screws (torque specs: 0.15-0.2 N·m)

5. **Post-repair Validation**
   - Calibrate color/gamma
   - Test touch sensitivity across entire display
   - Verify device powers on correctly
   - Check for any light bleed or dead pixels

## 8.3 Battery Replacement Workflow

1. **Safety First**
   - Discharge battery below 25% before removal
   - Work in well-ventilated area
   - Keep fire extinguisher accessible
   - Wear safety glasses

2. **Battery Removal**
   - Remove adhesive securing battery
   - Use battery pull tabs if available
   - Never puncture or bend battery
   - Disconnect battery connector from board

3. **New Battery Installation**
   - Verify battery matches OEM specifications
   - Check connector alignment before seating
   - Apply new adhesive properly
   - Secure with gentle pressure

4. **Validation**
   - Power on and check charge cycle count
   - Verify battery health in diagnostic mode
   - Run battery calibration (full charge/discharge)

## 8.4 Water Damage Protocol

1. **Immediate Response**
   - Do NOT power on device
   - Remove SIM, battery (if accessible)
   - Dry exterior immediately
   - Place in silica gel or rice

2. **Assessment Phase**
   - Inspect liquid damage indicators (LDI)
   - Check for corrosion on connectors
   - Identify affected components

3. **Cleaning Phase**
   - Ultrasonic cleaning with 99% isopropyl alcohol
   - Brush corrosion with soft toothbrush
   - Clean connectors with contact cleaner
   - Allow full drying (24-48 hours minimum)

4. **Component Testing**
   - Test individual components before reassembly
   - Replace corroded parts as needed
   - Verify board integrity

## 8.5 Quality Control Checklist

| Check | Pass Criteria |
|-------|---------------|
| Screen | No dead pixels, even backlight, responsive touch |
| Battery | Charges correctly, no swelling, proper health reading |
| Cameras | Both front/rear functional, autofocus working |
| Speakers | Clear audio, no distortion at max volume |
| Microphone | Clear recording, noise cancellation working |
| Connectivity | WiFi, Bluetooth, Cellular all functional |
| Buttons | All physical buttons respond correctly |
| Sensors | Proximity, ambient light, accelerometer |
