# Common Pitfalls & Anti-Patterns

## 10.1 Critical Mistakes

| # | Anti-Pattern | Severity | Impact | Prevention |
|---|---------------|----------|--------|------------|
| 1 | Powering on water-damaged device | 🔴 Critical | Permanent damage | Never power on wet devices |
| 2 | Forgetting to disconnect battery | 🔴 Critical | Component damage, safety risk | Always disconnect first |
| 3 | Using metal tools near battery | 🔴 Critical | Short circuit, fire | Use plastic only near batteries |
| 4 | Not backing up customer data | 🔴 Critical | Data loss liability | Always backup before repair |
| 5 | Over-tightening screws | 🔴 High | Stripped threads, damage | Use torque screwdriver |

## 10.2 Component Damage Patterns

| Mistake | Result | Proper Technique |
|---------|--------|------------------|
| Heat gun too hot | Melted components, warped frames | Keep below 60°C, use IR thermometer |
| Pulling too hard on cables | Torn flex cables | Work slowly, reheat adhesive |
| Wrong adhesive type | Poor repair quality | Use manufacturer-specified adhesive |
| Skipping grounding | ESD damage to components | Wear ESD strap, use ESD mat |
| Reusing old screws | Screw damage, short circuits | Use new screws from repair kit |

## 10.3 Misdiagnosis Issues

| Incorrect Diagnosis | Actual Problem | Check |
|--------------------|----------------|-------|
| Dead battery | Charging port debris | Clean port with wooden toothpick |
| Screen failure | Software freeze | Force restart first |
| No sound | Phone in silent mode | Check physical switch |
| Camera blurry | Lens protector bubble | Remove and inspect |
| Won't charge | Dirty USB-C/Lightning | Clean with compressed air |

## 10.4 Parts and Tools Pitfalls

| Issue | Problem | Solution |
|-------|---------|----------|
| Cheap screens | Poor color accuracy, fragile | Source OEM or high-quality aftermarket |
| Wrong adhesive | Gaps, dust entry | Match adhesive specs exactly |
| Mismatched batteries | Capacity mismatch, safety | Verify model number precisely |
| Improper tools | Scratches, component damage | Use manufacturer-recommended tools |
| Outdated software | Calibration errors | Update firmware before repair |

## 10.5 Customer Communication Errors

- **Promising unrealistic timelines** - Always add buffer time
- **Not documenting pre-repair condition** - Takes photos/video before any work
- **Skipping repair estimate approval** - Get written authorization
- **Not explaining warranty implications** - Aftermarket parts void OEM warranty
- **Forgetting to mention data backup need** - Always advise customer

## 10.6 Safety Violations

| Violation | Consequence | Safety Protocol |
|-----------|------------|-----------------|
| Working without ESD protection | Component damage | Always use ESD mat and wrist strap |
| Disposing batteries in regular trash | Environmental hazard, fire | Follow local battery disposal laws |
| Using damaged tools | Injury, poor quality work | Inspect and replace worn tools |
| Blocking emergency exits | OSHA violation | Maintain clear pathways |
| Improper lifting | Back injury | Use proper lifting techniques |

## 10.7 Quality Control Failures

- [ ] Not testing repair before returning device
- [ ] Missing screws (causes rattling, loose components)
- [ ] Leaving protective films on new screens
- [ ] Not calibrating repaired components
- [ ] Skipping final cleaning before return
- [ ] Forgetting to return removed parts to customer

## 10.8 Business Practice Mistakes

- Not documenting warranty repair conditions
- Failure to maintain repair logs
- Not calibrating equipment regularly
- Skipping tool maintenance
- Improper customer data handling (privacy concerns)
