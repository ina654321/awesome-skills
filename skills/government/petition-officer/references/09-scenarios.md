# Scenario Examples

## 9.1 Late Filing with Excuse

**User:** "Petitioner submits tax refund claim 3 months after the 1-year deadline. They claim illness prevented timely filing. How handle?"

**Expert:**
> **Timeliness Analysis:**
> 
> 1. **Statutory Check**:
>    - 26 USC 6511: Time limits for claims
>    - Identify specific deadline
>    - Calculate days late
> 
> 2. **Equitable Tolling**:
>    - Apply if extraordinary circumstances
>    - Document medical evidence
>    - Assess whether prejudice to government
>    - Consider IRM guidance
> 
> 3. **Decision**:
>    - If valid excuse: Process on merits
>    - If not: Issue timeliness rejection
>    - Document all reasoning
>    - Provide appeal rights

## 9.2 Duplicate Petitions

**User:** "We received 3 copies of the same petition from the same person, each with slightly different information. How to handle?"

**Expert:**
> **Duplicate Handling:**
> 
> 1. **Consolidation**:
>    - Identify the most complete version
>    - Combine all relevant information
>    - Note all submissions in file
> 
> 2. **Notification**:
>    - Ask petitioner which version to use
>    - Or accept most recent
>    - Document decision rationale
> 
> 3. **Processing**:
>    - Single case number
>    - Single response
>    - Retain all versions in file

## 9.3 Class Action Petition

**User:** "Petition filed on behalf of 500 similarly-situated individuals. How to process?"

**Expert:**
> **Class Petition Protocol:**
> 
> 1. **Standing Assessment**:
>    - Verify named petitioner has standing
>    - Assess typicality
>    - Evaluate adequacy of representation
> 
> 2. **Coordination**:
>    - Notify supervisor immediately
>    - Consider legal consultation
>    - Evaluate policy implications
> 
> 3. **Response Options**:
>    - Individual response to named petitioner
>    - Notice to all class members
>    - May require higher-level review
