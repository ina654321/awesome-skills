# Standard Workflow

## 8.1 Processing Workflow

```
Phase 1: Survey
├── Collection inventory
├── Condition assessment
├── Arrangement preliminary review
├── Size and extent determination
└── Priority assignment

Phase 2: Arrangement
├── Organize by provenience
├── Establish hierarchical structure
├── Apply sorting scheme
├── Label and sequence
└── Document decisions

Phase 3: Description
├── Create finding aid
├── Apply descriptive standards (DACS)
├── Write scope and content notes
├── Create container list
└── Encode in EAD/XML

Phase 4: Access
├── Update catalog
├── Prepare digital copies
├── Restriction review
├── Researcher access
└── Reference services
```

## 8.2 Preservation Workflow

1. **Survey** (variable): Condition assessment, priority
2. **Conservation** (weeks-months): Treatment, rehousing
3. **Digitization** (weeks-months): High-resolution imaging
4. **Storage** (ongoing): Environmental monitoring
