# Standard Workflow

## 8.1 Emergency Response Protocol

```
Phase 1: Scene Assessment (0-5 min)
├── Size-up: Type of incident, hazards present
├── Establish incident command (ICS)
├── Initial resource request
└── Safety officer assignment

Phase 2: Operational Deployment (5-30 min)
├── Search teams assignment
├── Rescue operations initiation
├── Medical triage establishment
├── Staging area designation
└── Communications setup

Phase 3: Victim Recovery (30 min - hours)
├── Systematic search pattern
├── Locating and stabilizing victims
├── Extrication operations
├── Casualty collection point
└── Transport coordination

Phase 4: Scene Stabilization
├── Hazard mitigation
├── Secondary search
├── Demobilization planning
└── After-action documentation
```

## 8.2 Search Pattern Types

1. **Parallels**: Linear search, parallel paths, 100% coverage
2. **Expanding Spiral**: Circular pattern from datum point
3. **Creeping Line**: Team advances in line formation
4. **Grid Search**: Systematic grid pattern for large areas

## 8.3 Rescue Technical Operations

| Operation | Application |
|-----------|-------------|
| **Rope Rescue** | Low angle, high angle, confined space |
| **Vehicle Extrication** | MVA, machinery entrapment |
| **Structural Collapse** | USAR Type I, II, III |
| **Water Rescue** | Swift water, dive operations |
