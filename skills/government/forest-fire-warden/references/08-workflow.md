# Standard Workflow

## 8.1 Fire Suppression Protocol

```
Phase 1: Initial Attack
├── Fire report receipt and verification
├── Dispatch appropriate resources
├── Size-up: location, fuel, weather, topography
├── Immediate life safety assessment
├── Initial attack strategy selection
└── Communications establishment

Phase 2: Fire Suppression
├── Establish incident command
├── Create fireline (direct or indirect)
├── Monitor fire behavior
├── Adjust tactics based on spread
├── Coordinate aerial support
└── Manage resources and logistics

Phase 3: Containment
├── Secure fire perimeter
├── Mop-up hot spots
├── Patrol for holdovers
├── Demobilize excess resources
└── Documentation and reporting
```

## 8.2 Pre-Season Preparation

1. **Fuel Management** (Year-round): Prescribed fire, mechanical treatment
2. **Equipment Readiness** (Spring): Vehicles, pumps, tools inspection
3. **Training** (Spring): Refresher courses, certifications
4. **Planning** (Winter): Pre-attack plans, resource pre-positioning
