# Standard Workflow

## 8.1 Collection Acquisition Workflow

```
Phase 1: Initial Review
├── Review donation offer or acquisition request
├── Check collection policy scope
├── Preliminary provenance research
└── Assess condition and storage needs

Phase 2: Formal Evaluation
├── Legal review (title, provenance)
├── Curatorial assessment (significance)
├── Conservation evaluation
├── Board review for significant acquisitions
└── Budget impact analysis

Phase 3: Decision & Documentation
├── Acquisitions committee recommendation
├── Board approval (if required)
├── Execute transfer of title
├── Complete accession documentation
└── Catalog and integrate into collection
```

## 8.2 Exhibition Development

1. **Concept Development** (Weeks 1-4): Theme selection, story development, scholarly review
2. **Object Selection** (Weeks 5-8): Inventory check, conservation assessment, loan coordination
3. **Design Phase** (Weeks 9-16): Spatial planning, graphics design, mounting fabrication
4. **Installation** (Weeks 17-20): Object mounting, lighting adjustment, label placement
5. **Opening** (Week 21): Preview, press materials, public programming

## 8.3 Deaccessioning Process

Per AAM Guidelines:
1. Review against collection policy
2. Document rationale (redundancy, condition, relevance)
3. Conservation review
4. Disposal method selection (transfer, exchange, sale)
5. Board approval required
6. Proceeds use restrictions
