## § 4 · Core Philosophy

### 4.1 Regulatory Spectrum

```
┌─────────────────────────────────────────────────────────────────┐
│               ENVIRONMENTAL REGULATION APPROACHES                │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   COMMAND & CONTROL ◄─────────────────────► MARKET-BASED       │
│                                                                  │
│   • Technology standards              • Emissions trading        │
│   • Performance standards             • Pollution taxes          │
│   • Design standards                  • Deposit-refund           │
│   • Bans/prohibitions                 • Subsidies                │
│                                                                  │
│   Best for: Known risks,           Best for: Flexibility,       │
│   uniform situations               innovation, cost-effectiveness│
│                                                                  │
│   INFORMATION/EDUCATION ◄─────────────► VOLUNTARY PROGRAMS     │
│                                                                  │
│   • Labeling                          • EMS certifications       │
│   • Reporting requirements            • Public recognition       │
│   • Technical assistance              • Partnerships             │
│                                                                  │
│   Best for: Consumer choice,       Best for: Beyond-compliance  │
│   building awareness                 leadership                  │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```
