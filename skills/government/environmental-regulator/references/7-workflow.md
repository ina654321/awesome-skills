## § 7 · Workflow

### 7.1 Enforcement Case Development

**Phase 1: Discovery (Weeks 1-4)**
```
□ Inspection findings or citizen complaint received
□ Preliminary investigation (records review, sampling)
□ Violation confirmation and classification
□ Case file opened; evidence preserved
Deliverable: Investigation report with violation summary
```

**Phase 2: Case Development (Weeks 5-12)**
```
□ Detailed evidence gathering
□ Document review and witness interviews
□ Sampling and laboratory analysis
□ Economic benefit calculation
□ Gravity factor assessment
Deliverable: Case file with complete evidence package
```

**Phase 3: Enforcement Action (Weeks 13-24)**
```
□ Notice of violation issued
□ Settlement negotiations (if appropriate)
□ Administrative order or complaint filed
□ Penalty determination per policy
□ Supplemental Environmental Project (SEP) negotiation
Deliverable: Final order or settlement agreement
```

**Phase 4: Compliance & Closure (Weeks 25-52)**
```
□ Compliance schedule monitoring
□ Progress reports review
□ Verification inspection
□ Final compliance determination
□ Case closure
Deliverable: Case closure memorandum
```
