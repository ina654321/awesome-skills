# Standard Workflow

## 8.1 Primary Inspection Protocol

```
Phase 1: Document Review
├── Verify passport validity (6+ months)
├── Check visa/ESTA status
├── Review I-94 requirements
├── Query lookout databases (NCIC, CCD)
└── Verify travel purpose

Phase 2: Interview
├── Purpose of travel
├── Duration of stay
├── Address in US
├── Financial means
├── Prior US visits
└── Return ticket (if required)

Phase 3: Determination
├── Admissible: Stamp I-94, issue admission
├── Referral: Secondary inspection
├── Visa rev: Place in removal proceedings
└── Expedited removal: If applicable
```

## 8.2 Secondary Inspection

1. **Extended Interview** (15-60 min): Detailed questioning
2. **Document Verification** (30 min): Consular verification
3. **Biometric Check** (10 min): Fingerprint/photo
4. **Database Review** (30 min): Full system check
5. **Supervisor Review** (15 min): Final decision

## 8.3 inadmissibility Grounds

| Ground | Authority | Action |
|--------|-----------|--------|
| **Health** | INA 212(a)(1) | Referral to USPH |
| **Criminal** | INA 212(a)(2) | Waiver or bar |
| **Public Charge** | INA 212(a)(4) | Bond or denial |
| **Labor Cert** | INA 212(a)(5) | Visa invalid |
