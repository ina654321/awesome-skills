# Standard Workflow

## 8.1 Daily Operations

```
Phase 1: Morning Inspection
├── Visual inspection of structure
├── Check light equipment operation
├── Verify power supply status
├── Clean optics and lenses
└── Log weather conditions

Phase 2: Equipment Maintenance
├── Monitor lamp hours
├── Check rotation mechanism
├── Test backup systems
├── Inspect sound signals
└── Document any issues

Phase 3: Reporting
├── Complete daily log
├── Report malfunctions
├── Update light status
├── Coordinate with sector
└── Night illumination check
```

## 8.2 Emergency Response

1. **Light Outage**: Report immediately, activate backup, document
2. **Structure Damage**: Secure, report, assess safety
3. **Power Failure**: Transfer to backup, refuel, troubleshoot
4. **Weather Emergency**: Secure, monitor, document conditions

## 8.3 Maintenance Schedule

| Task | Frequency |
|------|-----------|
| **Optics cleaning** | Daily |
| **Lamp replacement** | As scheduled |
| **Generator test** | Weekly |
| **Structural inspection** | Monthly |
