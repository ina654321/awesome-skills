# Standard Workflow

## 8.1 Statistical Project Lifecycle

```
Phase 1: Requirements Definition
├── Define research questions
├── Identify data needs
├── Determine methodology
├── Specify quality standards
└── Establish timeline

Phase 2: Data Collection/Acquisition
├── Survey design (if primary)
├── Secondary data sourcing
├── Data transfer agreements
├── Initial data quality checks
└── Secure storage setup

Phase 3: Data Processing
├── Data cleaning
├── Variable coding
├── Missing data treatment
├── Weighting adjustments
└── Derived variables

Phase 4: Analysis
├── Descriptive statistics
├── Hypothesis testing
├── Regression/modeling
├── Sensitivity analysis
└── Results interpretation

Phase 5: Dissemination
├── Technical documentation
├── Public-use files
├── Disclosure review
├── Release materials
└── Data archiving
```

## 8.2 Survey Implementation

1. **Sampling**: Frame construction, sample selection, coverage assessment
2. **Instrument Design**: Questionnaire development, cognitive testing
3. **Data Collection**: Interviewer training, field procedures, monitoring
4. **Processing**: Editing, imputation, weighting
5. **Estimation**: Point estimates, variance estimation, benchmarking

## 8.3 Quality Assurance

| Check | Purpose |
|-------|---------|
| **Edit Checks** | Identify invalid values |
| **Consistency** | Cross-variable validation |
| **Trends** | Compare to historical data |
| **Benchmarking** | Align with other sources |
