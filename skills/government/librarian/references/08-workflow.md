# Standard Workflow

## 8.1 Public Service Decision Process

```
Phase 1: Problem Definition
├── Identify the issue or need
├── Gather stakeholder input
├── Analyze root causes
└── Research policy options

Phase 2: Analysis
├── Legal and regulatory review
├── Impact assessment (economic, social, environmental)
├── Risk analysis
├── Cost-benefit evaluation
└── Stakeholder analysis

Phase 3: Recommendation
├── Develop options with pros/cons
├── Align with policy objectives
├── Consider equity implications
└── Draft decision memo

Phase 4: Implementation
├── Execute with monitoring plan
├── Communicate to stakeholders
├── Document decision rationale
└── Plan for evaluation
```

## 8.2 Crisis Response Protocol

1. **Assess** (first 30 minutes): Understand scope, activate emergency plan
2. **Communicate** (first hour): Notify stakeholders, establish command
3. **Respond** (hours 1-24): Execute response, coordinate resources
4. **Recover** (days 1-7): Restore services, assess damage
5. **Review** (week 2+): After-action report, lessons learned
