# Standard Workflow

## 8.1 Bridge Watch-Keeping Workflow

```
Step 1: Pre-Watch Preparation (15 minutes before)
├── Review chart: intended track, traffic, weather
├── Check position: GPS, radar, visual bearings
├── Verify ECDIS is updated: Notice to Mariners
├── Brief with off-going watch: traffic, operational issues, orders
├── Check communications: VHF channel, AIS, GMDSS
└── Log position and time at watch start

Step 2: Continuous Watch (during duty)
├── Monitor position: hourly position plots on chart
├── Traffic awareness: radar, visual lookouts, AIS targets
├── Weather monitoring: wind, visibility, sea state
├── Respond to traffic: COLREGS application, sound signals
├── Respond to alarms: engine, navigation, fire
└── Communicate anomalies: call captain if any doubt

Step 3: Watch Relief
├── Brief on-coming watch: traffic, position, weather, course
├── Verify position: GPS/radar cross-check
├── Review passage plan: remaining legs, ETA
├── Transfer watch: verbal confirmation
└── Log position and time at watch relief
```

## 8.2 Cargo Operations Workflow

```
Phase 1: Pre-Cargo Planning
├── Review cargo plan: stowage sequence, tank/hold assignments
├── Verify cargo documentation: manifest, dangerous goods declaration
├── Safety meeting: cargo hazards, PPE requirements, communication
├── Check equipment: hoses, cranes, pumps, inert gas
├── Verify compatibility: tank cleaning status, previous cargo
└── Confirm environmental conditions: wind speed limits for cargo ops

Phase 2: Loading/Discharging
├── Gangway watch: access control, PPE enforcement
├── Pump room watch: leak detection, pump monitoring
├── Ullage/temperature measurement: at start, mid, end of operation
├── Communication with shore: agreed signal codes
├── Draft survey: before, during (if mid-stream), after
└── Emergency stop procedures understood by all

Phase 3: Post-Cargo
├── Verify quantity transferred: mass/volume from instruments
├── Complete documentation: mate's receipts, bill of lading
├── Tank/hold inspection: confirm clean, sealed
├── Disconnect and secure: hoses, arms, manifolds
└── Notify port authority if required
```

## 8.3 Emergency Response Workflow

| Emergency | Immediate Action |
|-----------|------------------|
| **Fire/Explosion** | Alarm, locate fire, muster crew, $\mathrm{CO_2}$/foam discharge per fire plan |
| **Man overboard** | Shout MOB (starboard side), throw buoys, plot position, return and recover |
| **Collision** | Stop engines, assess damage, call captain, check tanks for leakage |
| **Grounding** | Stop engines, assess grounding, do not refloat without assessment |
| **Flooding** | Isolate flooding compartment, activate pumps, notify captain |
| **Abandon ship** | General alarm, muster, launch lifeboats, activate EPIRB, survival suit |
| **Oil spill** | Stop source, notify USCG/NMSZ, contain with booms if possible |
