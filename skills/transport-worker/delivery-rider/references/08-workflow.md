# Standard Workflow

## 8.1 Daily Pre-Delivery Workflow

```
Step 1: Vehicle Inspection (Before Loading)
├── Walk around inspection: lights, tires, mirrors, reflectors
├── Check fluid levels: fuel, engine oil, coolant, washer fluid
├── Verify registration, insurance, and ELD device operational
├── Inspect cargo area: clean, secure, free of hazards
├── Test brakes: parking and service brake operation
├── Verify backup alarm and camera operational
└── Sign vehicle inspection report

Step 2: Route Planning
├── Review manifest: delivery stops, special instructions
├── Organize load: heaviest and first stops loaded last
├── Identify time-window deliveries: priority scheduling
├── Note signature-required, access code, or restricted deliveries
├── Plan alternate routes for known construction or closures
└── Verify GPS and communication devices

Step 3: Safe Loading Procedures
├── Team lift for packages > 50 lbs
├── Secure packages: strapping, blocking, bracing
├── Verify hazmat packages: placarding, shipping papers, PPE
├── Check temperature-controlled shipments
└── Confirm package count vs. manifest

Step 4: En-Route Protocol
├── Follow designated route; report delays immediately
├── Maintain safe following distance: 3–4 seconds
├── Complete all required logs: ELD, delivery records
├── Secure vehicle at every stop: remove keys, lock cargo area
├── Report any package damage or discrepancy
└── Contact dispatch if running > 30 min behind schedule

Step 5: End-of-Day Procedures
├── Complete all delivery confirmations in handheld device
├── Return to depot: unload all packages, document exceptions
├── Report vehicle maintenance needs
├── Clean cargo area; remove personal items
├── Attend end-of-day safety meeting if scheduled
└── Verify next-day route assignment
```

## 8.2 Hazmat Delivery Protocol

1. **Verify hazmat certification:** HAZMAT endorsement required for Class A CDL
2. **Inspect shipping papers:** Emergency response guidebook (ERG) on board
3. **Verify placarding:** Match placards to shipping papers
4. **Load separation:** Compatible classes separated per 49 CFR 177.848
5. **En route:** No cell phone use in hazmat zones; no eating/drinking
6. **Emergency response:** Call 911; evacuate; reference ERG for hazmat code

## 8.3 Delivery Exception Handling

| Exception | Action |
|-----------|--------|
| **Customer not home** | Leave notice; re-attempt next business day |
| **Wrong address** | Return to depot; update address in system |
| **Refused delivery** | Return to depot; document refusal with photo |
| **Damaged package** | Document damage; return; notify dispatch |
| **Access restricted** | Contact customer; reschedule or leave at secure location |
| **Weather delay** | Contact dispatch; log delay; ensure cargo security |
