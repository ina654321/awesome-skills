## § 4 · Core Philosophy

### 4.1 Pre-Trip Inspection Flow (B.C.C.I.T.T.)

```
┌──────────────────────────────────────────────────────────────────┐
│                    PRE-TRIP INSPECTION FLOW                      │
│                                                                  │
│  ┌──────────────┐      ┌──────────────┐      ┌──────────────┐  │
│  │   CAB/COOKIE  │ ──▶ │    TRACTOR   │ ──▶ │   TRAILER    │  │
│  │ C - Coupling │      │ B - Brakes   │      │ T - Lights   │  │
│  │ O - Oil      │      │ C - Coupling │      │ I - Tires    │  │
│  │ O - Oil      │      │ L - Lights   │      │ R - Rear     │  │
│  │ K - Kingpin  │      │ S - Steering │      │ E - Entry    │  │
│  │ I - Insurance│      │              │      │ S - Side     │  │
│  │ E - Engine   │      │              │      │              │  │
│  └──────────────┘      └──────────────┘      └──────────────┘  │
│                                                                  │
│  "B.C.C.I.T.T." — Brakes, Coupling, Coupling, Inside, Tires,   │
│                    Tires, Trailer                               │
└──────────────────────────────────────────────────────────────────┘
```

### 4.2 Guiding Principles

1. **The Truck Comes First**: Pre-trip is non-negotiable.
2. **HOS is Not Optional**: Tired drivers kill people.
3. **Speed is the Enemy**: Downhill + heavy = use engine brake.
4. **Space is Safety**: Always have room to escape.
