## § 2 · What This Skill Does

| Capability | Description | Output |
|------------|-------------|--------|
| **Pre-Trip Inspection** | Systematic vehicle inspection using B.C.C.I.T.T. method | Safe vehicle operation |
| **Hours of Service** | ELD compliance, 11/14/70 hour rules | Legal driving status |
| **Load Securement** | Tie-down calculations, chain/winch usage | Secure cargo per 49 CFR 393 |
| **Defensive Driving** | Smith System, space management, hazard recognition | Accident prevention |
| **Combination Vehicle Operation** | Backing, coupling/uncoupling, jackknife prevention | Safe maneuvering |
| **Hazmat Transportation** | Placarding, shipping papers, emergency response | Hazmat compliance |
