## § 6 · Domain Knowledge

### 6.1 Reference Service Types

| Type | Description | Example |
|------|-------------|---------|
| **Ready Reference** | Quick, factual answers | "What is the capital of Finland?" |
| **Research Support** | In-depth assistance | "I need sources for my thesis on climate policy" |
| **Reader Advisory** | Reading recommendations | "What should I read after enjoying The Night Circus?" |
| **Instruction** | Teaching information skills | Database workshop; citation class |
| **Technical Support** | Device and software help | "How do I download an ebook?" |
| **Referral** | Connecting to other services | "You need the genealogy center; here's how to reach them" |

### 6.2 Collection Development Criteria

| Factor | Consideration | Weight |
|--------|---------------|--------|
| **Community demand** | Holds; circulation trends; requests | High |
| **Quality** | Reviews; awards; author reputation | High |
| **Diversity** | Representation; own voices | High |
| **Cost** | Budget constraints; price-per-use | Medium |
| **Format** | Print; ebook; audiobook; streaming | Medium |
| **Space** | Physical collection constraints | Medium |
| **Curriculum alignment** | Educational support (academic) | High |

### 6.3 Information Literacy Frameworks

**ACRL Framework for Information Literacy:**
| Frame | Concept | Skill |
|-------|---------|-------|
| **Authority** | Who created this? Why trust them? | Evaluate sources |
| **Information Creation** | Format reflects process | Match need to format |
| **Information Value** | Information has economic, social value | Ethical use; access issues |
| **Research as Inquiry** | Research is iterative questioning | Develop research questions |
| **Scholarship as Conversation** | Disciplines communicate | Citation; synthesis |
| **Searching as Strategic Exploration** | Search requires planning | Search strategies |
