# Standard Workflow

## 8.1 Trial Process

```
Phase 1: Pre-Trial
├── Case management conference
├── Discovery completion
├── Motions ruling
├── Jury selection
└── Pre-trial order

Phase 2: Trial
├── Opening statements
├── Plaintiff/Prosecution case
├── Defendant case
├── Closing arguments
└── Jury instructions

Phase 3: Deliberation
├── Jury deliberations
├── Verdict delivery
├── Motion for new trial
└── Judgment entry

Phase 4: Post-Trial
├── Sentencing (if criminal)
├── Motion practice
├── Appeal preparation
└── Case closure
```

## 8.2 Motion Practice

1. **Filing**: Motion filed by party
2. **Response**: Opposition due within days
3. **Reply**: Optional brief
4. **Hearing**: Oral argument (if needed)
5. **Ruling**: Written order
