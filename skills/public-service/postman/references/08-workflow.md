# Standard Workflow

## 8.1 Daily Delivery Process

```
Phase 1: Office Preparation (6:00-8:00 AM)
├── Clock in and retrieve route assignment
├── Review days of week adjustments (Monday heavier)
├── Pull mail from cases by route
├── Check for Express, Priority, Certified mail
├── Prepare accountable mail for signature
├── Load vehicle with packages and mail
└── Final route check before leaving

Phase 2: Line of Travel (8:00 AM-4:00 PM)
├── Deliver in prescribed sequence
├── Attempt delivery at every address
├── Leave notice (PS Form 3849) for:
│   ├── Undeliverable as addressed
│   ├── Insufficient address
│   ├── Missorted or misrouted
│   └── Package too large for box
├── Obtain signature for accountable mail
└── Return undeliverable mail to office

Phase 3: End of Day (4:00-5:00 PM)
├── Return with accountable mail and coins
├── Complete required forms
├── Document any problems encountered
├── Update case and leave for next day
└── Clock out accurately
```

## 8.2 Address Verification

| Issue | Action |
|-------|--------|
| **No such address** | Return to sender with endorsement |
| **Vacant address** | Check if new resident; hold if needed |
| **No receptacle** | Leave notice for customer to get one |
| **Dog in yard** | Attempt later, dog notice if aggressive |
| **Blocked path** | Leave notice, attempt at different time |
| **Business closed** | Attempt different time, leave notice |

## 8.3 Special Mail Handling

```
Certified Mail:
├── Obtain signature on delivery
├── Show tracking scan at delivery
└── Retain signature card

Restricted Delivery:
├── Verify recipient ID
├── Only deliver to named person
└── No proxy signature

Hold for Pickup:
├── Retain at post office
├── Notify customer via PS Form 3849
├── Hold for 15 days
└── Return to sender if unclaimed
```
