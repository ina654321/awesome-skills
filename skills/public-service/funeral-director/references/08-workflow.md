# Standard Workflow

## 8.1 Arrangements Process

```
Phase 1: Initial Contact
├── Family meeting scheduled
├── Gather deceased information
├── Review services available
├── Explain pricing (FTC requirements)
└── Discuss preferences

Phase 2: Service Planning
├── Select burial or cremation
├── Choose merchandise (casket, urn)
├── Schedule service date/time
├── Coordinate with cemetery/crematory
└── Arrange clergy/officiant

Phase 3: Documentation
├── Death certificate preparation
├── Authorizations obtained
├── Vital statistics filed
├── Social Security notification
└── Military benefits (if applicable)

Phase 4: Service Execution
├── Embalming (if applicable)
├── Preparation of remains
├── Visitations/viewing
├── Funeral/memorial service
└── Final disposition
```

## 8.2 Death Certificate Process

1. **Physician/ME Completion**: Medical certification
2. **Registrar Filing**: Local vital statistics
3. **Burial Transit Permit**: For final disposition
4. **Certified Copies**: For family

## 8.3 Pre-Planning Option

| Service | Description |
|---------|-------------|
| **Preneed** | Pre-arranged, pre-funded |
| **At-need** | Immediate arrangements |
| **Direct disposition** | No ceremony option |
