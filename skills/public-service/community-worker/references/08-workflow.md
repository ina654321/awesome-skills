# Standard Workflow

## 8.1 Client Intake Process

```
Phase 1: Initial Contact
├── Verify eligibility for program
├── Collect intake information
├── Complete required forms
├── Assign case number
└── Schedule comprehensive assessment

Phase 2: Comprehensive Assessment
├── Identify presenting needs (housing, food, income, health)
├── Assess immediate safety concerns
├── Evaluate support system
├── Screen for trauma history
└── Determine acuity level

Phase 3: Service Planning
├── Prioritize immediate needs
├── Develop individualized service plan
├── Identify goals with client
├── Establish timeline
└── Assign case manager responsibilities

Phase 4: Referral & Connection
├── Make warm referrals to partner agencies
├── Assist with applications
├── Provide direct financial assistance if available
└── Schedule follow-up appointments
```

## 8.2 Case Management Workflow

1. **Document**: Every client interaction, assessment, referral
2. **Follow up**: Track referrals, confirm appointments kept
3. **Reassess**: Periodically update needs assessment
4. **Coordinate**: Multi-agency case conferences for complex cases
5. **Advocate**: Help clients navigate systems
6. **Close**: When goals met or client exits program

## 8.3 Emergency Response Protocol

```
Safety Assessment Triggered:
├── Immediate: Is client in danger? Call 911 if yes
├── Assess: Is there imminent risk of homelessness?
├── Stabilize: Emergency shelter, food, clothing
├── Document: Complete incident report
├── Escalate: Notify supervisor for high-risk cases
└── Follow up: Daily check-ins until stabilized
```
