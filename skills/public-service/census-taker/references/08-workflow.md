# Standard Workflow

## 8.1 Census Enumeration Process

```
Phase 1: Pre-Enumeration Preparation
├── Obtain assignment listing
├── Review address register for area
├── Map and plan route
├── Prepare ID badge, materials, equipment
└── Know questionnaire thoroughly

Phase 2: Initial Contact
├── Travel to assigned address
├── Make first attempt — all hours of day
├── If no answer: leave Notice of Visit (NC-1)
├── Record attempt in system (date, time, result)
└── Maximum 6 attempts per address

Phase 3: Enumeration Interview
├── Greet respondent professionally
├── Verify address information
├── Enumerate all household members
├── Ask all applicable questions
├── Record proxy information if needed
└── Thank respondent and confirm completion

Phase 4: Non-Response Follow-Up
├── Attempt at different times of day
├── Attempt on different days
├── Use proxy respondents (neighbors, landlords)
├── Document all attempts
└── Refer to Field Supervisor for problem cases
```

## 8.2 Questionnaire Administration

| Question Type | Handling |
|--------------|----------|
| **Population questions** | Count all persons, usual residence rule |
| **Relationship** | How each person relates to Person 1 |
| **Demographics** | Age, sex, race, Hispanic origin |
| **Housing** | Ownership, occupancy, utilities |
| **Houseless** | Enumerate at shelters, outdoor locations |

## 8.3 Difficult Situations

1. **Language barrier**: Use language assistance card, schedule return visit
2. **Refusal**: Call supervisor, attempt to convince, leave materials
3. **Dangerous area**: Report to supervisor, do not enter alone
4. **Locked building**: Get contact information from landlord/manager
5. **Vacant unit**: Verify vacancy, document condition
