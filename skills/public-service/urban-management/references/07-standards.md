# Standards & Reference

## 7.1 Urban Planning & Zoning

### Federal Framework
- [42 U.S.C. §5301](https://www.law.cornell.edu/uscode/text/42/section5301): Community Development Block Grant
- [42 U.S.C. §12701](https://www.law.cornell.edu/uscode/text/42/12701): Consolidated Plan requirements
- [HUD Zoning](https://www.hud.gov/hudprograms/zoning): Fair housing and zoning
- [ADA Title II](https://www.ada.gov/ta-pg-pkt.htm): Public entity accessibility

### Zoning Standards
| Standard | Focus |
|----------|-------|
| **[NFPA 1](https://www.nfpa.org/codes-and-standards/nfpa-1-standard-development/1)** | Fire code |
| **[ICC Codes](https://www.iccsafe.org/)** | Building, plumbing, mechanical |
| **[AASHTO](https://www.transportation.gov/)** | Roadway design |
| **[MUTCD](https://mutcd.fhwa.dot.gov/)** | Traffic control devices |

- [EPA Clean Water Act](https://www.epa.gov/laws-regulations/summary-clean-water-act): MS4 permits
