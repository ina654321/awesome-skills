# Standard Workflow

## 8.1 Code Enforcement Workflow

```
Phase 1: Complaint Intake
├── Log complaint with date, time, caller
├── Document exact address and description
├── Photograph condition if possible
├── Assign to inspector by zone
└── Set response priority (emergency, urgent, routine)

Phase 2: Field Inspection
├── Visit property, document all violations
├── Photograph from multiple angles
├── Identify property owner (assessor's records)
├── Determine applicable codes
├── Estimate cost to cure
└── Complete inspection report

Phase 3: Notice & Compliance
├── Issue Notice of Violation (NOV)
├── Allow compliance period (30-60 days typical)
├── Document service of notice
├── Conduct re-inspection at compliance deadline
├── Assess penalties if not cured
└── Escalate to legal if needed
```

## 8.2 Permit Review Process

1. **Pre-application**: Meeting with applicant, zoning verification
2. **Formal submission**: Plans, applications, fees
3. **Staff review**: Zoning, traffic, environmental, fire
4. **Comments**: Issues requiring revision
5. **Approval/Conditions**: With conditions or denial rationale
6. **Appeal period**: Applicant can appeal to board

## 8.3 Emergency Response

```
Natural Disaster (Flood, Fire, Earthquake):
├── Damage assessment teams deployed
├── Priority: Life safety structures
├── Triage: Habitable, repairable, condemn
├── Debris removal coordination
├── Temporary housing if needed
└── Long-term rebuilding coordination

Public Health Emergency:
├── Coordinate with health department
├── Issue emergency orders
├── Close affected facilities
├── Deploy resources
└── Public communication
```
