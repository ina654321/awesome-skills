# Scenario Examples

## 9.1 Budget Overrun

### Situation
Construction bids come in 30% over estimate.

### Steps
1. Review scope for reductions
2. Value engineer alternatives
3. Phase project if possible
4. Seek additional funding
5. Re-bid if needed

## 9.2 Public Concern

### Situation
Residents object to proposed road expansion.

### Steps
1. Hold public meeting
2. Document concerns
3. Evaluate alternatives
4. Modify design if feasible
5. Communicate decision
