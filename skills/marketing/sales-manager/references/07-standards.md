# Standards & Reference

## 7.1 Marketing Metrics

### KPIs by Channel
| Channel | Metrics |
|---------|---------|
| **SEO** | Organic traffic, rankings, CTR |
| **PPC** | CPC, ROAS, conversion rate |
| **Email** | Open rate, CTR, unsubscribe |
| **Social** | Engagement, reach, follower growth |
| **Content** | Page views, time on page |

### Funnel Metrics
| Stage | Metric | Target |
|-------|--------|--------|
| Awareness | Impressions, reach | — |
| Interest | Engagement, time on site | > 2 min |
| Consideration | Add to cart, demo request | 2-5% |
| Conversion | Purchase, sign-up | 1-3% |
| Retention | LTV, repeat purchase | > 30% |

## 7.2 Budget Allocation

| Channel | Typical % | Consideration |
|---------|-----------|---------------|
| Digital | 40-60% | Performance focus |
| Content | 15-25% | SEO, thought leadership |
| Events | 10-20% | Trade shows, webinars |
| Brand | 10-15% | Awareness building |

## 7.3 Attribution Models

- **First touch**: Credit to first interaction
- **Last touch**: Credit to last interaction
- **Linear**: Equal credit all touchpoints
- **Time decay**: More credit recent touches
- **Position-based**: 40% first, 40% last, 20% middle
