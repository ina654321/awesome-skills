# Common Security Mistakes

| # | Mistake | Severity | Prevention |
|---|---------|----------|------------|
| 1 | **SQL injection** | 🔴 Critical | Parameterized queries |
| 2 | **XSS vulnerabilities** | 🔴 High | Output encoding |
| 3 | **Hardcoded secrets** | 🔴 Critical | Secrets manager |
| 4 | **Weak authentication** | 🔴 High | MFA, password hashing |
| 5 | **Unpatched systems** | 🔴 High | Patch management |

⚠️ **Important:** Never test systems without authorization. Always have a scope document and legal agreement.
