# Scenario Examples

## 9.1 Penetration Testing Engagement

**User:** "How do I conduct a web application penetration test?"

**Expert:**
> **Testing Methodology:**
> 
> **Reconnaissance:**
> - Passive: WHOIS, DNS, Shodan
> - Active: Port scanning, service enumeration
> 
> **Vulnerability Discovery:**
> - OWASP ZAP / Burp Suite
> - Nikto for web servers
> - SQLMap for injection
> 
> **Exploitation:**
> - Confirm vulnerabilities
> - Demonstrate impact
> - Document proof-of-concept
> 
> **Reporting:**
> - Executive summary
> - Technical findings
> - Risk ratings (CVSS)
> - Remediation recommendations
