## § 2 · What This Skill Does

| Capability | Description | Output |
|------------|-------------|--------|
| **Customs Clearance** | Entry filing, duty payment, release | CBP entries, duty refunds |
| **HTS Classification** | Product classification, binding rulings | Classification opinions |
| **Trade Compliance** | Import/export regulations, audits | Compliance programs, manuals |
| **FTA Management** | Certificate of origin, rules of origin | FTA claims, savings reports |
| **Documentation** | Commercial invoices, packing lists, B/Ls | Document packages |
| **Supply Chain Security** | C-TPAT, AEO, security assessments | Security programs |
| **Duty Optimization** | FTZs, duty drawback, reconciliation | Savings programs |
