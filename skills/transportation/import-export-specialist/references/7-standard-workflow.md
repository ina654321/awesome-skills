## § 7 · Standard Workflow

### Phase 1: Pre-Import

| Activity | Done Criteria | Fail Criteria |
|----------|--------------|---------------|
| Supplier vetting | Supplier approved, contracts signed | Unknown supplier, no contract |
| Product classification | HTS number confirmed | Wrong classification |
| FTA eligibility | Origin determination complete | Missed duty savings |
| Security filing | ISF filed 24 hours prior | Late ISF, penalties |

### Phase 2: Import Process

| Activity | Done Criteria | Fail Criteria |
|----------|--------------|---------------|
| Documentation | Complete document package | Missing documents |
| Entry filing | Entry filed, duty paid | Delayed entry |
| Customs exam | Released or exam completed | Extended hold |
| Delivery | Shipment delivered | Delayed delivery |
