# Common Pitfalls & Anti-Patterns

## 10.1 Regulatory Compliance Violations

| # | Pitfall | Severity | Consequence | Prevention |
|---|---------|----------|-------------|------------|
| 1 | Releasing aircraft without proper authorization | 🔴 Critical | FAA violation, grounding | Verify credentials |
| 2 | Using non-approved parts | 🔴 Critical | Aircraft damage, liability | Use approved parts only |
| 3 | Skipping required inspections | 🔴 Critical | Safety incident, violation | Follow maintenance program |
| 4 | Falsifying maintenance records | 🔴 Critical | Criminal charges, termination | Document accurately |
| 5 | Working beyond certification | 🔴 High | Quality escapes | Know your limits |

## 10.2 Technical Errors

| Mistake | Result | Correct Approach |
|---------|--------|------------------|
| Wrong torque values | Fastener failure | Reference manual specs |
| Incorrect parts installed | System failure | Verify part numbers |
| Improper safety wiring | Lost hardware | Follow standard practice |
| Skipping step in procedure | Defective work | Complete all steps |
| Wrong fluid/oil | Contamination | Check specifications |

## 10.3 Documentation Errors

| Error | Impact | Prevention |
|-------|--------|------------|
| Missing logbook entries | Untraceable history | Complete immediately |
| Incomplete work orders | Quality escapes | Detailed descriptions |
| Wrong part numbers | Wrong repair | Double-check documentation |
| Forgetting sign-offs | Regulatory issue | Checklist verification |
| Altering original entries | Fraud | New entry for corrections |

## 10.4 Safety Violations

| Violation | Risk | Protocol |
|-----------|------|----------|
| Not wearing PPE | Injury | Mandatory equipment |
| Working under suspended load | Fatality | Proper jacking/shoring |
| Hot work without clearance | Fire | Hot work permit |
| Ignoring electrical hazards | Shock, fire | De-energize and verify |
| Oxygen system negligence | Explosion | Follow oxygen safety |

## 10.5 Quality Control Failures

| Mistake | Result | Fix |
|---------|--------|-----|
| Skipping inspection step | Defects released | Proper inspection process |
| Not following MPD | Airworthiness issue | Use approved maintenance program |
| Ignoring deferred items | MEL violation | Track all deferred items |
| Poor handover | Missed work | Structured handoff |
| Incomplete functional test | Missed defects | Full functional check |

## 10.6 Tool and Equipment Issues

| Problem | Consequence | Prevention |
|---------|-------------|------------|
| Uncalibrated test equipment | False readings | Current calibration |
| Missing special tools | Improper repair | Proper tooling |
| Damaged precision tools | Quality issues | Regular inspection |
| Improper gauges | Wrong tolerance | Verify accuracy |
| Unauthorized substitutions | Compliance | Use approved tools |

## 10.7 Human Factors Errors

| Factor | Impact | Mitigation |
|--------|--------|------------|
| Fatigue | Mistakes | Rest requirements |
| Distraction | Skipped steps | Focus on task |
| Complacency | Missed defects | Inspection discipline |
| Pressure | shortcuts | Proper planning |
| Communication failure | Errors | Clear handoffs |

## 10.8 Parts Management Errors

| Error | Result | Best Practice |
|-------|--------|--------------|
| Installing expired parts | In-flight failure | Check dates |
| Wrong part number | Rework | Verify match |
| Damage during handling | Defective parts | Proper storage |
| Counterfeit parts | Unknown quality | Authorized sources |
| Untraceable parts | No warranty | Full traceability |

## 10.9 Procedure Adherence

- **Assuming instead of verifying** - Always confirm
- **Memory shortcuts** - Follow written procedures
- **Rushing** - Allow adequate time
- **Multi-tasking** - Focus on one task
- **Ignoring warnings** - Investigate all alerts

## 10.10 Engineering Judgment Errors

| Mistake | Impact | Prevention |
|---------|--------|------------|
| Deviating without approval | Unapproved modification | Get engineering approval |
| Over-repairing | Unnecessary cost | Minimum acceptable repair |
| Under-repairing | Reoccurrence | Thorough assessment |
| Ignoring service bulletins | Liability | Track and evaluate SBs |
| Not consulting OEM | Wrong repair | Engage manufacturer |
