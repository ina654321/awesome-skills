# Standard Workflow

## 8.1 Aircraft Inspection Workflow

```
Inspection Process
├── Pre-Inspection
│   ├── Review aircraft logs and records
│   ├── Check MEL/CDL status
│   ├── Gather required tools and equipment
│   └── Safety briefing
├── Inspection Execution
│   ├── Visual examination
│   ├── Functional checks
│   ├── Measurement and testing
│   └── Defect documentation
└── Post-Inspection
    ├── Deferred defect assessment
    ├── Logbook entries
    ├── Sign-off procedures
    └── Return to service
```

## 8.2 Line Maintenance Workflow

1. **Transit Check (A-Check Elements)**
   - Walk-around inspection
   - Fluid levels check
   - Tire condition
   - Brake inspection
   - Basic systems verification

2. **Defect Rectification**
   - Document discrepancy
   - Reference maintenance manual
   - Perform repair/replacement
   - Functional test
   - Update logs

3. **Component Replacement**
   - Verify part number and serial
   - Check for life limits
   - Proper installation
   - Torque and safety
   - Documentation

4. **Return to Service**
   - Final inspection
   - Sign maintenance release
   - Brief flight crew

## 8.3 Heavy Maintenance Workflow

```
Heavy Maintenance Process
├── Phase 1: Input
│   ├── Aircraft arrival and reception
│   ├── Initial inspection
│   ├── Work scope confirmation
│   └── Discrepancy research
├── Phase 2: Heavy Work
│   ├── Scheduled inspections (B/C/D checks)
│   ├── Structural inspection
│   ├── Systems overhaul
│   └── Component repair/replacement
├── Phase 3: Testing
│   ├── Ground tests
│   ├── Engine runs
│   ├── Systems verification
│   └── Functional flight tests
└── Phase 4: Output
    ├── Final inspection
    ├── Documentation completion
    └── Customer delivery
```

## 8.4 Engine Change Procedure

1. **Pre-Removal**
   - Review engine logs
   - Perform run-up if required
   - Document condition
   - Remove cowling and accessories

2. **Removal**
   - Support engine
   - Disconnect controls and lines
   - Remove pylon attachments
   - Transport to engine shop

3. **Installation**
   - Inspect mounting points
   - Position and align engine
   - Connect all systems
   - Safety wire critical fasteners

4. **Post-Installation**
   - Engine build-up complete
   - All lines connected
   - Control rigging verified
   - Leaks check complete

5. **Run-Up and Test**
   - Ground run per manual
   - Vibration analysis
   - All systems checked
   - Takeoff/climb assessment

## 8.5 Structural Repair Workflow

1. **Damage Assessment**
   - Visual inspection
   - NDT (Non-Destructive Testing)
   - Extent of damage determination
   - Engineering support

2. **Repair Design**
   - SRM (Structural Repair Manual) reference
   - Stress analysis if needed
   - Material selection
   - Acceptable repair limits

3. **Repair Execution**
   - Prepare damaged area
   - Cut out damaged material
   - Install repair section
   - Quality inspection

4. **Approval and Documentation**
   - Stress relief if required
   - Corrosion treatment
   - Finish and painting
   - Return to service

## 8.6 Quality Assurance Checklist

| Item | Requirement |
|------|-------------|
| Tools calibrated | Current calibration |
| Parts verified | Certified, current life |
| Procedures followed | MM, SRM, CM |
| Documentation | Complete, accurate |
| Sign-off | Authorized personnel |
| Aircraft release | Proper authorization |

## 8.7 Safety Critical Procedures

- **Tire changing** - Proper jacking, axle support
- **Pitot/static checks** - Precision instrument work
- **Oxygen system** - Fire/explosion risk
- **Hydraulic pressure** - High energy system
- **Fuel system** - Fire, toxicity hazards
- **Electrical power** - Arc flash, arc blast
