# CNC Operator — Glossary

## Programming Terms

**G-Code** — The universal language of CNC machines. Numerical control programming language for tool path and machine functions.

**M-Code** — Machine function codes controlling spindle, coolant, tool changes, and program flow.

**CAM (Computer-Aided Manufacturing)** — Software that generates toolpaths from CAD models. Examples: Mastercam, Fusion 360, SolidWorks CAM.

**Post-Processor** — Converts generic CAM toolpaths to machine-specific G-code.

**Toolpath** — The planned route the cutting tool will follow through the material.

**Cutter Compensation (G41/G42)** — Offsetting the tool path to account for actual tool diameter vs. programmed diameter.

** canned Cycle** — Pre-programmed drilling or milling sequence (G81-G89). Simplifies repetitive operations.

## Tool Geometry Terms

**Sweep** — The curvature radius of a gouge or endmill. Higher sweep number = deeper curve.

**Bevel Angle** — The angle of the cutting edge relative to the tool axis. carving chisels: 15-25°.

**Rake Angle** — The angle of the tool face relative to the workpiece surface.

**Relief Angle** — The angle behind the cutting edge that prevents rubbing.

**Helix Angle** — The spiral angle of flutes on an endmill. Higher helix = faster chip evacuation.

**Lead Angle** — The angle at which the cutting edge approaches the workpiece.

## Machining Parameters

**SFM (Surface Feet per Minute)** — The speed at which the tool circumference travels through the material.

**RPM (Revolutions Per Minute)** — The rotational speed of the spindle.

**IPR (Inches Per Revolution)** — Feed rate per spindle revolution. Also called "chip load per tooth."

**DOC (Depth of Cut)** — The axial depth of material removed per pass.

**WOC (Width of Cut)** — The radial engagement of the tool in the workpiece.

**MRR (Material Removal Rate)** — Cubic inches of material removed per minute.

**ADOC (Axial Depth of Cut)** — Depth along the axis of the tool.

**Stepover** — The lateral distance between adjacent toolpath passes.

**Chip Load** — The amount of material each cutting tooth removes per revolution.

## Machine Components

**Spindle** — The rotating assembly holding the tool. Types: belt-driven, gear-driven, direct-drive.

**Chuck** — Workholding device that grips the workpiece. Types: 3-jaw universal, 2-jaw, collet.

**Collet Chuck** — Tapered sleeve that grips round shank tools or workpieces precisely.

**Turret** — Rotating tool holder on lathes or mill-turn machines.

**ATC (Automatic Tool Changer)** — Mechanism that swaps tools automatically.

**Coolant System** — Fluid delivery for heat removal, chip evacuation, and surface finish.

**Way Covers** — Sealed covers protecting linear guides from chips and debris.

## Measurement & Quality

**CMM (Coordinate Measuring Machine)** — Precision 3D measurement device for verifying machined parts.

**DRO (Digital Readout)** — Digital display showing axis positions.

**Runout** — How much a rotating surface deviates from true center. Measured with dial indicator.

**TIR (Total Indicator Runout)** — Maximum deviation of a surface from a reference axis over one rotation.

**Tramming** — Adjusting a mill spindle to be perpendicular to the table surface.

**Datum** — The reference point or surface from which all measurements are made.

## Machining Operations

**Facing** — Cutting the top surface of a workpiece to create a flat face.

**Turning** — Rotating the workpiece while a stationary tool removes material.

**Milling** — Rotating tool removing material from stationary workpiece.

**Drilling** — Creating cylindrical holes with a drill point.

**Reaming** — Finishing holes to close tolerance with a reamer tool.

**Tapping** — Creating internal threads with a tap tool.

**Boring** — Enlarging or finishing existing holes to precise diameter.

**Profiling** — Cutting the outside or inside contour of a part.

**Pocketing** — Removing material from an enclosed pocket area.

## Tool Types

**Endmill** — Cylindrical cutting tool with flutes on the end and sides.

**Face Mill** — Large cutting head with multiple insert positions for facing operations.

**Drill** — Pointed tool for creating holes.

**Tap** — Threaded tool for creating internal threads.

**Reamer** — Multi-flute tool for finishing holes to close tolerance.

**Form Tool** — Tool with a specific profile for creating non-standard contours.

**Ball Endmill** — Endmill with hemispherical tip for 3D contouring.

## Workholding

**Fixture** — Custom-designed workholding device for a specific part.

**Vise** — Clamping device for holding workpieces on the mill table.

**Soft Jaws** — Aluminum or plastic jaws in a vise that are machined to match part contour.

**Mandrel** — Internal workholding device that expands to grip inside diameters.

**Step Jaws** — Vise jaws with steps for holding parts at different heights.

**Magnetic Chuck** — Electromagnet or permanent magnet for holding ferrous materials.
