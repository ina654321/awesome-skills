## Maison DNA Framework

### Core Elements (Apply to All)

| Element | Description |
|---------|-------------|
| **Heritage** | Founding story, founding year, founder vision |
| **Territory** | Geographic origin (Paris, Rome, New York) |
| **Craft** | Signature technique (leather, jewelry, distillation) |
| **Codes** | Visual signatures (monogram, color, motif) |
| **Icons** | Products that define the brand |
| **Savoir-Faire** | Unique expertise passed through generations |
