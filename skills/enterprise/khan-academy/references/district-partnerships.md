# Khan Academy Districts Program

## Program Overview

Khan Academy Districts provides comprehensive support for school systems implementing Khan Academy at scale. The program serves 500+ districts across the United States.

## Partnership Components

### Core Offerings

| Component | Description |
|-----------|-------------|
| **Implementation Support** | Dedicated success manager and rollout planning |
| **Professional Learning** | Teacher training, administrator workshops |
| **Data & Analytics** | District-wide progress dashboards |
| **Rostering** | Automated student/teacher account creation |
| **Curriculum Alignment** | Standards mapping (CCSS, state, district) |
| **Technical Support** | Priority help desk and troubleshooting |

### Program Tiers

**Tier 1: Essential (Free)**
- Basic Khan Academy access
- Self-service implementation resources
- Community support

**Tier 2: Supported (District Partnership)**
- All Essential features
- Dedicated implementation specialist
- Professional learning workshops
- Data dashboards
- Priority technical support

**Tier 3: Comprehensive**
- All Supported features
- Khanmigo AI integration
- MAP Accelerator
- Ongoing coaching
- Custom reporting

## Key Tools for Districts

### MAP Accelerator

**Partnership:** NWEA (Northwest Evaluation Association)

**Function:** Automatically places students at appropriate skill level based on MAP Growth assessment scores

**Features:**
- Personalized learning pathways
- Targeted skills recommendations
- Progress monitoring aligned to MAP goals
- Teacher reports on skill mastery

**Research Outcomes:**
- Students using MAP Accelerator show 1.5x expected growth
- Reduces teacher planning time for differentiation
- Improves at-risk student outcomes

### Khan Academy Kids Districts

**Early Learning Program:**
- Ages 2-8 (preschool through 2nd grade)
- District dashboard for progress monitoring
- Professional learning for early educators
- Family engagement resources

**Content Areas:**
- Early math and literacy
- Social-emotional learning
- Executive function skills
- Creative expression

### Khanmigo for Districts

**AI Implementation:**
- District-wide Khanmigo access
- Teacher training on AI tools
- Student safety monitoring
- Usage analytics dashboard

**Pricing Models:**
- Per-student licensing
- Site-based licensing
- Multi-year agreements

## Implementation Roadmap

### Phase 1: Planning (Months 1-2)

**Leadership Alignment**
- [ ] Superintendent and cabinet buy-in
- [ ] Board of education presentation
- [ ] Budget approval
- [ ] Success metrics definition

**Stakeholder Identification**
- [ ] District implementation lead
- [ ] School site champions (1 per school)
- [ ] IT/technical coordinator
- [ ] Professional learning coordinator

### Phase 2: Setup (Months 2-3)

**Technical Configuration**
- [ ] Clever or CSV rostering setup
- [ ] SSO integration (if applicable)
- [ ] District dashboard configuration
- [ ] Teacher account verification

**Curriculum Mapping**
- [ ] Identify priority courses/subjects
- [ ] Align Khan Academy content to scope/sequence
- [ ] Determine assignment expectations
- [ ] Create pacing guides

### Phase 3: Launch (Months 3-5)

**Professional Learning**
- [ ] Administrator orientation (2 hours)
- [ ] Teacher launch workshops (3 hours)
- [ ] Khan Academy 101 for all educators
- [ ] Khanmigo training (if applicable)

**Pilot Implementation**
- [ ] Select pilot schools/teachers
- [ ] Begin student onboarding
- [ ] Weekly check-ins with pilot teachers
- [ ] Gather feedback and adjust

### Phase 4: Scale (Months 6-12)

**District-Wide Rollout**
- [ ] Expand to all schools
- [ ] Regular data review meetings
- [ ] Peer learning communities
- [ ] Student and family communication

**Sustainability Planning**
- [ ] Integrate into instructional expectations
- [ ] Include in teacher evaluation
- [ ] Annual review and goal-setting
- [ ] Ongoing professional development

## Success Metrics

### Quantitative Indicators

| Metric | Target | Measurement |
|--------|--------|-------------|
| Weekly Active Users | 70%+ of enrolled students | Khan Academy dashboard |
| Learning Minutes/Week | 30-45 minutes per student | Platform analytics |
| Skills Leveled Up | 5+ per month per student | Mastery tracking |
| Mastery Completion | 60%+ of assigned skills | Course progress |
| Teacher Active Users | 80%+ of enrolled teachers | Login analytics |

### Qualitative Indicators

- Teacher confidence with platform
- Student engagement and motivation
- Parent satisfaction
- Integration into instructional practice
- Perception of learning impact

## Equity Focus

### Priority Populations

Khan Academy Districts prioritizes partnerships serving:
- Title I schools (high poverty)
- Rural communities
- English Language Learners
- Students with learning differences
- Historically underperforming schools

### Equity Strategies

**Device Access**
- 1:1 device programs
- After-school computer lab access
- Mobile-friendly implementation
- Offline capability promotion

**Language Access**
- Multilingual family communications
- Native language support where available
- ESL/ELL-specific implementation guidance

**Support Differentiation**
- Additional professional learning for high-need schools
- Intensive implementation support
- Khanmigo scholarships for economically disadvantaged students
