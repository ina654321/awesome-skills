# Mastery Learning Framework

## Definition

Mastery learning is an educational approach where students must achieve a high level of understanding (typically 80-90%) in prerequisite knowledge before moving on to advanced concepts.

## Khan Academy Implementation

### Skill Progression Levels

```
Not Started → Attempted → Familiar → Proficient → Mastered
     ↓            ↓           ↓            ↓            ↓
   0%        Some work    Some correct  Mostly correct  Consistently
                                        (70%+)       correct (100%)
```

### The Five Levels Explained

| Level | Description | Criteria |
|-------|-------------|----------|
| **Not Started** | No interaction with skill | — |
| **Attempted** | Started but not achieved Familiar | <70% correct |
| **Familiar** | Basic understanding demonstrated | 70% correct on practice |
| **Proficient** | Strong understanding, some errors | Correct streak in varied contexts |
| **Mastered** | Complete mastery demonstrated | 100% on mastery challenge |

### Core Principles

#### 1. Prerequisite Structure
- Skills organized in dependency trees
- Cannot access advanced skills until prerequisites mastered
- Prevents gaps in foundational knowledge

#### 2. Immediate Feedback
- Real-time correctness indication
- Step-by-step explanations for errors
- Hints that gradually reveal solution path

#### 3. Unlimited Attempts
- No penalty for incorrect answers
- Encourages experimentation
- Focus on learning, not testing

#### 4. Multiple Representations
- Visual models
- Symbolic notation
- Verbal explanations
- Real-world contexts

## Learning Science Foundation

### Benjamin Bloom's Research (1968)

Bloom's original mastery learning experiments showed:
- **1 standard deviation improvement** over conventional instruction
- Effect size equivalent to moving from 50th to 84th percentile
- Requires: clear objectives, formative assessment, corrective procedures

### Khan Academy Adaptations

| Bloom's Component | Khan Academy Implementation |
|-------------------|----------------------------|
| Clear objectives | Skill-specific learning goals |
| Formative assessment | Embedded practice with instant feedback |
| Corrective procedures | Hints, videos, related exercises |
| Parallel assessment | Mastery challenges, unit tests |

### Spacing and Interleaving

**Mastery Challenges:**
- Appear 16 hours after skill practice
- Mix recently practiced skills
- Promote long-term retention
- Prevent cramming

## Implementation Best Practices

### For Students

1. **Diagnostic First**
   - Take Course Challenge to identify gaps
   - Focus on lowest unmastered skills
   - Use mastery map for visual progress

2. **Practice Protocol**
   - Watch video if stuck
   - Use hints strategically
   - Aim for 3-5 correct in a row
   - Retry after incorrect answers

3. **Spaced Review**
   - Complete mastery challenges when available
   - Review mastered skills periodically
   - Connect new learning to mastered concepts

### For Teachers

1. **Assignment Strategy**
   - Assign skills at appropriate level
   - Allow time for mastery (not just completion)
   - Celebrate mastery achievements

2. **Intervention Points**
   - Identify students stuck at "Attempted"
   - Provide small-group support
   - Use Khanmigo for personalized help

3. **Data-Driven Decisions**
   - Monitor class mastery percentages
   - Identify common misconception patterns
   - Adjust pacing based on mastery rates

## Comparison to Traditional Models

| Aspect | Traditional | Mastery Learning |
|--------|-------------|------------------|
| Pace | Fixed (time-based) | Variable (mastery-based) |
| Assessment | Summative (end-of-unit) | Formative (embedded) |
| Advancement | Time-based promotion | Competency-based |
| Feedback | Delayed (days/weeks) | Immediate |
| Remediation | After failure | Continuous support |

## Common Misconceptions

**Myth:** Mastery learning means perfect scores
**Reality:** Mastery means consistent demonstration of understanding, not perfection

**Myth:** It slows down fast learners
**Reality:** Fast learners accelerate through skills they already know; enrichment available

**Myth:** Only works for math
**Reality:** Successfully applied to science, humanities, languages, and skills
