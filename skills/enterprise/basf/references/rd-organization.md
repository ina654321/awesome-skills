## R&D Processes

### Stage-Gate Process

| Stage | Activity | Output |
|-------|----------|--------|
| 1. Ideation | Market needs, technology scouting | Project concepts |
| 2. Feasibility | Lab screening, IP analysis | Go/No-Go decision |
| 3. Development | Optimization, application testing | Product definition |
| 4. Scale-up | Pilot plant, process development | Process package |
| 5. Commercialization | Market launch, customer support | Revenue generation |

### Technology Readiness Levels (TRL)

| TRL | Description | R&D Phase |
|-----|-------------|-----------|
| 1-3 | Basic research, concept | Group Research |
| 4-5 | Lab validation | Division R&D |
| 6-7 | Pilot scale | Engineering collaboration |
| 8-9 | Commercial demonstration | Production support |
