## 1. Upstream Operations

### 1.1 Permian Basin Deep Dive

**Geology:**

The Permian Basin is a sedimentary basin in West Texas and southeastern New Mexico, 
covering approximately 75,000 square miles. It's actually three basins:
- **Delaware Basin**: Western portion, deeper, multiple stacked pay zones
- **Midland Basin**: Eastern portion, most active drilling
- **Central Basin Platform**: Separates the two, conventional production

**Key Formations:**

| Formation | Depth (ft) | Lithology | Oil Quality | Activity Level |
|-----------|------------|-----------|-------------|----------------|
| **Spraberry** | 5,000-8,000 | Siltstone, sandstone | 35-42° API | High |
| **Wolfcamp A** | 6,000-10,000 | Shale, limestone | 38-45° API | Very High |
| **Wolfcamp B** | 8,000-12,000 | Shale | 35-42° API | Very High |
| **Wolfcamp C** | 10,000-14,000 | Shale | 35-40° API | Moderate |
| **Wolfcamp D** | 12,000-16,000 | Shale | 30-38° API | Emerging |
| **Bone Spring** | 7,000-12,000 | Sandstone, shale | 40-50° API | High |
| **Delaware Mountain** | 5,000-9,000 | Sandstone | 40-48° API | Moderate |

## 5. Financial & Economic Frameworks

### 5.1 Upstream Economics

**Key Metrics:**

| Metric | Formula | Target |
|--------|---------|--------|
| **Finding Cost** | Exploration & reserve adds $ / BOE added | <$10/BOE |
| **Lifting Cost** | Operating expense / BOE produced | <$8/BOE (Permian) |
| **DD&A** | Depreciation, depletion, amortization / BOE | $12-15/BOE |
| **F&D Cost** | (Exploration + Development) / Reserve adds | <$15/BOE |
| **Recycle Ratio** | Operating cash flow / F&D cost | >2.0x |

**Break-even Analysis:**

```
Permian Shale Break-even (2025):
├── Well Cost: $8-10 million
├── EUR: 1.2-1.8 MMboe
├── Operating Cost: $6-8/BOE
├── Royalties & Taxes: 25-30%
└── Break-even Price: $35-45/bbl WTI

Deepwater Break-even (Guyana):
├── Development Cost: $3-5 billion
├── Peak Production: 200,000+ bpd
├── Operating Cost: $8-12/BOE
├── Break-even Price: $25-35/bbl Brent
```
