# § 8 · Workflow Phases - Full Details

## Phase 1: Concept Development

### Objective

Develop a compelling story idea with clear emotional stakes.

### Key Activities

1. **Idea Generation**
   - What if? questions
   - Personal experiences
   - Universal themes
   - Character concepts

2. **Logline Creation**
   - One sentence summary
   - Character + Goal + Obstacle
   - Emotional hook

3. **Theme Exploration**
   - What is this story about?
   - Universal human experience
   - Personal connection

4. **Character Basics**
   - Protagonist want and need
   - Antagonist force
   - Supporting characters
   - Character relationships

### Done Criteria

- [✓] Clear logline (1 sentence)
- [✓] Identified theme
- [✓] Protagonist defined
- [✓] Emotional stakes established
- [✓] Unique angle identified

### Fail Criteria

- [✗] Vague premise
- [✗] No emotional core
- [✗] Passive protagonist
- [✗] Unclear stakes
- [✗] Generic concept

---

## Phase 2: Treatment Writing

### Objective

Create a 1-2 page outline of the complete story.

### Key Activities

1. **Three-Act Structure**
   - Act 1: Setup (25%)
   - Act 2: Confrontation (50%)
   - Act 3: Resolution (25%)

2. **Story Spine Application**
   - Once upon a time...
   - And every day...
   - Until one day...
   - And because of that...
   - Until finally...

3. **Key Scenes**
   - Inciting incident
   - Plot points (1, 2, midpoint)
   - Climax
   - Resolution

4. **Character Arc**
   - Beginning state
   - Transformation journey
   - Ending state

### Done Criteria

- [✓] 1-2 page document
- [✓] Complete story arc
- [✓] Clear three-act structure
- [✓] Character growth evident
- [✓] Cause-and-effect logic

### Fail Criteria

- [✗] Unfocused narrative
- [✗] Missing act breaks
- [✗] No character growth
- [✗] Coincidence-driven plot
- [✗] Theme preached, not shown

---

## Phase 3: Storyboarding

### Objective

Visualize the story through sequential art.

### Key Activities

1. **Thumbnail Sketches**
   - Rough composition
   - Pacing exploration
   - Shot variety

2. **Rough Boards**
   - Key scenes visualized
   - Character acting
   - Camera movement

3. **Pitching**
   - Present to team
   - Get feedback
   - Iterate

4. **Revision**
   - Cut what doesn't work
   - Strengthen weak sequences
   - Clarify storytelling

### Done Criteria

- [✓] All key sequences boarded
- [✓] Visual storytelling clear
- [✓] Pacing established
- [✓] Character emotions visible
- [✓] Ready for reels

### Fail Criteria

- [✗] All talking heads
- [✗] Static compositions
- [✗] Unclear action
- [✗] Missing visual variety
- [✗] Rushed execution

---

## Phase 4: Braintrust Review

### Objective

Get honest peer feedback to strengthen the story.

### Key Activities

1. **Preparation**
   - Compile reels
   - Set screening
   - Invite Braintrust

2. **Screening**
   - Watch in full
   - No interruptions
   - Take notes

3. **Feedback Session**
   - Start with positives
   - Specific problem identification
   - Suggestion offering
   - Director listens

4. **Processing**
   - Sleep on notes
   - Identify patterns
   - Decide what to implement

### Done Criteria

- [✓] Complete screening
- [✓] Specific feedback received
- [✓] Director notes documented
- [✓] Implementation plan created

### Fail Criteria

- [✗] Defensive responses
- [✗] Vague feedback
- [✗] Ignoring major issues
- [✗] Implementing everything blindly

---

## Phase 5: Iteration and Polish

### Objective

Refine the story based on feedback until emotional truth shines through.

### Key Activities

1. **Revision Planning**
   - Prioritize changes
   - Test solutions
   - Keep what works

2. **Implementation**
   - Rewrite sequences
   - Reboard scenes
   - Update reels

3. **Testing**
   - Internal screenings
   - Audience testing
   - Emotional response measurement

4. **Final Polish**
   - Fine-tune pacing
   - Clarify moments
   - Lock the story

### Done Criteria

- [✓] Emotional truth clear
- [✓] Story flows smoothly
- [✓] Character arcs complete
- [✓] Theme emerges naturally
- [✓] Ready for production

### Fail Criteria

- [✗] Superficial changes
- [✗] Avoiding hard problems
- [✗] Theme still preached
- [✗] Characters unchanged
- [✗] Rushing to lock
