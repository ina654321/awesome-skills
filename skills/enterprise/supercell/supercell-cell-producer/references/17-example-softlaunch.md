# Example 2: Soft Launch Analysis

## Context

Strategy RPG has been in soft launch for 3 months in Canada and Philippines.

## User Request

"Our soft launch metrics: D1 52%, D7 22%, D30 8%. ARPDAU $0.08. Team loves the game. Players say it's fun but 'gets grindy after a week.' What should we do?"

## Metric Analysis

| Metric | Value | Assessment | Threshold |
|--------|-------|------------|-----------|
| D1 Retention | 52% | ✅ Good | > 50% |
| D7 Retention | 22% | ✅ Good | > 20% |
| D30 Retention | 8% | ❌ Below | > 10% |
| ARPDAU | $0.08 | ✅ Good | ~$0.10 |

### Interpretation

The D7 to D30 drop (22% → 8%) is steep. Player feedback about "getting grindy after a week" explains this perfectly—you have a **progression wall problem**, not a fun problem.

## Recommendation: CONTINUE with Focused Iteration

### Action Plan

#### Phase 1: Fix Progression (2-4 weeks)

- Analyze where players drop off (likely days 8-14)
- Smooth the difficulty curve
- Add mid-game goals and variety
- Reduce repetitive grinding

**Target**: Improve D30 to 10%+

#### Phase 2: Content Expansion (4-6 weeks)

- Add new game modes for variety
- Implement daily/weekly challenges
- Create collection systems (long-term goals)

**Target**: Improve D30 to 12%+

#### Phase 3: Re-test (4-6 weeks)

- Soft launch update to existing players
- Measure retention improvement
- Decision point: Kill, continue, or global launch

### Continue Criteria

- D30 must reach 10% within 2 months
- Player sentiment on "grindy" must improve
- Team must maintain enthusiasm

### Kill Criteria

If D30 doesn't improve after these changes, we kill. The monetization is solid, so if we can fix retention, we have a potential hit.

## Alternative Scenarios

**What if D30 was 12% already?**
→ Global Launch. Metrics are great, team loves it, monetization works.

**What if D1 was 40%?**
→ Kill. Core loop isn't working. D1 is hardest to fix.

**What if team didn't love it?**
→ Kill. Team passion is non-negotiable.
