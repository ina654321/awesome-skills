# Marriott Bonvoy Loyalty Program Guide

## Program Overview

Marriott Bonvoy is the world's largest hotel loyalty program with **271 million+ members** globally. Launched in February 2019, it unified three legacy programs: Marriott Rewards, Starwood Preferred Guest (SPG), and The Ritz-Carlton Rewards.

### Key Statistics (2025)

| Metric | Value |
|--------|-------|
| **Total Members** | 271 million+ |
| **Active Members** | ~75% (engaged in last 24 months) |
| **Elite Members** | ~25 million (Platinum+) |
| **Credit Card Revenue** | $716 million annually |
| **Point Liability** | ~$2.1 billion |
| **Properties Participating** | 9,800+ |
| **Countries** | 145+ |

### Elite Bonus Multipliers

| Tier | Bonus | Total Points per $1 |
|------|-------|---------------------|
| Member | 0% | 10x |
| Silver | 10% | 11x |
| Gold | 25% | 12.5x |
| Platinum | 50% | 15x |
| Titanium | 75% | 17.5x |
| Ambassador | 75% | 17.5x |

### Points Expiration

- Points expire after **24 months of inactivity**
- Activity includes: earning, redeeming, or transferring points
- Paid stays count as earning activity
- Credit card spending counts as earning activity

## Program Economics

### Point Valuation

| Redemption Type | Value per Point |
|-----------------|-----------------|
| **Hotel stays (optimal)** | 0.7-1.0 cents |
| **Hotel stays (average)** | 0.5-0.7 cents |
| **Airline transfers (premium cabins)** | 0.8-1.2 cents |
| **Airline transfers (economy)** | 0.3-0.5 cents |
| **Cash + Points** | 0.6-0.8 cents |
| **Gift cards** | 0.25-0.4 cents |
| **Merchandise** | 0.2-0.3 cents |

### Breakage

- **Current breakage rate**: ~22%
- Breakage = points that expire or go unredeemed
- Marriott recognizes breakage as revenue
- Program optimized to encourage redemption while maintaining healthy breakage

## Recent Program Changes

### 2025 Updates

1. **Nightly Upgrade Awards**: Formerly "Suite Night Awards," now apply to any premium room upgrade
2. **Lufthansa Partnership**: New status point earning for Miles & More members
3. **Nectar Partnership**: UK members earn Nectar points on stays
4. **Ambassador Plus One**: Gift Ambassador status to household member at $40K spend

### 2024 Updates

1. **Soft Landing Policy**: Elite members retain status one tier lower if they don't requalify
2. **Automated Complimentary Upgrades (ACU)**: AI-powered room upgrades
3. **Choice Benefits Deadline**: February 1 deadline for annual selection

## Program Strategy Insights

### Why Bonvoy Succeeds

1. **Scale**: 9,800+ properties create ubiquitous earning opportunities
2. **Brand Portfolio**: From Fairfield to Ritz-Carlton, covers all segments
3. **Credit Card Economics**: $716M annual revenue with high margins
4. **Breakage Optimization**: ~22% unredeemed points improve profitability
5. **Partnership Ecosystem**: Airline, car rental, retail partners extend value

### Comparison to Competitors

| Program | Members | Key Advantage | Key Weakness |
|---------|---------|---------------|--------------|
| **Marriott Bonvoy** | 271M | Largest network, 30+ brands | Dynamic pricing reduces value |
| **Hilton Honors** | 180M | Easy elite status, consistent value | Smaller global footprint |
| **World of Hyatt** | 40M | Best elite benefits, luxury focus | Smallest network |
| **IHG One Rewards** | 100M | Good value redemptions | Inconsistent elite benefits |
