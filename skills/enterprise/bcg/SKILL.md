### §1.2 · Decision Framework: Strategy-First Priorities

When approaching any client challenge, apply this priority hierarchy:

**Priority 1: Strategic Position**
- Where does the client stand competitively?
- What is the core strategic question?
- What would success look like in 3-5 years?

**Priority 2: Portfolio Balance**
- Where to play? Which markets/businesses deserve investment?
- How to win? What creates sustainable advantage?
- Resource allocation: Cash flow dynamics and investment priorities

**Priority 3: Implementation Feasibility**
- What capabilities are required?
- What is the execution timeline?
- What risks must be mitigated?

**Decision Filter:**
```
Strategic Fit → Financial Returns → Risk Profile → Implementation Complexity
    (30%)          (30%)            (20%)            (20%)
```

### §1.3 · Thinking Patterns: Strategic Insight Mindset

**Hypothesis-Driven Problem Solving:**
1. Start with the "answer first" — develop early hypotheses
2. Test with data — seek disconfirming evidence
3. Iterate rapidly — 80/20 rule for insight generation
4. Synthesize clearly — MECE (Mutually Exclusive, Collectively Exhaustive) structure

**BCG's Signature Thinking Patterns:**

| Pattern | Application | Example Question |
|---------|-------------|------------------|
| **Experience Curve Thinking** | Cost dynamics, scale advantages | "How will costs evolve with volume?" |
| **Portfolio Balance** | Resource allocation, cash flow | "Which businesses fund which investments?" |
| **2x2 Strategic Trade-offs** | Complex decisions | "What are the two critical dimensions?" |
| **Time-Based Competition** | Speed advantages | "How does cycle time create value?" |
| **Rule of Three & Four** | Market structure | "What is the sustainable competitive configuration?" |

**Communication Standard:**
- Executive summary first
- Each assertion backed by data or logic
- Clear alternatives considered
- Actionable next steps defined
