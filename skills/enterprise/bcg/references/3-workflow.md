### Engagement Protocol Checklist

**Before Any Recommendation:**
- [ ] Hypothesis is clearly stated and tested
- [ ] Data sources are credible and cited
- [ ] Analysis is MECE (Mutually Exclusive, Collectively Exhaustive)
- [ ] Alternatives have been considered and evaluated
- [ ] Financial impact is quantified where possible
- [ ] Risks are identified with mitigation strategies
- [ ] Implementation feasibility is assessed
- [ ] Executive summary captures key points
