## §3 · Workflow: Pharma Portfolio Management

### §3.1 · Portfolio Strategy Process

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                BMS PORTFOLIO STRATEGY DEVELOPMENT FLOW                      │
└─────────────────────────────────────────────────────────────────────────────┘

PHASE 1: PORTFOLIO ASSESSMENT (Annual + Quarterly)
┌─────────────────────────────────────────────────────────────────────────────┐
│ • Revenue trajectory analysis (Growth vs. Legacy mix)                       │
│ • LOE timeline mapping (Revlimid, Opdivo, Eliquis cliffs)                   │
│ • Competitive dynamics assessment                                           │
│ • Pipeline value and probability-adjusted NPV                               │
│ • Gap analysis: therapeutic areas, modalities, geographies                  │
│                                                                              │
│ OUTPUT: Portfolio Dashboard with risk-adjusted revenue projections          │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
PHASE 2: STRATEGIC OPTIONS GENERATION
┌─────────────────────────────────────────────────────────────────────────────┐
│ • Organic growth initiatives (label expansion, new indications)             │
│ • Business development targets (M&A, partnerships, licensing)               │
│ • Lifecycle management strategies (formulations, combinations)              │
│ • Productivity and cost optimization                                        │
│ • Geographic expansion priorities                                           │
│                                                                              │
│ OUTPUT: Strategic options with financial impact and risk assessment         │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
PHASE 3: DECISION & RESOURCE ALLOCATION
┌─────────────────────────────────────────────────────────────────────────────┐
│ • R&D budget allocation across therapeutic areas                            │
│ • Commercial investment prioritization                                      │
│ • M&A capital deployment decisions                                          │
│ • Launch resource planning                                                  │
│ • Productivity initiative implementation                                    │
│                                                                              │
│ OUTPUT: Annual Operating Plan + Strategic Priorities                        │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
PHASE 4: EXECUTION & MONITORING
┌─────────────────────────────────────────────────────────────────────────────┐
│ • Quarterly business reviews                                                │
│ • Pipeline milestone tracking                                               │
│ • Launch performance monitoring                                             │
│ • Competitive response planning                                             │
│ • Investor communication (guidance, updates)                                │
│                                                                              │
│ OUTPUT: Quarterly earnings, investor updates, course corrections            │
└─────────────────────────────────────────────────────────────────────────────┘
```


### §3.3 · Launch Excellence Framework

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ BMS LAUNCH EXCELLENCE MODEL                                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│ PRE-LAUNCH (12-18 months prior)                                              │
│ ┌─────────────────────────────────────────────────────────────────────────┐ │
│ │ • Market research and patient journey mapping                            │ │
│ │ • KOL engagement and medical education                                   │ │
│ │ • Payer value proposition development                                    │ │
│ │ • Supply chain and manufacturing readiness                               │ │
│ │ • Field force training and deployment                                    │ │
│ └─────────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│ LAUNCH PHASE (Months 1-12)                                                   │
│ ┌─────────────────────────────────────────────────────────────────────────┐ │
│ │ • Rapid formulary access (target: 80%+ commercial coverage)              │ │
│ │ • HCP awareness and education campaigns                                  │ │
│ │ • Patient support program activation                                     │ │
│ │ • Real-world evidence generation initiation                              │ │
│ │ • Competitive monitoring and response                                    │ │
│ └─────────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│ POST-LAUCH OPTIMIZATION (Year 2+)                                            │
│ ┌─────────────────────────────────────────────────────────────────────────┐ │
│ │ • Label expansion opportunities                                          │ │
│ │ • Geographic expansion                                                   │ │
│ │ • Combination development                                                │ │
│ │ • Lifecycle management (formulations, dosing)                            │ │
│ │ • Real-world evidence publications                                       │ │
│ └─────────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│ KEY METRICS:                                                                 │
│ • Time to formulary access                                                 │
│ • NBRx (New to Brand Prescriptions) trajectory                             │
│ • Market share vs. competitors                                             │
│ • Patient persistency rates                                                │
│ • Revenue vs. forecast                                                     │
└─────────────────────────────────────────────────────────────────────────────┘
```
