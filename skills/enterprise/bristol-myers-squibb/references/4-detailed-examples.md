### Example 1: Growth Portfolio vs. Legacy Portfolio Strategy

```

Pomalyst, Sprycel) facing generic erosion to Growth Portfolio (Opdivo, Eliquis, 
Reblozyl, Camzyos, Sotyktu, Breyanzi). 2025 results show Growth Portfolio at 
$26.4B (+17% YoY) and Legacy at $21.8B (-15% YoY).

USER REQUEST: "How should we allocate resources between defending Legacy cash 
flows and investing in Growth Portfolio acceleration?"

├─────────────────────────────────────────────────────────────────────────────┤

│ Portfolio                │ Revenue     │ Growth      │ Resource Strategy   │
├──────────────────────────┼─────────────┼─────────────┼─────────────────────┤
│ Growth Portfolio         │ $26.4B      │ +17% YoY    │ INVEST & EXPAND     │
│ • Opdivo franchise       │ ~$9.0B      │ Stable+     │ Defend & innovate   │
│ • Eliquis                │ ~$12.5B     │ +10-15%     │ Maximize to LOE     │
│ • New products           │ ~$4.9B      │ +50%+       │ Accelerate launches │
├──────────────────────────┼─────────────┼─────────────┼─────────────────────┤
│ Legacy Portfolio         │ $21.8B      │ -15% YoY    │ HARVEST & OPTIMIZE  │
│ • Revlimid               │ Declining   │ -44%        │ Minimize SG&A       │
│ • Pomalyst/Sprycel       │ Declining   │ -24%/-53%   │ Managed decline     │
└──────────────────────────┴─────────────┴─────────────┴─────────────────────┘

GROWTH PORTFOLIO (70% of commercial investment):
┌─────────────────────────────────────────────────────────────────────────────┐
│ 1. Opdivo Franchise Defense ($X Million)                                    │

│    • Yervoy combination promotion                                           │

│    • Camzyos: nHCM indication launch                                        │

│    • Cobenfy: Schizophrenia launch build-out                                │

┌─────────────────────────────────────────────────────────────────────────────┐

│    • Focus on markets with limited generic competition                      │

├─────────────────────────────────────────────────────────────────────────────┤

```

│ ELIQUIS LOE NAVIGATION STRATEGY                                               │
├─────────────────────────────────────────────────────────────────────────────┤

│ 2027-2028: Generic entry expected | Post-2028: Multi-generic competition   │

STRATEGIC PILLARS:

PILLAR 1: VOLUME GROWTH TO OFFSET PRICE PRESSURE
┌─────────────────────────────────────────────────────────────────────────────┐
│ • Target: Maintain +10-15% WW revenue growth through 2026 (guidance)        │

│                           │ Delayed generic entry                           │
│                           │ Strong international performance                │
