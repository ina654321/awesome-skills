# Meta AI & Llama Models

## Overview

Meta is positioning itself as the leader in open AI, using Llama models to power its products and the broader developer ecosystem.

## Investment

- **2025 Capital Expenditure**: $60-65 billion (primarily AI infrastructure)
- **Data Center**: Building 2 gigawatt facility (size of significant Manhattan area)
- **GPU Clusters**: 100K+ H100s planned for training

## Llama Model Family

### Llama 3.1 (Released July 2024)
| Model | Parameters | Context | Use Case |
|-------|------------|---------|----------|
| 8B | 8 billion | 128K | Edge devices, fast inference |
| 70B | 70 billion | 128K | Balanced performance/cost |
| 405B | 405 billion | 128K | Maximum capability |

### Llama 3.2 (Released September 2024)
- **Vision Models**: 11B and 90B with image understanding
- **Edge Models**: 1B and 3B for on-device use
- **Multilingual**: Support for 8 languages

### Llama 3.3 (Released December 2024)
- 70B parameters
- Performance of 405B model with smaller size
- Improved reasoning and instruction following

### Llama 4 (Expected 2025)
- Native multimodal capabilities
- Stronger reasoning abilities
- Faster inference
- New modalities beyond text/image

## Technical Architecture

### Training
- **Data**: Trillions of tokens from public sources
- **Compute**: Custom GPU clusters
- **Alignment**: RLHF (Reinforcement Learning from Human Feedback)
- **Safety**: Red teaming, safety classifiers

### Inference
- **On-Device**: Llama 3.2 1B/3B for mobile
- **Cloud**: Full model for complex queries
- **Hybrid**: Routing based on query complexity

### Open Source Approach

#### Benefits
- Community innovation
- Wider adoption
- Standardization
- Talent attraction

#### License Terms
- Free for commercial use (with limits)
- Must comply with acceptable use policy
- Attribution required for derivatives
- Some cloud restrictions (addressed in 3.1+)

## Competitive Position

### vs. OpenAI
| Aspect | Meta Llama | OpenAI GPT |
|--------|------------|------------|
| Model Access | Open weights | API only |
| Cost | Free (self-host) | Per-token pricing |
| Customization | Full fine-tuning | Limited via API |
| Performance | Competitive | Leading (GPT-4) |
| Safety Control | User responsibility | Managed by OpenAI |

### vs. Google Gemini
- Llama more widely adopted in open source community
- Gemini integrated with Google products
- Both offer on-device options

### vs. DeepSeek
- DeepSeek R1 challenged Llama on reasoning
- Meta responded with "war rooms" to incorporate learnings
- Competition driving rapid innovation
