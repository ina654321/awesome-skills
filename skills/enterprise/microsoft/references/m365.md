## Microsoft Graph API

### Key Endpoints
| Endpoint | Purpose |
|----------|---------|
| `/me` | Current user profile |
| `/users` | User directory |
| `/groups` | Microsoft 365 Groups |
| `/teams` | Teams and channels |
| `/sites` | SharePoint sites |
| `/drives` | OneDrive/SharePoint files |
| `/calendar` | Events and meetings |
| `/mail` | Email messages |

### Permissions
| Scope | Access |
|-------|--------|
| `User.Read` | Read user profile |
| `Mail.Read` | Read email |
| `Calendars.ReadWrite` | Full calendar access |
| `Files.Read.All` | Read all files |
| `Group.Read.All` | Read all groups |

### Sample Query
```http
GET https://graph.microsoft.com/v1.0/me/messages?$filter=importance eq 'high'
Authorization: Bearer {access_token}
```

### Teams JS SDK
```javascript
// Initialize Teams SDK
microsoftTeams.initialize();

// Get context
microsoftTeams.getContext((context) => {
  console.log(context.userPrincipalName);
});

// Authenticate
microsoftTeams.authentication.authenticate({
  url: window.location.origin + "/auth",
  width: 600,
  height: 535
});
```
