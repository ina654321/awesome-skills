# Microsoft History Reference

## Company Timeline

### Founding Era (1975-1985)
| Year | Event |
|------|-------|
| 1975 | Founded by Bill Gates and Paul Allen in Albuquerque, NM |
| 1976 | Microsoft trademark registered |
| 1979 | Moved to Bellevue, Washington |
| 1981 | MS-DOS released for IBM PC |
| 1983 | Paul Allen leaves Microsoft |
| 1985 | Windows 1.0 released |

### PC Dominance (1986-2000)
| Year | Event |
|------|-------|
| 1986 | IPO at $21/share |
| 1989 | Office suite launched |
| 1990 | Windows 3.0 - major commercial success |
| 1995 | Windows 95 - Start menu debut |
| 1998 | Windows 98, antitrust case begins |
| 2000 | Windows 2000, Steve Ballmer becomes CEO |

### Internet & Enterprise (2001-2013)
| Year | Event |
|------|-------|
| 2001 | Windows XP, Xbox launched |
| 2007 | Windows Vista, Office 2007 |
| 2008 | Azure announced (launched 2010) |
| 2009 | Windows 7 |
| 2010 | Windows Phone 7 |
| 2011 | Skype acquisition ($8.5B) |
| 2012 | Windows 8, Surface launch |
| 2013 | Xbox One, Steve Ballmer announces retirement |

### Nadella Transformation (2014-Present)
| Year | Event |
|------|-------|
| 2014 | Satya Nadella becomes CEO |
| 2015 | Windows 10, "Cloud-first, mobile-first" strategy |
| 2016 | LinkedIn acquisition ($26.2B) |
| 2018 | GitHub acquisition ($7.5B) |
| 2019 | OpenAI partnership begins ($1B) |
| 2020 | Azure Arc, Microsoft 365 rebrand |
| 2021 | Windows 11, ZeniMax acquisition ($7.5B) |
| 2022 | Activision Blizzard acquisition announced ($68.7B) |
| 2023 | GPT-4 integration, Copilot launch |
| 2024 | Copilot proliferation across all products |
| 2025 | Microsoft 50th anniversary, AI business >$13B run rate |

## Product Evolution

|---------|------|------------------|
| Windows 1.0 | 1985 | GUI, tiled windows |
| Windows 3.1 | 1992 | Program Manager |
| Windows 95 | 1995 | Start menu, plug and play |
