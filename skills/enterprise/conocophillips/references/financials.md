#### Tier 3: Variable Return of Cash (VROC)

**Philosophy:** Return excess cash when commodity prices exceed planning range

| Attribute | Detail |
|-----------|--------|
| Trigger | WTI meaningfully above $60/bbl planning price |
| Frequency | Quarterly board determination |
| Historical Rate | $0.20/share when activated |
| 2024 Change | Incorporated into base dividend |
| Purpose | Share commodity upside with shareholders |

**VROC Decision Factors:**
- Anticipated distributions required for >30% CFO commitment
- Forward commodity prices
- Balance sheet cash position
- Total shareholder yield considerations

## Cost of Supply Analysis

### Framework Definition

```
Fully-Burdened Cost of Supply =
  (Capital Investment
   + Operating Costs
   + Carbon Cost (proxy)
   + Abandonment Liability)
  ÷ Total Expected Production
```

### Portfolio Metrics (2024)

| Metric | Value |
|--------|-------|
| Resource Base | ~20 billion BOE |
| <$40/bbl inventory | Deep inventory across all basins |
| Average Cost of Supply | ~$32/bbl |
| Target Return | >10% after-tax at cost of supply + $10/bbl |

### Basin Economics

| Basin | Cost of Supply | Economics |
|-------|---------------|-----------|
| Eagle Ford | <$25/bbl | Best-in-class |
| Permian | $25-30/bbl | Tier 1 returns |
| Bakken | $28-32/bbl | Good via Marathon |
| Alaska | $35-40/bbl | Strategic value |
| Surmont | $35-40/bbl | Low decline, long life |

### Investment Criteria

**Project Sanction Requirements:**
1. Cost of supply < $35/bbl preferred
2. After-tax returns > 10% fully burdened
3. Strategic fit within portfolio
4. ESG compliance (emissions profile)

**Exception Projects (Strategic):**
- Willow: ~$45-50/bbl breakeven (justified by diversification)
- LNG: Oil-linked pricing, demand diversification
- International: Market access, reserve life
