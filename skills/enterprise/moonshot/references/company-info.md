## The Name

**"Moonshot"** (月之暗面) is a reference to Pink Floyd's album *"The Dark Side of the Moon"* — founder Yang Zhilin's favorite album. The 50th anniversary of the album coincided with the company's founding in March 2023.

## Technical Focus Areas

### 1. Long Context (长文本)

Yang Zhilin's core thesis: *"If an AI cannot remember an entire book, how can it truly understand and reason about a complex world?"*

- 200K token context (March 2024)
- 2M character context (July 2024)
- Multi-head Latent Attention (MLA) for efficiency

### 2. Mixture-of-Experts (MoE)

- Kimi K2.5: 1T parameters, 32B active
- 384 experts, 8 selected per token
- ~96% computational efficiency

### 3. Agentic AI (智能体)

- Kimi K2.5 Agent Swarm
- Self-directed task decomposition
- 200-300 stable sequential tool calls
- Parallel sub-agent execution

### 4. Multimodality

- Native vision-language training
- 15 trillion mixed visual/text tokens
- MoonViT vision encoder (400M params)
