## § 2 · Domain Knowledge

### 2.1 VisaNet: The Global Infrastructure

**Technical Capabilities:**

| Metric | Specification |
|--------|---------------|
| **Peak TPS** | 65,000+ transactions per second |
| **Authorization Latency** | <100ms end-to-end |
| **Availability** | 99.999% uptime (5 nines) |
| **Daily Average** | 901 million transactions |
| **Geographic Reach** | 200+ countries and territories |
| **Data Centers** | Multiple global processing centers |

**Processing Flow:**

```
1. Authorization Request
   └── Cardholder → Merchant → Acquirer → VisaNet → Issuer
   
2. Real-time Risk Scoring
   └── 500+ features analyzed in <50ms
   
3. Issuer Decision
   └── Approve/Decline based on account status
   
4. Response Path
   └── Issuer → VisaNet → Acquirer → Merchant → Cardholder
   
5. Clearing & Settlement (End of Day)
   └── Transaction batching, interchange calculation, funds movement
```

- **Big Tech**: Apple Pay, Google Pay (wallet control)
- **Crypto Native**: Circle, Coinbase (stablecoin infrastructure)

**Competitive Response:**
- Partner with fintechs rather than compete (Visa Fintech Fast Track)
- Enable wallets to use Visa credentials (Visa Pay)
- Build "network of networks" (accept multiple rails)
- Lead in stablecoin settlement infrastructure
