## § 3 · Workflow: McKinsey Engagement Approach

### Standard Engagement Process

```
┌─────────────────────────────────────────────────────────────────┐
│                    ENGAGEMENT LIFECYCLE                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  WEEK 1        WEEKS 2-4       WEEKS 5-8       WEEKS 9-12+      │
│    │              │               │               │             │
│    ▼              ▼               ▼               ▼             │
│ ┌──────┐      ┌──────┐       ┌──────┐       ┌──────┐          │
│ │DEFINE│  →   │STRUCT│   →   │ANALYZ│  →    │DELIVER│         │
│ └──┬───┘      └──┬───┘       └──┬───┘       └──┬───┘          │
│    │              │               │               │             │
│ • Problem    • Issue trees    • Data gather   • Final pres    │
│   alignment  • Hypotheses     • Analysis      • Impl plan     │
│ • Scope      • Work plan      • Synthesis     • Change mgmt   │
│ • Kickoff    • 80/20 prior.   • Insights      • Value track   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Daily Consulting Workflow

**Morning Routine (8:00 AM - 9:00 AM)**
1. Review overnight client communications
2. Check team dashboard for deliverable status
3. Prioritize day's objectives using 80/20 rule
4. Brief 15-minute team standup

**Core Work Blocks (9:00 AM - 12:00 PM; 2:00 PM - 6:00 PM)**
1. Deep analysis work (interviews, modeling, research)
2. Hypothesis testing and refinement
3. Slide/exhibit development
4. Internal team reviews

**Client Interaction (Scheduled throughout)**
1. Weekly steering committee meetings
2. Daily check-ins with key stakeholders
3. Working sessions for joint problem-solving

**Evening Routine (6:00 PM - 8:00 PM)**
1. Team synthesis session
2. Next-day planning
3. Senior partner update preparation

### Quality Gates

| Gate | Timing | Criteria |
|------|--------|----------|
| **Kickoff Quality** | Day 3 | Clear problem statement, aligned scope, defined hypotheses |
| **Mid-Engagement** | Week 4 | Key insights validated, hypotheses refined, direction clear |
| **Final Review** | Week 10 | Recommendations actionable, implementation feasible, client-ready |
