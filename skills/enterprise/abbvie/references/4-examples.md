## 4. Examples

### Example 1: Skyrizi IBD Launch Strategy

**Context:** Skyrizi received FDA approval for ulcerative colitis in June 2024, adding to existing Crohn's disease approval. This positions Skyrizi as the only IL-23 inhibitor covering both major IBD indications.

**Strategic Approach:**

```
SITUATION:
- IBD market: ~$20B globally, dominated by anti-TNFs (Humira, Remicade)
- Entyvio (Takeda) leads biologic-naïve UC; Stelara (J&J) strong in CD
- Skyrizi needs to establish position in crowded market

ABBVIE RESPONSE:
┌────────────────────────────────────────────────────────────────┐
│ 1. CLINICAL DIFFERENTIATION                                     │
│    • Emphasize complete histologic remission data               │
│    • Highlight dual IBD coverage (CD + UC)                      │
│    • Leverage established dermatology relationships             │
├────────────────────────────────────────────────────────────────┤
│ 2. MARKET ACCESS                                                │
│    • Aggressive contracting with major PBMs                     │
│    • Biosimilar defense: position as upgrade from Humira        │
│    • GI specialist targeting (3,000+ high-volume prescribers)   │
├────────────────────────────────────────────────────────────────┤
│ 3. PATIENT SERVICES                                             │
│    • Streamlined prior authorization support                    │
│    • Nurse educator network for injection training              │
│    • Co-pay assistance programs                                 │
├────────────────────────────────────────────────────────────────┤
│ 4. LAUNCH TARGETS                                               │
│    • Year 1: 15% biologic-naïve UC share                        │
│    • Year 3: $3B+ incremental IBD revenue                       │
│    • Maintain >90% formulary access in top 10 plans             │
└────────────────────────────────────────────────────────────────┘

EXPECTED OUTCOME:
Skyrizi IBD franchise reaches $5B+ by 2027, contributing significantly
to the $20B+ Skyrizi total revenue target.
```

### Example 2: Managing Humira Biosimilar Erosion

**Context:** Humira faced US LOE in January 2023. Multiple biosimilars launched with significant discounts (15-85% off WAC). US Humira revenue fell 41% in 2024 to $7.1B.

**AbbVie's Multi-Pronged Defense:**

```
THE CHALLENGE:
• 10+ biosimilars competing on price
• PBMs removing Humira from preferred lists
• Patients switching to lower-cost alternatives

ABBVIE'S RESPONSE FRAMEWORK:

┌─────────────────────────────────────────────────────────────────┐
│ TIER 1: CONTRACTING STRATEGY                                    │
│ • Negotiated value-based contracts with major insurers          │
│ • Volume guarantees in exchange for formulary position          │
│ • Indication-specific pricing (maintain value in niche areas)   │
├─────────────────────────────────────────────────────────────────┤
│ TIER 2: PATIENT RETENTION                                       │
│ • Humira Complete patient support program                       │
│ • $5 co-pay card for commercially insured                       │
│ • Nurse ambassador adherence support                            │
│ • Injection training and device familiarity                     │
├─────────────────────────────────────────────────────────────────┤
│ TIER 3: SWITCHING PROGRAMS                                      │
│ • Proactive Skyrizi/Rinvoq conversion for appropriate patients  │
│ • Internal "Humira to next-gen" migration pathways              │
│ • HCP education on IL-23/JAK advantages in specific populations │
├─────────────────────────────────────────────────────────────────┤
│ TIER 4: OPERATIONAL EFFICIENCY                                  │
│ • Reduced Humira sales force by 40%                             │
│ • Redeployed resources to Skyrizi/Rinvoq                        │
│ • Manufacturing optimization                                    │
└─────────────────────────────────────────────────────────────────┘

RESULTS (2024):
• Humira still generated $9B globally ($8B US contribution)
• Skyrizi + Rinvoq growth ($6B+ increase) offset Humira decline
• Portfolio transition on track for "best-in-class" execution
```

### Example 4: Neuroscience Pipeline Build-Out (Cerevel Acquisition)

**Context:** AbbVie acquired Cerevel Therapeutics for $8.7B in 2024, gaining emraclidine (M4/M1 selective agonist) for schizophrenia and Parkinson's psychosis.

**Strategic Analysis:**

```
ACQUISITION THESIS:
• Cerevel focused on neuroscience with differentiated mechanisms
• Emraclidine: potential best-in-class antipsychotic without weight gain/diabetes risk
• Complemented existing Vraylar franchise in psychiatric care

POST-ACQUISITION DEVELOPMENTS:
• December 2024: Emraclidine Phase 2 failures in schizophrenia
• $4.5B impairment charge recorded
• Pipeline re-prioritization underway

ABBVIE RESPONSE TO SETBACK:
┌─────────────────────────────────────────────────────────────────┐
│ 1. PORTFOLIO RISK MANAGEMENT                                    │
│    • Diversified pipeline reduces single-asset risk             │
│    • Other Cerevel assets remain valuable (tavapadon, etc.)     │
│    • Core growth drivers (Skyrizi/Rinvoq) unaffected            │
├─────────────────────────────────────────────────────────────────┤
│ 2. STRATEGIC ADJUSTMENTS                                        │
│    • Increased focus on tavapadon (Parkinson's - under review)  │
│    • Vyalev international rollout acceleration                  │
│    • Continued BD activity in neuroscience (30+ deals in 2024)  │
├─────────────────────────────────────────────────────────────────┤
│ 3. FINANCIAL DISCIPLINE                                         │
│    • Maintained investment-grade rating                         │
│    • Reaffirmed 2029 growth targets                             │
│    • Share buyback program continuation                         │
└─────────────────────────────────────────────────────────────────┘

LESSON: High-risk, high-reward pipeline bets require portfolio
balance. AbbVie's diversification across 4 therapeutic areas
provides resilience against individual asset failures.
```
