# AMD Strategic Acquisitions Reference

## Xilinx Acquisition (2022)

### Overview
| Attribute | Value |
|-----------|-------|
| **Announced** | October 2020 |
| **Closed** | February 2022 |
| **Value** | $49 billion (all-stock) |
| **Strategic Rationale** | Adaptive computing, data center expansion |

### Xilinx Portfolio Integration
| Product Line | Description | AMD Integration |
|--------------|-------------|-----------------|
| **Versal** | Adaptive SoC with AI engines | Data center AI acceleration |
| **Zynq** | FPGA + ARM processor | Embedded, aerospace |
| **Kintex/Virtex** | High-end FPGAs | Communications, defense |
| **Alveo** | Data center accelerators | Instinct companion cards |

### Strategic Impact
- **Market Expansion**: Embedded, aerospace, defense, communications
- **Technology**: FPGA fabric, adaptive computing expertise
- **Revenue**: ~$3B annual embedded segment
- **Customers**: Added Cisco, Nokia, Samsung relationships

## Historical Acquisitions

### Sea Micro (2012) - $334M
- **Product**: Microserver fabric technology
- **Outcome**: Technology integrated, Sea Micro brand discontinued 2015
- **Lesson**: Early cloud/hyperscale bet, timing was early

### ATI Technologies (2006) - $5.4B
- **Product**: Graphics processors (Radeon)
- **Outcome**: Core to AMD GPU strategy, gaming + pro graphics
- **Legacy**: Foundation for RDNA, CDNA architectures

### Altera (attempted 2024)
- **Status**: Competing bid with Intel
- **Outcome**: Not pursued; Xilinx acquisition sufficient

## Acquisition Strategy Analysis

### Pattern
1. **Technology Gaps**: Fill portfolio holes (FPGA, DPU, systems)
2. **Market Expansion**: Enter new verticals (embedded, aerospace)
3. **Customer Access**: Acquire customer relationships
4. **Talent**: Technical expertise in key areas

### Financial Discipline
| Acquisition | Multiple | Integration Status |
|-------------|----------|-------------------|
| Xilinx | ~10x revenue | Complete, accretive |
| Pensando | ~20x revenue | Integrated, growing |
| ZT Systems | ~2x revenue | Pending close |

### Competitive Response
| AMD Move | Competitive Response |
|----------|---------------------|
| Xilinx | Intel acquired Altera (2015) |
| Pensando | NVIDIA BlueField, Intel IPU |
| ZT Systems | NVIDIA DGX, HPE Cray |
