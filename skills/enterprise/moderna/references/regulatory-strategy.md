## mRNA-4157 Cancer Vaccine Regulatory Pathway

### Current Status
- Phase 3 (INTerpath-001) ongoing in melanoma
- Phase 2b KEYNOTE-942: 49% recurrence reduction
- Potential Breakthrough Therapy Designation (BTD) candidate

### Regulatory Considerations
1. **Personalized medicine**: Each dose is patient-specific
2. **Manufacturing**: 6-week turnaround from biopsy to dose
3. **Endpoints**: Disease-free survival (DFS) in adjuvant setting
4. **Companion diagnostic**: Tumor sequencing required

### Potential Pathways
- **Accelerated approval**: Based on Phase 2b data (previously considered)
- **Traditional approval**: Phase 3 data required (current path)
- **Priority review**: Expected given unmet need

## CMC (Chemistry, Manufacturing, Controls)

### Key CMC Considerations for mRNA

| Element | Challenge | Moderna Approach |
|---------|-----------|------------------|
| mRNA integrity | Degradation sensitive | -80°C storage, optimized UTRs |
| LNP consistency | Batch variability | Microfluidic manufacturing, strict QC |
| Raw materials | Lipid supply security | Dual sourcing, strategic inventory |
| Analytics | Novel methods | Platform assays, extensive validation |

### Manufacturing Changes
- **Post-approval changes**: Requires regulatory notification
- **Process improvements**: Continuous optimization
- **Facility changes**: Prior approval supplements
