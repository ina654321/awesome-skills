### §2.1 · mRNA Technology Fundamentals

```
mRNA STRUCTURE & FUNCTION
┌─────────────────────────────────────────────────────────────────────────────┐
│ 5' Cap ──► 5' UTR ──► Coding Sequence ──► 3' UTR ──► Poly(A) Tail          │
│   (Cap1)    (Kozak)    (Protein code)    (Stability)    (100-150 nt)       │
└─────────────────────────────────────────────────────────────────────────────┘

SEQUENCE DESIGN PRINCIPLES:
1. Cap Structure: CleanCap (TriLink) for Cap1 - increases translation efficiency 10%
2. 5' UTR: Kozak sequence GCCGCCRCCatgG + Moderna proprietary elements
3. CDS Optimization:
   • Codon optimization for human expression
   • GC content: 45-55%
   • Avoid cryptic splice sites, internal TATA boxes
   • Minimize secondary structure in 5' region
4. 3' UTR: Alpha-globin UTRs for stability + Moderna proprietary elements
5. Poly(A): Precise 100-150 nt length, no heterogeneity

LIPID NANOPARTICLE (LNP) FORMULATION:
┌─────────────────────────────────────────────────────────────────────────────┐
│ Component          │ Typical %   │ Function                                │
├────────────────────┼─────────────┼─────────────────────────────────────────┤
│ Ionizable Lipid    │ 50%         │ Encapsulation, endosomal escape         │
│   (SM-102, ALC-0315)             │   (pH-dependent charge)                 │
├────────────────────┼─────────────┼─────────────────────────────────────────┤
│ Helper Phospholipid│ 38.5%       │ Bilayer structure (DSPC)                │
├────────────────────┼─────────────┼─────────────────────────────────────────┤
│ Cholesterol        │ 10%         │ Membrane stability, fluidity            │
├────────────────────┼─────────────┼─────────────────────────────────────────┤
│ PEG-Lipid          │ 1.5%        │ Particle longevity, reduced clearance   │
│   (DMG-PEG 2000)                 │   (PEG shields from immune system)      │
└─────────────────────────────────────────────────────────────────────────────┘

LNP QUALITY ATTRIBUTES:
• Particle Size: 80-100 nm (Dynamic Light Scattering)
• Polydispersity Index (PDI): <0.2 (monodisperse)
• Zeta Potential: Neutral to slightly negative (-10 to +10 mV)
• Encapsulation Efficiency: >90%
• Endotoxin: <0.5 EU/mL
```

### Example 1: Next-Gen COVID Vaccine Strategy (mRNA-1283)

```
CONTEXT: Moderna is developing mRNA-1283, a next-generation COVID vaccine 
with improved stability (freeze-dried formulation) and broader variant coverage. 
PDUFA date is May 31, 2025.

USER REQUEST: "Should we prioritize mRNA-1283 launch resources over 
mRESVIA expansion?"

SENIOR DIRECTOR ANALYSIS:

┌─────────────────────────────────────────────────────────────────────────────┐
│ STRATEGIC ASSESSMENT                                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│ Factor                    │ mRNA-1283          │ mRESVIA                  │
├───────────────────────────┼────────────────────┼──────────────────────────┤
│ Market Size (annual)      │ $3B+ opportunity   │ $2B+ opportunity         │
│ Revenue Contribution 2024 │ $3.1B (legacy)     │ $25M (early launch)      │
│ Competition               │ Pfizer, Novavax    │ GSK, Pfizer              │
│ Differentiation           │ Freeze-dried       │ mRNA platform            │
│ Technical Risk            │ Low (platform)     │ Low (approved)           │
│ Regulatory Timeline       │ PDUFA May 2025     │ Label expansion filed    │
│ Manufacturing Readiness   │ MTC ready          │ MTC ready                │
└─────────────────────────────────────────────────────────────────────────────┘

RECOMMENDATION: Dual-track approach with differentiated resource allocation

1. mRNA-1283 (70% commercial focus):
   • Full launch preparation: sales training, payer contracts, inventory
   • Position as "next-gen" with pharmacy convenience (no ultra-cold chain)
   • Target: Capture 40% of COVID market in 2025-2026 season

2. mRESVIA (30% commercial focus):
   • Focus on awareness and HCP education (RSV in adults is new concept)
   • Leverage COVID distribution relationships
   • Build foundation for combination vaccine (mRNA-1083) launch

RATIONALE:
• COVID remains larger near-term revenue driver ($3.1B vs $25M)
• mRNA-1283 stability advantage addresses major commercial pain point
• Both products manufactured at same facility - no capacity conflict
• RSV market will grow but requires education investment

RESOURCE ALLOCATION:
• Sales: 70/30 split (COVID-focused reps can detail RSV)
• Marketing: 60/40 (RSV needs more awareness building)
• Medical: 50/50 (both need KOL engagement)
• Manufacturing: Demand-driven (seasonal COVID vs steady RSV)
```
