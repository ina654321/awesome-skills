## §3 · Workflow

### §3.1 · Automotive Development Lifecycle

**Phase 1: Market & Customer Definition**

```
┌─────────────────────────────────────────────────────────────┐
│  ACTIVITIES                                                 │
│  • Voice of Customer (VOC) research                         │
│  • Competitive benchmarking                                 │
│  • Target cost/price analysis                               │
│  • Concept development                                      │
├─────────────────────────────────────────────────────────────┤
│  DELIVERABLES                                               │
│  • Product concept document                                 │
│  • Target specifications (QCD: Quality, Cost, Delivery)     │
│  • Business case approval                                   │
│  • Chief Engineer assignment                                │
└─────────────────────────────────────────────────────────────┘
```

**Phase 2: Design & Engineering**

```
┌─────────────────────────────────────────────────────────────┐
│  ACTIVITIES                                                 │\n│  • Set-based concept exploration                            │
│  • Trade-off curve development                              │
│  • Supplier integration (early involvement)                 │
│  • Digital validation (CAD/CAE)                             │
│  • Prototype build planning                                 │
├─────────────────────────────────────────────────────────────┤
│  CHECKPOINTS                                                │
│  □ Concept freeze                                           │
│  □ Engineering release                                      │
│  □ Supplier contracts                                       │
│  □ Tooling kickoff                                          │
└─────────────────────────────────────────────────────────────┘
```

**Phase 3: Prototype & Testing**

| Test Type | Purpose | Duration |
|-----------|---------|----------|
| Mule testing | Powertrain integration | 3-6 months |
| Alpha prototypes | Design validation | 6-9 months |
| Beta prototypes | Production validation | 6-9 months |
| Durability | Long-term reliability | 12+ months |
| Certification | Regulatory compliance | Parallel |

**Phase 4: Production Preparation**

```
┌─────────────────────────────────────────────────────────────┐
│  TPS PREPARATION                                            │
│  • Process design (Standardized Work)                       │
│  • Jidoka implementation (error-proofing)                   │
│  • Kanban system design                                     │
│  • Supplier production readiness                            │
├─────────────────────────────────────────────────────────────┤
│  RAMP SEQUENCE                                              │
│  Pilot → Tryout → Mass Production Trial → Full Production   │
│   ↓         ↓              ↓                    ↓           │
│  10/day   100/day        1000/day           Full rate       │
└─────────────────────────────────────────────────────────────┘
```
