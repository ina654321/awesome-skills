# AT&T Network Technology Reference

## 5G Network Architecture

### Spectrum Portfolio

| Band | Frequency | Use Case | Coverage |
|------|-----------|----------|----------|
| **Low-band** | 850 MHz | Nationwide coverage | 300M+ POPs |
| **C-band (n77)** | 3.7-3.98 GHz | Capacity & speed | 270M+ POPs |
| **mmWave (n260/n261)** | 28/39 GHz | Dense urban hotspots | 100+ cities |
| **FirstNet (Band 14)** | 700 MHz | Public safety | Nationwide |

### 5G Deployment Status (2024)

- **5G Nationwide**: Sub-6 GHz coverage for broad availability
- **5G+ (C-band)**: Ultra-fast speeds in dense areas
- **Dynamic Spectrum Sharing (DSS)**: 5G on existing LTE spectrum for rapid coverage
- **Standalone 5G (SA)**: Core network modernization underway

## AT&T Fiber Technology

### Network Specifications

| Parameter | Specification |
|-----------|---------------|
| **Technology** | XGS-PON (10G symmetric) |
| **Download Speeds** | 300 Mbps to 5 Gbps |
| **Upload Speeds** | Symmetric (matching download) |
| **Latency** | <5ms typical |
| **Passings** | 28.9M locations (2024) |
| **Subscribers** | 9.3M fiber broadband |

### Fiber Build Economics

- Cost per passing: $1,000-$1,500 (aerial), $2,000+ (underground)
- Target penetration: 35-40%
- Payback period: 4-5 years
- Annual adds: 1M+ (7 consecutive years)

## Fixed Wireless Access (Internet Air)

### Technical Overview

| Parameter | Specification |
|-----------|---------------|
| **Technology** | 5G NR (C-band, n77) |
| **Download** | 100-300 Mbps typical, 500+ Mbps peak |
| **Upload** | 20-50 Mbps |
| **Latency** | 30-50ms |
| **CPE** | Self-install gateway |
| **Use Cases** | Copper migration, rural, temporary |

### Deployment Strategy

- Targets 10% of copper footprint not in fiber plan
- Faster deployment than fiber (weeks vs. 12-18 months)
- Lower CapEx per customer
- Complements fiber in convergence strategy

## FirstNet (Public Safety Network)

### Partnership Details

- **Partner**: First Responder Network Authority (federal government)
- **Spectrum**: Band 14 (700 MHz) dedicated + AT&T commercial spectrum
- **Coverage**: Nationwide, including rural/remote
- **Subscribers**: 2.8M+ (first responders and agencies)

### Capabilities

- Priority and preemption for public safety users
- Network hardening for disaster resilience
- Dedicated security operations center
- 99.99% availability target

## IP/MPLS Core Network

### Architecture

| Layer | Technology | Capacity |
|-------|------------|----------|
| **Core** | IP/MPLS | 100G/400G DWDM |
| **Metro** | Ethernet/OTN | 10G/100G |
| **Access** | Fiber/PON | 1G/10G |

### Convergence Benefits

- Shared fiber backhaul for 5G and wireline
- Unified IP core reduces complexity
- Common operational systems
- Cost synergies
