# Thermo Fisher Scientific — M&A History

## Overview

Thermo Fisher Scientific has completed 200+ acquisitions since its formation in 2006, using M&A as a core strategic capability to expand market leadership, enter new markets, and extend customer value propositions.

## Landmark Acquisitions

### 1. Life Technologies Corporation (2014)

| Attribute | Details |
|-----------|---------|
| **Acquisition Price** | $13.6 billion |
| **Announcement** | April 2013 |
| **Close** | February 2014 |
| **Strategic Rationale** | Create dedicated Life Sciences Solutions segment; gain leadership in genetic analysis and bioproduction |
| **Key Assets** | Genetic analysis platforms, qPCR, sequencing, cell culture |
| **Integration** | Formed new reporting segment; retained key brands (Applied Biosystems, Invitrogen, Gibco) |
| **Outcome** | Transformed TMO into life sciences leader; segment now generates $10B+ annually |

### 2. FEI Company (2016)

| Attribute | Details |
|-----------|---------|
| **Acquisition Price** | $4.2 billion |
| **Announcement** | May 2016 |
| **Close** | September 2016 |

### Integration Excellence

Thermo Fisher's integration playbook emphasizes:

1. **Speed to Value**
   - 100-day integration plan
   - Quick wins identification
   - Synergy tracking rigor

2. **Cultural Integration**
   - Respect acquired company strengths
   - Selective PPI Business System deployment
   - Aligned incentive structures

3. **Commercial Leverage**
   - Cross-selling opportunity capture
   - Channel expansion for acquired products
   - Unified customer experience

4. **Operational Synergies**
   - Procurement consolidation
   - Shared services integration
   - Best practice sharing
