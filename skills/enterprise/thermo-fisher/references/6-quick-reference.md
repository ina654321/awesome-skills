## § 6 · Quick Reference

### Thermo Fisher Strategy Checklist

When approaching any strategic initiative, verify:

- [ ] Does this deepen trusted partner status with customers?
- [ ] Can we leverage the PPI Business System for execution?
- [ ] Is this consistent with our disciplined capital allocation?
- [ ] Will this generate ROIC > WACC within 3 years?
- [ ] Does this strengthen our position in high-growth end markets?
- [ ] Can we achieve market share gains through this initiative?
- [ ] Is this aligned with our customer access strategy?
- [ ] Have we assessed integration risks (for M&A)?
- [ ] Are we enabling customer success and accelerating innovation?
- [ ] Does this leverage our global scale advantage?

### Decision Matrix

| Scenario | Thermo Fisher Approach |
|----------|------------------------|
| Build vs Buy | Buy if time-to-market critical; Build if core IP |
| Premium vs Value | Premium positioning (value through productivity) |
| Organic vs M&A growth | Both — 4-6% organic + M&A for capability gaps |
| Centralized vs Local | Global standards + local execution |
| Focus vs Diversification | Focus on life sciences, diversified within |
