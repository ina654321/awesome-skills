### Late-Stage Pipeline

#### Orforglipron (LY3502970)
| Attribute | Details |
|-----------|---------|
| **Mechanism** | Oral, non-peptide GLP-1 receptor agonist |
| **Differentiation** | First truly oral GLP-1 (small molecule, not peptide) |
| **Key Trials** | ATTAIN-1, ATTAIN-2, ACHIEVE-1 |
| **Efficacy** | 12.4% weight loss (ATTAIN-1), 9.6% (ATTAIN-2 with T2D) |
| **Status** | Submitted U.S., Japan, EU (2025) |
| **Expected Approval** | Q2 2026 (U.S.) |
| **Peak Sales Potential** | $16B+ (2031) |
| **Advantages** | No injection, no cold chain, lower manufacturing cost |

#### Retatrutide (LY3437943)
| Attribute | Details |
|-----------|---------|
| **Mechanism** | GGG tri-agonist (GLP-1/GIP/Glucagon) |
| **Differentiation** | Triple hormone action, next-gen after tirzepatide |
| **Key Trials** | TRIUMPH (obesity/OSA/OA), TRANSCEND (T2D) |
| **Efficacy** | 24.2% weight loss in Phase 2 |
| **Status** | Phase 3 ongoing |
| **Expected Filing** | 2026 |
| **Expected Launch** | 2028 |
| **Peak Sales Potential** | $30B+ (2031) |
