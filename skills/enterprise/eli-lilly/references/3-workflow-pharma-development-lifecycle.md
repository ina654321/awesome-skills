## §3 · Workflow: Pharma Development Lifecycle

### §3.1 · Clinical Development Process

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    ELI LILLY CLINICAL DEVELOPMENT FLOW                      │
└─────────────────────────────────────────────────────────────────────────────┘

PHASE 1: DISCOVERY TO IND (Years 1-3)
┌─────────────────────────────────────────────────────────────────────────────┐
│ Target Validation → Lead Optimization → Preclinical → IND Submission        │
│                                                                              │
│ Key Activities:                                                              │
│ • Target identification and validation                                      │
│ • Medicinal chemistry / biologics engineering                               │
│ • In vitro and in vivo efficacy/safety studies                              │
│ • GLP toxicology studies                                                    │
│ • CMC development for clinical supplies                                     │
│                                                                              │
│ Decision Gate: IND approval → First human dose                              │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
PHASE 2: EARLY CLINICAL (Years 2-4)
┌─────────────────────────────────────────────────────────────────────────────┐
│ Phase 1 (Safety/PK) → Phase 2a (PoC) → Phase 2b (Dose-ranging)              │
│                                                                              │
│ Key Activities:                                                              │
│ • First-in-human safety and tolerability                                    │
│ • Pharmacokinetics/pharmacodynamics                                         │
│ • Proof-of-concept efficacy signals                                         │
│ • Dose selection for Phase 3                                                │
│ • Biomarker development                                                     │
│                                                                              │
│ Decision Gate: Positive Phase 2b → Phase 3 initiation                       │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
PHASE 3: PIVOTAL TRIALS (Years 3-6)
┌─────────────────────────────────────────────────────────────────────────────┐
│ Phase 3 Program Design → Execution → Data Lock → Submission                 │
│                                                                              │
│ Key Activities:                                                              │
│ • Pivotal efficacy trials (regulatory endpoints)                            │
│ • Safety database compilation                                               │
│ • Health economics and outcomes research                                    │
│ • Parallel regulatory strategy (FDA, EMA, global)                           │
│ • Commercial launch preparation                                             │
│                                                                              │
│ Decision Gate: Positive Phase 3 → BLA/MAA submission                        │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ↓
PHASE 4: LAUNCH & LIFECYCLE (Years 6+)
┌─────────────────────────────────────────────────────────────────────────────┐
│ Regulatory Approval → Launch → Real-World Evidence → Label Expansion        │
│                                                                              │
│ Key Activities:                                                              │
│ • Regulatory approval and reimbursement negotiations                        │
│ • Commercial launch and market access                                       │
│ • Post-marketing safety surveillance                                        │
│ • Real-world evidence generation                                            │
│ • Lifecycle management (new indications, formulations)                      │
│                                                                              │
│ Decision Gate: Label expansion opportunities, next-gen development          │
└─────────────────────────────────────────────────────────────────────────────┘
```

### §3.2 · Regulatory Strategy

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ ELI LILLY REGULATORY APPROACH                                                │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│ PRIORITY REVIEW PROGRAMS:                                                    │
│ • Breakthrough Therapy Designation (BTD) - expedited development            │
│ • Fast Track - serious conditions with unmet need                           │
│ • Priority Review - 6-month FDA review vs. 10-month standard                │
│ • Accelerated Approval - surrogate endpoints                                │
│                                                                              │
│ GLOBAL SUBMISSION STRATEGY:                                                  │
│ • U.S. (FDA): Primary submission, PDUFA target                              │
│ • EU (EMA): Parallel CHMP process                                           │
│ • Japan (PMDA): Collaborative development                                   │
│ • China (NMPA): Separate development path, growing importance               │
│                                                                              │
│ REGULATORY MILESTONES 2025-2026:                                             │
│ • Orforglipron: U.S. approval expected Q2 2026                              │
│ • Retatrutide: Phase 3 readouts 2026, filing 2026-2027                      │
│ • Jaypirca CLL: U.S. approval expected 2025                                 │
└─────────────────────────────────────────────────────────────────────────────┘
```
