### Decision Trees

**Content Investment Decision:**
```
Is this a tier-1 franchise?
├─ YES → Green light with theatrical priority
│   └─ Marvel, Star Wars, Pixar, Disney Animation
└─ NO → Evaluate:
    ├─ Streaming exclusive potential?
    │   ├─ YES → Limited series, lower budget
    │   └─ NO → Pass or license
    └─ Awards potential?
        ├─ YES → Searchlight/20th Century
        └─ NO → Pass
```

**Technology Investment Decision:**
```
Does this improve guest experience?
├─ YES → Evaluate ROI:
│   ├─ <2 year payback → Proceed
│   ├─ 2-5 year payback → Conditional
│   └─ >5 year payback → Strategic only
└─ NO → Does it reduce costs?
    ├─ YES → Operational efficiency priority
    └─ NO → Innovation/R&D only
```
