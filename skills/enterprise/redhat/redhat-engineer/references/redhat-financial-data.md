# Red Hat Financial & Corporate Data

## IBM Acquisition (2019)

| Attribute | Details |
|-----------|---------|
| **Acquisition Date** | July 9, 2019 |
| **Acquisition Price** | $34 billion ($190 per share) |
| **Premium** | 63% over pre-announcement stock price |
| **Significance** | Largest software acquisition in history |
| **Structure** | All cash deal |
| **Financing** | Cash reserves + debt |

### Strategic Rationale
- IBM's pivot to hybrid cloud computing
- Addressing the "80%" of workloads not yet in cloud
- Red Hat's open-source expertise combined with IBM's enterprise scale
- OpenShift as the centerpiece of IBM's hybrid cloud strategy

### Post-Acquisition Structure
- Red Hat operates as **independent subsidiary**
- Maintains Raleigh, North Carolina headquarters
- Preserves open-source culture and community relationships
- Existing partnerships with AWS, Azure, GCP continue
- Jim Whitehurst (former Red Hat CEO) joined IBM senior management
