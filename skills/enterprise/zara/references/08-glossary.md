# Zara & Fast Fashion Glossary

**Industry Terminology & Inditex Acronyms**

---

## Fashion Industry Terms

### A-C

| Term | Definition |
|------|------------|
| **A-line** | Silhouette that is narrow at top and flares toward bottom |
| **Assortment** | Range of products offered |
| **Basic** | Core, timeless item not tied to trends |
| **Capsule collection** | Limited, curated selection of items |
| **CMT** | Cut-Make-Trim; manufacturing model where brand provides fabric, contractor provides labor |
| **Collection** | Group of garments designed for a season |
| **Colorway** | A style in different color options |
| **Conversion rate** | Percentage of store visitors who make a purchase |
| **Core collection** | Permanent, non-seasonal items |
| **Cutting** | Process of cutting fabric according to patterns |

### D-F

| Term | Definition |
|------|------------|
| **Delivery** | Shipment of new merchandise to stores |
| **Diffusion line** | Lower-priced secondary brand |
| **Drop** | Release of new products |
| **DTD** | Design-to-Delivery; time from concept to store |
| **Fast fashion** | Rapid production model with quick trend turnaround |
| **Fashion cycle** | Lifecycle of a trend from introduction to decline |
| **Flagship** | Large, showcase store with full assortment |
| **Flat lay** | Product photography style with items laid flat |
| **FOB** | Free on Board; shipping term |
| **Full-price sell-through** | Percentage of goods sold at original price |

### G-I

| Term | Definition |
|------|------------|
| **Grading** | Scaling pattern to different sizes |
| **Gross margin** | Revenue minus cost of goods sold |
| **Hangtag** | Label attached to garment with price/information |
| **Inventory turn** | Number of times inventory is sold/replaced annually |
| **Item** | Individual product/SKU |
| **Key item** | Bestselling or strategic product |
| **Lead time** | Time from order to delivery |

### J-L

| Term | Definition |
|------|------------|
| **Join Life** | Zara's sustainable product label |
| **Knitwear** | Garments made from knitted fabric |
| **Label** | Brand tag sewn into garment |
| **Last** | Footwear mold determining shape |
| **Launch** | Introduction of new product/collection |
| **Line sheet** | Document showing product range with details |
| **Lookbook** | Collection of styled outfits for season |
| **LS** | Long Sleeve |

### M-O

| Term | Definition |
|------|------------|
| **Markdown** | Price reduction to clear inventory |
| **Mark-up** | Difference between cost and selling price |
| **Merchandising** | Presentation and promotion of products |
| **MOQ** | Minimum Order Quantity |
| **Net selling space** | Retail area excluding storage/office |
| **Omnichannel** | Integrated online and offline retail |
| **Open-to-buy** | Budget for new merchandise purchases |
| **OTB** | Open-to-Buy |

### P-R

| Term | Definition |
|------|------------|
| **Pattern** | Template for cutting fabric pieces |
| **Pick and pack** | Order fulfillment process |
| **Placement** | Location of print/design on garment |
| **POP** | Point of Purchase display |
| **Pre-collection** | Early delivery before main season |
| **Pre-order** | Ordering before production |
| **Quality control** | Inspection for defects |
| **Replenishment** | Restocking sold items |
| **RTW** | Ready-to-Wear |
| **Runway** | Fashion show presentation |

### S-U

| Term | Definition |
|------|------------|
| **Sample** | Prototype garment for approval |
| **Seam** | Line where fabric pieces join |
| **SKU** | Stock Keeping Unit; unique product identifier |
| **Sourcing** | Finding and purchasing materials/products |
| **Sportswear** | Athletic-inspired casual clothing |
| **Stockout** | Running out of inventory |
| **Style number** | Unique identifier for design |
| **Supply chain** | Network from raw materials to customer |
| **Sustainability** | Environmental and social responsibility |
| **Trend** | Direction in which fashion is moving |
| **UPT** | Units Per Transaction |
| **US** | United States size or United Sizes |

### V-Z

| Term | Definition |
|------|------------|
| **Value chain** | Sequence of value-adding activities |
| **Vertical integration** | Owning multiple supply chain stages |
| **Visual merchandising** | Display and presentation of products |
| **Wovens** | Fabrics made by weaving (vs. knitting) |
| **Wholesale** | Selling to retailers (not consumers) |
| **Window display** | Storefront presentation |
| **Working capital** | Operating liquidity |
