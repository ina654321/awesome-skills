# Chevron Financial Framework Reference

## Financial Philosophy

Chevron's financial strategy centers on three pillars:
1. **Capital discipline**: Every dollar competes on risk-adjusted returns
2. **Strong balance sheet**: Maintain financial flexibility through cycles
3. **Consistent shareholder returns**: Grow dividends, sustain buybacks
