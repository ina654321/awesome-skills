# Equipment Specifications

## Wheel Gun (Pneumatic Impact Wrench)

### Technical Specifications

| Parameter | Value | Notes |
|---|---|---|
| **RPM** | 10,000 - 12,000 | Optimal range for nut engagement |
| **Torque** | 3,000 - 4,000 Nm | Controlled torque to prevent hub damage |
| **Weight** | 3.5 - 4.5 kg | Balance between power and operator fatigue |
| **Air pressure** | 8-10 bar | Consistent across all team guns |
| **Cycle time** | 0.3 - 0.5s per wheel | Includes engage and release |
| **Threading speed** | ~1.5mm per revolution | Most nuts: 8-10 revolutions to secure |

### Threading Direction

Critical: F1 uses different thread directions on different corners:

| Corner | Thread Direction | Reason |
|---|---|---|
| Right Front (RF) | **Left-hand thread** | Wheel rotation tightens nut |
| Left Front (LF) | Right-hand thread | Wheel rotation tightens nut |
| Right Rear (RR) | Right-hand thread | Wheel rotation tightens nut |
| Left Rear (LR) | **Left-hand thread** | Wheel rotation tightens nut |

**Why?** When the wheel spins forward, friction between tire and ground creates torque on the hub. Left-hand thread on right-side wheels prevents self-loosening. Get this wrong: catastrophic.

### Operator Grip

- **Left-hand operators**: Right-front (left-hand thread) and Left-rear (left-hand thread)
- **Right-hand operators**: Right-rear (right-hand thread) and Left-front (right-hand thread)
- Operators use dominant hand for gun trigger
- Off-hand stabilizes the gun on the wheel nut

## Wheel Nut

### Specifications

| Parameter | Value |
|---|---|
| **Diameter** | 1-inch (25.4mm) |
| **Material** | Titanium alloy or high-strength steel |
| **Weight** | ~80g per nut |
| **Contact surface** | 8-point or 12-point internal spline |
| **Coating** | Molybdenum disulfide (dry lubricant) |

### Design Requirements

1. **Quick engagement**: Spline design allows fast seating
2. **Controlled torque**: Gun stops at predetermined torque setting
3. **Visual confirmation**: Operator can see when seated properly
4. **No cross-threading**: Design includes anti-cross-thread features

## Tire Handler Equipment

### Wheel Tray

- Holds new tire in exact position for rapid transfer
- Positioned at optimal angle for handler's body mechanics
- Color-coded for compound identification

### Tire Warming

- **Pre-race**: Tires on warming blankets (80-90°C)
- **Target temperature**: 80-100°C for optimal grip on out-lap
- **Wet tires**: Supplied at ambient temperature (no warming needed)

## Traffic Light System

### Specifications

| Parameter | Value |
|---|---|
| **Type** | LED traffic light (road-legal standard) |
| **Colors** | Red / Green (yellow rarely used in F1) |
| **Position** | Lollipop stick at driver eye level |
| **Manual override** | Crew chief can disable if needed |
| **FIA requirement** | Automatic system with manual backup |

### Lollipop Man Role

- Position: Front-left of car, driver's sight line
- Responsibility: Ultimate safety authority
- Protocol:
  1. Car arrives → Green light (or held red if unsafe)
  2. Any hazard → Red light immediately
  3. All clear → Green light
  4. Driver must acknowledge before moving
