# Training Methodology

## Physical Conditioning

### Position-Specific Training

**Wheel Gun Operators:**
- Grip strength: 60-80 kg (sustained)
- Arm endurance: 100+ repetitions without fatigue
- Core stability: Prevent sway during gun operation
- Reaction time: <150ms to visual stimulus

**Jack Operators:**
- Explosive power: 800kg lift capacity
- Lower body: Squat strength 1.5x body weight
- Reaction time: <200ms to car arrival
- Stamina: Multiple stops per race, 20+ races per season

**Tire Handlers:**
- Grip strength: Sustained tire hold
- Upper body: Pulling mechanics
- Coordination: Both hands, timed release
- Spatial awareness: Position in tight pit box

### Daily Training Schedule

```
Pre-Season (Winter, 4-6 weeks):
- Physical: 5 days/week, 2-3 hours/day
- Technical: 3 days/week, practice stops
- Mental: Visualization, pressure management

Race Season (20+ races):
- Day before race: Light practice, team meeting
- Race day: Warm-up, visualization, single practice stop
- Between races: 2-3 practice days/week

Training Breakdown:
- Physical conditioning: 40%
- Technical practice: 40%
- Mental training: 20%
```

## Stop Practice Protocol

### Repetition Framework

**Target: 50-100 stop repetitions per week during season**

| Session Type | Reps | Focus |
|---|---|---|
| Speed work | 10-15 | Pure timing, minimize all movements |
| Precision work | 15-20 | Perfect gun angle, nut alignment |
| Recovery drills | 10-15 | Error recovery, cross-thread handling |
| Full stop simulation | 10-15 | Race-day conditions |
| Team coordination | 5-10 | Timing across all 22 crew |

### High-Speed Camera Analysis

Every practice stop recorded at 240fps minimum:

1. **Frame-by-frame breakdown**:
   - Identify micro-delays (<0.05s)
   - Check body mechanics for efficiency
   - Verify simultaneous operations

2. **Error capture**:
   - Near-misses documented
   - Improvement tracked over time

3. **Benchmarking**:
   - Compare against team's best stops
   - Compare against rival team published data

### Biomechanical Optimization

Goal: Shave 0.01-0.05s per movement through efficiency

**Areas of optimization:**
- Gun operator: Arm angle, grip pressure, trigger pull
- Jack operator: Entry point, lift trajectory, release timing
- Tire handler: Grip position, pull angle, body rotation
- All positions: Foot placement, stance width

## Mental Training

### Pressure Simulation

**Techniques:**

1. **Crowd noise**: Replicate race-day atmosphere with 100+ dB audio
2. **Time pressure**: Random stop calls, simulating race conditions
3. **Consequence framing**: Frame practice as "championship-deciding stop"
4. **Visualization**: Daily mental rehearsal of perfect stop

### Error Recovery Training

Practice the four recovery scenarios:

1. **Cross-threaded nut** (most common)
2. **Stuck wheel** (hub contamination)
3. **Gun malfunction** (trigger failure)
4. **Unsafe release** (near-miss)

Goal: Make recovery response as automatic as normal operation.

## Team Coordination Drills

### 22-Person Synchronization

**Drills:**

1. **Rhythm exercise**: All 22 in circle, clap in sequence — train timing
2. **Silent stop**: No verbal commands during full stop — pure visual coordination
3. **Fault injection**: Random "problems" inserted — team must adapt
4. **Multi-car simulation**: Two cars pitting simultaneously

### Role-Specific Communication

| Call | Meaning |
|---|---|
| "Box, box, box" | Pit immediately |
| "In the window" | Approaching pit box |
| "Ready" | All crew in position |
| "Green" | Safe to release |
| "Hold" | Do not release |
| "Problem [corner]" | Issue at specific wheel |
| "Clear" | Issue resolved |
| "Abort" | Cancel pit stop |

## Recovery Protocols Mastery

### Cross-Thread Recovery (Standard)

```
1. Feel resistance → Immediate trigger release (0.1s)
2. Back gun off nut (0.3s)
3. Re-position perpendicular to hub (0.2s)
4. Re-engage nut (0.5s)
5. Verify seated (0.1s)
6. Complete torque (0.3s)
Total recovery: 1.5s (vs. 0.45s normal)
```

### Stuck Wheel Recovery

```
1. Both operators on same nut (0.3s)
2. Maximum torque application (2.0s)
3. If fails: Manual breaker bar (10-15s)
4. If fails: Hub inspection, possible DNF
Prevention: Pre-race hub cleaning protocol
```
