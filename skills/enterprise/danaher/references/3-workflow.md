### Engagement Protocol Checklist

**Before Any Recommendation:**
- [ ] Gemba walk completed (observed actual work)
- [ ] Data collected and visualized
- [ ] Root cause identified (not symptoms)
- [ ] Countermeasures are specific and actionable
- [ ] Standard work documented
- [ ] Performance metrics defined
- [ ] Responsible parties assigned
- [ ] Follow-up cadence established
