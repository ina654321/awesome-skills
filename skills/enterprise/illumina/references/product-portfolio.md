# Illumina Product Portfolio Reference

## Sequencing Platforms

### NovaSeq X Series (Flagship)

**NovaSeq X Plus**
- Throughput: Up to 20 billion reads per run
- Output: Up to 8Tb per run (25B flow cell)
- Cost per genome: ~$200 (30x coverage)
- Run time: ~48 hours
- Chemistry: XLEAP-SBS (2-color)
- Use cases: Population genomics, biobanks, large-scale WGS

**NovaSeq X**
- Throughput: Up to 10 billion reads per run
- Output: Up to 4Tb per run
- Cost per genome: ~$250-300
- Smaller footprint than X Plus
- Use cases: Large research projects, clinical genomics

### NovaSeq 6000 (Established)

- Throughput: Up to 6Tb per run (S4 flow cell)
- Output: Up to 20 billion reads
- Chemistry: SBS (4-color)
- Use cases: High-throughput research, clinical, NIPT
- Status: Remains in portfolio; slower migration than expected

### NextSeq 1000/2000 (Mid-Throughput)

**NextSeq 2000**
- Throughput: 330Gb - 1.2Tb per run
- Chemistry: XLEAP-SBS
- Run time: 12-48 hours
- Use cases: Core labs, clinical research, targeted panels

**NextSeq 1000**
- Throughput: 40Gb - 330Gb per run
- Entry point to NextSeq family
- Use cases: Smaller labs, targeted applications

### MiSeq Series (Benchtop)

**MiSeq i100 Series** (Newest)
- Throughput: Up to 1.1Gb per run
- Run time: As fast as 4 hours
- Use cases: Small genome, amplicons, targeted panels

**MiSeq**
- Throughput: 0.3Gb - 15Gb per run
- Most widely deployed benchtop sequencer
- Use cases: Targeted sequencing, 16S, QC

**MiSeq Dx** (IVD version)
- FDA-cleared diagnostic platform
- Use cases: Regulated clinical assays

### iSeq 100 (Entry)

- Throughput: 1.2Gb per run
- Lowest cost entry point
- Use cases: QC, small projects, educational

- Nextera DNA: Transposome-based, fastest

**RNA Library Prep**
- TruSeq Stranded Total RNA: Complete transcriptome
- TruSeq Stranded mRNA: Poly-A selected
- Illumina Stranded Total RNA Prep: Ribo-zero compatible

- TruSight Oncology 500: Comprehensive cancer profiling
