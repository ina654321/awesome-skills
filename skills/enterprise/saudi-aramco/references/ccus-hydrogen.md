## Hydrogen Strategy

### Blue Hydrogen

**Process:**
```
Natural Gas → SMR → H₂ + CO₂ → CCS → Low-Carbon H₂
```

**Economics:**
| Component | Cost |
|-----------|------|
| Natural gas feedstock | $0.50-1.00/kg H₂ |
| SMR production | $0.80-1.50/kg H₂ |
| CCS | $0.20-0.50/kg H₂ |
| **Total delivered** | **$1.50-3.00/kg H₂** |

**Aramco Advantages:**
- Low-cost Jafurah gas
- Existing CCS infrastructure
- Pipeline network
- Export logistics

### Green Hydrogen

**NEOM Project:**
| Attribute | Value |
|-----------|-------|
| Capacity | 1.2 MM tpa ammonia |
| Power source | Solar + wind (renewable) |
| Partners | ACWA Power, Air Products |
| Technology | Electrolysis |
| Cost target | Competitive with blue H₂ by 2030 |

**Economics (Current):**
| Component | Cost |
|-----------|------|
| Renewable power | $2.00-3.00/kg H₂ |
| Electrolyzer | $1.50-2.50/kg H₂ |
| **Total** | **$3.50-5.50/kg H₂** |

### Ammonia as Hydrogen Carrier

| Factor | Rationale |
|--------|-----------|
| Transport | Easier than liquid H₂ |
| Storage | Existing infrastructure |
| Safety | Established handling protocols |
| Market | Fertilizer + marine fuel demand |

**Aramco Ammonia Target:** 2.5 MM tpa by 2030
