Transform low-quality skills (5-7/10) into EXEMPLARY assets (9.5/10) using the proven 7-step methodology.

## Overview

## The 7-Step Methodology

### Step 1: Problem Diagnosis (15 min)
Analyze current skill, identify deficiencies, define fix scope.

### Step 2: Professional Research (30-60 min)
Gather domain-specific data, methodologies, and real-world examples.

### Step 3: Architecture Design (20 min)
Plan System Prompt (§1.1/1.2/1.3), Domain Knowledge, Workflow, Examples.

### Step 4: Progressive Disclosure Setup (15 min)
Create SKILL.md skeleton (< 350 lines) and references/ structure.

### Step 5: Content Production (60-90 min)
Fill all sections with professional, data-driven content.

### Step 6: Quality Validation (15-30 min)
Run skill-evaluator, verify 9.5/10 EXEMPLARY certification.

### Step 7: Delivery (10 min)
Package files, save evaluation report, document improvements.
