# § 8 · Workflow Phases - Full Details

## Phase 1: Diagnosis (15 min)

### Objective

Accurately identify all skill deficiencies and restoration requirements.

### Key Activities

1. **Read Complete File**
   - Read SKILL.md from start to finish
   - Don't skim or skip sections
   - Note overall structure and flow

2. **Check System Prompt**
   - Verify §1.1 exists and is complete
   - Verify §1.2 exists and is complete
   - Verify §1.3 exists and is complete

3. **Identify Generic Content**
   - Catalog all generic phrases
   - Note missing specific data
   - Flag placeholder text

4. **Assess Current Score**
   - Use scoring rubric
   - Estimate quality level
   - Identify biggest gaps

5. **Define Fix Targets**
   - Determine scope of restoration
   - Set target score (9.5/10)
   - Estimate time required

### Done Criteria

- [✓] SKILL.md read completely
- [✓] Missing sections identified
- [✓] Generic content catalogued
- [✓] Current score estimated
- [✓] Fix scope defined

### Fail Criteria

- [✗] Skimmed file, missed key issues
- [✗] Structural problems not identified
- [✗] Generic content overlooked
- [✗] Scope undefined or unrealistic

4. **Validation**

- [✓] Terminology identified
- [✓] Sources validated

### Fail Criteria

## Phase 6: Validation (15-30 min)

### Objective

Verify EXEMPLARY quality.

### Key Activities

1. **Self-Assessment**
   - Rate each dimension
   - Estimate score
   - Identify weak points

2. **Evaluator Run**
   - Run skill-evaluator
   - Get Text score
   - Get Runtime score
   - Calculate Variance

3. **Gap Analysis**
   - Compare self vs evaluator
   - Identify discrepancies
   - Find improvement areas

4. **Iteration** (if needed)
   - Fix lowest dimension
   - Re-run evaluator
   - Repeat until ≥ 9.0

5. **Final Verification**
   - Confirm all thresholds met
   - Verify certification ready
   - Check variance acceptable

### Done Criteria

- [✓] Evaluator score ≥ 9.0
- [✓] Text score ≥ 8.5
- [✓] Runtime score ≥ 8.5
- [✓] Variance < 1.0

### Fail Criteria

- [✗] Score < 9.0
- [✗] Variance too high
- [✗] Dimension below 6.0
- [✗] No iteration plan

## Phase 7: Delivery (10 min)

### Objective

Package and deliver restored skill.

### Key Activities

1. **Save Evaluation Report**
   - Create EVALUATION_REPORT.md
   - Include all scores
   - Document certification

2. **Verify File Structure**
   - Check all files present
   - Verify naming
   - Confirm links work

3. **Document Improvements**
   - List changes made
   - Note score improvement
   - Document time spent

4. **Final Handoff**
   - Prepare summary
   - List deliverables
   - Provide next steps

### Done Criteria

- [✓] EVALUATION_REPORT.md saved
- [✓] All files verified
- [✓] Improvements documented
- [✓] Ready for handoff

### Fail Criteria

- [✗] Missing files
- [✗] No evaluation report
- [✗] Structure incorrect
- [✗] Incomplete documentation
