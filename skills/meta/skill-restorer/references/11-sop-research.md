# SOP 2: Professional Research

## Purpose

Gather domain-specific data and knowledge to replace generic content.

## Input

Diagnosis Report (know what domain to research)

## Output

Research Notes with specific data

## Process

### Step 1: Identify Research Targets (5 min)

From the Diagnosis Report, identify what needs research.

**Common Research Needs**:
- Company/organization financials
- Methodology specifics
- Industry benchmarks
- Real-world examples
- Key terminology

### Step 2: Source Selection (5 min)

Choose appropriate sources for the domain.

**Source Quality Hierarchy**:
1. Company annual reports / SEC filings
2. Academic papers / Books
3. Industry analysis (Gartner, McKinsey)
4. Verified news (Reuters, Bloomberg)
5. Official blogs / Conference talks

### Step 3: Data Collection (30-60 min)

**Company Data**:
```
Founded: [Year, Location]
Revenue: [Amount, Timeframe]
Valuation: [Amount, Date]
Employees: [Number]
Users/Customers: [Number]
Key Products: [Names, Launch dates, Metrics]
Leadership: [Key people, Quotes]
```

**Methodology Data**:
```
Framework Name: 
Creator: 
Key Principles: 
Process Steps: 
Success Metrics: 
Real-world Applications: 
```

**Industry Data**:
```
Market Size: 
Growth Rate: 
Key Players: 
Benchmarks: 
Trends: 
```

### Step 4: Validation (10 min)

Cross-reference data from multiple sources.

**Validation Checklist**:
- [ ] Data found in 2+ independent sources
- [ ] Numbers are consistent
- [ ] Dates align
- [ ] Quotes are accurate and attributed

### Step 5: Documentation (10 min)

Compile findings into Research Notes.
