# Example 4: New Skill Creation

## Context

**Skill**: New domain (not restoration)
**Starting Point**: From scratch
**Target Score**: 9.5/10 from day one

## Process Adaptation

### Step 1: Domain Analysis
Instead of diagnosis, analyze the domain:
- What is unique about this field?
- Who are the experts?
- What data is available?
- What are common pitfalls?

### Step 2: Research (Extended)
Spend 60-90 min on research:
- Find 15+ data points
- Identify 3+ methodologies
- Document 10+ key terms
- Collect 5+ real examples

### Steps 3-7: Same as Restoration
Architecture → Structure → Production → Validation → Delivery

### Key Difference

New skills require more research (no existing content to leverage), but have no legacy issues to fix.

**Time Estimate**: 3-4 hours (similar to restoration)
