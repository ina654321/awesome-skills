# Glossary

## Vault Terms

| Term | Definition |
|------|------------|
| **Secret** | Sensitive data |
| **Engine** | Secret backend |
| **Policy** | Access control |
| **Token** | Auth credential |
| **KV** | Key-Value store |

## Concepts

| Term | Definition |
|------|------------|
| **Mount** | Secret engine path |
| **Namespace** | Isolation |
| **Transit** | Encryption service |
| **PKI** | Certificate authority |
