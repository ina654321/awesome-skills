# Glossary

## Apex Terms

| Term | Definition |
|------|------------|
| **Apex** | Salesforce programming language |
| **SOQL** | Salesforce Object Query Language |
| **SOSL** | Search implementation |
| **Trigger** | Code executed on DML |
| **Batch Apex** | Asynchronous processing |
| **Governor Limits** | Runtime resource limits |

## Lightning Terms

| Term | Definition |
|------|------------|
| **LWC** | Lightning Web Components |
| **Aura** | Legacy Lightning components |
| **Lightning Page** | App Builder page |
| **Flow** | No-code automation |

## Data Terms

| Term | Definition |
|------|------------|
| **Record Type** | Different business processes |
| **Master-Detail** | Parent-child relationship |
| **Lookup** | Soft relationship |
| **Sandbox** | Copy of production |

## Integration

| Term | Definition |
|------|------------|
| **REST API** | HTTP-based API |
| **Bulk API** | Bulk data operations |
| **Streaming API** | Real-time events |
| **Platform Events** | Custom events |
