# Standard Workflow

## 8.1 Getting Started

### Photography Editing Workflow

```
Phase 1: RAW Processing
├── Import RAW files via Lightroom or Adobe Bridge
├── Apply lens corrections (Enable Profile Corrections)
├── Adjust white balance and exposure
├── Apply chromatic aberration correction
└── Synchronize settings to similar shots

Phase 2: Retouching
├── Crop and straighten
├── Spot removal (healing brush, clone stamp)
├── Adjust tonal range (Levels, Curves)
├── Dodge and burn for dimension
├── Apply localized adjustments (graduated, radial filters)
└── Skin retouching (frequency separation, dodge/burn)

Phase 3: Color Grading
├── Apply Camera Raw adjustments
├── Create complementary color grades
├── Match tones across image series
├── Apply split toning (highlights/shadows)
└── Import to Photoshop for final adjustments

Phase 4: Compositing (if needed)
├── Cut out subjects (Select > Subject, Quick Selection)
├── Refine edges (Select and Mask)
├── Place elements on separate layers
├── Match lighting and color
├── Apply layer masks for blending
└── Add atmospheric effects

Phase 5: Export
├── Convert to appropriate color space (sRGB for web)
├── Resize for destination (social, print, web)
├── Apply sharpening (Output sharpening)
├── Export with File > Export As
└── Save master file (PSD/TIFF)
```

### Digital Painting Workflow

```
Phase 1: Canvas Setup
├── Create document at appropriate resolution
├── Set up layers (sketch, lineart, color, shading)
├── Configure brush preferences
└── Set up reference panel

Phase 2: Sketching
├── Rough gesture sketch (low opacity)
├── Block in major shapes
├── Refine proportions
└── Create clean lineart layer

Phase 3: Base Coloring
├── Flat color blocking
├── Define light source
├── Base shadow and highlight colors
└── Establish color harmony

Phase 4: Rendering
├── Shading with soft brush
├── Build up form with value transitions
├── Add details and textures
├── Refine edges
└── Add highlights

Phase 5: Finishing
├── Add atmospheric effects
├── Apply color adjustments
├── Add texture overlays
├── Clean up layers
└── Export to final format
```

## 8.2 Asset Production Workflow

### Social Media Graphics

1. **Research**: Check platform specs (Instagram 1080x1080, 1080x1350, Stories 1080x1920)
2. **Create**: Set up artboard with proper dimensions
3. **Brand**: Apply brand colors, fonts, and logos
4. **Compose**: Arrange visual hierarchy elements
5. **Export**: Generate multiple sizes via Artboard export or Export As
6. **Optimize**: Compress for web, verify file sizes

### Product Mockups

1. **Source**: Find or photograph base product images
2. **Prepare**: Extract product using Select > Subject or Pen Tool
3. **Position**: Place artwork on smart object layer
4. **Transform**: Apply perspective and distortion to match product
5. **Polish**: Add shadows, highlights, texture to integrate
6. **Present**: Create presentation mockup with multiple angles

### UI/Screen Mockups

1. **Screen Capture**: Gather device frames or capture screens
2. **Extract**: Isolate screen area from device frame
3. **Artwork**: Create UI design on screen-sized canvas
4. **Composite**: Place UI onto screen layer with proper scaling
5. **Atmosphere**: Add glare, reflections, shadows
6. **Batch**: Create variants using layer comps or artboards

### Icon Sets

1. **Grid**: Set up consistent artboard grid (24x24, 32x32, 48x48)
2. **Style**: Choose stroke or filled style
3. **Outline**: Create master icon on pixel grid
4. **Scale**: Duplicate and scale for each size
5. **Refine**: Adjust for clarity at each size
6. **Export**: SVG export or slice for raster formats

## 8.3 Collaboration Workflow

### Client Review Workflow

```
Step 1: Preparation
├── Flatten visible for review (if needed)
├── Save version with client naming (ClientName_v01.psd)
├── Export review formats (PDF, JPEG high quality)
└── Organize layers with clear naming

Step 2: Sharing
├── Create PDF presentation or export for review
├── Use Adobe Creative Cloud sharing
├── Or export to shared folder with clear naming
└── Note specific areas for feedback

Step 3: Feedback Integration
├── Mark feedback areas with comments or notes
├── Create new layer for revisions
├── Version incrementally (v01, v02, v03)
└── Document changes in layer naming

Step 4: Approval
├── Get written approval (email confirmation)
├── Archive final approved version
└── Keep project folder for future reference
```

### Team Workflow with Libraries

1. **Shared Libraries**: Create team library for colors, brushes, graphics
2. **Component Libraries**: Store reusable elements as linked assets
3. **Version Control**: Use descriptive file names with dates (Project_YYYYMMDD)
4. **Layer Organization**: Use folders, color-coded layers, layer groups
5. **Smart Object Linking**: Keep frequently edited elements as linked Smart Objects
6. **Documentation**: Include README with project specs, fonts used, asset sources

### Handoff Workflow

1. **Audit**: Verify all layers named, no hidden surprises
2. **Organize**: Group layers logically (Background, Subject, Effects, Text)
3. **Flatten Styles**: Rasterize layer styles if handoff requires
4. **Collect**: Use File > Generate > Image Assets for exported layers
5. **Package**: Use File > Generate > Photoshop Droid to package all linked files
6. **Document**: Create spec sheet with dimensions, color codes, fonts
