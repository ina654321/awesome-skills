## 7.3 Common Tasks & Workflows

### Design Creation
1. **Choose Template or Start Blank** - Select from template gallery or create custom size
2. **Add Elements** - Drag from Elements panel, search library
3. **Upload Media** - Upload photos, videos, PDFs to your library
4. **Edit Elements** - Resize, rotate, recolor, adjust transparency
5. **Add Text** - Choose font, size, color, alignment
6. **Apply Brand Kit** - Apply brand colors, fonts, logos
7. **Export** - Download in desired format and quality

### Brand Kit Management
- **Colors**: Add up to 50 brand colors with hex codes
- **Fonts**: Upload custom fonts or use Canva's library (max 500 fonts)
- **Logos**: Store primary, secondary, and icon versions
- **Templates**: Create reusable templates with locked brand elements

### Bulk Create (Canva Pro)
1. **Create design** or open existing
2. **Connect to data**: Google Sheets, CSV, or manual entry
3. **Map fields**: Link text, images to design elements
4. **Preview**: Check sample data
5. **Generate**: Create all variations
6. **Download**: Batch export all versions

### Export Formats

| Format | Use Case | Options |
|--------|----------|---------|
| PNG | Images with transparency | Quality: Standard/Pro, Transparent BG |
| JPG | Photos, smaller file size | Quality slider (1-100) |
| PDF Print | High-quality print | Bleed, crop marks, CMYK |
| PDF Standard | Documents, sharing | Interactive or flat |
| MP4 | Video animations | 720p, 1080p, 4K |
| GIF | Animated graphics | Quality, loop settings |
| SVG | Vector graphics | Editable paths |

## 7.5 Performance Tips

### Large Team Management
- Organize designs in nested folders
- Use consistent naming conventions
- Set folder-level permissions
- Archive old projects to reduce clutter

### Asset Optimization
- Upload images at final display size
- Use PNG for graphics, JPG for photos
- Keep video files under 30 seconds for smooth editing
- Clear uploaded media library periodically

### Design Speed Tips
- Duplicate existing designs as starting points
- Use keyboard shortcuts for faster editing
- Apply consistent spacing with grid guides
- Lock frequently reused elements
- Save as template for recurring designs
