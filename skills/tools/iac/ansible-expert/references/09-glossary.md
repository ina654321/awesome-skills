# Glossary

## Core Terms

| Term | Definition |
|------|------------|
| **Playbook** | YAML file defining automation tasks |
| **Role** | Reusable collection of tasks |
| **Inventory** | List of hosts to manage |
| **Module** | Script executed on target |
| **Task** | Single action to perform |
| **Handler** | Task triggered by notification |

## Terms

| Term | Definition |
|------|------------|
| **Fact** | System information gathered |
| **Vault** | Encrypted variable storage |
| **Template** | File with variables |
| **Loop** | Repeat task with items |
| **When** | Conditional execution |
| **Include** | Import external file |

## Inventory Types

| Term | Definition |
|------|------------|
| **INI** | Simple host file |
| **YAML** | Structured inventory |
| **Dynamic** | Script-generated hosts |
