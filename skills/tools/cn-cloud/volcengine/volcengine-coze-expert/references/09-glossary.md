# Glossary

## 9.1 扣子核心概念

| 术语 | 定义 |
|------|------|
| 扣子/Coze | 字节跳动的AI应用平台 |
| Bot | AI机器人/智能体 |
| 技能 | Bot可调用的工具集合 |
| 工作流 | 可视化的流程编排 |

## 9.2 功能术语

| 术语 | 定义 |
|------|------|
| 知识库 | 存储文档用于RAG检索 |
| 变量 | Bot的上下文状态 |
| 插件 | 集成外部API的工具 |
| 开场白 | Bot首次对话的欢迎语 |
| 预置问题 | 快捷入口问题 |

## 9.3 发布术语

| 术语 | 定义 |
|------|------|
| Bot ID | Bot的唯一标识 |
| API Token | 调用API的认证令牌 |
| 渠道 | Bot发布的平台(豆包/飞书等) |
