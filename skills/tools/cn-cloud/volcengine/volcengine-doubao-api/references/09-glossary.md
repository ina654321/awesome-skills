# Glossary

## 9.1 豆包模型术语

| 术语 | 定义 |
|------|------|
| 豆包 | 字节跳动大模型系列 |
| doubao-pro | 高性能版模型 |
| doubao-lite | 轻量版，性价比高 |
| Ark | 火山引擎大模型服务平台 |

## 9.2 API术语

| 术语 | 定义 |
|------|------|
| AccessKey | 访问密钥ID |
| SecretKey | 访问密钥私钥 |
| Region | 服务区域(ap-beijing/ap-shanghai等) |
| Token | 文本处理单位 |

## 9.3 调用参数

| 术语 | 定义 |
|------|------|
| temperature | 控制生成随机性(0-1) |
| top_p | 核采样参数 |
| max_tokens | 最大生成Token数 |
| stream | 流式输出开关 |
