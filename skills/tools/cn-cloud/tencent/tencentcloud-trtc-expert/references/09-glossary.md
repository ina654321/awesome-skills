# Glossary

## 9.1 TRTC核心概念

| 术语 | 定义 |
|------|------|
| TRTC | Tencent Real-Time Communication，实时音视频 |
| RoomId | 房间ID，用户进入同一房间进行通话 |
| UserId | 用户ID，唯一标识参与者 |
| UserSig | 用户签名，用于身份验证 |

## 9.2 通话模式

| 术语 | 定义 |
|------|------|
| 视频通话 | 1v1或多人视频通话 |
| 互动直播 | 主播+观众连麦模式 |
| 语音通话 | 仅音频通话 |
| 屏幕分享 | 共享屏幕内容 |

## 9.3 质量指标

| 术语 | 定义 |
|------|------|
| 丢包率 | 传输过程中丢失的数据包比例 |
| 延迟 | 数据从发送到接收的时间 |
| FPS | 帧率，画面每秒更新次数 |
| 码率 | 视频数据传输速率 |
