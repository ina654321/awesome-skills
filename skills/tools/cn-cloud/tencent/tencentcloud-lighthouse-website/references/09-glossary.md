# Glossary

## 9.1 轻量服务器术语

| 术语 | 定义 |
|------|------|
| Lighthouse | 腾讯云轻量应用服务器 |
| 应用镜像 | 预装应用(如WordPress)的镜像 |
| 系统镜像 | 仅包含操作系统的镜像 |
| 防火墙 | 实例级别的访问控制 |

## 9.2 建站术语

| 术语 | 定义 |
|------|------|
| LNMP | Linux + Nginx + MySQL + PHP |
| LAMP | Linux + Apache + MySQL + PHP |
| 伪静态 | URL重写，隐藏后缀 |
| SSL证书 | HTTPS加密证书 |
