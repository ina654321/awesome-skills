# Glossary

## 9.1 云开发核心概念

| 术语 | 定义 |
|------|------|
| 云开发 | 腾讯云提供的小程序/Web后端服务 |
| 环境 | 云资源隔离的逻辑单元 |
| 数据库 | JSON格式的NoSQL文档数据库 |
| 云存储 | 文件存储服务(图片/音视频) |
| 云函数 | 服务端无状态函数 |

## 9.2 数据库术语

| 术语 | 定义 |
|------|------|
| 集合 | 类似关系型数据库的表 |
| 记录 | 类似关系型数据库的行 |
| _id | 记录唯一ID |
| _openid | 用户唯一标识 |
| 安全规则 | 基于用户身份的数据库访问控制 |

## 9.3 云函数术语

| 术语 | 定义 |
|------|------|
| entry | 云函数入口函数 |
| event | 触发云函数的事件数据 |
| context | 云函数运行上下文 |
| 定时触发 | 按Cron表达式定时执行 |

## 9.4 云存储术语

| 术语 | 定义 |
|------|------|
| fileID | 云存储文件的唯一标识 |
| tempFileURL | 临时访问链接 |
| cloudPath | 云存储中的文件路径 |
