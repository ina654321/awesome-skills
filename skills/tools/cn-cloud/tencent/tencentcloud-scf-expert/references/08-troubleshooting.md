# Troubleshooting

## 8.1 函数调用问题

### 8.1.1 函数超时

**排查:**
```bash
# 查看函数日志
# 腾讯云控制台 → 云函数 → 函数管理 → 日志查询
```

**解决方案:**
- 增加超时时间(最大900秒)
- 检查代码是否有死循环
- 优化冷启动

### 8.1.2 触发器不触发

**排查步骤:**
1. 确认触发器配置正确
2. 检查触发器是否启用
3. 验证事件源配置

```python
# COS触发器示例事件
event = {
    "Records": [{
        "cos": {
            "cosSchemaVersion": "1.0",
            "cosObject": {
                "key": "/input/file.jpg",
                "size": 1024
            }
        }
    }]
}
```

## 8.2 权限问题

### 8.2.1 无权访问COS等资源

**解决方案:**
```python
# 在函数配置中绑定角色
# 腾讯云控制台 → 云函数 → 函数管理 → 权限配置

# 角色需包含相关权限策略
# QcloudCOSFullAccess (COS读写权限)
# QcloudVPCReadOnlyAccess (VPC权限)
```

## 8.3 冷启动问题

### 8.3.1 首次调用延迟高

**原因:** 函数实例需要初始化

**解决方案:**
- 开启预置并发
- 精简依赖
- 使用更轻量的运行时
