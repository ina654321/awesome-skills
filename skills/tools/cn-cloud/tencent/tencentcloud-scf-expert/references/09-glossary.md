# Glossary

## 9.1 云函数术语

| 术语 | 定义 |
|------|------|
| SCF | Serverless Cloud Function，云函数 |
| 触发器 | 触发函数执行的事件源 |
| 事件 | 触发函数的数据负载 |
| 上下文 | 函数运行时的环境和信息 |

## 9.2 性能术语

| 术语 | 定义 |
|------|------|
| 冷启动 | 函数实例从创建到执行 |
| 热启动 | 已有实例执行函数 |
| 预置并发 | 提前启动实例避免冷启动 |
| 并发限制 | 同时处理的最大请求数 |

## 9.3 触发器类型

| 术语 | 定义 |
|------|------|
| API网关触发器 | HTTP请求触发 |
| COS触发器 | 对象存储事件触发 |
| 定时触发器 | Cron表达式触发 |
| CKafka触发器 | 消息队列触发 |
