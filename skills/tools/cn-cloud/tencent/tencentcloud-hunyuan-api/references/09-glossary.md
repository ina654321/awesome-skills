# Glossary

## 9.1 混元模型术语

| 术语 | 定义 |
|------|------|
| 混元 | 腾讯自研大模型系列 |
| hunyuan-standard | 标准版模型 |
| hunyuan-pro | 高性能版，128K上下文 |
| hunyuan-lite | 轻量版，性价比高 |
| hunyuan-vision | 多模态图文理解 |
