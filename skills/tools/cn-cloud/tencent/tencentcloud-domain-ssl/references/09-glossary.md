# Glossary

## 9.1 域名术语

| 术语 | 定义 |
|------|------|
| DNSPod | 腾讯云DNS解析服务 |
| 域名注册 | 向注册局申请域名的过程 |
| 实名认证 | 验证域名所有者身份 |
| WHOIS | 域名注册信息查询协议 |

## 9.2 DNS记录类型

| 术语 | 定义 |
|------|------|
| A记录 | 域名指向IPv4地址 |
| AAAA记录 | 域名指向IPv6地址 |
| CNAME | 域名别名指向 |
| MX记录 | 邮件交换服务器 |
| TXT记录 | SPF/DKIM等验证 |
| NS记录 | 域名服务器 |

## 9.3 SSL证书术语

| 术语 | 定义 |
|------|------|
| DV证书 | 仅验证域名所有权 |
| OV证书 | 验证组织身份 |
| EV证书 | 增强验证，绿色地址栏 |
| 证书链 | 根证书→中间证书→服务器证书 |
| CSR | 证书签名请求文件 |

## 9.4 安全术语

| 术语 | 定义 |
|------|------|
| HSTS | 强制HTTPS访问策略 |
| OCSP | 在线证书状态协议 |
| CAA | 证书颁发机构授权 |
