# Glossary

## 9.1 直播核心概念

| 术语 | 定义 |
|------|------|
| CSS | Cloud Streaming Services，云直播 |
| RTMP | Real-Time Messaging Protocol，推流协议 |
| HLS | HTTP Live Streaming，苹果的流媒体协议 |
| FLV | Flash Video，流媒体格式 |
| 推流 | 将视频流发送到直播服务器 |
| 拉流 | 从直播服务器获取视频流播放 |

## 9.2 直播相关

| 术语 | 定义 |
|------|------|
| 推流地址 | 用于发送视频流的地址 |
| 播放地址 | 用于接收视频流的地址 |
| AppName | 直播应用名，默认live |
| StreamName | 流名称，唯一标识一路直播 |
| 防盗链签名 | 基于时间戳的URL鉴权 |

## 9.3 编码相关

| 术语 | 定义 |
|------|------|
| 码率 | 视频数据传输速率(kbps) |
| 分辨率 | 视频画面尺寸 |
| 帧率 | 每秒画面数(fps) |
| GOP | Group of Pictures，关键帧间隔 |

## 9.4 录制相关

| 术语 | 定义 |
|------|------|
| HLS录制 | 生成.ts切片文件 |
| MP4录制 | 生成完整MP4文件 |
| 录制回调 | 录制完成后的通知 |
| 录制存储 | COS对象存储 |
