# Glossary

## 9.1 COS核心概念

| 术语 | 定义 |
|------|------|
| COS | Cloud Object Storage，对象存储服务 |
| Bucket | 存储桶，COS的文件容器 |
| Object | 对象，存储在COS中的文件 |
| Region | 地域，存储桶所在的数据中心位置 |
| APPID | 腾讯云账户ID，Bucket名称包含APPID |

## 9.2 存储类型

| 术语 | 定义 |
|------|------|
| 标准存储 | 高频访问场景 |
| 低频存储 | 较低访问频率(30天+) |
| 归档存储 | 长期归档(90天+) |
| 深度归档 | 超长期存储(180天+) |

## 9.3 访问控制

| 术语 | 定义 |
|------|------|
| 公有读私有写 | 所有人可读，Owner可写 |
| 私有读写 | 仅授权用户可读写 |
| 防盗链 | 基于Referer限制访问 |
| CORS | 跨域资源共享 |
| ACL | Access Control List，访问控制列表 |

## 9.4 SDK术语

| 术语 | 定义 |
|------|------|
| SecretId | API密钥ID |
| SecretKey | API密钥Key |
| 预签名URL | 带时效性的访问链接 |
| 分片上传 | 大文件分块上传 |
