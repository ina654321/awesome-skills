# Glossary

## 9.1 VOD核心概念

| 术语 | 定义 |
|------|------|
| VOD | Video on Demand，视频点播 |
| 转码 | 将视频转换为多分辨率/格式 |
| 超级播放器 | VOD提供的播放器SDK |
| fileId | 视频文件的唯一标识 |

## 9.2 视频处理

| 术语 | 定义 |
|------|------|
| 转码模板 | 定义输出视频的规格 |
| 截图 | 生成视频封面、，雪碧图 |
| 审核 | 智能识别违规内容 |
| 分类 | 按类别管理视频 |

## 9.3 防盗链

| 术语 | 定义 |
|------|------|
| Key防盗链 | 基于Key+过期时间的签名 |
| Referer防盗链 | 基于HTTP Referer |
| IP黑白名单 | 限制访问IP范围 |

## 9.4 播放器

| 术语 | 定义 |
|------|------|
| HLS | HTTP Live Streaming |
| DASH | Dynamic Adaptive Streaming |
| 自适应码率 | 根据网络自动切换画质 |
