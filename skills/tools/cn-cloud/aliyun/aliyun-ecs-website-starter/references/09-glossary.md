# Glossary

## 9.1 服务器术语

| 术语 | 定义 |
|------|------|
| ECS | Elastic Compute Service，弹性云服务器 |
| SSH | Secure Shell，远程登录协议 |
| VNC | Virtual Network Computing，图形化远程桌面 |
| 快照 | 磁盘在某一时刻的完整拷贝 |
| 镜像 | 服务器预装操作系统和软件 |
| 安全组 | 云服务器的虚拟防火墙 |

## 9.2 宝塔面板术语

| 术语 | 定义 |
|------|------|
| LNMP | Linux + Nginx + MySQL + PHP |
| LAMP | Linux + Apache + MySQL + PHP |
| phpMyAdmin | 基于Web的MySQL管理工具 |
| 计划任务 | 定时执行的备份/清理任务 |
| SSL证书 | 网站HTTPS加密证书 |

## 9.3 网站术语

| 术语 | 定义 |
|------|------|
| 虚拟主机 | 共享服务器资源的托管方式 |
| 伪静态 | 将动态URL伪装成静态URL |
| 固定链接 | WordPress的URL结构设置 |
| 子目录绑定 | 不同域名指向同一服务器不同目录 |
| 反向代理 | 代理请求到后端其他服务 |

## 9.4 WordPress术语

| 术语 | 定义 |
|------|------|
| 主题 | 控制网站外观的模板 |
| 插件 | 扩展WordPress功能的模块 |
| wp-content | WordPress内容目录(上传文件/插件/主题) |
| wp-admin | WordPress后台管理目录 |
| WP-CLI | WordPress命令行工具 |
