# Glossary

## 9.1 函数计算术语

| 术语 | 定义 |
|------|------|
| 函数 | 函数计算中的代码执行单元 |
| 触发器 | 触发函数执行的事件源 |
| 事件 | 触发函数的数据负载(JSON格式) |
| 上下文 | 函数运行时的环境和元信息 |
| 执行环境 | 函数运行的容器环境 |

## 9.2 性能术语

| 术语 | 定义 |
|------|------|
| 冷启动 | 函数实例从创建到执行的过程 |
| 热启动 | 已有实例执行函数 |
| 预置并发 | 提前启动函数实例避免冷启动 |
| 并发限制 | 函数同时处理的最大请求数 |
| 执行时长 | 函数实际运行的时间 |

## 9.3 触发器术语

| 术语 | 定义 |
|------|------|
| HTTP触发器 | 通过HTTP请求触发函数 |
| OSS触发器 | 文件上传/删除触发函数 |
| 定时触发器 | Cron表达式定时执行 |
| API网关触发器 | 通过API网关暴露函数 |
| MNS触发器 | 消息队列触发 |

## 9.4 计费术语

| 术语 | 定义 |
|------|------|
| 调用次数 | 函数被触发的总次数 |
| GB-秒 | 计费时长单位(内存GB × 执行秒数) |
| 付费调用 | 实际收费的调用(不含免费额度) |
| 函数实例 | 承载函数执行的容器 |
