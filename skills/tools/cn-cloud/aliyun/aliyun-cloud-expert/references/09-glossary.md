# Glossary

## 9.1 阿里云核心概念

| 术语 | 英文 | 定义 |
|------|------|------|
| ECS | Elastic Compute Service | 弹性计算服务，云服务器 |
| RDS | Relational Database Service | 关系型数据库服务 |
| OSS | Object Storage Service | 对象存储服务 |
| VPC | Virtual Private Cloud | 私有网络，逻辑隔离的网络环境 |
| SLB | Server Load Balancer | 负载均衡，分发流量 |
| CDN | Content Delivery Network | 内容分发网络 |
| ACK | Container Service for Kubernetes | 容器服务Kubernetes版 |
| NAS | Network Attached Storage | 文件存储 |
| API网关 | API Gateway | API托管服务 |
| ROS | Resource Orchestration Service | 资源编排 |

## 9.2 计费相关

| 术语 | 定义 |
|------|------|
| 按量付费 | 按实际使用量计费，按秒结算 |
| 包年包月 | 预付费方式，最低1个月 |
| 预留实例券 | 抵扣按量付费实例的账单 |
| 抢占式实例 | 低价获取闲置实例，可能被回收 |
| 资源组 | 按项目/部门管理云资源 |

## 9.3 网络相关

| 术语 | 定义 |
|------|------|
| 可用区 | 同一区域内物理隔离的数据中心 |
| 安全组 | 实例级别的虚拟防火墙 |
| 网络ACL | 子网级别的访问控制列表 |
| 弹性公网IP | 可绑定到任意ECS的固定公网IP |
| NAT网关 | 私网访问公网的网关服务 |
| 高速通道 | 跨地域私网互联服务 |

## 9.4 性能指标

| 术语 | 定义 |
|------|------|
| IOPS | Input/Output Operations Per Second，存储每秒读写次数 |
| 吞吐量 | 单位时间内数据传输量(MB/s) |
| 命中率 | 缓存命中的请求比例 |
| P99延迟 | 99%请求的响应时间上限 |
