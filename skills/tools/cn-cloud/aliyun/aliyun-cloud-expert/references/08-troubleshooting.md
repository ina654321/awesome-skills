# Troubleshooting

## 8.1 架构设计问题

### 8.1.1 ECS实例选型不当

**症状:** 网站响应慢、CPU持续100%、内存不足

**排查步骤:**
1. 登录控制台查看监控 → 选择实例 → 基础监控
2. 查看CPU/内存/带宽使用率
3. 确认实例规格与业务负载匹配

**解决方案:**
- CPU高 → 升级规格或使用抢占式实例+自动弹性
- 内存高 → 开启Swap或升级规格
- 带宽瓶颈 → 升级带宽或接入CDN

### 8.1.2 无法连接ECS实例

**排查步骤:**
1. 检查安全组是否放行对应端口
2. 确认实例状态是"运行中"
3. 测试本地网络到云服务器的连通性

```bash
# 测试端口连通性
telnet 1.2.3.4 22
nc -zv 1.2.3.4 80

# 查看安全组规则
# 控制台 → ECS → 安全组 → 入方向规则
```

## 8.2 网络问题

### 8.2.1 VPC内网不通

**症状:** 同VPC下两台ECS无法互通

**排查步骤:**
1. 确认两台ECS在同一VPC
2. 检查安全组入方向规则是否允许互访
3. 确认交换机ACL是否有限制

**解决方案:**
- 安全组添加入方向规则: 允许同安全组或IP段访问
- 检查NACL(网络ACL)规则

### 8.2.2 外网访问ECS缓慢

**原因:** 带宽不足、地域距离远、CDN未启用

**解决方案:**
- 接入CDN减轻源站压力
- 使用对象存储OSS存放静态资源
- 考虑使用全球加速

## 8.3 费用问题

### 8.3.1 账单异常

**常见原因:**
- 按量付费实例未及时释放
- 公网流量超出预期
- 快照存储费用累积

**排查:**
```bash
# 查看账单明细
# 控制台 → 费用中心 → 费用账单 → 账单明细
# 筛选: 产品名称 / 计费方式
```

**优化策略:**
- 开启自动快照策略并设置保留期限
- 及时释放测试环境按量实例
- 使用OSS生命周期规则降低存储成本

## 8.4 服务不可用

### 8.4.1 RDS连接失败

**排查步骤:**
1. 确认RDS白名单是否包含ECS IP
2. 检查连接地址是否正确
3. 验证账号密码

```bash
# 测试RDS连通性
mysql -h rm-xxxxx.mysql.rds.aliyuncs.com -P 3306 -u username -p
```

**解决方案:**
- 内网访问: 确保ECS与RDS在同一VPC，白名单添加100.64.0.0/10
- 外网访问: 开启RDS外网地址，限制IP访问
