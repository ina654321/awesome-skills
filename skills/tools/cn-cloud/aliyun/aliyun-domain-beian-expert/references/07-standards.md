### 7.2.2 实名认证材料

| 主体类型 | 所需材料 |
|---------|---------|
| 个人 | 身份证正反面照片、手持身份证照 |
| 企业 | 营业执照、法人身份证、授权书 |

### 7.2.3 实名认证流程

```
1. 购买域名 → 进入"域名控制台"
2. 实名认证 → 上传认证材料
3. 等待审核 → 1-3个工作日
4. 认证通过 → 可进行DNS解析
```

## 7.3 DNS解析配置

### 7.3.1 常用记录类型

| 记录类型 | 用途 | 示例值 |
|---------|------|--------|
| A | IPv4地址 | 1.2.3.4 |
| AAAA | IPv6地址 | 2001:db8::1 |
| CNAME | 域名别名 | www.yourdomain.com → web.yourdomain.com |
| MX | 邮件服务器 | 10 mail.yourdomain.com |
| TXT | SPF/DKIM验证 | v=spf1 include:_spf.yourdomain.com ~all |
| NS | nameserver | ns1.alidns.com |

### 7.3.2 TTL设置

| 场景 | 建议TTL | 说明 |
|------|--------|------|
| 正常解析 | 600秒(10分钟) | 平衡缓存和变更需求 |
| 变更频繁 | 300秒 | 快速生效 |
| 长期稳定 | 3600秒+ | 减少查询压力 |
