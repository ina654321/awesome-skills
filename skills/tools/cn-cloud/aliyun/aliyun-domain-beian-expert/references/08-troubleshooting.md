# Troubleshooting

## 8.1 域名问题

### 8.1.1 域名实名认证失败

**症状:** 实名认证审核不通过

**常见原因:**
- 身份证照片不清晰/模糊
- 营业执照信息与注册信息不一致
- 个人姓名与证件姓名不符

**解决方案:**
- 重新拍摄高清证件照片
- 企业用户确保营业执照最新且在有效期内
- 联系域名注册商客服确认具体驳回原因

### 8.1.2 DNS解析不生效

**排查步骤:**
1. 检查域名是否已完成实名认证
2. 确认DNS服务器是否已修改为阿里云DNS
3. 查看解析记录是否正确配置

```bash
# 检查DNS服务器
whois yourdomain.com | grep "Name Server"

# 检查解析是否生效
nslookup yourdomain.com 223.5.5.5
dig yourdomain.com @ns1.alidns.com

# 常见DNS: 
# 阿里云: ns1.alidns.com / ns2.alidns.com
# DNSPod: f1g1ns1.dnspod.net / f1g1ns2.dnspod.net
```

**TTL影响:** 修改DNS后，全球生效最长需要48小时

## 8.2 备案问题

### 8.2.1 备案被退回

**常见退回原因:**
- 网站名称不符合规范(如包含"最"、"第一"等夸大词)
- 负责人手机号无法接通或非本人
- 资料不清晰或缺失

**解决方案:**
- 登录备案系统查看具体退回原因
- 按要求修改后重新提交
- 提前准备: 幕布照、身份证原件、域名证书

### 8.2.2 备案服务号获取失败

**原因:**
- ECS实例未付费购买(试用不行)
- 实例已过期
- 该实例已生成过备案服务号

**解决方案:**
```bash
# 确认实例状态
# 控制台 → ECS → 实例 → 查看实例状态和计费方式
# 确保是预付费(包年包月)或已购买按量付费实例
# 备案服务号在ECS控制台 → 实例详情 → 备案服务号
```

## 8.3 SSL证书问题

### 8.3.1 证书申请验证失败

**排查步骤:**
1. 确认域名已在阿里云注册
2. 检查DNS验证记录是否已添加
3. 验证记录值是否与CA提供的一致

```bash
# 查看TXT记录
nslookup -qt=TXT _dnsauth.yourdomain.com 223.5.5.5
```

**解决方案:**
- 手动添加DNS验证TXT记录
- 等待5-10分钟后重新验证
- 文件验证: 下载验证文件放到域名对应路径
