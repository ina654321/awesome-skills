# Glossary

## 9.1 OSS核心概念

| 术语 | 定义 |
|------|------|
| Bucket | 存储桶，OSS的顶层容器，类似"根目录" |
| Object | 对象，存储在Bucket中的文件 |
| Endpoint | 访问域名，不同区域有不同Endpoint |
| Region | 地域，OSS数据中心所在区域 |

## 9.2 存储相关

| 术语 | 定义 |
|------|------|
| 标准存储 | 高频访问，毫秒级响应 |
| 低频访问存储 | 较低频(30天未访问)，价格更低 |
| 归档存储 | 长期存档(90天+)，需要解冻 |
| 冷归档存储 | 超长期(180天+)，解冻需数小时 |

## 9.3 权限相关

| 术语 | 定义 |
|------|------|
| ACL | Access Control List，访问控制列表 |
| 私有Bucket | 仅Owner有权限访问 |
| 公共读Bucket | 任何人可读，Owner可写 |
| 防盗链 | 基于Referer限制访问来源 |
| CORS | 跨域资源共享配置 |
| RAM | 资源访问管理，细粒度权限控制 |

## 9.4 访问相关

| 术语 | 定义 |
|------|------|
| 签名URL | 带时效性和权限的访问链接 |
| STS | Security Token Service，临时凭证服务 |
| OSSFS | OSS File System，基于FUSE的挂载工具 |
| 跨区域复制 | 自动同步Bucket数据到其他区域 |

## 9.5 计费相关

| 术语 | 定义 |
|------|------|
| 存储量费用 | 按存储大小计费 |
| 流量费用 | 外网出流量计费 |
| 请求费用 | PUT/GET等请求次数计费 |
| 生命周期 | 自动转换存储类型/删除的规则 |
