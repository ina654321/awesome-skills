# Glossary

## 9.1 CDN核心概念

| 术语 | 定义 |
|------|------|
| CDN | Content Delivery Network，内容分发网络，通过边缘节点加速内容分发 |
| 边缘节点 | 部署在各地的缓存服务器，直接响应用户请求 |
| 回源 | 当缓存未命中时，CDN节点向源站请求资源的行为 |
| 命中率 | 缓存命中的请求占总请求的比例 |
| 回源率 | 需要回源的请求占总请求的比例 |

## 9.2 域名解析相关

| 术语 | 定义 |
|------|------|
| CNAME | 域名别名记录，将域名指向另一个域名 |
| DNS | 域名系统，将域名解析为IP地址 |
| 解析生效时间 | DNS记录变更后全球生效所需时间(通常5分钟-48小时) |
| TTL | DNS记录的缓存时间，影响解析变更生效速度 |

## 9.3 缓存相关

| 术语 | 定义 |
|------|------|
| Cache-Control | HTTP缓存控制头，max-age指定缓存时长 |
| Expires | 资源过期时间(GMT格式) |
| ETag | 资源版本标识，用于缓存校验 |
| Last-Modified | 资源最后修改时间 |
| Vary | 决定缓存key的请求头，如Vary: Accept-Encoding |
| no-cache | 每次请求都向源站验证缓存有效性 |
| no-store | 完全不缓存 |
| private | 只允许浏览器缓存，不允许CDN缓存 |

## 9.4 性能指标

| 术语 | 定义 |
|------|------|
| TTFB | Time To First Byte，首字节时间 |
| 首屏时间 | 页面可见内容加载完成的时间 |
| 带宽峰值 | 单位时间内最大数据传输量 |
| QPS | Queries Per Second，每秒请求数 |

## 9.5 安全相关

| 术语 | 定义 |
|------|------|
| Referer防盗链 | 基于HTTP Referer头限制访问来源 |
| IP黑名单 | 禁止特定IP/段访问 |
| HTTPS回源 | CDN节点用HTTPS协议向源站请求内容 |
| 证书链 | 从服务器证书到根证书的完整信任链 |
