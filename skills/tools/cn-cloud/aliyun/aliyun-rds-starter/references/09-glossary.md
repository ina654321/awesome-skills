# Glossary

## 9.1 RDS核心概念

| 术语 | 定义 |
|------|------|
| RDS | Relational Database Service，关系型数据库服务 |
| 实例 | 数据库运行的服务器，一个实例可包含多个数据库 |
| 数据库 | 逻辑上的数据集合 |
| 表空间 | 数据存储的逻辑区域 |
| Binlog | MySQL二进制日志，记录所有DDL和DML |

## 9.2 高可用相关

| 术语 | 定义 |
|------|------|
| 主从复制 | 主库写入，从库同步读取 |
| 高可用版 | 双机热备，自动故障切换 |
| 集群版 | 一主多从，读写分离 |
| 独享规格 | 独享CPU和内存资源 |

## 9.3 备份恢复相关

| 术语 | 定义 |
|------|------|
| 自动备份 | RDS按策略自动备份 |
| 手动备份 | 用户主动创建的备份 |
| 数据备份 | 全量数据备份 |
| 日志备份 | Binlog日志备份，支持按时间点恢复 |
| PITR | Point In Time Recovery，基于时间点的恢复 |

## 9.4 性能相关

| 术语 | 定义 |
|------|------|
| IOPS | 存储每秒读写次数 |
| 连接数 | 同时连接到数据库的客户端数量 |
| max_connections | 最大允许连接数 |
| slow_query_log | 慢查询日志 |
| EXPLAIN | SQL执行计划分析 |

## 9.5 安全相关

| 术语 | 定义 |
|------|------|
| 白名单 | 允许访问RDS的IP列表 |
| SSL加密 | 数据库连接加密 |
| SQL审计 | 记录所有SQL执行日志 |
| 读写权限 | SELECT/INSERT/UPDATE/DELETE权限 |
