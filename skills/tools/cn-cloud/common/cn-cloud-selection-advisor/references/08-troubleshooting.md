# Troubleshooting

## 8.1 服务不可用问题

### 8.1.1 阿里云服务被限制

**症状:** 无法访问阿里云控制台或特定服务

**常见原因:**
- 账号欠费
- 安全合规审查
- 操作违规

**解决方案:**
- 登录阿里云检查账户余额
- 查看消息中心通知
- 联系客服了解详情

### 8.1.2 腾讯云微信相关服务无法接入

**症状:** 微信小程序无法调用腾讯云服务

**排查:**
- 确认已在微信公众平台配置服务器域名
- 检查微信AppID是否正确配置
- 确认云开发环境ID正确

## 8.2 价格问题

### 8.2.1 账单超出预算

**优化策略:**
- 使用按量付费测试，确认需求后再包年
- 开启消费预警
- 使用成本优化工具
- 及时释放闲置资源

## 8.3 迁移问题

### 8.3.1 从阿里云迁出

**步骤:**
1. 导出所有数据(OSS、RDS等)
2. 在新平台创建对应服务
3. 配置新平台DNS
4. 逐步切换流量
5. 确认无误后取消阿里云服务

### 8.3.2 跨平台迁移注意事项

- 快照/镜像可能不通用
- 数据库迁移需注意字符集差异
- CDN需要重新配置域名
- SSL证书需要重新申请
