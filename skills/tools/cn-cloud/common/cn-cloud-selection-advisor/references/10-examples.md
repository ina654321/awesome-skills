# Examples

## 10.1 初创企业选型

```
推荐: 火山引擎(成本优先) 或 腾讯云(新用户优惠)

入门配置:
- 火山引擎轻量应用服务器: ¥30/月 (2核2G)
- TOS对象存储: ¥5/月 (100GB)
- CDN: ¥0.18/GB

或:
- 腾讯云轻量应用服务器: ¥30/年 (首年特惠)
- COS对象存储: ¥9.9/月
```

## 10.2 电商网站选型

```
推荐: 阿里云(电商生态成熟)

典型架构:
- ECS × 2 (Web服务): ¥240/月 × 2
- RDS MySQL: ¥200/月
- Redis: ¥150/月
- OSS + CDN: ¥50/月
- SLB负载均衡: ¥30/月
总预算: ~¥920/月
```

## 10.4 AI应用开发

```
推荐: 火山引擎(豆包API性价比高)

配置:
- 云服务器: ¥100/月
- 豆包API调用: ¥50/月(基础用量)
- OSS/TOS存储: ¥10/月
- CDN: ¥20/月

对比:
- 阿里云通义千问: ¥0.004/千token
- 火山引擎豆包: ¥0.0008/千token
- 火山引擎成本低约80%
```

## 10.5 多云策略

```
推荐组合: 阿里云(核心业务) + 腾讯云(微信相关)

架构示例:
核心业务(阿里云):
  - ECS + RDS + OSS
  - VPC内网通信

微信业务(腾讯云):
  - 小程序云开发
  - COS + CDN

数据同步:
  - 使用API网关做数据桥接
  - 异步同步关键数据
```
