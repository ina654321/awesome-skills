# Glossary

## 9.1 云服务术语

| 术语 | 定义 |
|------|------|
| IaaS | Infrastructure as a Service，基础设施即服务 |
| PaaS | Platform as a Service，平台即服务 |
| SaaS | Software as a Service，软件即服务 |
| BaaS | Backend as a Service，后端即服务 |
| FaaS | Function as a Service，函数即服务 |

## 9.2 计费模式

| 术语 | 定义 |
|------|------|
| 按量付费 | 用多少付多少，按秒/小时计费 |
| 包年包月 | 预付费，价格优惠 |
| 预留实例 | 承诺使用换取折扣 |
| 抢占式实例 | 低价竞拍，可能被回收 |
| 资源组 | 按项目/部门管理资源的逻辑容器 |

## 9.3 性能术语

| 术语 | 定义 |
|------|------|
| 可用区 | 同一区域内物理隔离的数据中心 |
| Region | 地理区域，如华东1、华北2 |
| 弹性伸缩 | 根据负载自动增减资源 |
| 负载均衡 | 分发流量到多台服务器 |
| CDN | 内容分发网络，加速分发 |

## 9.4 安全术语

| 术语 | 定义 |
|------|------|
| 安全组 | 实例级别的虚拟防火墙 |
| IAM | Identity and Access Management，身份和访问管理 |
| VPC | Virtual Private Cloud，私有网络 |
| 加密服务 | 密钥管理/证书管理服务 |
