## 7.2 备案类型

### 7.2.1 备案分类

| 类型 | 适用场景 | 所需时间 |
|------|---------|---------|
| 首次备案 | 新域名+新服务器 | 15-30天 |
| 新增网站 | 已有备案，新增域名 | 10-20天 |
| 接入备案 | 域名在其他平台接入当前服务器 | 7-15天 |
| 变更备案 | 修改备案主体/网站信息 | 5-10天 |
| 注销备案 | 域名不再使用 | 3-5天 |

### 7.2.2 前置审批

| 许可证类型 | 说明 | 审批部门 |
|-----------|------|---------|
| 经营性ICP许可证 | 有偿服务 | 通信管理局 |
| 新闻许可证 | 新闻发布 | 网信办 |
| 出版许可证 | 图书/音像 | 新闻出版署 |
| 药品信息服务许可证 | 药品信息 | 药监局 |
| 医疗器械许可证 | 医疗器械 | 药监局 |

## 7.3 材料清单

### 7.3.1 个人备案材料

| 材料 | 要求 |
|------|------|
| 身份证正反面 | 电子版，清晰可见 |
| 域名证书 | 域名注册商处下载 |
| 网站真实性核验单 | 在线填写或下载签字 |
| 幕布照 | 特定背景幕布拍照 |

### 7.3.2 企业备案材料

| 材料 | 要求 |
|------|------|
| 营业执照副本 | 加盖公章的电子版 |
| 法人身份证 | 正反面 |
| 网站负责人身份证 | 可与法人不同 |
| 域名证书 | 加盖公章 |
| 真实性核验单 | 加盖公章 |
| 授权书 | 负责人非法人时需要 |

## 7.4 域名要求

| 要求 | 说明 |
|------|------|
| 域名后缀 | 需在工信部已备案后缀列表中 |
| 实名认证 | 域名需在申请者名下实名认证满3天 |
| 注册商 | 需为国内注册商 |
| 有效期 | 域名需在有效期内(建议>3个月) |

## 7.5 审核流程

```
提交资料 → 阿里云审核(1-3天) → 短信核验(24小时) → 管局审核(10-20天) → 备案成功
```

| 省份 | 管局的通常审核时间 |
|------|-------------------|
| 北京 | 15-20个工作日 |
| 上海 | 10-15个工作日 |
| 广东 | 7-15个工作日 |
| 浙江 | 10-15个工作日 |
