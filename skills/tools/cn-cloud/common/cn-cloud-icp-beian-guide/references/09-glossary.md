# Glossary

## 9.1 备案核心概念

| 术语 | 定义 |
|------|------|
| ICP备案 | Internet Content Provider备案，互联网内容提供者备案 |
| 备案主体 | 备案申请人(个人或企业) |
| 网站负责人 | 具体负责网站管理的人员 |
| 接入商 | 提供服务器接入的服务商(阿里云/腾讯云等) |
| 管局 | 通信管理局，各省通信管理局 |
| 备案号 | 备案成功后的唯一编号 |

## 9.2 备案类型

| 术语 | 定义 |
|------|------|
| 首次备案 | 新域名+新服务器的首次备案 |
| 新增网站 | 已有备案，新增网站/域名 |
| 接入备案 | 将域名从其他接入商转入当前接入商 |
| 变更备案 | 修改备案主体或网站信息 |
| 注销备案 | 主动注销不再使用的备案 |

## 9.3 材料术语

| 术语 | 定义 |
|------|------|
| 真实性核验单 | 签字盖章的纸质/电子核验材料 |
| 域名证书 | 证明域名注册信息的文件 |
| 授权书 | 非法人为网站负责人时的授权文件 |
| 幕布照 | 带特定背景的半身照 |

## 9.4 审批相关

| 术语 | 定义 |
|------|------|
| 前置审批 | 部分行业需先获得特定许可证才能备案 |
| 经营性ICP | 有偿提供信息服务的许可证 |
| 短信核验 | 备案信息真实性手机短信验证 |
| 公安联网备案 | 取得ICP备案号后，还需到公安网监备案 |
