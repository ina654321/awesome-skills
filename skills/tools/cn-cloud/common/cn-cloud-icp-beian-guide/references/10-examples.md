# Examples

## 10.1 首次备案流程

```
第一步: 准备工作
1. 完成阿里云账号实名认证
2. 购买符合备案条件的服务器(包年包月)
3. 获取备案服务号
4. 准备材料: 身份证正反面照片、手持身份证照

第二步: 提交备案
1. 登录阿里云备案系统
2. 选择备案类型: 首次备案
3. 填写主体信息
4. 填写网站信息
5. 上传资料照片
6. 提交初审

第三步: 短信核验
1. 收到短信验证码
2. 访问工信部短信核验入口: beian.miit.gov.cn
3. 输入验证码完成核验

第四步: 管局审核
1. 阿里云初审: 1-3天
2. 管局审核: 10-20天(各省不同)
3. 备案号下发
```

## 10.2 企业备案材料清单

```markdown
企业备案所需材料:
1. 营业执照副本(加盖公章)
2. 法人身份证正反面
3. 网站负责人身份证正反面
4. 真实性核验单(加盖公章)
5. 信息安全承诺书(部分省份需要)
6. 域名证书
7. 前置审批文件(如有)
```

## 10.3 接入备案流程

```
适用场景: 域名已在其他平台备案，现在要迁移到阿里云

步骤1: 确保域名原备案还在有效期
步骤2: 在阿里云购买服务器并获取备案服务号
步骤3: 在阿里云备案系统选择"接入备案"
步骤4: 填写原备案信息(备案号、主体信息)
步骤5: 上传阿里云服务器的验证材料
步骤6: 提交审核
注意: 原备案不会被注销，只是增加新的接入商
```

## 10.4 变更备案信息

```markdown
可变更内容:
- 企业名称/法人变更 → 需要重新实名认证
- 网站负责人变更 → 上传新负责人资料
- 网站名称变更 → 符合规范即可
- 联系方式变更 → 填写新手机号

不可变更内容:
- 备案类型(个人→企业需重新备案)
- 备案省份(需注销后重新备案)
```
