# Glossary

## 9.1 宝塔核心术语

| 术语 | 定义 |
|------|------|
| 面板 | 宝塔提供的Web管理界面 |
| 一键部署 | 快速部署常用应用(WordPress等) |
| 伪静态 | URL重写规则，隐藏.php等后缀 |
| PHP-FPM | PHP FastCGI进程管理器 |
| 计划任务 | 定时执行的备份/清理任务 |

## 9.2 Web服务器术语

| 术语 | 定义 |
|------|------|
| Nginx | 高性能Web服务器和反向代理服务器 |
| Apache | 老牌Web服务器，.htaccess配置 |
| 反向代理 | 代理客户端请求到后端服务 |
| 负载均衡 | 分发请求到多台服务器 |
| 证书链 | 根证书+中间证书+服务器证书 |

## 9.3 网站管理术语

| 术语 | 定义 |
|------|------|
| 子目录绑定 | 多个域名指向不同子目录 |
| 默认站点 | 未匹配域名时的默认站点 |
| 网站目录 | 网站文件存放的根目录 |
| 运行目录 | PHP等脚本的执行目录 |
| 防御CC | 抗CC攻击设置 |
