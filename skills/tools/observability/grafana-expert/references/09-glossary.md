# Glossary

## Grafana Concepts

| Term | Definition |
|------|------------|
| **Dashboard** | Collection of panels |
| **Panel** | Visualization component |
| **Datasource** | Data source connection |
| **Query** | Data request |
| **Variable** | Dynamic value |
| **Annotation** | Event markers |

## Alerting

| Term | Definition |
|------|------------|
| **Alert Rule** | Condition to trigger |
| **Notification Channel** | Alert destination |
| **Alert Manager** | Handles alerts |
| **Silences** | Mute alerts |

## Visualization Types

| Term | Definition |
|------|------------|
| **Graph** | Time series |
| **Stat** | Single value |
| **Table** | Tabular data |
| **Heatmap** | Density |
| **Geo Map** | Geographic |
