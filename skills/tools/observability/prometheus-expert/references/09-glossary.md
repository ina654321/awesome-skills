# Glossary

## Core Terms

| Term | Definition |
|------|------------|
| **Metric** | Time series data point |
| **Label** | Key-value pair |
| **Sample** | Metric value at timestamp |
| **Scrape** | Pull metrics |
| **Target** | Metric source |

## Prometheus

| Term | Definition |
|------|------------|
| **PromQL** | Prometheus Query Language |
| **Recording Rule** | Pre-computed query |
| **Alert Rule** | Condition for alert |
| **Exporter** | Metrics collector |
| **Node Exporter** | System metrics |

## Alerting

| Term | Definition |
|------|------------|
| **Alertmanager** | Alert routing |
| **Receiver** | Alert destination |
| **Inhibition** | Suppress related alerts |
| **Silences** | Mute alerts |
