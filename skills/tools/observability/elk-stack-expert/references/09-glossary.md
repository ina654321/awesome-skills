# Glossary

## Elasticsearch

| Term | Definition |
|------|------------|
| **Index** | Collection of documents |
| **Shard** | Index partition |
| **Replica** | Shard copy |
| **Cluster** | Group of nodes |
| **Node** | Elasticsearch server |
| **Document** | JSON record |

## Logstash

| Term | Definition |
|------|------------|
| **Input** | Data source |
| **Filter** | Data transformation |
| **Output** | Data destination |
| **Pipeline** | Complete processing |

## Kibana

| Term | Definition |
|------|------------|
| **Discover** | Log exploration |
| **Visualize** | Chart builder |
| **Dashboard** | Visualizations collection |
| **Dev Tools** | Query console |

## Beats

| Term | Definition |
|------|------------|
| **Filebeat** | Log files |
| **Metricbeat** | Metrics |
| **Heartbeat** | Availability |
| **Packetbeat** | Network |
