# Troubleshooting Guide

## 8.1 API Issues

### 401 Unauthorized
- Check API token valid
- Verify base URL correct
- Check authentication method

### Rate Limiting
- Check rate limits
- Use pagination
- Add retry logic

## 8.2 Page Creation Issues

### Page Not Created
- Verify space key
- Check page title unique
- Review content format

## 8.3 Search Issues

### Search Not Working
- Re-index content
- Check search permissions
- Verify CQL syntax
