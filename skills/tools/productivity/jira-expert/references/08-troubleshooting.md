# Troubleshooting Guide

## 8.1 Authentication

### 401 Error
- Check API token
- Verify email/user correct
- Check permissions

## 8.2 JQL Issues

### No Results
- Check field names correct
- Verify operators valid
- Review syntax

## 8.3 Webhooks

### Webhook Not Firing
- Verify URL correct
- Check webhook enabled
- Review event filters
