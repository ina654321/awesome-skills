# Glossary

## Spreadsheet Core Terms

| Term | Definition |
|------|------------|
| **Cell** | Intersection of row and column |
| **Range** | Selection of cells (A1:B10) |
| **Sheet** | Individual tab within workbook |
| **Workbook** | Entire spreadsheet file |
| **Formula** | Expression that calculates values |
| **Function** | Pre-built formula (SUM, VLOOKUP) |
| **Reference** | Pointer to cell/range (A1, Sheet2!B2) |

## Excel-Specific Terms

| Term | Definition |
|------|------------|
| **Pivot Table** | Dynamic data summarization tool |
| **Power Query** | Data connection and transformation |
| **Power Pivot** | Data modeling with DAX |
| **Named Range** | Defined name for cell/range |
| **Table** | Structured data range with headers |
| **Macro** | VBA automated actions |
| **Conditional Formatting** | Dynamic cell styling |
| **Data Validation** | Input restrictions |

## Google Sheets-Specific Terms

| Term | Definition |
|------|------------|
| **IMPEXTRANGE** | Import from another spreadsheet |
| **QUERY** | SQL-like data manipulation |
| **ARRAYFORMULA** | Apply formula to array |
| **Named Functions** | Custom reusable functions |
| **Connected Sheets** | BigQuery integration |
| **Apps Script** | JavaScript automation |
| **Add-ons** | Third-party extensions |

## Function Categories

### Lookup & Reference
| Function | Purpose |
|----------|---------|
| **VLOOKUP** | Vertical lookup (legacy) |
| **HLOOKUP** | Horizontal lookup (legacy) |
| **XLOOKUP** | Modern flexible lookup |
| **INDEX/MATCH** | Advanced lookup combo |
| **MATCH** | Find position in range |
| **CHOOSE** | Select from list |
| **INDIRECT** | Reference from text |

### Math & Trig
| Function | Purpose |
|----------|---------|
| **SUM** | Add values |
| **AVERAGE** | Calculate mean |
| **COUNT/COUNTA** | Count cells |
| **MAX/MIN** | Find extremes |
| **ROUND** | Round to digits |
| **SUMPRODUCT** | Sum of products |
| **ABS** | Absolute value |

### Conditional
| Function | Purpose |
|----------|---------|
| **IF** | Single condition |
| **IFS** | Multiple conditions |
| **IFERROR** | Error handling |
| **SWITCH** | Multiple matches |
| **AND/OR/NOT** | Logical operations |

### Text Functions
| Function | Purpose |
|----------|---------|
| **CONCATENATE/CONCAT** | Join text |
| **TEXTJOIN** | Join with delimiter |
| **LEFT/RIGHT/MID** | Extract text |
| **LEN** | Text length |
| **TRIM** | Remove spaces |
| **SUBSTITUTE** | Replace text |
| **TEXT** | Format as text |

### Date & Time
| Function | Purpose |
|----------|---------|
| **TODAY/NOW** | Current date/time |
| **DATE** | Create date |
| **DATEDIF** | Date difference |
| **YEAR/MONTH/DAY** | Extract parts |
| **WORKDAY** | Business days |
| **NETWORKDAYS** | Count business days |
