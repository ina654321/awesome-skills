# Glossary

## Azure Terminology

### A

| Term | Definition |
|------|------------|
| **ARM** | Azure Resource Manager — Deployment and management layer |
| **ARM Template** | JSON template for infrastructure as code |
| **ASR** | Azure Site Recovery — Disaster recovery service |
| **Availability Set** | Logical grouping for VM redundancy |
| **Availability Zone** | Physically isolated data centers in a region |

### B

| Term | Definition |
|------|------------|
| **Blob Storage** | Object storage for unstructured data |
| **Bicep** | DSL for ARM templates, cleaner syntax |
| **BYOK** | Bring Your Own Key — Customer-managed encryption |

### C

| Term | Definition |
|------|------------|
| **Cognitive Services** | Azure AI/ML APIs |
| **Cosmos DB** | Globally distributed, multi-model database |
| **CSM** | Cloud Shell — Browser-based shell |

### D

| Term | Definition |
|------|------------|
| **DDoS Protection** | Distributed denial of service protection |
| **DevOps** | Development + Operations practices |
| **DNS** | Domain Name System — Domain resolution |
| **DR** | Disaster Recovery |
| **DSC** | Desired State Configuration |

### E

| Term | Definition |
|------|------------|
| **EA** | Enterprise Agreement — Azure billing agreement |
| **EB** | Event Grid — Event routing service |
| **EH** | Event Hubs — Big data streaming platform |
| **EIP** | Elastic IP — Static public IP |
| **EIA** | ExpressRoute — Dedicated private connection |
| **ESI** | ExpressRoute — Direct connection to Azure |

### F

| Term | Definition |
|------|------------|
| **FA** | Functions — Serverless compute |
| **Firewall** | Azure Firewall — Managed network security |

### G

| Term | Definition |
|------|------------|
| **Graph** | Azure AD Graph API |
| **Guard** | Azure Bastion — Secure VM access |

### H

| Term | Definition |
|------|------------|
| **HB** | Health Bot — AI-powered health assistant |
| **HBI** | High Business Impact — Sensitive data tier |

### I

| Term | Definition |
|------|------------|
| **IaaS** | Infrastructure as a Service — VMs, networking |
| **Identity** | Azure AD / Entra ID |
| **IoT Hub** | IoT device management |
| **IR** | Instance Run — Single VM billing |
| **Icing** | Infrastructure as code |

### J

| Term | Definition |
|------|------------|
| **Jumpbox** | Bastion host for accessing private VMs |

### K

| Term | Definition |
|------|------------|
| **Key Vault** | Managed HSM — Secrets, keys, certificates |
| **KEDA** | Kubernetes Event-driven Autoscaling |

### L

| Term | Definition |
|------|------------|
| **LB** | Load Balancer — Traffic distribution |
| **Log Analytics** | Log aggregation and querying |

### M

| Term | Definition |
|------|------------|
| **MI** | Managed Identity — Azure AD identity for resources |
| **MFA** | Multi-Factor Authentication |
| **MSI** | Managed Service Identity — See MI |
| **MSSQL** | Azure SQL Database — Managed SQL Server |

### N

| Term | Definition |
|------|------------|
| **NAT GW** | NAT Gateway — Outbound internet for private subnets |
| **Network Watcher** | Network diagnostics and monitoring |
| **Nsg** | Network Security Group — Firewall rules |

### O

| Term | Definition |
|------|------------|
| **OMS** | Operations Management Suite — Monitoring (now LA) |

### P

| Term | Definition |
|------|------------|
| **PaaS** | Platform as a Service — Managed platforms |
| **PIM** | Privileged Identity Management — Just-in-time access |
| **Policy** | Azure Policy — Compliance enforcement |
| **Private Link** | Private endpoint for PaaS services |
| **Public IP** | Internet-accessible IP address |

### R

| Term | Definition |
|------|------------|
| **RBAC** | Role-Based Access Control |
| **Resource Group** | Logical container for resources |
| **RI** | Reserved Instance — Capacity reservation |
| **RM** | Resource Manager — See ARM |

### S

| Term | Definition |
|------|------------|
| **SA** | Storage Account — Container for Azure Storage |
| **S2S** | Site-to-Site — VPN connection type |
| **SAS** | Shared Access Signature — Delegated access token |
| **Sentinel** | Azure Sentinel — SIEM service |
| **SKU** | Stock Keeping Unit — Service tier |
| **SLA** | Service Level Agreement — Uptime guarantee |
| **SQL MI** | SQL Managed Instance — Managed SQL Server |
| **SSIS** | SQL Server Integration Services |

### T

| Term | Definition |
|------|------------|
| **Tags** | Metadata for resource organization |
| **Template** | ARM Template — IaC definition |

### U

| Term | Definition |
|------|------------|
| **UD** | User Defined — Custom routing |
| **UDN** | User Defined Network — Custom route table |

### V

| Term | Definition |
|------|------------|
| **VNet** | Virtual Network — Isolated Azure network |
| **VN** | Virtual Network — See VNet |
| **VPN GW** | VPN Gateway — Site-to-site VPN |
| **VCN** | Virtual Cloud Network — (Same as VNet) |

### W

| Term | Definition |
|------|------------|
| **WAF** | Web Application Firewall |
| **WVD** | Windows Virtual Desktop — Now AVD |
| **AVD** | Azure Virtual Desktop — Desktop virtualization |

### Z

| Term | Definition |
|------|------------|
| **Zone** | DNS zone — Domain hosting |
| **ZRS** | Zone-Redundant Storage — 3 AZs |
| **GZRS** | Geo-Zone-Redundant Storage — Cross-region |
