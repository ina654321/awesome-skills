# Glossary

## Azure Services

| Term | Definition |
|------|------------|
| **Azure Resource Manager (ARM)** | Deployment and management service for Azure resources |
| **Resource Group** | Logical container for Azure resources |
| **Subscription** | Billing unit for Azure services |
| **Tenant** | Azure AD organization instance |
| **Management Group** | Hierarchical organization above subscriptions |

## Identity & Access

| Term | Definition |
|------|------------|
| **Azure AD** | Microsoft Entra ID - cloud-based identity service |
| **Service Principal** | Application identity for automation |
| **Managed Identity** | Auto-managed identity for Azure resources |
| **RBAC** | Role-Based Access Control |
| **Azure AD B2B** | Business-to-business identity collaboration |
| **Azure AD B2C** | Consumer identity management |
| **Conditional Access** | Policy-based access control |
| **MFA** | Multi-Factor Authentication |
| **PIM** | Privileged Identity Management |

## Compute

| Term | Definition |
|------|------------|
| **VM Scale Set** | Set of identical VMs for scaling |
| **Azure Kubernetes Service (AKS)** | Managed Kubernetes service |
| **Azure Functions** | Serverless compute service |
| **Azure Container Instances (ACI)** | Container hosting without infrastructure |
| **App Service** | PaaS for web apps and APIs |

## Networking

| Term | Definition |
|------|------------|
| **Virtual Network (VNet)** | Isolated network in Azure |
| **VNet Peering** | Connect two Vnets privately |
| **ExpressRoute** | Dedicated private connectivity |
| **VPN Gateway** | VPN connection to Azure |
| **Azure Load Balancer** | Layer 4 load balancing |
| **Application Gateway** | Layer 7 load balancing with WAF |
| **Azure Front Door** | Global load balancing and CDN |
| **Private Endpoint** | Private access to PaaS services |

## Storage

| Term | Definition |
|------|------------|
| **Azure Storage** | Blob, Files, Queue, Table storage |
| **Managed Disks** | Block storage for VMs |
| **Azure Files** | Fully managed file shares |
| **Azure Blob Storage** | Object storage for unstructured data |
| **Cool/Archive Tiers** | Storage tiers for cost optimization |

## Databases

| Term | Definition |
|------|------------|
| **Azure SQL** | Managed SQL Server |
| **Azure Cosmos DB** | Globally distributed NoSQL DB |
| **Azure Database for PostgreSQL** | Managed Postgres service |
| **Azure Database for MySQL** | Managed MySQL service |
| **Azure Cache for Redis** | In-memory caching |

## Monitoring & Security

| Term | Definition |
|------|------------|
| **Azure Monitor** | Full stack monitoring |
| **Log Analytics** | Log aggregation and analysis |
| **Azure Sentinel** | SIEM solution |
| **Azure Defender** | Cloud workload protection |
| **Azure Policy** | Governance and compliance |
| **Azure Blueprint** | Reproducible environment templates |

## DevOps & Deployment

| Term | Definition |
|------|------------|
| **ARM Templates** | Infrastructure as Code (JSON) |
| **Bicep** | Domain-specific language for ARM |
| **Azure DevOps** | CI/CD and collaboration tools |
| **GitHub Actions** | CI/CD workflow automation |
| **Azure Pipelines** | Build and release pipelines |

## Cost Management

| Term | Definition |
|------|------------|
| **Pay-as-you-go** | On-demand pricing |
| **Reserved Capacity** | 1 or 3 year commitment discount |
| **Spot VMs** | Discounted interruptible VMs |
| **Azure Cost Management** | Billing and cost analysis tool |
| **Budget** | Cost threshold alerts |
| **Azure Advisor** | Cost optimization recommendations |
