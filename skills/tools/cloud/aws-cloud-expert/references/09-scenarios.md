# Glossary

## AWS Terminology

### A

| Term | Definition |
|------|------------|
| **AMI** | Amazon Machine Image — Pre-configured template for launching EC2 instances |
| **ASG** | Auto Scaling Group — Manages EC2 instances across AZs based on policies |
| **ARN** | Amazon Resource Name — Unique identifier for AWS resources |
| **AZ** | Availability Zone — One or more discrete data centers with redundant power, networking, and cooling |

### B

| Term | Definition |
|------|------------|
| **Bucket** | S3 container for storing objects (files) |
| **Bastion Host** | Jump server for accessing instances in private subnets |
| **BYOL** | Bring Your Own License — Use existing software licenses on AWS |

### C

| Term | Definition |
|------|------------|
| **CIDR** | Classless Inter-Domain Routing — IP address notation (e.g., 10.0.0.0/16) |
| **CloudFront** | AWS CDN service for global content delivery |
| **CloudFormation** | IaC service using JSON/YAML templates |
| **CloudTrail** | Audit logging for AWS API calls |
| **CloudWatch** | Monitoring and observability service |
| **CUR** | Cost and Usage Report — Detailed billing data |

### D

| Term | Definition |
|------|------------|
| **DDoS** | Distributed Denial of Service — Malicious traffic flood |
| **DynamoDB DAX** | DynamoDB Accelerator — In-memory cache for DynamoDB |
| **DX** | Direct Connect — Dedicated network connection to AWS |

### E

| Term | Definition |
|------|------------|
| **EBS** | Elastic Block Store — Persistent block storage for EC2 |
| **EC2** | Elastic Compute Cloud — Virtual servers in the cloud |
| **ECS** | Elastic Container Service — Container orchestration |
| **EFS** | Elastic File System — Managed NFS file storage |
| **EKS** | Elastic Kubernetes Service — Managed Kubernetes |
| **ELB** | Elastic Load Balancer — Distributes traffic across targets |
| **ENI** | Elastic Network Interface — Virtual network interface |
| **ES** | Elasticsearch Service — Managed Elasticsearch |

### F

| Term | Definition |
|------|------------|
| **Fargate** | Serverless compute for containers (ECS/EKS) |
| **Flow Logs** | VPC Flow Logs — Captures network traffic |

### G

| Term | Definition |
|------|------------|
| **GuardDuty** | Managed threat detection service |

### H

| Term | Definition |
|------|------------|
| **HVM** | Hardware Virtual Machine — Virtualization type for EC2 |

### I

| Term | Definition |
|------|------------|
| **IAM** | Identity and Access Management — User/role/policy management |
| **IGW** | Internet Gateway — Connects VPC to the internet |
| **IoT** | Internet of Things — Device connectivity service |

### K

| Term | Definition |
|------|------------|
| **KMS** | Key Management Service — Managed encryption keys |
| **Kinesis** | Real-time data streaming platform |

### L

| Term | Definition |
|------|------------|
| **Lambda** | Serverless compute — Run code without servers |
| **LB** | Load Balancer — Distributes application traffic |

### M

| Term | Definition |
|------|------------|
| **mTLS** | Mutual TLS — Two-way authentication |
| **MTU** | Maximum Transmission Unit — Largest packet size |

### N

| Term | Definition |
|------|------------|
| **NAC** | Network Access Control |
| **NACL** | Network Access Control List — Stateless firewall at subnet level |
| **NAT Gateway** | Network Address Translation — Allows private subnet internet access |
| **NLB** | Network Load Balancer — Layer 4 load balancing |

### O

| Term | Definition |
|------|------------|
| **OG** | Option Group — Database feature configuration |
| **OpsWorks** | Managed Chef/Puppet operations |
| **Organizations** | Multi-account management with SCPs |

### P

| Term | Definition |
|------|------------|
| **PBR** | Policy-Based Routing |
| **PG** | Parameter Group — Database parameter configuration |
| **Public Subnet** | Subnet with route table pointing to IGW |

### R

| Term | Definition |
|------|------------|
| **RDS** | Relational Database Service — Managed database |
| **Regional Endpoint** | Service endpoint at region level (default) |
| **RI** | Reserved Instance — Capacity reservation with discount |
| **Route 53** | Managed DNS service |
| **RTB** | Route Table — Routes for subnets |

### S

| Term | Definition |
|------|------------|
| **S3** | Simple Storage Service — Object storage |
| **S3 Glacier** | Long-term archival storage |
| **SCP** | Service Control Policy — Organization-level restrictions |
| **SDK** | Software Development Kit |
| **SES** | Simple Email Service — Managed email service |
| **SG** | Security Group — Stateful firewall for instances |
| **SNS** | Simple Notification Service — Pub/sub messaging |
| **SOP** | Standard Operating Procedure |
| **SQS** | Simple Queue Service — Managed message queue |
| **SSM** | Systems Manager — Instance management and automation |
| **STS** | Security Token Service — Temporary credentials |

### T

| Term | Definition |
|------|------------|
| **TGW** | Transit Gateway — Hub for VPC connectivity |
| **TCO** | Total Cost of Ownership |

### V

| Term | Definition |
|------|------------|
| **VPC** | Virtual Private Cloud — Isolated cloud network |
| **VPCE** | VPC Endpoint — Private connection to AWS services |
| **VPN** | Virtual Private Network — Encrypted tunnel |

### W

| Term | Definition |
|------|------------|
| **WAF** | Web Application Firewall — Protects web apps |
| **WAFv2** | Latest version of AWS WAF with regional/global support |
| **WAN** | Wide Area Network |
| **WebACL** | Web Access Control List — WAF rule set |
