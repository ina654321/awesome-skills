# Glossary

## Cloudflare Terminology

### A

| Term | Definition |
|------|------------|
| **Anycast** | Routing method where multiple servers share same IP |
| **API Shield** | API security product with mTLS, Schema Validation |
| **Access** | Zero Trust Network Access (ZTNA) |

### B

| Term | Definition |
|------|------------|
| **BYOIP** | Bring Your Own IP — Use your own IP ranges |
| **Bandwidth Alliance** | Reduced egress fees with partners |

### C

| Term | Definition |
|------|------------|
| **Cache** | Edge-stored copies of content |
| **Cache Hit** | Content served from cache |
| **Cache Miss** | Content fetched from origin |
| **CDN** | Content Delivery Network — Edge caching |
| **CNAME** | Canonical Name — DNS alias record |
| **Content Shield** | Advanced caching policies |
| **CORS** | Cross-Origin Resource Sharing |

### D

| Term | Definition |
|------|------------|
| **D1** | Cloudflare's SQLite database |
| **DDoS** | Distributed Denial of Service |
| **DNS** | Domain Name System |
| **DoH** | DNS over HTTPS |
| **DoT** | DNS over TLS |
| **Durable Objects** | Strongly-consistent, single-threaded storage |

### E

| Term | Definition |
|------|------------|
| **Edge** | Cloudflare's global network of data centers |
| **Edge Functions** | Serverless compute at the edge |
| **Egress** | Outbound traffic from Cloudflare |

### F

| Term | Definition |
|------|------------|
| **Firewall Rules** | Custom firewall rules |
| **Full Strict** | SSL mode with origin certificate verification |

### G

| Term | Definition |
|------|------------|
| **Gateway** | Cloudflare's secure web gateway |

### H

| Term | Definition |
|------|------------|
| **Health Checks** | Origin monitoring |
| **HTTP/3** | QUIC-based protocol, faster than HTTP/2 |

### I

| Term | Definition |
|------|------------|
| **Image Resizing** | On-the-fly image optimization |
| **Instant Play** | Video optimization |
| **IoT** | Internet of Things |

### L

| Term | Definition |
|------|------------|
| **Load Balancer** | Global server load balancing |
| **Logs** | Cloudflare Logpush/Logs |

### M

| Term | Definition |
|------|------------|
| **Magic Transit** | DDoS protection + network security |
| **Magic WAN** | SD-WAN over Cloudflare |
| **mTLS** | Mutual TLS — Client certificate authentication |

### N

| Term | Definition |
|------|------------|
| **Network Interconnect** | Direct connection to Cloudflare |
| **Nginx** | Origin web server |

### O

| Term | Definition |
|------|------------|
| **Origin** | Your web server / backend |
| **Origin Certificate** | Cloudflare-issued SSL for origin |

### P

| Term | Definition |
|------|------------|
| **Pages** | Static site hosting and SSR |
| **Polish** | Image optimization service |
| **PoP** | Point of Presence — Cloudflare data center |
| **Proxy** | Cloudflare's intermediate server |

### Q

| Term | Definition |
|------|------------|
| **QUIC** | Modern transport protocol (HTTP/3) |

### R

| Term | Definition |
|------|------------|
| **R2** | Cloudflare object storage (S3-compatible) |
| **Railgun** | Dynamic content optimization (deprecated) |
| **Rate Limiting** | Request throttling rules |
| **Registrar** | Domain registration service |
| **Routing** | Traffic routing and load balancing |

### S

| Term | Definition |
|------|------------|
| **SASE** | Secure Access Service Edge |
| **Schema Validation** | API Shield request validation |
| **Scrape Shield** | Protection from scraping |
| **Spectrum** | TCP/UDP proxy |
| **SSL/TLS** | Secure Socket Layer / Transport Layer Security |
| **Stream** | Video streaming platform |

### T

| Term | Definition |
|------|------------|
| **Tiered Cache** | Multi-layer caching strategy |
| **Tunnel** | Secure connection (Cloudflare Tunnel) |

### V

| Term | Definition |
|------|------------|
| **Virtual DNS** | Cloudflare's anycast DNS |
| **VoD** | Video on Demand |

### W

| Term | Definition |
|------|------------|
| **WAF** | Web Application Firewall |
| **WARP** | VPN client by Cloudflare |
| **Workers** | Serverless JavaScript runtime |
| **Workers KV** | Key-value store for Workers |

### Z

| Term | Definition |
|------|------------|
| **Zero Trust** | Security model requiring verification |
| **Zone** | Your domain in Cloudflare |
| **Zone file** | DNS records for a domain |
