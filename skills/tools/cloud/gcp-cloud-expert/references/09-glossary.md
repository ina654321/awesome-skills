# Glossary

## GCP Core Concepts

| Term | Definition |
|------|------------|
| **Project** | Logical container for resources |
| **Organization** | Top-level container for enterprise |
| **Folder** | Hierarchical grouping under organization |
| **Resource Manager** | API for managing resources |
| **Cloud IAM** | Identity and Access Management |

## Compute Services

| Term | Definition |
|------|------------|
| **Compute Engine** | Virtual machines |
| **GKE** | Google Kubernetes Engine |
| **Cloud Run** | Serverless containers |
| **App Engine** | PaaS for apps |
| **Cloud Functions** | Serverless functions |
| **Kubernetes Engine** | See GKE |

## Data & Analytics

| Term | Definition |
|------|------------|
| **BigQuery** | Serverless data warehouse |
| **Cloud Storage** | Object storage |
| **Dataflow** | Stream/batch processing |
| **Dataproc** | Managed Spark/Hadoop |
| **Pub/Sub** | Messaging service |
| **Bigtable** | NoSQL wide-column |

| Term | Definition |

| **AutoML** | Automated ML |
| **AI Platform** | Legacy ML service |
