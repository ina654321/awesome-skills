# Glossary

## GCP Terminology

### A

| Term | Definition |
|------|------------|
| **API** | Application Programming Interface |
| **Apigee** | API management platform |
| **Artifact Registry** | Container and language package registry |
| **Autoscaler** | Automatically adjusts instance count |

### B

| Term | Definition |
|------|------------|
| **Backend** | Instance group or service handling traffic |
| **BigQuery** | Serverless data warehouse |
| **Bigtable** | Wide-column NoSQL database |
| **Bucket** | GCS container for objects |

### C

| Term | Definition |
|------|------------|
| **CA** | Cloud Armor — DDoS protection and WAF |
| **CDN** | Content Delivery Network via Cloud CDN |
| **CDN** | Cloud DNS — Managed DNS service |
| **Cmek** | Customer-Managed Encryption Key |
| **Cloud Build** | CI/CD platform |
| **Cloud Functions** | Serverless functions |
| **Cloud Run** | Serverless containers |
| **Cloud Scheduler** | Cron job service |
| **Cloud Shell** | Browser-based shell |
| **Cloud Storage** | Object storage |
| **Cloud SQL** | Managed relational database |
| **Cluster** | GKE managed Kubernetes cluster |
| **CUD** | Committed Use Discount |
| **CUJ** | Critical User Journey |

### D

| Term | Definition |
|------|------------|
| **Datastore** | NoSQL document database |
| **DLP** | Data Loss Prevention |
| **DNS** | Domain Name System |
| **DR** | Disaster Recovery |
| **DRA** | Disaster Recovery as a Service |
| **DSC** | Deployment Manager — IaC |

### E

| Term | Definition |
|------|------------|
| **Editor** | IAM role with edit access |
| **Egress** | Outbound network traffic |

### F

| Term | Definition |
|------|------------|
| **Firestore** | Document database (new Datastore) |
| **Firewall rule** | Network access control |
| **Folder** | Resource hierarchy container |

### G

| Term | Definition |
|------|------------|
| **GCE** | Google Compute Engine — VMs |
| **GCLB** | Google Cloud Load Balancer |
| **GCP** | Google Cloud Platform |
| **gcloud** | GCP CLI tool |
| **GKE** | Google Kubernetes Engine |
| **GPU** | Graphics Processing Unit |
| **Granular** | Fine-grained access control |
| **gsutil** | GCS CLI tool |

### H

| Term | Definition |
|------|------------|
| **Health Check** | Determines if instance is healthy |
| **Horizons** | GCP console UI |

### I

| Term | Definition |
|------|------------|
| **IAM** | Identity and Access Management |
| **Instance** | Virtual machine |
| **Instance Group** | Group of VM instances |
| **Instance Template** | Template for creating VMs |
| **Instance Zone** | Zone where instance runs |

### K

| Term | Definition |
|------|------------|
| **Kubernetes** | Container orchestration |
| **KMS** | Key Management Service |

### L

| Term | Definition |
|------|------------|
| **Labels** | Key-value pairs for resource organization |
| **LB** | Load Balancer |
| **Leader** | Primary in HA setup |
| **Load Balancer** | Distributes traffic |
| **Location** | Region or multi-region |

### M

| Term | Definition |
|------|------------|
| **MIG** | Managed Instance Group |
| **MFA** | Multi-Factor Authentication |
| **Monitoring** | Cloud Monitoring |

### N

| Term | Definition |
|------|------------|
| **NAT** | Network Address Translation |
| **NAT Gateway** | Cloud NAT for private instances |
| **Network** | VPC network |
| **Node** | Worker machine in GKE |
| **Node Pool** | Group of nodes in GKE |

### O

| Term | Definition |
|------|------------|
| **Object** | File in Cloud Storage |
| **Organization** | Root of resource hierarchy |
| **OS Login** | OS-level SSH management |

### P

| Term | Definition |
|------|------------|
| **Partition** | Query partition on BigQuery |
| **Path** | URL path for routing |
| **Peering** | VPC Network Peering |
| **Policy** | IAM policy document |
| **Project** | Container for GCP resources |
| **Proxy** | Layer 7 proxy for routing |
| **Pub/Sub** | Messaging and ingestion service |

### Q

| Term | Definition |
|------|------------|
| **Quota** | Limit on resource usage |

### R

| Term | Definition |
|------|------------|
| **RAM** | Random Access Memory |
| **Region** | Geographic area with zones |
| **Replica** | Copy for HA |
| **Resource** | GCP service entity |
| **Role** | Collection of permissions |
| **Router** | Routes traffic between networks |
| **Runtime** | Execution environment |

### S

| Term | Definition |
|------|------------|
| **SA** | Service Account |
| **Scope** | OAuth scope for access |
| **Service** | Managed service |
| **SLA** | Service Level Agreement |
| **Slot** | BigQuery compute capacity |
| **SNIP** | Subnetwork IP range |
| **Snapshot** | Point-in-time copy of disk |
| **Source** | Backend origin |
| **Spanner** | Globally distributed relational DB |
| **Spot** | Preemptible VMs at discount |
| **Stackdriver** | Monitoring and logging (unified) |
| **Subnet** | Subdivision of VPC network |
| **SUD** | Sustained Use Discount |

### T

| Term | Definition |
|------|------------|
| **Target** | Where traffic is sent |
| **Task** | Cloud Tasks job |
| **Terraform** | HashiCorp IaC (via Google provider) |
| **Throughput** | Data transfer rate |
| **Traceroute** | Network path analysis |
| **Transit** | Network transit service |

### U

| Term | Definition |
|------|------------|
| **Uptime Check** | Monitoring health check |
| **URL Map** | LB path routing rules |

### V

| Term | Definition |
|------|------------|
| **VPC** | Virtual Private Cloud |
| **Vertex AI** | ML platform |
| **View** | Saved BigQuery query |
| **VPN** | Virtual Private Network |

### W

| Term | Definition |
|------|------------|
| **WAF** | Web Application Firewall |
| **WIF** | Workload Identity Federation |
| **Workflows** | Serverless workflow orchestration |

### X

| Term | Definition |
|------|------------|
| **XPN** | Shared VPC |

### Z

| Term | Definition |
|------|------------|
| **Zone** | Single data center location |
| **Zonal** | Resource in single zone |
| **Regional** | Resource across zones |
