## 7.2 Soil & Crop Management Standards

| Metric | Optimal Range | Measurement |
|--------|--------------|-------------|
| **Soil organic matter** | 3–5% (temperate); 1–2% (sandy) | Combustion method, LOI |
| **Soil pH** | 6.0–7.0 (most crops); 5.5–6.5 (acid-loving) | 1:1 soil:water or buffer method |
| **Bulk density** | 1.0–1.4 g/cm³ | Core sampler method |
| **Available water capacity** | 0.15–0.25 in/in | Pressure plate apparatus |
| **Nitrogen (total)** | 0.1–0.3% | Dumas method or Kjeldahl |
| **Phosphorus (Olsen P)** | 20–50 ppm (most crops) | Sodium bicarbonate extraction |
| **Potassium** | 150–300 ppm | Ammonium acetate extraction |
| **Cation exchange capacity** | 10–25 meq/100g | Ammonium acetate method |

## 7.3 USDA Program Eligibility Standards

| Program | Key Requirements | Payment Rate |
|---------|-----------------|--------------|
| **ARC-CO** (Agriculture Risk Coverage) | Base acres, 5-year Olympic average | County-level guarantee |
| **PLC** (Price Loss Coverage) | Base acres | Price trigger below reference price |
| **Crop Insurance (APH)** | Actual production history | Indemnity at guarantee - actual |
| **EQIP** (Environmental Quality Incentives) | Conservation plan required | 50–75% cost share |
| **CSP** (Conservation Stewardship Program) | Existing stewardship baseline | Per-acre payments |
| **CRP** (Conservation Reserve Program) | Marginal cropland | Rental payment + cost share |

## 7.4 Fertilizer Application Standards

| Nutrient | Deficiency Symptoms | Application Rate (typical) |
|----------|--------------------|---------------------------|
| **Nitrogen (N)** | Yellowing lower leaves, stunted growth | 40–200 lbs N/acre (row crop) |
| **Phosphorus (P)** | Purple leaves, poor root development | 40–80 lbs P₂O₅/acre |
| **Potassium (K)** | Leaf edge burn, weak stems | 40–120 lbs K₂O/acre |
| **Zinc (Zn)** | Stunted new growth, small leaves | 5–10 lbs Zn/acre (soil); foliar spray |
| **Sulfur (S)** | Uniform yellowing, similar to N | 15–30 lbs S/acre |
| **Boron (B)** | Poor flowering, hollow stems | 0.5–2 lbs B/acre (soil); foliar |
