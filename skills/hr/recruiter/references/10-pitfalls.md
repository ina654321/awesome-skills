# Common Pitfalls & Anti-Patterns

## 10.1 High-Severity Anti-Patterns

### 🔴 Anti-Pattern 1: Submitting Unvetted Candidates
```
❌ BAD: Sending anyone who vaguely fits to fill req fast
✅ GOOD: Quality over quantity; only submit candidates you'd stake your reputation on
```
**Why it matters**: Damages reputation with hiring managers; wastes everyone's time; burns pipeline.

### 🔴 Anti-Pattern 2: Misrepresenting the Role
```
❌ BAD: "It's mostly coding, minimal meetings" when it's 50% meetings
✅ GOOD: Be honest; it's okay if role isn't perfect match for everyone
```
**Why it matters**: Candidates who feel misled withdraw or quit early; creates bad reputation.

### 🔴 Anti-Pattern 3: Ghosting Candidates
```
❌ BAD: Stopping communication after interview without update
✅ GOOD: Close every loop; candidates deserve response
```
**Why it matters**: Ghosting damages employer brand; rejected candidates talk about experience.

### 🔴 Anti-Pattern 4: Salary Mismatch at Offer
```
❌ BAD: Offering below range because "we can get them cheaper"
✅ GOOD: Offer within band; respect candidate's time and expectations
```
**Why it matters**: Lose candidates at offer; damages relationships; wastes everyone's time.

## 10.2 Medium-Severity Anti-Patterns

### 🟡 Anti-Pattern 5: Ignoring Red Flags
```
❌ BAD: "Job hopping is normal in tech" — dismissing pattern
✅ GOOD: Probe job-hopping; understand context; check references
```

### 🟡 Anti-Pattern 6: Leading with Compensation
```
❌ BAD: Starting calls with salary range before fit
✅ GOOD: Interest first; money conversation after fit established
```

### 🟡 Anti-Pattern 7: No Interview Feedback Loop
```
❌ BAD: Interviewers making decisions without input
✅ GOOD: Structured debrief; collective decision-making
```

### 🟡 Anti-Pattern 8: Reactive vs. Proactive Sourcing
```
❌ BAD: Posting and waiting; only sourcing when req opens
✅ GOOD: Build talent pools; nurture relationships year-round
```

## 10.3 Best Practices Checklist

### Job Intake
- [ ] Clear requirements documented
- [ ] Compensation band confirmed
- [ ] Interview plan established
- [ ] Timeline agreed with HM

### Candidate Experience
- [ ] Timely communication at every stage
- [ ] Clear expectations set
- [ ] Feedback collected and shared
- [ ] Every candidate thanked

### Quality Control
- [ ] Only submit A-players
- [ ] Red flags investigated
- [ ] References checked
- [ ] Offer within compensation band

## 10.4 Red Flags to Watch For

| Red Flag | Risk Level | Action |
|----------|------------|--------|
| Job hopping with no explanation | Medium | Probe; check references |
| Salary expectations way off | Medium | Align early; don't waste time |
| Vague about current employer | Medium | Understand reason; verify |
| Can't articulate impact | Medium | Press for specifics |
| Bad attitude in screening | High | Don't submit; quality matters |
