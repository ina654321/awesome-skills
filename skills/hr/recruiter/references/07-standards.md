# Standards & Reference

## 7.1 Sourcing Channels

### Active Channels
| Channel | Best For | Reach |
|---------|----------|-------|
| **LinkedIn Recruiter** | Professional, passive candidates | 900M+ members |
| **Indeed** | Volume, mid-market roles | 250M+ users |
| **Glassdoor** | Company-researching candidates | Active job seekers |
| **ZipRecruiter** | SMB, faster volume | 3M+ daily visitors |

| Source | Niche | Method |
|--------|-------|--------|
| **GitHub** | Software engineers | Search by language, contributions |
| **Stack Overflow** | Developers | Talent search, job ads |
| **XING** | European professionals | Networking |

### Diversity Sourcing
- **HBCU Connect**: Historically Black colleges

## 7.2 Interview Frameworks

### STAR Method
| Component | Description |
|-----------|-------------|
| **Situation** | Set the context |
| **Task** | Your responsibility |
| **Action** | What you specifically did |
| **Result** | Outcome and impact |

### Structured Interview Format
1. **Opening** (2 min): Build rapport
2. **Core Questions** (30 min): Behavioral/competency
3. **Candidate Questions** (10 min): Your turn to ask
4. **Close** (3 min): Timeline, next steps

### Common Competency Questions
- Leadership: "Tell me about leading a difficult project"
- Problem-solving: "Describe a complex problem you solved"
- Communication: "Explain technical concept to non-technical person"
- Collaboration: "Work through disagreement with teammate"

## 7.3 Compensation Standards

### Total Compensation Components
| Component | Typical Range |
|-----------|---------------|
| **Base Salary** | Fixed cash |
| **Annual Bonus** | 0-30% of base |
| **Equity/RSU** | Startup: 0.1-2%; Public: varies |
| **Signing Bonus** | 0-20% of base |
| **Benefits** | Healthcare, 401k, PTO |

### Negotiation Leverage Points
- Multiple offers
- Counter offer from current employer
- High demand skills (market timing)
- Relocation needs
- Signing timeline
