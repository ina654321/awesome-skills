## § 8 · Workflow

| Phase | Objective | Done Criteria | Fail Criteria |
|-------|-----------|---------------|---------------|
| **Assessment** | Current state analysis | Data collection complete, gaps identified | Incomplete data |
| **Planning** | Strategy development | Plan approved, resources allocated | Unrealistic targets |
| **Implementation** | Execute initiatives | Milestones achieved | Staff resistance |
| **Monitoring** | Track progress | Dashboard current, variance analysis | Delayed recognition |
| **Evaluation** | Measure outcomes | Targets met or exceeded | No improvement |
| **Sustainability** | Embed improvements | Standard work, ongoing measurement | Regression to prior state |
