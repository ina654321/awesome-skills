# Standard Workflow

## 8.1 Pre-Vaccination Assessment

```
Phase 1: Screening
├── Review vaccination history
├── Screen for contraindications
├── Assess current health status
├── Verify age-appropriate vaccines
├── Check for needed vaccines
└── Document screening

Phase 2: Patient Education
├── Explain benefits and risks
├── Discuss common side effects
├── Provide VIS (Vaccine Information Statement)
├── Answer questions
├── Obtain informed consent
└── Document discussion

Phase 3: Documentation
├── Record consent
├── Document screening
├── Verify patient identity
└── Prepare vaccination record
```

## 8.2 Vaccine Administration

```
Standard Protocol:
```
Injection Sites:
├── IM: Deltoid (adults), anterolateral thigh (infants)
├── Subcutaneous: Fatty tissue (triceps, thigh)
├── Intradermal: As per specific vaccine
└── Oral: Per vaccine instructions

Needle Selection:
├── IM (adult deltoid): 22-25G, 1-1.5"
├── IM (infant thigh): 22-25G, 1"
├── Subcutaneous: 23-25G, 5/8"
└── Syringe size matches dose
```

Process:
1. Verify vaccine name and dose
2. Check expiration date
3. Prepare vaccine per route
4. Identify correct site
5. Administer using proper technique
6. Document administration
7. Monitor for adverse reactions
8. Provide aftercare instructions
```

## 8.3 Emergency Response

```
Anaphylaxis Protocol:
```
Signs:
├── Urticaria, angioedema
├── Respiratory distress
├── Hypotension
├── GI symptoms
└── Altered consciousness

Immediate Actions:
├── Stop vaccine administration
├── Position patient (supine, legs elevated)
├── Administer epinephrine IM
├── Call for help
├── Monitor vitals
└── Prepare for transport
```
```

## 8.4 Documentation Requirements

```
Required Documentation:
├── Vaccine name (brand + generic)
├── Manufacturer
├── Lot number and expiration
├── Dose and route
├── Site administered
├── Administrator name/title
├── VIS date provided
├── Precautions screened
└── Patient/guardian signature

Reporting:
├── IIS (Immunization Information System)
├── VAERS for adverse events
├── Internal record
└── Patient vaccination card
```
