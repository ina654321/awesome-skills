## 7.2 First Trimester Standards

```
Dating Ultrasound:
├── Crown-rump length 7-13+6 weeks
├── Multiple gestations assessed
├── Fetal heart rate documented
├── Nuchal translucency 11-14 weeks
└──chorionicity if twins

Viability:
├── Intrauterine pregnancy confirmed
├── Fetal poles at 5.5-6 weeks
├── Cardiac activity at 6-7 weeks
└──EDD by CRL most accurate
```

## 7.3 Second Trimester Anatomy

```
Complete Anatomy Survey:
├── Head: BPD, HC, ventricles, cerebellum, CSP
├── Face: Lips, profile
├── Neck: Cystic hygroma
├── Chest: Heart, lungs
├── Abdomen: Stomach, bowel, kidneys, bladder
├── Spine: Vertebrae
├── Extremities: Arms, legs, feet
├── Umbilical cord: 3 vessels
├── Placenta: Location, grade
└── Amniotic fluid: AFI or MVP

Cervical Assessment:
├── Transvaginal for cervical length
├── Length <25mm = short
├── Risk for preterm delivery
└── Funneling if present
```

## 7.4 Doppler Standards

```
Indications:
├── FGR surveillance
├── Umbilical artery Doppler
├── Middle cerebral artery (MCA)
├── Ductus venosus
├── Maternal uterine arteries
└── IAC if indicated

Measurements:
├── Umbilical artery: Systolic/diastolic ratio
├── MCA: Peak systolic velocity
├── CPR = MCA/UA ratio
└── Absent/reversed end-diastolic flow
```
