# Standard Workflow

## 8.1 Pre-Op Phase

```
Phase 1: Patient Arrival
├── Verify patient identity (2 identifiers)
├── Confirm NPO status
├── Verify surgical consent signed
├── Check allergy band
├── Complete pre-op checklist
└── Verify H&P on chart

Phase 2: Preparation
├── Start IV access
├── Administer pre-op medications
├── Verify surgical site marking
├── Complete nursing assessment
├── Ensure all documentation complete
└── Hand off to OR team
```

## 8.2 Intraoperative Phase

```
Scrub Role:
├── Set up sterile field
├── Maintain instrument count
├── Anticipate surgeon needs
├── Pass instruments safely
└── Maintain sterile technique

Circulating Role:
├── Coordinate with OR team
├── Specimen handling
├── Specimen labeling
├── Documentation
├── Environment management
└── Counts verification
```

## 8.3 Count Protocol

```
Standard Counts:
├── Before incision ("initial count")
├── Before closure of cavity ("additional count")
├── Before skin closure ("final count")
├── Before leaving OR ("closing count")
└── After skin closure ("final count")

Count Documentation:
├── Count performed with another person
├── Verbal confirmation
├── Document count numbers
├── Document counts correct
└── Report discrepancies immediately
```

## 8.4 Post-Anesthesia Phase

```
Phase 1: Transfer to PACU
├── Verbal report to PACU nurse
├── Patient assessment
├── IV and airway status
├── Drains and dressings
├── Implants and specimens
└── Documentation complete

Phase 2: PACU Care
├── Airway and breathing
├── Vital signs q5-15 min
├── Pain assessment
├── Nausea/vomiting management
├── Discharge criteria met
└── Modified Aldrete score ≥9
```
