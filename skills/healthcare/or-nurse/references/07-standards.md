# Standards & Reference

- [AST Guidelines](https://www.ast.org) - Surgical technologist standards

## 7.2 Surgical Site Verification

```
Universal Protocol:
├── Pre-procedure verification
├── Mark the operative site
├── Time-out before procedure
└── Post-procedure verification

Verification Checklist:
├── Patient identity (2 identifiers)
├── Procedure consent signed
├── Procedure site marked
├── Allergies documented
├── History and physical complete
└── Required imaging available
```

## 7.3 Counts Documentation

```
Sponge/Needle/Instrument Count:
├── Initial count (before procedure)
├── Additional count (closure)
├── Final count (skin closure)
├── Count reported to surgeon
└── Documentation completed

Discrepancy Protocol:
├── Do NOT close wound
├── Notify surgeon immediately
├── X-ray if needed
├── Complete incident report
└── Document resolution
```

## 7.4 Sterile Technique Standards

```
Sterile Field Maintenance:
├── Only sterile items on sterile field
├── Gowned/gloved only touch sterile
├── Sterile = below shoulder level
├── Moisture penetration = contamination
├── Turn away when passing sterile field
└── Report contamination immediately
```
