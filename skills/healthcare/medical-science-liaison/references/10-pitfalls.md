# Common Pitfalls & Anti-Patterns

## 10.1 Promotional Conduct Violations

| Pitfall | Severity | Prevention |
|---------|----------|------------|
| Discussing unapproved indications | 🔴 Critical | Script adherence |
| Providing dosing guidance off-label | 🔴 Critical | Compliance training |
| Offering incentives for prescribing | 🔴 Critical | Ethics education |
| Using sales-oriented language | 🟡 Medium | Medical language focus |

## 10.2 Communication Errors

| Error | Impact | Solution |
|-------|--------|----------|
| Overpromising | Trust damage | Scientific accuracy |
| Speculating | Compliance risk | Stick to data |
| Deflecting legitimate inquiries | Missed opportunity | Balance compliance/engagement |
| Inconsistent messaging | Confusion | Territory alignment |

## 10.3 KOL Management Failures

```
Common Issues:
├── Only calling on prescribing physicians
├── Missing non-prescriber influencers
├── Inconsistent engagement
├── Not following up on commitments
└── Poor scientific credibility

Solutions:
├── Stakeholder mapping beyond prescribers
├── Consistent engagement plan
├── Track all commitments in CRM
├── Continuous scientific development
```

## 10.4 Compliance Gaps

| Gap | Risk | Mitigation |
|-----|------|------------|
| Inadequate documentation | Audit findings | CRM compliance |
| Missing Sunshine reporting | Legal risk | Expense tracking |
| Inappropriate venue | Violation | Event guidelines |
| No CoI disclosure | Ethics breach | Disclosure protocols |

## 10.5 Territory Management Issues

```
Common Problems:
├── Inefficient travel
├── Missing strategic accounts
├── Poor prioritization
├── Uneven coverage
└── Lack of insights documentation

Solutions:
├── Data-driven territory planning
├── Regular territory reviews
├── RACI matrix for accounts
├── Insights collection framework
```

## 10.6 Quality Checklist

```
Scientific Exchange:
├── Purpose clearly defined
├── Content scientifically balanced
├── Compliance reviewed
├── Outcome documented
└── Follow-up tracked

KOL Engagement:
├── Stakeholder mapped
├── Engagement tier assigned
├── Strategic plan documented
├── Commitments tracked
└── Insights captured

Compliance:
├── Sunshine reported
├── Gifts documented
├── CoI disclosed
├── HCP autonomy respected
└── Appropriate venue
```
