# Standard Workflow

## 8.1 Territory Planning

```
Phase 1: Account Analysis
├── Identify key accounts
├── Map stakeholders
├── Assess engagement opportunities
├── Prioritize accounts (tier system)
└── Set strategic objectives

Phase 2: Engagement Planning
├── Scientific exchange topics
├── Preferred communication channels
├── Frequency of interactions
├── Key messages aligned
└── Resource requirements

Phase 3: Execution
├── Schedule HCP interactions
├── Attend scientific meetings
├── Coordinate with sales team
├── Document all interactions
└── Track outcomes
```

## 8.2 KOL Engagement

```
Identification (1-3 tier system):
├── Tier 1: National/international influence
├── Tier 2: Regional influence
├── Tier 3: Local influence
└── Thought leaders vs. advocates

Engagement Framework:
```
Tier 1 KOLs:
├── Advisory board participation
├── Speaker programs
├── Investigator-initiated studies
├── Congress symposia
└── Publication collaboration

Tier 2 KOLs:
├── Regional advisory input
├── Speaker training
├── Clinical trial site
└── Peer-to-peer education
```

## 8.3 Scientific Exchange

```
Interaction Types:
├── One-on-one office visits
├── Virtual scientific discussions
├── Congress interactions
├── Advisory board meetings
└── Medical information requests

Documentation:
├── CRM system entry
├── Scientific exchange summary
├── Follow-up commitments
├── Insights gathered
└── Compliance review if needed
```

## 8.4 Medical Information

```
Request Handling:
├── Log request in system
├── Identify appropriate response
├── Provide balanced information
├── Document delivery
└── Follow up if needed

Response Sources:
├── Prescribing information
├── Clinical data
├── Publication data
├── Medical affairs materials
└── External medical literature
```
