## 7.2 Assessment Tools

| Tool | Purpose | Score Interpretation |
|------|---------|----------------------|
| Subjective Global Assessment (SGA) | Nutritional status | A (well), B (moderate), C (severely malnourished) |
| MUST (Malnutrition Universal Screening Tool) | Malnutrition risk | 0 = low, 1 = medium, ≥2 = high risk |
| NRS-2002 | Hospital malnutrition screening | ≥3 indicates nutritional risk |
| BMI | Body composition | <18.5 underweight, 18.5-24.9 normal, 25-29.9 overweight, ≥30 obese |

## 7.3 Energy Calculation References

**Mifflin-St Jeor Equation (REE):**
- Male: REE = 10(weight) + 6.25(height) - 5(age) + 5
- Female: REE = 10(weight) + 6.25(height) - 5(age) - 161

**Activity/Injury Factors:**
| Condition | Factor |
|-----------|--------|
| Sedentary | 1.0-1.2 |
| Light activity | 1.3-1.4 |
| Moderate activity | 1.5-1.6 |
| Critical illness | 1.2-1.5 |
| Major surgery | 1.1-1.3 |
| Burns | 1.5-2.0 |
| Sepsis | 1.2-1.6 |

## 7.4 Disease-Specific Nutrition Guidelines

| Condition | Energy | Protein | Special Considerations |
|-----------|--------|---------|----------------------|
| CKD without dialysis | 25-30 kcal/kg | 0.6-0.8 g/kg | Limit Na, K, phosphorus |
| Hemodialysis | 30-35 kcal/kg | 1.2 g/kg | Replace lost micronutrients |
| Diabetes | Individualized | 1.0-1.2 g/kg | Carb counting, GI, glycemic load |
| COPD | 30-35 kcal/kg | 1.2-1.5 g/kg | Small frequent meals; fluid management |
| Liver disease | 25-40 kcal/kg | 1.2-1.5 g/kg | High BCAA if encephalopathy |
| Wound healing | 30-35 kcal/kg | 1.2-1.5 g/kg | Increase Zn, vitamin C, protein |
