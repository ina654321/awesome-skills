# Standard Workflow

## 8.1 Initial Evaluation

```
Phase 1: Chart Review
├── Diagnosis and medical history
├── Prior level of function
├── Medication review
├── Prior therapy history
└── Patient goals

Phase 2: Clinical Assessment
├── Strength and ROM
├── Balance and coordination
├── Sensation and pain
├── Functional mobility
├── ADL assessment
└── Safety awareness

Phase 3: Plan Development
├── Identify impairments
├── Set functional goals
├── Develop treatment plan
├── Determine frequency/duration
├── Establish home program
└── Communicate with team
```

## 8.2 Treatment Sessions

```
Session Structure:
├── Reassess current status
├── Review home program
├── Manual therapy (if indicated)
├── Therapeutic exercise
├── Functional training
├── Patient education
└── Assign home program

Documentation:
├── Treatment provided
├── Patient response
├── Progress toward goals
├── Barriers/challenges
└── Plan for next session
```

## 8.3 Progress Monitoring

```
Weekly Progress Note:
├── Objective measurements
├── Treatment response
├── Goal progress
├── Barriers to progress
├── Plan adjustment
└── Continued medical necessity

Reassessment (every 10 visits):
├── Full reassessment
├── Goal status
├── Progress documentation
├── Continued need
└── Plan for discharge
```

## 8.4 Discharge Planning

```
Discharge Criteria:
├── Goals met or plateau
├── Independent with home program
├── Patient ready for discharge
├── Maximal benefit achieved
└── Alternative setting more appropriate

Discharge Process:
├── Final assessment
├── Home program education
├── Equipment recommendations
├── Follow-up recommendations
├── Patient satisfaction
└── Discharge summary
```
