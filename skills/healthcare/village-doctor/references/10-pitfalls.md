# Common Pitfalls & Anti-Patterns

## 10.1 Diagnostic Errors

| Pitfall | Severity | Prevention |
|---------|----------|------------|
| Underestimating severity | 🔴 High | Systematic assessment |
| Not considering common diagnoses | 🟡 Medium | Differential diagnosis |
| Missing red flags | 🔴 High | Checklists |
| Anchoring bias | 🟡 Medium | Open-ended assessment |

## 10.2 Resource Limitations

```
Challenges:
├── Limited diagnostics
├── Far from hospital
├── Limited medications
├── Staffing shortages
└── Equipment constraints

Solutions:
├── Point-of-care testing
├── Telemedicine consultation
├── Clear referral protocols
├── Standing orders
└── Community health workers
```

## 10.3 Treatment Errors

| Error | Impact | Prevention |
|-------|--------|------------|
| Inappropriate prescribing | Harm | Follow guidelines |
| Not referring when needed | Delayed | Know scope |
| Over-treating minor illness | Waste | Evidence-based |
| Medication errors | Harm | Check twice |

## 10.4 Communication Failures

| Failure | Impact | Prevention |
|---------|--------|------------|
| Not documenting | Continuity | Real-time |
| Poor referral communication | Delayed care | Standard handoff |
| Language barriers | Misunderstanding | Interpreter if available |
| Not following up | Worse outcomes | Tracking system |

## 10.5 Emergency Response

```
Common Errors:
├── Delaying transport
├── Under-treating emergencies
├── Not activating EMS
├── Poor communication
└── Not documenting

Prevention:
├── Clear protocols
├── Regular training
├── Good communication
├── Proper documentation
```

## 10.6 Quality Checklist

- [ ] Comprehensive history
- [ ] Complete exam
- [ ] Red flags assessed
- [ ] Appropriate testing
- [ ] Evidence-based treatment
- [ ] Referrals when needed
- [ ] Follow-up arranged
- [ ] Documentation complete
- [ ] Patient educated
- [ ] Return precautions given
