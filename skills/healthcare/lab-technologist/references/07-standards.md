# Standards & Reference

- [CLSI Guidelines](https://clsi.org) - Clinical and Laboratory Standards Institute
- [CAP Accreditation](https://www.cap.org) - College of American Pathologists
- [CMS CLIA](https://www.cms.gov/regulations-and-guidance/legislation/clia/index.html) - Laboratory certification
- [FDA Device Standards](https://www.fda.gov/medical-devices) - IVD regulations
- [WHO EML](https://list.afro.who.org) - Laboratory quality standards

## 7.2 Quality Control Standards

```
Westgard Rules (QC Interpretation):
├── 1-2s: Warning, investigate
├── 1-3s: Reject run
├── 2-2s: Reject if same level
├── 4-1s: Reject if 4 consecutive >1SD same direction
├── 10x: Reject if 10 consecutive same side of mean
└── R-4s: Reject if range >4SD between levels
```

## 7.3 Instrument Calibration

| Frequency | Type | Verification |
|-----------|------|--------------|
| Daily | Calibration verify | QC materials |
| Weekly | Full calibration | Manufacturer specs |
| Monthly | Performance check | Reference methods |
| As needed | Corrective | Out-of-spec results |

## 7.4 Reference Ranges

```
Standard Reference Ranges (Adult):
├── Hemoglobin: M 13.5-17.5, F 12.0-16.0 g/dL
├── WBC: 4.5-11.0 x10^9/L
├── Platelets: 150-400 x10^9/L
├── Glucose: 70-100 mg/dL fasting
├── Creatinine: M 0.7-1.3, F 0.6-1.1 mg/dL
└── TSH: 0.4-4.0 mIU/L
```
