# Common Pitfalls & Anti-Patterns

## 10.1 Medical Knowledge Errors

| Pitfall | Impact | Prevention |
|---------|--------|------------|
| Not keeping up with literature | Outdated practice | Daily reading |
| Relying on memory | Errors | Check references |
| Ignoring guidelines | Variation | Know guidelines |
| Not learning from errors | Repeated mistakes | Reflection |

## 10.2 Patient Care Errors

| Error | Impact | Prevention |
|-------|--------|------------|
| Incomplete H&P | Missed diagnosis | Thorough evaluation |
| Not following up labs | Delayed diagnosis | Check results |
| Ignoring patient concerns | Poor care | Listen carefully |
| Not examining patient | Missed findings | Physical exam |

## 10.3 Communication Failures

```
Common Issues:
├── Poor sign-out
├── Not updating patients/family
├── Conflict with team
├── Not escalating
├── Documentation unclear

Prevention:
├── Standardized handoff (I-PASS)
├── Family meetings scheduled
├── Professional communication
├── Escalation policy
├── Clear documentation
```

## 10.4 Professionalism Issues

| Issue | Impact | Prevention |
|-------|--------|------------|
| Duty hour violations | Patient safety | Follow rules |
| Impairment | Safety | Get help if impaired |
| Honesty | Trust | Always truthful |
| Boundaries | Professionalism | Maintain boundaries |

## 10.5 Documentation Errors

| Error | Consequence | Prevention |
|-------|-------------|------------|
| Copy-forward | Propagation error | Fresh documentation |
| Missing time | Legal risk | Date/time all entries |
| Incomplete | Quality | Template use |
| Late entries | Legal | Real-time documentation |

## 10.6 Quality Checklist

- [ ] H&P completed within 24 hours
- [ ] Daily progress notes documented
- [ ] All labs/imaging reviewed
- [ ] Consults communicated
- [ ] Patient/family updated
- [ ] Discharge planning begun
- [ ] Attending informed of issues
- [ ] Sign-out complete
- [ ] Duty hours documented
- [ ] Procedures documented
