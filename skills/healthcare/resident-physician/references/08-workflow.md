# Standard Workflow

## 8.1 Admission Workflow

```
Phase 1: Initial Assessment
├── H&P completed within 24 hours
├── Verify patient identity
├── Introduce yourself and role
├── Complete comprehensive H&P
├── Develop assessment and plan
└── Present to attending

Phase 2: Orders
├── Write admission orders
├── Attending approval required
├── Medication reconciliation
├── Diet, activity, nursing orders
├── Labs and imaging
└── Consults as needed

Phase 3: Ongoing Care
├── Daily progress notes
├── Labs and imaging review
├── Family discussions
├── Teaching rounds
├── Multidisciplinary rounds
└── Discharge planning
```

## 8.2 Procedure Workflow

```
Pre-Procedure:
├── Indication documented
├── Informed consent obtained
├── NPO verified
├── Orders placed
├── Timeout completed
└── Attending aware

Post-Procedure:
├── Recovery monitoring
├── Procedure note documented
├── Family update
├── Orders adjusted
└── Follow-up plan

Common Procedures:
├── Central line placement
├── Lumbar puncture
├── Arterial blood gas
├── Thoracentesis/paracentesis
├── Intubation
└── Arthrocentesis
```

## 8.3 Call Workflow

```
Handoff Preparation:
├── Update sign-out list
├── Prioritize patients
├── Anticipate issues
├── Update task list
└── Prepare cross-cover information

During Call:
├── New patient assessment
├── Cross-cover issues
├── Acute changes
├── Family questions
├── Orders as needed
└── Document all decisions

Handoff:
├── Verbal sign-out
├── Patient summary
├── Pending issues
├── Anticipated problems
└── Contact information
```

## 8.4 Discharge Workflow

```
Preparation:
├── Medical stability confirmed
├── Follow-up arranged
├── Medications reconciled
├── Patient education complete
├── Transportation arranged
└── Discharge summary drafted

Documentation:
├── Discharge summary
├── Procedure notes
├── Discharge medications
├── Follow-up instructions
└── Patient instructions

Teaching:
├── Diagnosis explanation
├── Medication instructions
├── Red flag signs
├── When to return
└── Follow-up plan
```
