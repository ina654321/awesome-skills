# Standard Workflow

## 8.1 Medication Therapy Management Workflow

```
Phase 1: Medication Reconciliation
├── Obtain best possible medication history (BPMH)
├── Compare with admission orders
├── Identify discrepancies (omissions, duplications, errors)
├── Clarify indication for each medication
└── Reconcile and document changes

Phase 2: Comprehensive Medication Review
├── Identify drug-related problems (DRPs)
├── Assess indication, effectiveness, safety, adherence
├── Review drug-drug, drug-disease, drug-food interactions
├── Evaluate renal/hepatic dosing appropriateness
├── Screen for Beers Criteria medications (≥65 years)
└── Assess cost-effective alternatives

Phase 3: Intervention & Monitoring
├── Prioritize interventions by clinical significance
├── Communicate recommendations to prescriber
├── Document interventions and outcomes
├── Develop monitoring plan for each DRP
└── Schedule follow-up

Phase 4: Follow-up & Documentation
├── Assess patient adherence and understanding
├── Evaluate therapeutic outcomes
├── Adjust plan based on response
└── Document in MTR (medication therapy review) note
```

## 8.2 Drug Interaction Analysis Workflow

1. **Identify**: List all medications including OTC, supplements, herbals
2. **Screen**: Check interaction databases (Lexicomp, Micromedex)
3. **Classify**: Determine severity (contraindicated/major/moderate/minor)
4. **Mechanism**: Understand PK (CYP450, P-gp) vs. PD (additive, synergistic, antagonistic)
5. **Act**: Avoid combination, separate administration, adjust dose, or monitor
6. **Document**: Note interaction in chart with management plan

## 8.3 Renal Dosing Workflow

1. **Calculate CrCl**: Cockcroft-Gault equation
   - Male: CrCl = [(140 - age) × weight × 0.85 if female] / (72 × SCr)
   - Use ideal body weight if obese; adjust for amputation
2. **Review Medications**: Identify drugs requiring renal adjustment
3. **Adjust**: Dose reduction or increased interval per guidelines
4. **Monitor**: Check drug levels and renal function
5. **Document**: Note dosing changes and rationale

## 8.4 Antimicrobial Stewardship Workflow

| Step | Action |
|------|--------|
| Empiric | Broad-spectrum based on likely pathogens and local antibiogram |
| Streamline | Narrow to pathogen-directed therapy when culture results available |
| De-escalate | Stop redundant coverage; remove redundant agents |
| IV to Oral | Convert when patient meets criteria (stable, absorbing) |
| Duration | Optimize treatment duration per IDSA guidelines |

## 8.5 Documentation Standards

| Note Type | Required Elements |
|-----------|------------------|
| Medication Reconciliation | Complete BPMH, discrepancies found/resolved, sources |
| Drug Information Query | Question, answer, references, clinical significance |
| MTR (Comprehensive) | DRPs identified, interventions, monitoring plan |
| Anticoagulation | Heparin-induced thrombocytopenia assessment, monitoring parameters |
