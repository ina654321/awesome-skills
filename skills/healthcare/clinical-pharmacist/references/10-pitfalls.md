# Common Pitfalls & Anti-Patterns

## 10.1 Drug Interaction Pitfalls

| # | Pitfall | Severity | Prevention |
|---|---------|----------|------------|
| 1 | Missing OTC/herbal interactions | 🔴 High | Ask specifically about supplements, herbals, vitamins |
| 2 | Ignoring food-drug interactions | 🔴 High | Review administration instructions for each drug |
| 3 | Not checking drug-disease interactions | 🟡 Medium | Consider comorbidities; avoid in hepatic/renal impairment |
| 4 | Assuming brand/generic equivalent | 🟡 Medium | Some drugs have clinically significant differences (levothyroxine, digoxin) |
| 5 | Over-reliance on interaction checkers | 🟡 Medium | Clinical judgment required; check primary literature |

## 10.2 Dosing Pitfalls

| Anti-Pattern | Correct Approach |
|--------------|------------------|
| Ignoring renal function | Always calculate CrCl before dosing renally cleared drugs |
| Standard dosing in obesity | Use adjusted body weight for some drugs (aminoglycosides) |
| Fixed-dose ignoring weight | Calculate weight-based doses for pediatrics, oncology, anticoagulation |
| Not accounting for drug interactions | Adjust doses when CYP450 interactions present |

## 10.3 Monitoring Pitfalls

| Pitfall | Correct Approach |
|---------|------------------|
| Vancomycin trough only | Use AUC/MIC target (400-600) with Bayesian software |
| Ignoring drug half-life | Time blood draws appropriately |
| Not monitoring for adverse effects | Know common ADEs for high-risk drugs |
| Assuming stable function | Reassess renal/hepatic function when clinical status changes |

## 10.4 Documentation Anti-Patterns

| Anti-Pattern | Correct Approach |
|--------------|------------------|
| "Accept" without explanation | Document why interaction accepted vs. avoided |
| Generic recommendations | Be specific: "reduce dose to X mg" not "use caution" |
| Not documenting interventions | Track time spent, recommendations made, acceptance rate |
| Missing follow-up plan | Document when to reassess and parameters to monitor |

## 10.5 Quality Assurance Checklist

- [ ] Complete medication history obtained (BPMH)
- [ ] Drug interactions screened (OTC, herbals included)
- [ ] Renal/hepatic function assessed for dosing
- [ ] Beers Criteria medications reviewed (≥65 years)
- [ ] Indication verified for each medication
- [ ] Cost-effective alternatives considered
- [ ] Recommendations communicated to prescriber
- [ ] Monitoring plan documented
- [ ] Patient counseling provided

## 10.6 High-Alert Medication Categories

1. **Anticoagulants**: Warfarin, heparin, LMWH, DOACs
2. **Insulin**: All insulin products (high-alert insulin)
3. **Opioids**: IV, high-dose, transdermal
4. **Chemotherapy**: Hazardous drugs requiring special handling
5. **IV Potassium**: Concentrated potassium chloride
6. ** neuromuscular blockers**: Paralytic agents
7. ** thrombolytics**: tPA, tenecteplase
8. **Inotropes**: High-dose dopamine, epinephrine
