# Common Pitfalls & Anti-Patterns

## 10.1 Product Development Errors

| Pitfall | Severity | Prevention |
|---------|----------|------------|
| Designing for lab conditions only | 🔴 High | Real-world testing with target users |
| Ignoring cognitive decline | 🔴 High | Include users with mild impairment |
| Over-complicated interfaces | 🟡 Medium | Iterative testing with seniors |
| Insufficient contrast ratios | 🟡 Medium | Follow WCAG 2.1 AAA standards |
| Small touch targets | 🟡 Medium | 48x48dp minimum for elderly |

## 10.2 Regulatory Mistakes

| Mistake | Impact | Correction |
|---------|--------|------------|
| Unclear device classification | Delayed approval | Pre-submission meeting with FDA |
| Incomplete risk analysis | Recall risk | Systematic risk management |
| Missing usability validation | 510(k) rejection | Formative + summative testing |
| Inadequate shelf-life data | Launch delay | Real-time stability testing |

## 10.3 User Research Failures

```
Common Issues:
├── Recruiting too homogeneous sample
├── Testing with younger staff, not actual users
├── Lab testing vs real-world conditions
├── Ignoring input from caregivers

Solutions:
├── Diverse recruitment (age, cognition, tech literacy)
├── Hire actual target users for testing
├── Field testing in home environments
├── Include care team perspectives
```

## 10.4 Accessibility Oversights

| Oversight | Effect | Mitigation |
|-----------|--------|------------|
| No voice output | Inaccessible to visually impaired | Text-to-speech features |
| Complex gestures | Motor skill challenges | Simple tap/swipe only |
| Small fonts | Vision impairment | Minimum 16pt, scalable |
| Color-coded info only | Color blindness | Add icons/text labels |

## 10.5 Clinical Validation Errors

| Error | Result | Prevention |
|-------|--------|------------|
| Testing with wrong population | Invalid data | Match to intended user profile |
| No longitudinal testing | Miss wearability issues | Extended use testing |
| Ignoring edge cases | User failures | Edge case testing |
| Poor documentation | Regulatory issues | Complete study records |

## 10.6 Market Launch Pitfalls

```
Common Issues:
├── Inadequate training materials
├── Poor customer support for seniors
├── Pricing too high for target market
├── Insufficient beta testing

Solutions:
├── Multi-format training (video, print, phone support)
├── Dedicated senior-friendly support line
├── Tiered pricing or insurance partnerships
├── Extended pilot program before full launch
```
