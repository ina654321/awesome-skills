# Standard Workflow

## 8.1 Emergency Response Workflow

```
Phase 1: Scene Size-up
├── Ensure scene safety (BSI precautions)
├── Determine number of patients
├── Request additional resources if needed
├── Identify mechanism of injury/illness
└── Consider hazardous materials/C-spine

Phase 2: Primary Assessment (ABCs)
├── Airway: Is it patent? C-spine control if trauma
├── Breathing: Rate, depth, quality, oxygenation
├── Circulation: Pulse, BP, bleeding control
├── Disability: GCS, pupillary response, motor function
└── Exposure: Full body exposure, temperature control

Phase 3: Secondary Assessment
├── Focused history (SAMPLE/OPQRST)
├── Head-to-toe examination
├── Vital signs (BP, pulse ox, ECG if indicated)
├── Baseline neurological status
└── Identify time-sensitive conditions

Phase 4: Transport Decision
├── Load-and-go vs. stay-and-play
├── Closest appropriate facility
├── Alert receiving facility
└── Ongoing monitoring and treatment
```

## 8.2 Cardiac Emergency Workflow

### Adult Cardiac Arrest (ACLS)

1. **Check responsiveness; activate code; get AED/defibrillator**
2. **CPR: Push hard (2-2.4"), fast (100-120/min), avoid hyperventilation**
3. **Check rhythm**: Shockable vs. non-shockable
4. **Shockable (VF/pVT)**: Defibrillate ASAP (200J biphasic)
5. **CPR 2 minutes; Epi 1mg IV/IO every 3-5 min**
6. **Consider H's and T's**: Hypovolemia, hypoxia, hyper/hypokalemia, hypothermia, tension pneumo, tamponade, toxins, thrombosis
7. **Reversible causes addressed; advanced airway when appropriate**

### Acute Coronary Syndrome

1. **MONA**: Morphine, Oxygen (if SpO2 <94%), Nitro, Aspirin (324mg chewable)
2. **P2Y12**: Clopidogrel 600mg, ticagrelor 180mg, or prasugrel 60mg
3. **12-lead ECG within 10 minutes**
4. **STEMI criteria met**: Activate cath lab; door-to-balloon goal <90 min
5. **Ongoing**: Monitor, establish IV, pain management, beta-blocker if stable

## 8.3 Trauma Workflow

1. **Primary Survey**: Identify and treat life threats
2. **Airway**: Jaw thrust if C-spine; definitive airway if GCS ≤8
3. **Breathing**: Decompress tension pneumo; seal open pneumo
4. **Circulation**: Control bleeding; 2 large bore IVs; TXA if indicated
5. **Disability**: GCS, pupils, glucose check
6. **Exposure**: Full trauma assessment; prevent hypothermia
7. **Transport**: Consider direct to trauma center; early notification

## 8.4 Documentation Standards

| Report Element | Required Content |
|----------------|------------------|
| Dispatch | Time, location, nature of call |
| Scene | Hazards, bystanders, mechanism |
| Assessment | VS, GCS, pertinent findings |
| Treatment | All interventions with times |
| Transport | Destination, patient response |
| Transfer | Verbal report to receiving RN |
