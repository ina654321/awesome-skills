# Standards & Reference

## 7.1 Epidemiology Standards

- [CDC Guidelines](https://www.cdc.gov/infectioncontrol/guidelines/index.html) - Infection control guidelines
- [WHO Disease Outbreak News](https://www.who.int/emergencies/disease-outbreak-news) - International reporting
- [APHA Control of Communicable Diseases](https://www.apha.org) - Reference manual
- [Epi Info](https://www.cdc.gov/epiinfo/index.html) - CDC software tool
- [CIDRAP](https://www.cidrap.umn.edu) - Antimicrobial resistance and outbreak data

## 7.2 Case Definitions (CDC/WHO)

| Classification | Criteria |
|----------------|----------|
| Suspected case | Presenting symptoms consistent with disease |
| Probable case | Suspected + epidemiological link OR positive lab without confirmation |
| Confirmed case | Laboratory confirmation (gold standard) |
| Contact | Exposed to confirmed case during infectious period |

## 7.3 Basic Reproduction Number (R0)

```
R0 Interpretation:
├── R0 < 1: Outbreak will self-terminate
├── R0 = 1: Endemic steady state
└── R0 > 1: Epidemic potential

R0 Estimation Factors:
├── Contact rate (frequency and duration)
├── Transmission probability per contact
└── Infectious period
```

## 7.4 Surveillance Indicators

| Indicator | Formula | Threshold |
|-----------|---------|-----------|
| Attack rate | New cases/population at risk × 100 | Outbreak definition |
| Secondary attack rate | Secondary cases/contacts × 100 | Household transmission |
| Case fatality rate | Deaths/cases × 100 | Severity assessment |
| Incidence density | Events/person-time | Dynamic measure |
