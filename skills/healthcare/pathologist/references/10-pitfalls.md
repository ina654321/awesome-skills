# Common Pitfalls & Anti-Patterns

## 10.1 Diagnostic Errors

| Pitfall | Impact | Prevention |
|---------|--------|------------|
| Sampling error | Missed diagnosis | Adequate sampling |
| Misinterpretation | Wrong diagnosis | Experience, consultation |
| Clinical correlation missing | Inappropriate | Always correlate |
| Not consulting | Uncertainty ignored | Expert consultation |

## 10.2 Communication Errors

| Error | Consequence | Prevention |
|-------|-------------|------------|
| Critical result not called | Delayed treatment | Call critical results |
| Misunderstanding by clinician | Wrong treatment | Clarify ambiguities |
| Delayed reporting | Treatment delay | Turnaround time |
| Illegible/ambiguous | Misinterpretation | Clear documentation |

## 10.3 Quality Issues

```
Common Problems:
├── Inadequate clinical history
├── Specimen misidentification
├── Improper fixation
├── Block mislabeling
├── Slide mix-up
└── Incomplete reports

Prevention:
├── Two identifiers
├── Barcode tracking
├── Tissue tracking
├── Standardized protocols
└── Quality metrics
```

## 10.4 Report Deficiencies

| Deficiency | Impact | Prevention |
|-----------|--------|------------|
| Missing staging | Incomplete | Checklist |
| No comment when needed | Uncertainty | Judgment call |
| Vague diagnosis | Unactionable | Be specific |
| Missing follow-up | Missed cancer | Alert system |

## 10.5 Safety Issues

| Issue | Risk | Mitigation |
|-------|------|------------|
| Specimen swap | Wrong diagnosis | Barcoding |
| Contamination | False positive | Clean tools |
| Slide mislabeling | Wrong patient | Verification |
| Report alteration | Fraud | Audit trail |

## 10.6 Quality Checklist

- [ ] Clinical history adequate
- [ ] Specimen labeled correctly
- [ ] Gross description complete
- [ ] Adequate sections submitted
- [ ] Fixation proper
- [ ] Diagnosis specific
- [ ] Staging complete
- [ ] Comments when needed
- [ ] Critical results called
- [ ] Report verified before signing
