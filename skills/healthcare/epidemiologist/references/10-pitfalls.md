# Common Pitfalls & Anti-Patterns

## 10.1 Study Design Pitfalls

| # | Pitfall | Severity | Prevention |
|---|---------|----------|------------|
| 1 | Wrong study design selection | 🔴 High | Match design to research question and outbreak context |
| 2 | Selection bias in controls | 🔴 High | Use population-based controls; matched design for rare exposures |
| 3 | Recall bias in interviews | 🟡 Medium | Use objective data when possible; validate self-report |
| 4 | Ignoring confounding | 🔴 High | Stratify or adjust for known confounders |
| 5 | Multiple comparisons without correction | 🟡 Medium | Bonferroni correction or FDR adjustment |

## 10.2 Analysis Pitfalls

| Anti-Pattern | Correct Approach |
|--------------|------------------|
| Focusing on p-value only | Report effect size and 95% CI; statistical ≠ clinical significance |
| Ignoring missing data | Multiple imputation; sensitivity analysis |
| Assuming linear relationships | Check linearity assumptions; use splines when appropriate |
| Overadjustment | Include confounders; don't adjust for intermediates |
| Ecologic fallacy | Individual-level data for individual-level conclusions |

## 10.3 Surveillance System Pitfalls

| Pitfall | Correct Approach |
|---------|------------------|
| Passive surveillance only | Supplement with active case finding |
| Case definition too narrow | Balance sensitivity and specificity |
| Not feeding back data | Regular reports to reporting sites |
| Ignoring underreporting | Estimate completeness; adjust for underascertainment |
| Data quality poor | Train reporters; implement data quality checks |

## 10.4 Communication Pitfalls

| Anti-Pattern | Correct Approach |
|--------------|------------------|
| Overstating certainty | Report uncertainty; use appropriate language |
| Releasing incomplete data | Wait for validated counts; correct promptly |
| Ignoring community concerns | Address misinformation; engage stakeholders |
| Delayed reporting | Timely communication is critical for outbreaks |

## 10.5 Quality Assurance Checklist

- [ ] Case definition established and documented
- [ ] Line list complete with key variables
- [ ] Epidemic curve constructed
- [ ] Appropriate statistical methods applied
- [ ] Effect measures reported with 95% CI
- [ ] Limitations acknowledged
- [ ] Findings validated with stakeholders
- [ ] Report prepared per STROBE/CONSORT guidelines
- [ ] Recommendations evidence-based and actionable
- [ ] Lessons learned documented

## 10.6 Ethical Considerations

1. **Informed Consent**: Protect participants in research; public health surveillance often exempt
2. **Privacy**: Protect identifiable information; follow HIPAA/GDPR
3. **Equity**: Ensure surveillance captures all populations, including vulnerable groups
4. **Conflict of Interest**: Disclose all funding sources and conflicts
5. **Responsible Reporting**: Avoid stigmatization; frame findings constructively
