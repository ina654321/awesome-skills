# Standard Workflow

## 8.1 Outbreak Investigation Workflow (WHO 10 Steps)

```
Phase 1: Preparation
├── Verify outbreak exists
├── Define case definition
├── Identify investigation team
└── Develop investigation plan

Phase 2: Descriptive Epidemiology
├── Count cases (who, when, where)
├── Create epidemic curve
├── Calculate attack rates
└── Generate hypotheses

Phase 3: Analytical Epidemiology
├── Select study design (case-control, cohort)
├── Select controls (population-based, hospital)
├── Calculate OR/RR with 95% CI
└── Identify risk factors

Phase 4: Environmental/Laboratory Investigation
├── Environmental assessment
├── Laboratory testing of specimens
├── Trace-back/trace-forward studies
└── Match findings to hypothesis

Phase 5: Control and Prevention
├── Implement control measures
├── Communication and reporting
└── Monitor intervention effectiveness

Phase 6: Follow-up and Communication
├── Document lessons learned
├── Prepare outbreak report
└── Share findings with stakeholders
```

## 8.2 Study Design Selection

| Question | Best Design | Why |
|----------|-------------|-----|
| What caused this outbreak? | Case-control | Rapid; good for rare outcomes |
| What is the risk ratio for exposure? | Cohort | Provides RR directly |
| Is the vaccine working? | Test-negative design | Efficient for ongoing surveillance |
| What is incidence over time? | Longitudinal cohort | Tracks new cases |

## 8.3 Statistical Analysis Workflow

1. **Descriptive Statistics**
   - Frequency distributions
   - Means, medians, proportions
   - Cross-tabulations by exposure/outcome

2. **Bivariate Analysis**
   - Chi-square or Fisher's exact for categorical
   - t-test or Mann-Whitney for continuous
   - Calculate OR or RR with 95% CI

3. **Stratified Analysis**
   - Mantel-Haenszel method for confounding
   - Effect modification identification

4. **Multivariable Analysis**
   - Logistic regression for case-control
   - Cox regression for survival
   - Poisson regression for incidence rates

5. **Sensitivity Analysis**
   - Multiple imputation for missing data
   - Propensity score matching
   - Probabilistic bias analysis

## 8.4 Surveillance System Evaluation

| Component | Questions |
|-----------|-----------|
| Sensitivity | What proportion of true cases are detected? |
| Specificity | How many false positives occur? |
| Representativeness | Does system capture all affected populations? |
| Timeliness | How quickly are cases detected and reported? |
| Stability | Is the system reliable and funded? |

## 8.5 Documentation Standards

| Document | Required Elements |
|---------|-------------------|
| Case Investigation Form | Demographics, exposure history, symptoms, lab results |
| Line List | Case variables for descriptive analysis |
| Epidemic Curve | Date of onset, case counts by time |
| Analytical Report | Study design, measures of association, conclusions |
| Outbreak Summary | Timeline, findings, recommendations |
