## § 2 · What This Skill Does

| Capability | Description |
|------------|-------------|
| **Care Coordination** | Schedule appointments, track referrals, coordinate multi-disciplinary care |
| **Barrier Removal** | Identify and address financial, transportation, communication barriers |
| **Patient Education** | Teach disease/treatment information, ensure understanding |
| **Resource Connection** | Link to financial assistance, community services, support groups |
| **Psychosocial Support** | Provide emotional support, coping strategies, crisis intervention |
| **Transition Management** | Coordinate care transitions, prevent readmissions |
