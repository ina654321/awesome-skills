# Standard Workflow

## 8.1 Assistive Technology Assessment

```
Phase 1: Referral and Intake
├── Client referral source
├── Medical diagnosis
├── Functional status
├── Living situation
├── Funding source
└── User goals

Phase 2: Comprehensive Evaluation
├── Physical assessment
├── Cognitive assessment
├── Sensory assessment
├── Task analysis
├── Environmental assessment
├── Trial equipment
└── Technology assessment

Phase 3: Device Selection
├── Match technology to needs
├── Trial and compare options
├── Obtain funding authorization
├── Order equipment
└── Schedule fitting/delivery
```

## 8.2 Wheelchair Assessment

```
Seating Assessment:
├── Postural support needs
├── Skin integrity risk
├── Range of motion
├── Strength
├── Balance
└── Propulsion ability

Equipment Selection:
```
Ultra-lightweight wheelchair:
├── Active users
├── Independent propulsion
├── Custom configuration
├── Weight <15 lbs
└── Carbon fiber frame

Power wheelchair:
├── Limited upper extremity function
├── Long distance mobility
├── Complex positioning needs
├── Power seating (tilt, recline, elevating legrests)
└── Alternative access methods
```

## 8.3 Technology Implementation

```
Device Configuration:
├── Physical setup
├── Software configuration
├── Access method calibration
├── Training on features
├── Safety features
└── Maintenance plan

Training Sequence:
├── Basic operation
├── Daily use tasks
├── Troubleshooting
├── Maintenance
├── Emergency procedures
└── Caregiver training
```

## 8.4 Follow-up and Support

```
Post-Delivery:
├── 30-day follow-up
├── 90-day follow-up
├── Annual review
├── Equipment adjustment
├── Troubleshooting
└── Equipment upgrade assessment
```
