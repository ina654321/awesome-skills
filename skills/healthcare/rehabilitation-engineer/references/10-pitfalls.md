# Common Pitfalls & Anti-Patterns

## 10.1 Assessment Errors

| Pitfall | Impact | Prevention |
|---------|--------|------------|
| Incomplete assessment | Wrong device | Comprehensive eval |
| Ignoring future needs | Equipment obsolete | Consider progression |
| Not observing real use | Poor fit | Home/school observation |
| Funding-driven selection | User needs unmet | User needs first |

## 10.2 Equipment Selection Errors

| Error | Impact | Prevention |
|-------|--------|------------|
| Over-prescribing features | Wasted resources | Match to needs |
| Under-prescribing | Equipment inadequate | Address all needs |
| Not trialing | Poor fit | Multiple trials |
| Ignoring user preference | Abandonment | User involvement |

## 10.3 Implementation Failures

```
Common Issues:
├── Poor training
├── Inadequate follow-up
├── Caregiver not trained
├── Environmental barriers
├── Maintenance not addressed
└── Funding issues

Prevention:
├── Comprehensive training
├── Regular follow-up schedule
├── Family/caregiver training
├── Home evaluation
├── Maintenance plan
└── Funding support resources
```

## 10.4 Technology Issues

| Issue | Impact | Prevention |
|-------|--------|------------|
| Too complex | Abandonment | Match to ability |
| Not portable | Non-use | Consider environments |
| Poor support | Frustration | Vendor support |
| Not maintained | Failure | Maintenance plan |

## 10.5 Documentation Issues

| Error | Consequence | Prevention |
|-------|-------------|------------|
| Incomplete eval | Funding denial | Complete forms |
| No goal documentation | Denied | SMART goals |
| Missing medical necessity | Denial | Clinical justification |
| No trial documentation | Denial | Trial records |

## 10.6 Quality Checklist

- [ ] Comprehensive assessment complete
- [ ] User goals documented
- [ ] All environments assessed
- [ ] Equipment trials conducted
- [ ] Funding requirements met
- [ ] Training plan established
- [ ] Follow-up scheduled
- [ ] Maintenance plan documented
- [ ] Family/caregiver trained
- [ ] Outcomes measured
