# Standard Workflow

## 8.1 Office Visit Workflow

```
Phase 1: Rooming Patient
├── Verify demographics and contact info
├── Chief complaint documented
├── Vital signs collected
├── Medication reconciliation
├── Allergies confirmed
└── Update problem list

Phase 2: Clinical Encounter
├── History of present illness
├── Review of systems
├── Physical examination
├── Assessment and plan
└── Patient education

Phase 3: Documentation and Follow-up
├── Complete SOAP note
├── Order appropriate tests
├── Provide visit summary
├── Schedule follow-up
└── After-visit summary for patient
```

## 8.2 Chronic Disease Management

```
Diabetes Care Protocol:
├── HbA1c every 3-6 months (goal <7%)
├── Annual lipid panel (LDL <100)
├── Annual kidney function (creatinine, UACR)
├── Annual eye exam (retinopathy screen)
├── Annual foot exam
├── BP control (<140/90)
└── Tobacco cessation counseling

Hypertension Management:
├── BP at every visit
├── Home BP monitoring recommended
├── Lab workup (BMP, UA, lipid panel)
├── ECG for baseline
├── Lifestyle modification first
└── Medication titration per guidelines
```

## 8.3 Acute Care Decision Tree

```
Fever + Respiratory Symptoms:
├── Assess severity (vital signs, O2 sat)
├── Risk stratify (Centor criteria for strep)
├── Test appropriately (rapid strep, flu, COVID)
├── Treat symptomatically vs antibiotics
└── Follow-up if not improving in 48-72h

Chest Pain:
├── Rule out ACS (ECG, troponin, risk score)
├── HEART score for ED disposition
├── Stable patient: outpatient workup
└── Unstable: Emergency referral
```

## 8.4 Referral Workflow

| Referral Type | Timeframe | Documentation |
|---------------|-----------|---------------|
| Urgent (chest pain, stroke) | Same day | Direct handoff |
| Routine specialty | Within 1 week | Referral note |
| Mental health | Within 2 weeks | Warm handoff preferred |
| Social services | As needed | Community resource list |
