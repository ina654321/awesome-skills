# Common Pitfalls & Anti-Patterns

## 10.1 Diagnostic Errors

| Pitfall | Severity | Prevention |
|---------|----------|------------|
| Anchoring bias | 🔴 High | Consider alternatives |
| Premature closure | 🔴 High | Reassess before finalizing |
| Availability heuristic | 🟡 Medium | Broaden differential |
| Confirmation bias | 🟡 Medium | Seek disconfirming evidence |
| Anchoring on first impression | 🟡 Medium | Structured history review |

## 10.2 Treatment Errors

| Pitfall | Prevention |
|---------|------------|
| Prescribing cascade | Review medications before adding new |
| Inappropriate antibiotic use | Viral illness: no antibiotics |
| Drug-drug interactions | Check interaction database |
| Omission of indicated therapy | Clinical decision support |
| Inadequate follow-up | Schedule before leaving |

## 10.3 Communication Failures

```
Common Issues:
├── Using medical jargon
├── Not confirming understanding
├── Inadequate informed consent
├── Poor handoff communication
└── Failure to address patient concerns

Prevention:
├── Teach-back method
├── Plain language explanations
├── Motivational interviewing
├── Standardized handoff (SBAR)
└── Shared decision-making
```

## 10.4 Documentation Errors

| Error | Impact | Prevention |
|-------|--------|------------|
| Copy-forward errors | Wrong data propagated | Review before signing |
| Missing time stamps | Legal risk | Contemporaneous documentation |
| Vague assessments | Poor continuity | Specific diagnoses |
| Omitted relevant negatives | Incomplete picture | Systematic ROS |

## 10.5 Preventive Care Oversights

```
Common Misses:
├── Incomplete vaccination history
├── Overdue cancer screenings
├── Missed cardiovascular risk factors
├── Failure to address tobacco/alcohol
└── Mental health not screened

Prevention:
├── Use preventive care checklist
├── Standing orders for immunizations
├── Panel management tools
└── Annual wellness visit template
```

## 10.6 Quality Checklist

- [ ] Chief complaint addressed
- [ ] Vital signs reviewed
- [ ] Medication reconciliation complete
- [ ] Allergies confirmed
- [ ] Preventive care up to date
- [ ] Assessment specific
- [ ] Plan documented
- [ ] Patient education provided
- [ ] Follow-up arranged
- [ ] Referrals placed if needed
