# Standard Workflow

## 8.1 Site Initiation

```
Phase 1: Feasibility Assessment
├── Site capability evaluation
├── Staff qualification review
├── IRB/IEC submission
├── Contract and budget negotiation
└── Essential document collection

Phase 2: Site Initiation Visit (SIV)
├── Protocol review with investigators
├── GCP and ICH guidelines training
├── Safety reporting procedures
├── EDC system training
├── Investigational product handling
└── Delegation of authority finalization

Phase 3: Activation
├── IRB/IEC approval obtained
├── Investigational product shipped
├── Staff delegation logged
└── First subject ready screening
```

## 8.2 Subject Enrollment Workflow

```
Step 1: Screening
├── Confirm inclusion/exclusion criteria
├── Obtain informed consent
├── Complete screening assessments
└── Assign screening number

Step 2: Enrollment
├── Verify eligibility
├── Assign randomization number
├── Schedule baseline visit
└── Document enrollment date

Step 3: Study Visits
├── Protocol-specified assessments
├── Adverse event collection
├── Concomitant medication review
└── IP accountability
```

## 8.3 Monitoring Visit Workflow

```
Pre-Visit:
├── Review previous visit data
├── Identify outstanding queries
├── Plan site visit activities

During Visit:
├── Source document verification
├── IP accountability check
├── AE/SAE review
├── Protocol deviation assessment
└── Staff interview

Post-Visit:
├── Visit report within 5 business days
├── CAPA follow-up if needed
└── Update trial master file
```

## 8.4 Safety Reporting Process

```
SAE Identification:
├── Event serious? (death, life-threatening, hospitalization, disability, etc.)
├── Related to IP? (unrelated, possibly, probably, definitely)
└── Expected vs unexpected?

Reporting:
├── Suspected unexpected SAR (SUSAR) → Sponsor within 24h
├── IND safety report → FDA within 7/15 days
└── Deviation from protocol → Document and assess impact
```
