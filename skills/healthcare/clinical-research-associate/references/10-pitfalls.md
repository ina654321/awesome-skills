# Common Pitfalls & Anti-Patterns

## 10.1 Informed Consent Errors

| Pitfall | Severity | Prevention |
|---------|----------|------------|
| Missing consent pages | 🔴 High | Document checklist review |
| Using outdated consent version | 🔴 High | Verify version date each time |
| Consent obtained after procedures | 🔴 High | Schedule consent before any study activity |
| Subject unable to read consent | 🟡 Medium | Provide oral translation, document process |
| Witness not used when required | 🟡 Medium | Know protocol requirements |

## 10.2 Source Document Verification Errors

| Pitfall | Prevention |
|---------|------------|
| Retrospective documentation | Contemporaneous recording |
| Transcription errors | Double-entry verification |
| Missing data | Check for completeness before visit end |
| Protocol-required values not collected | Pre-visit checklist |

## 10.3 IP Accountability Issues

```
Common Issues:
├── Drug not stored at required temperature
├── Missing bottles/unused returns
├── Dispensation errors (wrong dose)
├── Subject non-compliance (missed doses)

Corrective Actions:
├── Temperature excursion reporting
├── Accountability log reconciliation
├── Re-education of site staff
└── Protocol deviation documentation
```

## 10.4 Monitoring Visit Deficiencies

| Finding | CAPA Required |
|---------|---------------|
| Missing source documents | Retrieve or document reason |
| Protocol deviation without documentation | Retrospective documentation if possible |
| Repeated data entry errors | Re-training, double-check process |
| Enrollment below projections | Recruitment action plan |

## 10.5 Safety Reporting Failures

| Pitfall | Severity | Corrective Action |
|---------|----------|-------------------|
| SAE reported late | 🔴 High | Implement 24-hour rule system |
| Missing SAE follow-up | 🟡 Medium | Set calendar reminder for follow-ups |
| Relatedness not assessed | 🟡 Medium | Protocol training, sponsor guidance |
| Unblinding without authorization | 🔴 High | Clear protocol communication |

## 10.6 Essential Document Management

```
Missing Documents (Before Trial Start):
├── Investigator CV (not dated/signed)
├── Financial disclosure forms
├── Lab normal ranges (not protocol-specific)
├── Site signature log incomplete

Prevention:
├── Use trial master file (TMF) checklist
├── Central document repository
└── Regular document reconciliation
```
