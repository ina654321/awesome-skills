# Common Pitfalls & Anti-Patterns

## 10.1 Variant Interpretation Errors

| Pitfall | Severity | Prevention |
|---------|----------|------------|
| Over-interpreting VUS | 🔴 High | Never treat VUS as pathogenic |
| Ignoring conflicting evidence | 🔴 High | Review all evidence equally |
| Missing mosaicism | 🟡 Medium | Consider technical limitations |
| Incomplete segregation | 🟡 Medium | Request family studies |
| Wrong inheritance assumption | 🔴 High | Verify genetic mechanism |

## 10.2 Technical Errors

| Error | Impact | Detection |
|-------|--------|----------|
| Sample swap | Wrong diagnosis | Cross-check identity |
| Contamination | False positives | QC metrics |
| Low coverage | Missed variants | Coverage thresholds |
| Poor alignment | Indel errors | Alignment QC |

## 10.3 Database Misuse

```
Common Mistakes:
├── Over-relying on population databases
├── Ignoring novel variants
├── Misinterpreting allele frequencies
├── Confusing variant with disease
└── Missing update to new versions

Best Practices:
├── Cross-reference multiple databases
├── Check recent literature
├── Consider local population
├── Distinguish variant significance
└── Update knowledge regularly
```

## 10.4 Reporting Errors

| Error | Consequence | Prevention |
|-------|-------------|------------|
| Wrong HGVS nomenclature | Confusion | Standard notation |
| Vague interpretations | Inactionable | Use standard classifications |
| Missing limitations | Overconfidence | Document test limitations |
| Omitting incidental findings | Missed diagnosis | Follow institutional policy |

## 10.5 Ethical Considerations

| Issue | Guidance |
|-------|----------|
| Incidental findings | Follow ACMG recommendations |
| Pediatric testing | Parental consent required |
| Genetic discrimination | GINA protection |
| Privacy | HIPAA compliance |
| Cascade testing | Family communication plan |

## 10.6 Quality Checklist

- [ ] Sample identity verified
- [ ] Quality metrics acceptable
- [ ] Coverage adequate
- [ ] Variants validated if needed
- [ ] ACMG criteria applied correctly
- [ ] Literature reviewed current
- [ ] Phenotype correlation done
- [ ] Report reviewed by qualified professional
- [ ] Recommendations actionable
- [ ] Patient/family informed appropriately
