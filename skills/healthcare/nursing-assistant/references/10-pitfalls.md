# Common Pitfalls & Anti-Patterns

## 10.1 Patient Safety Errors

| Pitfall | Severity | Prevention |
|---------|----------|------------|
| Falls during transfer | 🔴 High | Use gait belt, get help |
| Leaving confused patient alone | 🔴 Critical | Stay with patient |
| Medication error | 🔴 Critical | Never administer meds |
| Ignoring call light | 🟡 Medium | Answer promptly |
| Improper lifting | 🟡 Medium | Use proper body mechanics |

## 10.2 Infection Control Violations

| Violation | Impact | Prevention |
|-----------|--------|------------|
| Skipping hand hygiene | 🔴 High | Hand washing every time |
| Sharing equipment | 🔴 High | Dedicate or disinfect |
| Not wearing PPE | 🔴 High | Follow precaution signs |
| Poor perineal care | 🟡 Medium | Thorough cleaning |

## 10.3 Documentation Errors

```
Common Issues:
├── Missing documentation
├── Not documenting intake/output
├── Late entries
├── Illegible writing
├── Forgetting to sign

Prevention:
├── Document in real-time
├── Use proper forms
├── Sign with name and date
├── Use exact values, not "WNL"
```

## 10.4 Resident Rights Violations

```
Violations:
├── Privacy invasion
├── Rushing care
├── Using condescending tone
├── Not respecting preferences
├── Withholding food/drink

Prevention:
├── Knock before entering
├── Close doors during care
├── Allow adequate time
├── Respect choices
└── Offer alternatives
```

## 10.5 Communication Failures

| Failure | Impact | Solution |
|---------|--------|----------|
| Not reporting changes | Patient harm | Report to nurse |
| Incomplete handoff | Continuity | SBAR format |
| Ignoring complaints | Missed problems | Take seriously |
| Gossiping | Trust broken | Professionalism |

## 10.6 Quality Checklist

- [ ] Vital signs measured and documented
- [ ] Call light within reach
- [ ] Bed in lowest position
- [ ] Personal hygiene completed
- [ ] Residents repositioned q2h
- [ ] Safety devices working
- [ ] Hand hygiene performed
- [ ] PPE used appropriately
- [ ] Intake/output documented
- [ ] Changes reported to nurse
