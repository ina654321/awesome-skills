# Common Pitfalls & Anti-Patterns

## 10.1 Documentation Errors

| Pitfall | Impact | Prevention |
|---------|--------|------------|
| Upcoding | Fraud investigation | Accurate code selection |
| Under-coding | Revenue loss | Documentation education |
| Unbundling | Audit penalties | Bundle awareness |
| Missing CC/MCC | Lower DRG | Clinical review |

## 10.2 Sequencing Mistakes

| Error | Result | Correct Approach |
|-------|--------|------------------|
| Wrong principal Dx | Wrong DRG | Most resource-intensive |
| Missing complication | Lower weight | Document all conditions |
| Unbundled codes | Audit finding | Follow CCI edits |

## 10.3 Modifier Errors

```
Common Mistakes:
├── -25 used inappropriately
├── -59 misapplied
├── E/M modifier without basis
├── Ambulatory payment classification errors
└── Site-of-service errors

Prevention:
├── Modifier education
├── Clear documentation
├── Audit of modifier use
└── Follow CPT guidelines
```

## 10.4 Documentation Deficiencies

| Deficiency | Impact | Solution |
|-----------|--------|----------|
| Vague diagnoses | Query required | Clarify with provider |
| Missing signature | Unbillable | Signature requirements |
| Incomplete H&P | E/M level dispute | Template use |
| No treatment plan | Denials | Care plan documentation |

## 10.5 Compliance Issues

```
High-Risk Areas:
├── Cloned documentation
├── Template overuse
├── Inconsistency between notes
├── Auto-populated without review
└── Evaluation and management upgrades

Compliance Program:
├── Regular audits
├── Education and training
├── Documentation improvement
├── Query process
└── Corrective action plans
```

## 10.6 Quality Checklist

- [ ] Documentation supports codes assigned
- [ ] Diagnosis codes accurate and specific
- [ ] Procedure codes linked to diagnosis
- [ ] Modifiers applied correctly
- [ ] DRG assignment validated
- [ ] Queries documented when needed
- [ ] Compliance guidelines followed
- [ ] Timely completion within 24-48 hours
- [ ] Quality review performed
- [ ] Coding guidelines applied current year
