# Standard Workflow

## 8.1 Territory Planning

```
Phase 1: Account Analysis
├── Identify target accounts
├── Assess prescribing patterns
├── Determine call frequency
├── Map decision makers
└── Identify opportunities

Phase 2: Route Planning
├── Optimize visit sequence
├── Schedule appointments
├── Plan discussion topics
├── Prepare materials
└── Review call notes

Phase 3: Execution
├── Professional greeting
├── Establish rapport
├── Conduct business discussion
├── Leave appropriate materials
└── Document call
```

## 8.2 Sales Call Workflow

```
Step 1: Pre-Call Planning
├── Review account history
├── Set call objective
├── Prepare key messages
├── Gather materials
└── Schedule timing

Step 2: Call Execution
├── Introduce self and company
├── Identify needs/concerns
├── Present product benefits
├── Address objections
├── Close for next step

Step 3: Post-Call
├── Document call in CRM
├── Note follow-up needed
├── Process samples if given
├── Send thank you if appropriate
└── Prepare for next call
```

## 8.3 Objection Handling

```
Common Objections:
├── "Not interested"
├── "Already using competitor"
├── "Too expensive"
├── "Side effect concerns"
├── "Need more data"

Handling Framework:
├── Listen completely
├── Acknowledge concern
├── Clarify if needed
├── Respond with value
├── Confirm understanding
```

## 8.4 Sample Dispensing

```
Process:
1. Verify HCP license (if not on file)
2. Document sample request
3. Obtain signature on log
4. Dispense appropriate quantity
5. Record in perpetual inventory
6. Provide prescribing information
```

## 8.5 formulary Entry Process

```
Stakeholders:
├── P&T Committee members
├── Pharmacy director
├── Medical director
├── CFO/pharmacy buyer
└── Key opinion leaders

Process:
├── Identify formulary opportunities
├── Gather clinical data
├── Develop value proposition
├── Present to committee
├── Address concerns
└── Negotiate terms
```
