# Standard Workflow

## 8.1 Initial Clinical Assessment

```
Phase 1: Referral Intake & Triage
├── Verify referral information
├── Review presenting problem
├── Check previous psychiatric history
├── Assess urgency (safety concerns → immediate action)
└── Schedule initial appointment

Phase 2: Comprehensive Evaluation
├── Chief complaint clarification
├── History of present illness (SAD CAGES/OPQRST)
├── Psychiatric history (prior episodes, treatments, hospitalizations)
├── Medical history (conditions, medications)
├── Substance use assessment (AUDIT, DAST-10)
├── Trauma history (PCL-5)
├── Social history (occupation, relationships, support)
├── Family psychiatric history
├── Mental status examination (MSE)
└── Risk assessment (suicide, self-harm, violence)

Phase 3: Diagnostic Formulation
├── Differential diagnosis
├── Rule out medical causes
├── Consider comorbidities
├── Apply DSM-5-TR criteria
└── Document clinical impression

Phase 4: Treatment Planning
├── Evidence-based intervention selection
├── Goals and objectives (SMART)
├── Frequency and duration
├── Collaboration with prescriber if medication indicated
└── Informed consent process
```

## 8.2 Common Treatment Workflows

### Cognitive Behavioral Therapy Workflow

1. **Session 1-2**: Build therapeutic alliance; cognitive conceptualization
2. **Session 3-4**: Thought records and cognitive restructuring introduction
3. **Session 5-8**: Behavioral experiments; exposure work
4. **Session 9-12**: Relapse prevention; termination planning

### Crisis Intervention Workflow

1. **Immediate**: Assess safety (plan, intent, means access)
2. **Stabilize**: De-escalation; grounding techniques
3. **Connect**: Mobilize support systems; crisis resources
4. **Follow-up**: Safety plan documentation; next-day contact

## 8.3 Documentation Standards

| Document | Required Elements | Timing |
|----------|-------------------|--------|
| Initial Evaluation | All MSE domains, diagnoses, treatment plan | Within 48h |
| Progress Note | Subjective, Objective, Assessment, Plan (SOAP) | Each session |
| Risk Assessment | Specific ideation, plan, intent, means, protective factors | Each encounter |
| Treatment Plan | Goals, interventions, modalities, target dates | Annually |
