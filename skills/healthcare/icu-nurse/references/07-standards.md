## 7.2 Hemodynamic Monitoring Parameters

| Parameter | Normal Range | Interpretation |
|-----------|-------------|----------------|
| MAP | 70-100 mmHg | Maintain ≥65 for organ perfusion |
| CVP | 2-8 mmHg | Right heart preload indicator |
| PAOP/PCWP | 6-12 mmHg | Left heart preload indicator |
| Cardiac Output | 4-8 L/min | CO = HR × SV |
| SVR | 800-1200 dynes/sec/cm⁻⁵ | Systemic vascular resistance |
| PVR | 50-150 dynes/sec/cm⁻⁵ | Pulmonary vascular resistance |

## 7.3 Vasoactive Medication Reference

| Drug | Dose Range | Mechanism | Nursing Considerations |
|------|------------|-----------|----------------------|
| Norepinephrine | 0.01-3 mcg/kg/min | α1 > β1 | Central line preferred; monitor for extravasation |
| Dopamine | 2-20 mcg/kg/min | Dose-dependent | Low=renal, mid=cardiac, high=vasopressor |
| Epinephrine | 0.01-0.5 mcg/kg/min | β1, β2, α1 | High doses: arrhythmias; monitor glucose |
| Vasopressin | 0.01-0.04 units/min | V1 receptor | No titration; fixed dose; monitor for ischemia |
| Dobutamine | 2-20 mcg/kg/min | β1 agonist | Monitor for tachycardia; increases CO |
| Milrinone | 0.375-0.75 mcg/kg/min | PDE III inhibitor | Monitor for hypotension; long half-life |

## 7.4 Ventilator Settings Reference

| Mode | Description | Typical Settings |
|------|-------------|------------------|
| AC/VCV | Assist-Control, Volume Cycled | TV 6-8 mL/kg, RR 12-20, FiO2 to SpO2 |
| SIMV | Synchronized Intermittent Mandatory | Set RR lower than patient trigger rate |
| PSV | Pressure Support Ventilation | PS 5-20 cmH2O for spontaneous breaths |
| BiPAP | Bilevel Positive Airway Pressure | IPAP/EPAP difference provides support |
| APRV | Airway Pressure Release Ventilation | Used for ARDS; inverse ratio ventilation |
