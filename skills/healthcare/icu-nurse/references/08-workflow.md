# Standard Workflow

## 8.1 ICU Assessment Workflow

```
Phase 1: Shift Assessment (ABCDE Approach)
├── A-Airway: Patent? Artificial airway type and size?
├── B-Breathing: Ventilator settings, SpO2, breath sounds, RR
├── C-Circulation: HR, BP, MAP, vasoactive drips, rhythm, lines
├── D-Disability: GCS, pupil check, sedation score
├── E-Exposure: Skin, wounds, drains, I/O, lines
└── Review overnight events and pending orders

Phase 2: Critical Care Priorities
├── Assess for instability requiring immediate intervention
├── Review hemodynamics: CO, SVR, preload (CVP/PAOP)
├── Check ventilator synchrony and settings
├── Evaluate sedation/analgesia needs (RASS, CPOT)
└── Identify new problems or status changes

Phase 3: Care Planning
├── Prioritize nursing interventions by urgency
├── Coordinate with RT for ventilator adjustments
├── Communicate changes to physician using SBAR
└── Update care plan and documentation

Phase 4: Handoff
├── Use SBAR format for structured handoff
├── Include patient trajectory (improving, stable, declining)
├── Verify vasoactive drip titrations and plans
└── Confirm pending tests and family communication
```

## 8.2 Emergency Response Workflow

### Rapid Response / Code Blue Activation

1. **Assess**: Is patient deteriorating? Check ABCs, vital signs, alarms
2. **Call**: Activate Rapid Response for early intervention; Code Blue for arrest
3. **Begin**: Start BLS/ACLS as appropriate; establish IV access
4. **Document**: Times, interventions, medications, response
5. **Debrief**: Participate in post-code review

### Hemodynamic Instability Response

| Presentation | Possible Causes | Initial Actions |
|-------------|----------------|-----------------|
| Hypotension + tachycardia | Hypovolemia, sepsis, PE | Fluid bolus; identify cause |
| Hypotension + bradycardia | Drug effect, conduction block | Hold beta-blockers; notify MD |
| Hypertension + tachycardia | Pain, agitation, delirium | Treat pain/sedation; consider drips |
| Narrow pulse pressure | Cardiac tamponade, hypovolemia | Echo; urgent intervention |

## 8.3 Ventilator Management Workflow

1. **Assess Synchrony**: Is patient triggering breaths? Look for "fighting" ventilator
2. **Check Settings**: Review mode, TV, RR, FiO2, PEEP, trigger sensitivity
3. **Sedation Titration**: Use RASS to guide; maintain light sedation when appropriate
4. ** weaning Readiness**: Check criteria: P/F ratio >150, RSBI <105, minimal secretions
5. **Daily Spontaneous Breathing Trial (SBT)**: T-piece or low PS for 30-120 minutes
6. **Extubation Readiness**: GCS >8, gag reflex present, cuff leak test positive

## 8.4 Quality Assurance Checklist

- [ ] Patient identification verified
- [ ] Hourly vital signs and hemodynamic parameters documented
- [ ] Vasoactive drips titrated per orders; site assessed
- [ ] Ventilator settings correct; alarm parameters set
- [ ] Sedation/analgesia assessed and titrated (RASS/CPOT)
- [ ] Skin integrity assessed; turning schedule followed
- [ ] I/O balanced; daily weight obtained
- [ ] Critical values reported to MD
- [ ] Family updated; psychosocial needs addressed
