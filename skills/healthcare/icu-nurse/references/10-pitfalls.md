# Common Pitfalls & Anti-Patterns

## 10.1 Hemodynamic Monitoring Pitfalls

| # | Pitfall | Severity | Prevention |
|---|---------|----------|------------|
| 1 | Missing arterial line waveform dampening | 🔴 High | Flush system; assess for clot; transduce at phlebostatic axis |
| 2 | Incorrect MAP interpretation | 🟡 Medium | MAP = DBP + 1/3(SPB-DBP), not average of systolic and diastolic |
| 3 | Not trending CVP/PAOP | 🟡 Medium | Single values less useful than trends |
| 4 | Ignoring cardiac output trends | 🟡 Medium | Correlate with clinical status and other parameters |

## 10.2 Vasoactive Medication Anti-Patterns

| Anti-Pattern | Correct Approach |
|--------------|------------------|
| Titration without MAP assessment | Always check MAP before and after titration |
| Multiple drip adjustments simultaneously | Make one change at a time; document effect |
| Central line site infection | Daily chlorhexidine bath; hub care; minimize manipulation |
| Extravasation missed | Assess site regularly; avoid peripheral vasopressors |

## 10.3 Ventilator Management Pitfalls

| Pitfall | Consequence | Prevention |
|---------|-------------|------------|
| High PEEP without assessing hemodynamics | Decreased venous return, hypotension | Monitor BP closely with PEEP changes |
| Over-sedation delaying weaning | Prolonged ventilation | Daily sedation interruption; light sedation protocol |
| Not checking ETT position after turns | ETT displacement | Secure device; mark at lip; x-ray after repositioning |
| Ignoring ventilator alarms | Hypoxia, barotrauma | Never disable alarms; respond immediately |

## 10.4 Documentation Pitfalls

| Pitfall | Correct Approach |
|---------|------------------|
| Narrative documentation without assessment | Use SOAPIE format |
| Not documenting response to interventions | Document before and after vital signs |
| Missing alarm response documentation | Document all alarms and actions taken |
| Vague assessment language | Be specific: "BP 88/52" not "BP low" |

## 10.5 Quality Assurance Checklist

- [ ] Patient identification verified (2 identifiers)
- [ ] ABCs assessed hourly and with any change
- [ ] Hemodynamic parameters trended and documented
- [ ] Vasoactive drips at correct dose; site assessed
- [ ] Ventilator settings correct; alarms set appropriately
- [ ] Sedation/analgesia assessed (RASS/CPOT)
- [ ] Skin integrity monitored; turning schedule maintained
- [ ] I/O balanced; daily weights obtained
- [ ] Critical values reported and acknowledged
- [ ] Family communication documented
- [ ] SBAR handoff at change of shift

## 10.6 Safety Priorities

1. **Never disable alarms** without immediate replacement
2. **Verify high-risk medications** (vasopressors, insulin, heparin) with second nurse
3. **Maintain aseptic technique** for all line access
4. **Perform hand hygiene** before/after every patient contact
5. **Report near-misses** for system improvement
