# Standard Workflow

## 8.1 Case Receipt and Evaluation

```
Phase 1: Case Intake
├── Verify prescription completeness
├── Check impressions for defects
├── Photograph case conditions
├── Document stone model pour date
└── Enter case into tracking system

Phase 2: Case Planning
├── Determine appropriate material
├── Review shade information
├── Assess case complexity
├── Plan fabrication sequence
└── Estimate completion date
```

## 8.2 Crown and Bridge Workflow

```
Step 1: Model Preparation
├── Trim dies to proper contour
├── Section dies for access
├── Apply die spacer (2 coats)
└── Block out undercuts

Step 2: Framework Design
├── Design based on reduction requirements
├── Ensure adequate thickness
├── Plan connector size
└── Design for occlusion

Step 3: Framework Fabrication
├── Wax pattern fabrication
├── Sprue placement
├── Investing technique
├── Burnout protocol
└── Casting or milling

Step 4: Ceramic Application
├── Opaque application
├── Dentin build-up
├── Enamel layer
├── Characterization
└── Glaze firing
```

## 8.3 Denture Workflow

```
Step 1: Primary Cast
├── Cast primary impressions
├── Custom tray fabrication
├── Border molding
└── Final impression

Step 2: Master Cast
├── Pour master cast
├── Record base fabrication
├── Occlusal rims
└── Jaw relationship records

Step 3: Try-in
├── Teeth set-up
├── Wax try-in
├── Patient try-in verification
└── Adjustments recorded

Step 4: Processing
├── Flask press
├── Boil-out
├── Denture base processing
├── Finishing and polishing
└── Final delivery preparation
```

## 8.4 Quality Check Points

| Stage | Checkpoint | Acceptance Criteria |
|-------|------------|-------------------|
| Wax-up | Contour and occlusion | Full cont. contacts |
| Framework | Fit on die | No rock, adequate thickness |
| Ceramic | Shade match | ΔE <2.0 from Rx |
| Final | Surface finish | Smooth, polished |
| Delivery | Pre-delivery inspection | Complete, no damage |
