# Merck Engineer - Standards & Compliance Reference

## Regulatory Framework

### FDA Regulations
| Document | Scope | Key Requirements |
|----------|-------|------------------|
| **21 CFR Part 11** | Electronic Records & Signatures | Audit trails, electronic signatures, data integrity |
| **21 CFR Part 210/211** | cGMP for Pharmaceuticals | Manufacturing controls, documentation, quality |
| **ICH Q7** | GMP for APIs | Manufacturing controls, documentation |
| **ICH Q9** | Quality Risk Management | Risk-based approaches to validation |
| **ICH Q10** | Pharmaceutical Quality System | CAPA, change management, management review |

### EU Regulations
| Document | Scope | Key Requirements |
|----------|-------|------------------|
| **EU Annex 11** | Computerised Systems | Validation, data integrity, security |
| **EU GMP Chapter 4** | Documentation | Documentation requirements |
| **GDPR** | Data Protection | Patient data privacy, consent |

### GAMP 5 Categories
| Category | Description | Examples | Validation Approach |
|----------|-------------|----------|---------------------|
| **Category 1** | Infrastructure Software | Operating systems, databases | Record version, verify installation |
| **Category 3** | Non-Configured Products | COTS software | Verify installation, configure correctly |
| **Category 4** | Configured Products | Veeva Vault, SAP (configured) | Full lifecycle: URS, FS, DS, IQ/OQ/PQ |
| **Category 5** | Custom Applications | Custom code, AI/ML models | Full lifecycle + code review, testing |

## MSD-Specific Standards

### Manufacturing Standards
- **MSD Global Quality Standards**: Apply to all manufacturing sites
- **MSD Data Integrity Policy**: ALCOA+ requirements
- **MSD Computer System Validation SOPs**: Site-specific procedures
- **Animal Health GMP**: USDA requirements for veterinary products

### Clinical Standards
- **MSD Clinical Data Management SOPs**: EDC, CTMS operations
- **MSD Pharmacovigilance Standards**: Safety database requirements
- **MSD eTMF Standards**: Trial master file management
