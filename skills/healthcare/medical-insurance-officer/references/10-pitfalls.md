# Common Pitfalls & Anti-Patterns

## 10.1 Coding Errors

| Pitfall | Impact | Prevention |
|---------|--------|------------|
| Upcoding | Fraud investigation | Accurate code selection |
| Under-coding | Revenue loss | Documentation training |
| Unbundling | Audit penalties | Bundled code awareness |
| Mismatched codes | Denials | Code linkage review |

## 10.2 Documentation Deficiencies

| Deficiency | Consequence | Solution |
|-----------|-------------|----------|
| Incomplete H&P | Medical necessity denials | Checklist templates |
| Missing signatures | Claim denials | Signature requirements |
| Insufficient MDM | E/M level disputes | Documentation education |
| No treatment plan | Care denials | Care plan requirements |

## 10.3 Medical Necessity Failures

```
Common Issues:
├── Insufficient clinical evidence
├── Wrong setting of care
├── Frequency/quantity exceeded
├── No pre-authorization
├── Experimental treatment
└── Non-covered service

Prevention:
├── Payer policy review before service
├── Pre-authorization when required
├── Clinical documentation support
└── Peer-to-peer consultation early
```

## 10.4 Appeal Mistakes

| Mistake | Impact | Prevention |
|---------|--------|------------|
| Missing deadlines | Waiver lost | Calendar tracking |
| Incomplete documentation | Appeal denied | Checklist use |
| Wrong appeal level | Delay | Know payer process |
| No clinical rationale | Fail | Medical director involvement |

## 10.5 Eligibility/Eligibility Errors

| Error | Result | Prevention |
|-------|--------|------------|
| Patient not covered | No payment | Verify before service |
| Service not covered | Denial | Benefit verification |
| Provider not in network | Higher patient cost | Network verification |
| Timely filing missed | Claim rejected | Tracking system |

## 10.6 Quality Checklist

- [ ] Eligibility verified before service
- [ ] Prior auth obtained if needed
- [ ] Codes accurately reflect service
- [ ] Documentation supports medical necessity
- [ ] Claim submitted timely
- [ ] Denial reasons documented
- [ ] Appeals filed within deadlines
- [ ] Payment reconciled
- [ ] Provider notified of issues
- [ ] Process improvements identified
