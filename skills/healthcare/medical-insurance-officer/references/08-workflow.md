# Standard Workflow

## 8.1 Claims Review Workflow

```
Phase 1: Initial Processing
├── Verify patient eligibility
├── Confirm provider participation
├── Validate CPT/ICD-10 codes
├── Check for bundling rules
├── Apply timely filing limits
└── Assign to appropriate queue

Phase 2: Medical Review
├── Apply coding guidelines
├── Verify documentation
├── Assess medical necessity
├── Check coverage policies
└── Determine payment/findings

Phase 3: Resolution
├── Process payment
├── Issue denial with codes
├── Process appeals
├── Document decision rationale
└── Update provider communication
```

## 8.2 Denial Management

```
Step 1: Denial Analysis
├── Identify denial reason
├── Categorize (clinical vs coding vs administrative)
├── Review payer policies
├── Assess appeal viability
└── Determine priority

Step 2: Appeal Preparation
├── Gather clinical documentation
├── Write appeal letter
├── Include supporting guidelines
├── Attach peer-to-peer request if needed
└── Meet appeal deadlines

Step 3: Follow-Up
├── Track appeal status
├── Document conversations
├── Escalate if needed
├── Report patterns to leadership
└── Implement process improvements
```

## 8.3 Prior Authorization Workflow

```
Request Process:
├── Verify prior auth requirements
├── Gather clinical documentation
├── Complete payer-specific form
├── Submit with supporting records
├── Track authorization number
└── Communicate approval/denial

Denial Handling:
├── Review denial rationale
├── Gather additional information
├── Submit appeal within timeframe
├── Consider peer-to-peer review
└── Document all communications
```

## 8.4 Medical Necessity Determination

```
Clinical Review Criteria:
├── Condition warrants treatment setting
├── Less intensive setting not appropriate
├── Treatment consistent with guidelines
├── Duration medically necessary
└── Documentation supports need

Common Denials:
├── Outpatient vs inpatient level
├── Non-covered service
├── Experimental/investigational
├── Frequency exceeded
├── Missing authorization
```
