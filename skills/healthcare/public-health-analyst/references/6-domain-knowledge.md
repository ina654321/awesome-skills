## § 6 · Domain Knowledge

### Outbreak Investigation Steps

```
1. Prepare for Field Work
   - Review literature
   - Establish team
   - Prepare materials

2. Verify Diagnosis
   - Confirm cases
   - Rule out lab error

3. Establish Case Definition
   - Confirmed, probable, possible
   - Clinical and lab criteria

4. Find Cases Systematically
   - Active case finding
   - Line listing

5. Perform Descriptive Epidemiology
   - Person, place, time
   - Epidemic curve

6. Develop Hypotheses
   - Source, mode of transmission

7. Evaluate Hypotheses
   - Case-control or cohort study
   - Statistical testing

8. Refine Hypotheses and Execute Studies
   - Environmental sampling
   - Special studies

9. Implement Control Measures
   - Concurrent with investigation
   - Source control, exposure reduction

10. Communicate Findings
    - Reports, publications
    - Recommendations
```

### Program Evaluation Framework

| Component | Description | Example |
|-----------|-------------|---------|
| **Inputs** | Resources invested | Staff time, funding, materials |
| **Activities** | What program does | Education sessions, screening |
| **Outputs** | Direct products | # educated, # screened |
| **Outcomes** | Short-term changes | Knowledge increase, behavior change |
| **Impact** | Long-term results | Disease reduction, health improvement |
