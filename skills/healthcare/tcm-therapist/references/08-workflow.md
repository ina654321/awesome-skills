# Standard Workflow

## 8.1 Patient Assessment

```
Phase 1: History Taking
├── Chief complaint
├── HPI (location, quality, onset, triggers)
├── Systems review (10 questions)
├── Medical history
├── Family history
├── Lifestyle (diet, sleep, exercise, emotions)
└── Current medications

Phase 2: TCM Evaluation
├── Tongue diagnosis
├── Pulse diagnosis
├── Abdominal diagnosis
├── Observation
├── Listening/smelling
├── Inquiry
└── Palpation

Phase 3: Pattern Identification
├── Eight principles
├── Zang-fu pattern
├── Five element pattern
├── Meridian pattern
├── Six stages
└── Four levels
```

## 8.2 Treatment Planning

```
Treatment Principles:
├── Regulate yin/yang
├── Tonify deficiency
├── Disperse excess
├── Clear heat/cold
├── Move blood/qi
└── Transform damp

Treatment Methods:
```
Acupuncture:
├── Point selection based on pattern
├── Needle technique (tonifying/reducing)
├── Retention time (20-30 min)
├── Frequency (1-3x/week initially)
└── Reassessment every 4-6 weeks

Herbal Formula:
├── Classical formula base
├── Modifications for individual
├── Delivery method (decoction, pills, granules)
├── Dosage and timing
└── Duration and re-evaluation
```

## 8.3 Acupuncture Procedure

```
Pre-Treatment:
├── Explain treatment
├── Obtain consent
├── Position patient
├── Locate points
└── Prepare needles

During Treatment:
├── Insert technique
├── De qi obtained
├── Needle manipulation
├── Patient monitoring
└── Remove needles after retention

Post-Treatment:
├── Assess response
├── Document treatment
├── Schedule follow-up
├── Provide home instructions
└── Monitor for adverse effects
```

## 8.4 Safety Protocols

```
Infection Control:
├── Hand hygiene
├── Clean gloves when needed
├── Sterile field
├── Disposable needles
└── Proper sharps disposal

Adverse Event Management:
├── Needle breakage
├── Needle syncope
├── Pneumothorax prevention
├── Organ injury prevention
└── Allergic reactions
```
