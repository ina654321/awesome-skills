# Common Pitfalls & Anti-Patterns

## 10.1 Diagnostic Errors

| Pitfall | Impact | Prevention |
|---------|--------|------------|
| Western diagnosis focus | Wrong pattern | TCM differential |
| Ignoring tongue/pulse | Missed pattern | Systematic exam |
| Pattern hopping | Ineffective | Consistent pattern |
| Missing underlying pattern | Relapse | Treat root |

## 10.2 Treatment Errors

| Error | Impact | Prevention |
|-------|--------|------------|
| Wrong needle technique | Ineffective | Tonify/reduce |
| Too many points | Diluted effect | 8-15 points max |
| Ignoring contraindications | Adverse event | Check points |
| Prolonged retention | Infection risk | Time limits |

## 10.3 Herbal Medicine Issues

```
Common Mistakes:
├── Wrong herb for pattern
├── Inappropriate combinations
├── Ignoring interactions
├── Poor quality herbs
├── Incorrect preparation
└── Wrong dosage

Prevention:
├── Pattern-based prescribing
├── Check interactions
├── Use certified products
├── Follow preparation guidelines
└── Monitor for reactions
```

## 10.4 Safety Violations

| Violation | Risk | Prevention |
|-----------|------|------------|
| Reusing needles | Infection | Single-use only |
| Deep puncture | Organ injury | Anatomical knowledge |
| Improper disposal | Injury | Sharps protocol |
| No consent | Legal | Informed consent |

## 10.5 Communication Failures

| Failure | Impact | Prevention |
|---------|--------|------------|
| Medical advice without scope | Harm | Scope limitations |
| Ignoring Western diagnosis | Danger | Coordinate care |
| Not referring out | Delayed treatment | Know when to refer |
| Unrealistic claims | Trust | Evidence-based |

## 10.6 Documentation Checklist

- [ ] Complete intake form
- [ ] TCM diagnosis documented
- [ ] Pattern identification clear
- [ ] Treatment plan stated
- [ ] Point selection noted
- [ ] Needle technique recorded
- [ ] Patient response noted
- [ ] Follow-up scheduled
- [ ] Referrals if needed
- [ ] Adverse events documented
