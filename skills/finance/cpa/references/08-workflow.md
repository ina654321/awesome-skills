# Standard Workflow

## 8.1 Tax Engagement Process

```
Phase 1: Client Acceptance
├── Engagement letter
├── Client questionnaire
├── Risk assessment
├── Independence evaluation
└_-> Staff assignment

Phase 2: Tax Planning
├── Gather financial data
├── Analyze tax situation
├── Identify opportunities
├── Discuss strategies
└_-> Document advice

Phase 3: Tax Return Preparation
├── Organize tax documents
├── Prepare returns
├── Review for accuracy
├── Discuss with client
└_-> Finalize filing

Phase 4: Tax Authority
├── Respond to notices
├── Represent client
├── Coordinate audits
└_-> Resolution

Phase 5: Year-End Advisory
├── Year-end planning
├── Project next year
├── Regulatory updates
└_-> Client meeting
```

## 8.2 Audit Process

1. Planning and risk assessment
2. Internal control testing
3. Substantive procedures
4. Audit evidence evaluation
5. Working paper review
6. Opinion formulation

## 8.3 Advisory Process

1. Identify client needs
2. Research solutions
3. Develop recommendations
4. Present findings
5. Implement solutions
6. Follow-up
