# Citi Banking Workflows

## Cross-Border Cash Management Implementation

### Phase 1: Discovery (4-6 weeks)
1. **Global Footprint Mapping**
   - Entity chart across all jurisdictions
   - Bank account inventory
   - Current cash positioning
   - FX exposure analysis

2. **Pain Point Identification**
   - Visibility gaps
   - Manual processes
   - Liquidity fragmentation
   - Cost inefficiencies

3. **Regulatory Assessment**
   - Notional pooling feasibility by country
   - FX control restrictions
   - Tax implications
   - Data residency requirements

### Phase 2: Design (6-8 weeks)
1. **Structure Design**
   - Regional pooling architecture
   - Cross-border sweep mechanics
   - Virtual account framework
   - Reporting hierarchy

2. **Technology Integration**
   - API requirements
   - File format specifications
   - Security protocols
   - Testing protocols

3. **Legal Documentation**
   - Account opening packages
   - Pooling agreements
   - Sweep documentation
   - Security agreements

### Phase 3: Implementation (12-16 weeks)
1. **Account Setup**
   - Account opening (parallel processing)
   - Signatory updates
   - Online banking setup

2. **System Integration**
   - ERP integration
   - Testing (unit, integration, UAT)
   - Parallel running

3. **Training & Go-Live**
   - User training
   - Go-live support
   - Hypercare period

## Trade Finance Facility Setup

### Step-by-Step Process
1. **Credit Approval** (4-8 weeks)
   - Facility request submission
   - Credit analysis
   - Committee approval
   - Documentation

2. **LC Line Activation** (2-4 weeks)
   - LC agreement execution
   - Fee setup
   - Limit loading in systems
   - Client training

3. **Ongoing Management**
   - Monthly utilization reports
   - Annual review/renewal
   - Limit adjustments
   - Compliance monitoring

## Country Risk Assessment Workflow

### Initial Assessment
1. Sovereign rating review
2. IMF/World Bank data analysis
3. CDS spread monitoring
4. Local market intelligence

### Ongoing Monitoring
1. Monthly portfolio review
2. Quarterly country limit review
3. Annual comprehensive assessment
4. Ad-hoc event-driven reviews

### Escalation Triggers
- Sovereign downgrade
- Significant currency depreciation (>20%)
- Capital controls imposed
- Political instability events
