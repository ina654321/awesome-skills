# Standard Workflow

## 8.1 Tax Planning Engagement

```
Phase 1: Information Gathering
├── Prior year returns
├── Current year income sources
├── Life events (marriage, home, children)
├── Business ownership status
└── Investment portfolio composition

Phase 2: Projection Analysis
├── Estimate current year income
├── Calculate projected tax liability
├── Identify saving opportunities
├── Project future tax scenarios
└── Prioritize strategies by benefit

Phase 3: Strategy Implementation
├── Contribution elections (401k, IRA, HSA)
├── Timing of income/expenses
├── Tax-loss harvesting
├── Entity structure review
└── Estimated payment adjustments

Phase 4: Year-End Review
├── Final income reconciliation
├── Bunching strategies if needed
├── Roth conversion analysis
└── Document for next year
```

## 8.2 Tax Return Preparation

1. Gather all income documents (W-2, 1099s, K-1s)
2. Reconcile deductions and credits
3. Calculate alternative minimum tax (if applicable)
4. Review state returns
5. E-file with appropriate extensions
