# Standard Workflow

## 8.1 Pawn Transaction Process

```
Phase 1: Customer Intake
├── Customer identification (gov ID)
├── Verify age (18+)
├── Capture photo if required
├── Create/verify customer record
└── Explain terms and conditions

Phase 2: Item Evaluation
├── Inspect item condition
├── Verify ownership
├── Research value
├── Determine loan amount
└_-> Documentation (photos, description)

Phase 3: Loan Processing
├── Calculate loan amount
├── Determine interest rate
├── Explain redemption period
├── Process transaction
├── Provide pawn ticket
└── Record in system

Phase 4: Item Storage
├── Secure storage
├── Item tagging
├── Inventory management
└_-> Insurance coverage

Phase 5: Redemption/Disposition
├── Payment processing
├── Item release (with ID)
├── Extension processing
├── Default handling
└_-> Sale preparation
```

## 8.2 Loan Renewal Process

1. Customer presents pawn ticket
2. Verify customer identity
3. Accept payment of interest
4. Extend loan period
5. Update records
6. Provide new receipt

## 8.3 Default/Forfeiture Process

1. Review expired pawn tickets
2. Send notice to customer
3. Hold through redemption period
4. Transfer to inventory
5. Prepare for sale
6. Dispose per state law
