# Bank of America Workflows

## Consumer Banking Account Opening Workflow

### Phase 1: Discovery (15 minutes)
1. **Customer Needs Assessment**
   - Life stage identification
   - Financial goals discussion
   - Current banking relationship mapping
   - Pain point identification

2. **Product Matching**
   - Checking account tier selection (Advantage SafeBalance, Plus, Relationship)
   - Savings account options
   - Credit card recommendations
   - Digital enrollment

### Phase 2: Application (10 minutes)
1. **Identity Verification**
   - Government-issued ID
   - Social Security number/TIN
   - Address verification
   - Identity verification systems check

2. **Account Setup**
   - Account funding (minimum deposit)
   - Debit card ordering
   - Online/mobile banking enrollment
   - Erica AI activation
   - Direct deposit setup (if applicable)

### Phase 3: Onboarding (30 days)
1. **Digital Adoption**
   - Mobile app download and setup
   - Bill pay enrollment
   - Zelle registration
   - Alert preferences

2. **Relationship Deepening**
   - Preferred Rewards eligibility check
   - Credit card application (if qualified)
   - Savings goal setup
   - Financial wellness tools introduction

## Wealth Management Client Onboarding (Merrill/Private Bank)

### Phase 1: Initial Engagement (Week 1)
1. **Discovery Meeting**
   - Financial goals and objectives
   - Risk tolerance assessment
   - Time horizon analysis
   - Current asset inventory
   - Tax situation review
   - Estate planning considerations

2. **Data Gathering**
   - Account statements
   - Tax returns
   - Insurance policies
   - Estate documents
   - Business interests (if applicable)

### Phase 2: Analysis and Planning (Weeks 2-3)
1. **Financial Plan Development**
   - Cash flow analysis
   - Net worth statement
   - Retirement projections
   - Education funding analysis
   - Insurance needs assessment
   - Estate planning review

2. **Investment Policy Statement**
   - Investment objectives
   - Risk parameters
   - Asset allocation guidelines
   - Constraints and restrictions
   - Rebalancing protocols

### Phase 3: Implementation (Weeks 4-6)
1. **Account Establishment**
   - Account applications
   - Beneficiary designations
   - Authorized user setup
   - Trading authority documentation

2. **Asset Transfer and Investment**
   - ACATS transfers
   - New money investment
   - Tax-loss harvesting (if applicable)
   - Systematic investment setup

### Phase 4: Ongoing Management (Quarterly)
1. **Portfolio Review**
   - Performance vs. benchmark
   - Rebalancing needs
   - Tactical adjustments
   - Tax optimization

2. **Relationship Review**
   - Life changes assessment
   - Goal progress tracking
   - Additional service needs
   - Next generation engagement

## Business Banking Credit Workflow

### Phase 1: Origination
1. **Initial Consultation**
   - Business model understanding
   - Capital need identification
   - Financial history review
   - Ownership structure analysis

2. **Pre-Qualification**
   - Preliminary credit assessment
   - Structure options discussion
   - Documentation requirements
   - Timeline communication

### Phase 2: Underwriting
1. **Financial Analysis**
   - Historical financial statements (3 years)
   - Tax returns review
   - Cash flow analysis
   - Debt service coverage calculation
   - Collateral valuation

2. **Risk Assessment**
   - Industry analysis
   - Management assessment
   - Market position evaluation
   - Environmental risk (if applicable)

### Phase 3: Approval and Documentation
1. **Credit Approval**
   - Risk rating assignment
   - Authority approval
   - Terms finalization
   - Covenant establishment

2. **Documentation**
   - Loan agreement
   - Security agreements
   - Guarantor documents
   - Compliance certificates

### Phase 4: Monitoring
1. **Ongoing Surveillance**
   - Financial statement collection
   - Covenant compliance testing
   - Collateral monitoring
   - Site visits (if applicable)

2. **Annual Review**
   - Full credit analysis
   - Risk rating confirmation/update
   - Structure optimization
   - Cross-sell opportunity assessment

## Digital Implementation Workflow

### Erica AI Integration
1. **Customer Enablement**
   - Erica introduction during onboarding
   - Voice recognition setup
   - Preference configuration
   - Capability education

2. **Proactive Engagement**
   - Alert setup guidance
   - Financial insight review
   - Goal setting assistance
   - Cross-sell introductions (with consent)

### CashPro Implementation (Commercial)
1. **Needs Assessment**
   - Payment volume analysis
   - Workflow mapping
   - Integration requirements (ERP)
   - User role definition

2. **Platform Configuration**
   - User setup and entitlements
   - Approval workflow design
   - Account structure setup
   - Security configuration

3. **Training and Go-Live**
   - Administrator training
   - End-user training
   - Test transaction processing
   - Production cutover
