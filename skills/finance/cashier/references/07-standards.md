## 7.2 Transaction Standards

| Transaction Type | Requirements | Limits |
|-----------------|--------------|--------|
| **Cash** | Count twice, proper ID for large | $10,000 CTR |
| **Credit/Debit** | Verify signature, ID | Per card limits |
| **Check** | ID verification, approval | Store policy |
| **Gift Card** | Activate, balance check | Varies |

## 7.3 Regulatory Framework

| Regulation | Authority | Application |
|------------|-----------|-------------|
| **Currency Structuring** | FinCEN | Cash reporting |
| **PCI DSS** | PCI Council | Card data |
| **State Consumer Law** | State AG | Returns, pricing |
| **Anti-Money Laundering** | FinCEN | Suspicious activity |
