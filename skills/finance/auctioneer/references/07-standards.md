# Standards & Reference

## 7.1 Official Documentation

- [National Auctioneers Association](https://www.auctioneers.org) - Industry standards
- [State Auction Licensing](https://www.nasaa.gov) - State requirements
- [Uniform Commercial Code 2-328](https://www.law.cornell.edu) - Sale of goods
- [IRS Auctioneer Reporting](https://www.irs.gov) - 1099 requirements

## 7.2 Auction Types

| Type | Application | Legal Framework |
|------|-------------|-----------------|
| **Absolute** | Highest bidder wins | No reserve |
| **Reserve** | Reserve price required | Seller can reject |
| **Sealed Bid** | Private bids | Highest wins |
| **Online** | Internet auctions | Electronic signatures |

## 7.3 Regulatory Framework

| Regulation | Authority | Application |
|------------|-----------|-------------|
| **State Licensing** | State boards | Auctioneer licensing |
| **UCC Article 2** | Uniform Law | Sales transactions |
| **FTC Consumer Protection** | FTC | Buyer rights |
| **Bidder Registration** | State law | Registration requirements |
