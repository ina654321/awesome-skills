# Standard Workflow

## 8.1 Customer Transaction Process

```
Phase 1: Customer Identification
├── Verify identity (ID check)
├── Check against sanctions lists
├── Establish customer profile
├── Risk rating assignment
└── CIP completion

Phase 2: Transaction Processing
├── Receive transaction request
├── Verify account ownership
├── Process transaction
├── Generate receipt
└── Update records

Phase 3: Compliance Verification
├── Check transaction limits
├── Review for suspicious activity
├── Verify source of funds
├── Currency structuring check
└── SAR determination

Phase 4: Completion
├── Dispense funds if applicable
├── Secure documentation
├── File required reports
└── Log transaction
```

## 8.2 Cash Transaction Workflow

1. Customer presents identification
2. Verify account status
3. Process deposit/withdrawal
4. Review for CTR requirements
5. Complete transaction
6. Provide receipt

## 8.3 Suspicious Activity Workflow

1. Identify unusual activity
2. Complete SAR initial filing
3. Document observations
4. Notify supervisor
5. Preserve records
6. Continue monitoring

## 8.4 Currency Transaction Workflow

1. Verify customer identity
2. Determine reportable amount
3. Complete CTR if over $10K
4. Process transaction
5. File within 15 days
