# Interior Design — Troubleshooting

## Material Selection Issues

### Specified Material No Longer Available

**Symptom:** A specified material (tile, flooring, fabric) is discontinued or has a 16-week lead time.

**Fix:**
```bash
# Step 1: Check with supplier for stock status
# Contact manufacturer rep directly for availability

# Step 2: Identify equivalent alternatives
# Use Material Bank (materialbank.com) to search by:
# - Similar material type (porcelain → porcelain)
# - Matching color family
# - Comparable performance specs

# Step 3: Request samples of alternatives
# Order 3-5 samples of top alternatives

# Step 4: Present to client with pros/cons
# Document: Why original is unavailable, alternatives with comparison
```

**Prevention:** Always verify availability during specification. Maintain a "secondary specified" list for every primary specification.

### Material Color/Texture Doesn't Match Sample

**Symptom:** Installed material looks different from the approved sample.

**Diagnosis:**
```yaml
Common causes:
  - Batch variation: Different production lots have different dye lots
  - Lighting difference: Material looks different in warehouse vs. installed space
  - Substrate interaction: Paint looks different on primed wall vs. sample paper
  - Scale: Small sample doesn't show full pattern repeat
```

**Fix:**
```bash
# For paint:
# 1. Get paint mixed from the same tinting machine and batch
# 2. Paint a 4x4' sample on the actual wall; compare in natural + artificial light
# 3. If color is off, adjust: lighter → more colorant; darker → reduce colorant
# 4. Always get a sealed sample for future touch-ups

# For tile/stone:
# 1. Request current batch lot — inspect for variation
# 2. Blend tiles from multiple boxes during installation
# 3. Reserve extra material for future repairs (10-15% overage)

# For fabric/upholstery:
# 1. Request current dye lot number
# 2. Compare against original sample under D65 light source
# 3. Specify "dye lot variation acceptable" or require single lot
```

## Budget Issues

### FF&E Budget Exceeded by >15%

**Symptom:** The furniture, fixtures, and equipment budget is significantly over.

**Fix — Tiered approach:**
```yaml
# Tier 1: Swap, Don't Eliminate
# - Replace high-cost items with equivalent-look alternatives
# - Example: Custom sofa ($12,000) → High-end sectional from dealer ($8,500)
# - Keep the visual impact, reduce cost

# Tier 2: Reduce Scope, Preserve Key Pieces
# - Eliminate lower-priority items
# - Example: Remove 2 accent chairs, keep the statement sofa
# - Budget freed: $3,200

# Tier 3: Negotiate with Vendors
# - Request additional discounts (5-15% for volume orders)
# - Ask about floor stock or discontinued items
# - Request extended payment terms

# Tier 4: Value Engineering with Client
# - Present options for quality tiers
# - Example: Solid surface ($8,000) vs. quartz ($5,500) vs. laminate ($2,500)
# - Let client make informed decision
```

**Prevention:** Build 10% contingency into every FF&E budget from day one.

## Lighting Issues

### Room Too Dark Despite Adequate Fixture Count

**Symptom:** Installed lighting provides insufficient illumination.

**Diagnosis:**
```bash
# Step 1: Measure foot-candles with light meter
# Expected levels (IESNA):
#   Living room: 10-20 FC ambient
#   Kitchen counter: 50-75 FC task
#   Office: 30-50 FC ambient

# Step 2: Check fixture specifications
# Lumens output: Ensure fixtures are rated for room size
# Formula: Room SF × Recommended FC = Required Lumens
# Example: 300 SF kitchen × 50 FC = 15,000 lumens needed

# Step 3: Common issues:
# - Wrong Kelvin temperature: 2700K looks warm but may feel dim
# - Fixture placement: Lights placed too high, reducing intensity
# - Trim/color temperature mismatch: All fixtures should match
```

**Fix:**
```yaml
# Option 1: Add supplemental lighting
#   Under-cabinet LED strips in kitchen
#   Picture lights for artwork
#   Floor lamps in living areas

# Option 2: Change fixture specifications
#   Higher lumen output fixtures
#   Lower color temperature (3000K) for perceived brightness
#   Remove diffusers that reduce output by 15-20%

# Option 3: Adjust lighting controls
#   Increase dimmer output
#   Program scene to boost ambient levels
```

### Fixture Flickering or Intermittent Failure

**Fix:**
```yaml
# Cause 1: Incompatible dimmer
#   Solution: Replace with LED-compatible dimmer (Lutron Caseta, etc.)
#   Check fixture spec for dimmer compatibility list

# Cause 2: Voltage fluctuation
#   Solution: Install voltage stabilizer or check circuit loading

# Cause 3: Driver failure (LED fixtures)
#   Solution: Replace driver (often field-replaceable)
#   Match: Input voltage, output voltage, wattage capacity

# Cause 4: Loose connection
#   Solution: Have electrician check all wire nuts and terminal connections
```

## Procurement & Lead Time Issues

### Long-Lead Item Arrives Late

**Symptom:** Custom furniture (12-16 week lead time) hasn't arrived, causing installation delay.

**Fix:**
```bash
# Step 1: Contact manufacturer immediately
# - Get current production status and estimated ship date
# - Request expedited shipping (often available for 15-25% premium)

# Step 2: Adjust installation schedule
# - Work on areas not dependent on the item
# - Coordinate with other trades to minimize downtime

# Step 3: Interim solutions
# - Rent or borrow equivalent furniture for soft launch
# - Temporarily install substitute pieces

# Step 4: Future prevention
# - Order long-lead items at design development phase (not CDs)
# - Maintain 4-week buffer in project schedule after delivery
# - Confirm delivery date with manufacturer in writing
```

## Finish Issues

### Paint Peeling or Blistering

**Symptom:** Freshly painted walls show peeling, bubbling, or poor adhesion.

**Fix:**
```yaml
# Cause 1: Moisture behind paint
#   - Check for plumbing leaks, condensation, or rising damp
#   - Fix moisture source before repainting
#   - Use mold-resistant primer (e.g., Zinsser Mold Killing Primer)

# Cause 2: Improper surface prep
#   - Grease, dust, or contamination prevents adhesion
#   - Solution: Clean with TSP substitute, sand, prime, repaint

# Cause 3: Incompatible paint layers
#   - Latex over oil-based without proper primer
#   - Solution: Prime with bonding primer (e.g., INSL-X Stix)

# Cause 4: Painting over wet surface
#   - New plaster or drywall compound not fully cured
#   - Solution: Wait; test with moisture meter (<12% moisture)
```

### Wood Floor Gapping or Cupping

**Symptom:** hardwood floors show gaps between planks or edges raised higher than center.

**Fix:**
```yaml
# Gapping (gaps between planks):
# - Cause: Low humidity; wood has dried out
# - Fix: Use humidifier to raise RH to 35-55%
# - Small gaps (<1/8"): Monitor; often closes with humidity increase
# - Large gaps: Sand and refinish; fill with color-matched filler

# Cupping (edges higher than center):
# - Cause: Moisture differential; top of plank wetter than bottom
# - Fix: Increase ventilation; dehumidify
# - Let stabilize for weeks before sanding
# - DO NOT sand while cupped — will become crowned after adjustment

# Buckling (planks lifting from subfloor):
# - Cause: Severe moisture or flood
# - Fix: Remove baseboards; allow to dry; renail or glue; refinish
# - If subfloor is damaged, replace affected sections
```

## Specification Conflicts

### Architect's Finish Schedule Conflicts with Interior Design Spec

**Symptom:** Two design documents specify different finishes for the same space.

**Fix:**
```yaml
# Step 1: Review contract
# - Architect's fee includes design development through CDs
# - Interior design scope typically starts at Design Development
# - Check which document has contractual authority

# Step 2: Compare specifications
# - Note any differences in: material type, manufacturer, color/finish
# - Check if one document references the other

# Step 3: Resolution hierarchy:
# 1. Contract documents take precedence over specifications
# 2. Most recent approved submittal
# 3. In case of ambiguity: request RFI to architect

# Step 4: Document resolution
# - Issue a drawing revision or specification addendum
# - Get written approval from architect and client
# - File with project record set
```

## Accessibility Compliance Issues

### ADA Clearance Not Achieved

**Symptom:** Installed furniture or fixtures don't meet ADA reach range or clear floor space requirements.

**Fix:**
```yaml
# Common violations and fixes:

# Reach Range (forward or side):
# - Max height for forward reach (no obstruction): 48"
# - Max height for forward reach (obstruction ≤20" deep): 46"
# - Min height for low reach: 15"
# Fix: Adjust mounting height of all controls, shelves, counters

# Clear Floor Space:
# - 30"×48" required at each fixture
# Fix: Rearrange furniture to provide required clearance

# Turning Space (restrooms, kitchens):
# - 60" diameter circle or T-turn required
# Fix: Reposition or remove furniture to create turning space

# Door Clearance:
# - 32" minimum clear width
# Fix: Replace door, adjust hinges, or threshold
```

## Vendor/Contractor Issues

### Millwork Shop Drawings Don't Match Design Intent

**Symptom:** Custom millwork as-built drawings have dimensional or detail errors.

**Fix:**
```bash
# Step 1: Mark up shop drawings in red
# - Circle every error
# - Note: "Not per design intent"
# - Be specific: dimension, location, finish

# Step 2: Issue marked-up drawings to contractor
# - Request corrected drawings within 5 business days
# - Note: "Approval is conditional on addressing red-line comments"

# Step 3: Review corrected drawings
# - Verify all red marks are addressed
# - Check adjacent conditions still work

# Step 4: Approve corrected drawings
# - Written approval before fabrication proceeds
```

**Prevention:** Require 3D render or mockup for complex millwork before fabrication.
