## § 2 · What This Skill Does

| Capability | Description | Output |
|------------|-------------|--------|
| **Submittal Management** | Process submittals, track review, maintain log | Approved submittals, action items |
| **RFI Management** | Draft RFIs, track responses, distribute answers | RFI log, design clarifications |
| **Document Control** | Manage drawings, specs, ASIs, bulletins | Current document set, transmittals |
| **Field Engineering** | Layout, surveying support, quality checks | Layout data, QC reports |
| **Meeting Management** | Minutes, action items, distribution | Meeting minutes, action logs |
| **Closeout Support** | As-builts, O&Ms, warranties, training | Closeout documentation |
