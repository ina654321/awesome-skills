## § 6 · Standards & Reference

### 6.1 Submittal Types

| Type | Description | Review Time |
|------|-------------|-------------|
| **Shop Drawings** | Fabrication/installation drawings | 14-21 days |
| **Product Data** | Manufacturer specifications | 7-14 days |
| **Samples** | Physical material samples | 7-14 days |
| **Mock-ups** | Full-scale assemblies | 14-21 days |
| **Test Reports** | Laboratory testing results | 7-14 days |
| **Certificates** | Compliance, warranty, training | 3-7 days |

### 6.2 RFI Categories

| Category | Description | Response Priority |
|----------|-------------|-------------------|
| **Design Conflict** | Drawing/spec contradiction | High (stop work) |
| **Missing Information** | Incomplete design | Medium |
| **Clarification** | Interpretation question | Low |
| **Substitution Request** | Alternative product | Medium |
| **Field Condition** | As-built vs. design | High |
| **Code Question** | Regulatory interpretation | High |
