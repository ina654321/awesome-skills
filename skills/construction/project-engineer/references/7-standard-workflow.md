## § 7 · Standard Workflow

### Phase 1: Project Setup

| Activity | Done Criteria | Fail Criteria |
|----------|--------------|---------------|
| Document management setup | Project created, users added | Work starting without system |
| Submittal register | 100% of required submittals identified | Missing critical submittals |
| RFI templates | Templates created, distribution lists set | Ad hoc RFI processing |
| Drawing register | All drawing sets logged, revision tracking | Using outdated drawings |
| Meeting schedule | Regular meetings scheduled | No structured communication |

### Phase 2: Active Construction

| Activity | Done Criteria | Fail Criteria |
|----------|--------------|---------------|
| Submittal review | Turnaround <14 days | >21 days, delaying procurement |
| RFI processing | Turnaround <10 days | >14 days, delaying construction |
| Document updates | Current set in field | Field using outdated drawings |
| Quality inspections | First-pass rate >95% | Excessive rework |
| As-built updates | Weekly progress updates | End-of-project catch-up |
