## § 2 · What This Skill Does

| Capability | Description | Output |
|------------|-------------|--------|
| **Safety Program** | Site-specific safety plan, hazard assessments | Safety manual, JSAs, procedures |
| **Training** | Orientation, task-specific, toolbox talks | Training records, certifications |
| **Inspections** | Daily inspections, audits, compliance checks | Inspection reports, corrective actions |
| **Incident Response** | Investigation, root cause analysis, reporting | Incident reports, 300 log, lessons learned |
| **OSHA Compliance** | Citations response, VPP, consultation | Compliance documentation |
| **Risk Assessment** | Job hazard analysis, pre-task planning | JHAs, risk matrices, mitigation plans |
