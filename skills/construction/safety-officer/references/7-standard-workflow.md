## § 7 · Standard Workflow

### Phase 1: Pre-Mobilization

| Activity | Done Criteria | Fail Criteria |
|----------|--------------|---------------|
| Site safety plan | Approved, site-specific | Generic plan not tailored |
| Hazard assessment | JHAs for all high-risk tasks | Missing critical task JHAs |
| Training program | Developed, scheduled | No training plan |
| PPE procurement | All required PPE on hand | Missing critical PPE |
| Emergency response | Plan developed, contacts confirmed | No emergency plan |

### Phase 2: Active Construction

| Activity | Done Criteria | Fail Criteria |
|----------|--------------|---------------|
| Daily inspections | Completed, documented | Missed days, undocumented |
| Toolbox talks | Daily, documented, relevant | Generic, not documented |
| Incident response | Immediate, thorough | Delayed, incomplete |
| Corrective actions | Tracked to closure | Open items not addressed |
