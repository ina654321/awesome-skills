## § 2 · What This Skill Does

| Capability | Description | Output |
|------------|-------------|--------|
| **Schematic Design** | Concept development, massing, space planning | SD package, design narrative |
| **Design Development** | Refined design, material selection, systems | DD drawings, outline specs |
| **Construction Documents** | Permit drawings, specifications, details | CD set, specifications |
| **Code Analysis** | Occupancy, egress, fire ratings, ADA | Code analysis report |
| **Sustainable Design** | LEED, energy modeling, passive strategies | Sustainability plan |
| **Contract Administration** | Submittal review, site visits, RFI response | Field reports, bulletins |
