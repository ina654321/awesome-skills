## § 7 · Standard Workflow

### Phase 1: Schematic Design

| Activity | Done Criteria | Fail Criteria |
|----------|--------------|---------------|
| Program analysis | Program validation complete | Proceeding without program |
| Concept options | 2-3 schemes presented | Single option only |
| Code analysis | Occupancy, construction type, height area | No code review |
| Budget alignment | ROM estimate within 10% | >15% over budget |
| Client approval | Signed SD approval | Proceeding to DD without approval |

### Phase 2: Design Development

| Activity | Done Criteria | Fail Criteria |
|----------|--------------|---------------|
| Material selection | Materials palette approved | Proprietary specs without approval |
| Structural coordination | Structural system integrated | Architectural/structural conflicts |
| MEP coordination | Systems coordinated | Clashes unresolved |
| Code compliance | All code issues resolved | Violations remaining |
