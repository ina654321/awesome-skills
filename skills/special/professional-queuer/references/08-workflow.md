# Standard Workflow

## 8.1 Assessment Phase

### Step 1: Define the Target
- What specifically do you need? (Ticket, reservation, spot)
- What date/time?
- How many people?
- Budget constraints?
- What happens if you fail?

### Step 2: Research Requirements
- When do tickets go on sale? (Presale vs. general)
- What platforms sell these tickets?
- Are there presale codes? (Fan, Amex, Citi)
- What's the capacity/demand ratio?
- Any alternatives if primary fails?

### Step 3: Assess Difficulty
- Sold out in minutes? (Hard — multiple strategies needed)
- Same-day available? (Easy — just time it right)
- Lottery/rush system? (Fair chance with persistence)
- Reservation required? (Plan ahead)

## 8.2 Strategy Development

### For High-Demand Tickets
1. **Build presale access**
   - Join fan clubs
   - Get credit card presale access
   - Create multiple verified accounts
2. **Prepare devices**
   - Use fast computer/smartphone
   - Close unnecessary tabs
   - Have payment ready
3. **Time it precisely**
   - Know exact on-sale time
   - Sign in 10 minutes early
   - Be ready at T-0

### For Reservations
1. **Know the window**
   - Some book 30 days out, some 60
   - Some release at midnight, some at call opening
2. **Multiple attempts**
   - Call exactly when lines open
   - Refresh online continuously
   - Check day-before for cancellations
3. **Have alternatives**
   - Know backup dates/times
   - Have second-choice restaurants ready

## 8.3 Execution Phase

### Ticket Acquisition
1. Sign into all platforms 10 minutes before
2. Have cards ready (primary + backup)
3. Start transaction immediately when available
4. If one fails, try another platform/device immediately
5. If sold out, join official resale queue
6. Don't pay above face + fees on resale initially

### Reservation Acquisition
1. Call or refresh exactly at opening time
2. Have flexibility on time/date ready
3. If no availability, ask about cancellations
4. Check every few hours for openings
5. Consider waitlist if offered

## 8.4 Post-Acquisition

### Confirm Details
- Double-check date, time, number of tickets
- Verify seat locations
- Confirm email receipt
- Note confirmation number

### Plan Contingencies
- What if plans change? (Refund policy)
- What if event is cancelled? (Insurance)
- What if you're sick? (Transfer options)

### Follow Up
- Check ticket delivery status
- Add to calendar with reminder
- Print tickets if needed
- Note any pre-event requirements
