# Standard Workflow

## 8.1 Product Research Phase

### Step 1: Clarify Requirements
- Get exact product specifications
- Identify budget constraints
- Determine timeline (urgent vs. flexible)
- Note any must-have features vs. nice-to-have

### Step 2: Initial Search
- Research MSRP and typical retail price
- Identify authorized sellers
- Note release dates for upcoming products
- Check availability across major channels

### Step 3: Comparative Analysis
- List all found options with prices
- Factor in shipping costs
- Note authenticity guarantees
- Compare warranties and return policies

## 8.2 Sourcing Phase

### Step 4: Source Selection
- Prioritize authorized/verified sellers
- Compare total cost (price + shipping + import if applicable)
- Check seller ratings and reviews
- Verify return policy and customer service

### Step 5: Authentication Planning (for luxury/high-value)
- Identify professional authentication services
- Note authentication points for self-verification
- Budget for authentication fees if needed
- Plan for secure payment methods

### Step 6: Purchase Execution
- Time purchase for optimal pricing if flexible
- Use secure payment methods (credit card over wire)
- Keep all receipts and correspondence
- Track shipment and delivery

## 8.3 Verification Phase

### Step 7: Arrival Inspection
- Verify packaging matches authentic presentation
- Check for authentication markers
- Compare to product description/images
- Document any concerns immediately

### Step 8: Authentication
- Run through authentication checklist
- Use professional services if uncertain
- Compare to official product images
- Check serial numbers with manufacturer if applicable

### Step 9: Return/Resolution if Needed
- Contact seller within return window
- Document all issues clearly
- Use secure return methods
- Report to platforms if counterfeit suspected

## 8.4 Cross-Border Sourcing

### Special Considerations
1. **Import duties and taxes**: Factor into total cost
2. **Customs delays**: Build in buffer time
3. **Warranty coverage**: May not apply internationally
4. **Voltage differences**: Electronics may need converters
5. **Language barriers**: Use translation tools for support
6. **Currency conversion**: Use cards with good FX rates

### Trusted International Platforms
- **Amazon Global**: International shipping with customs handled
- **eBay Global Shipping**: International shipping program
- **Forwarders**: Use package forwarding for better rates
