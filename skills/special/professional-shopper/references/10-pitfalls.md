# Common Pitfalls & Anti-Patterns

## 10.1 High-Severity Anti-Patterns

### 🔴 Anti-Pattern 1: Skipping Authentication on High-Value Items
```
❌ BAD: Buying $2,000 watch without verification because "seller seems legitimate"
✅ GOOD: Always verify authenticity for items over $500; use professional services for $1000+
```
**Why it matters**: Counterfeit goods are common; even "legitimate" sellers can be fooled or deceive.

### 🔴 Anti-Pattern 2: Ignoring Total Cost on International Purchases
```
❌ BAD: Buying from Japan without calculating import duty, shipping, and currency fees
✅ GOOD: Always calculate total landed cost before deciding; sometimes domestic is cheaper
```
**Why it matters**: Hidden costs can eliminate savings; surprise customs fees cause problems.

### 🔴 Anti-Pattern 3: Wire Transfer to Unknown Sellers
```
❌ BAD: Sending bank wire to seller you've never dealt with
✅ GOOD: Use PayPal, credit card, or escrow services; never wire money to strangers
```
**Why it matters**: Wire transfers offer zero protection; scams are common and irreversible.

### 🔴 Anti-Pattern 4: Assuming Return Policy Applies
```
❌ BAD: Buying sale item assuming standard return policy applies
✅ GOOD: Read return policy before purchase; many sales items are final sale
```
**Why it matters**: Misunderstanding return policy leads to stuck purchases; read carefully.

## 10.2 Medium-Severity Anti-Patterns

### 🟡 Anti-Pattern 5: Not Comparing Prices Before Buying
```
❌ BAD: Buying at first price seen without checking alternatives
✅ GOOD: Spend 15 minutes researching across 3+ channels before purchasing
```

### 🟡 Anti-Pattern 6: Ignoring Seller History
```
❌ BAD: Buying from seller with 50 feedback because price was good
✅ GOOD: Check seller history; new sellers or low-feedback sellers are higher risk
```

### 🟡 Anti-Pattern 7: Forgetting Seasonal Pricing
```
❌ BAD: Paying full price for winter coat in December
✅ GOOD: Time purchases to seasonal sales; Black Friday, end of season, clearance
```

### 🟡 Anti-Pattern 8: Not Tracking Shipment
```
❌ BAD: Assuming package will arrive without tracking
✅ GOOD: Always pay for tracking; insure high-value items; monitor delivery
```

## 10.3 Best Practices Checklist

### Before Purchase
- [ ] Price compared across 3+ sources
- [ ] Authenticity verified (high-value items)
- [ ] Seller reputation checked
- [ ] Return policy understood
- [ ] Total cost calculated (including shipping/import)

### During Purchase
- [ ] Secure payment method used
- [ ] Receipt/order confirmation saved
- [ ] Tracking number recorded
- [ ] Insurance purchased if valuable

### After Purchase
- [ ] Package inspected upon arrival
- [ ] Authenticity verified if not done
- [ ] Any issues reported immediately
- [ ] Review left if appropriate

## 10.4 Red Flags to Watch For

| Red Flag | Risk Level | Action |
|----------|------------|--------|
| Price too good to be true | High | Walk away or authenticate thoroughly |
| New seller with great reviews | High | Ask questions; consider established sellers |
| Pressure to buy quickly | Medium | Legitimate sellers don't pressure; take time |
| Won't provide detailed photos | High | Walk away; insist on verification photos |
| Only accepts wire transfer | High | Use secure payment; wire = no protection |
