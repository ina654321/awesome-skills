# Common Pitfalls & Anti-Patterns

## 10.1 High-Severity Anti-Patterns

### 🔴 Anti-Pattern 1: Giving Advice Without Discovery
```
❌ BAD: "You should become a data analyst — it's a growing field with good pay"
✅ GOOD: First understand values, strengths, motivations, constraints — then explore options together
```
**Why it matters**: Advice without understanding client's context often leads to poor fit and client disengagement

### 🔴 Anti-Pattern 2: Letting Client Vent Without Commitment
```
❌ BAD: Continuing to explore problems without establishing commitment to act
✅ GOOD: "I hear your frustration. Before we dive deeper, are you ready to commit to making a change?"
```
**Why it matters**: Venting without action compounds frustration; coaching must drive to action

### 🔴 Anti-Pattern 3: Ignoring Market Reality
```
❌ BAD: "Follow your passion and the money will follow"
✅ GOOD: Explore passion AND demand — successful careers need both
```
**Why it matters**: Idealistic advice without market context sets clients up for failure

### 🔴 Anti-Pattern 4: Resume-Focused Career Coaching
```
❌ BAD: Treating career coaching as resume writing service
✅ GOOD: Address underlying career direction first, then position with resume
```
**Why it matters**: New resume won't help if client is heading in wrong direction

## 10.2 Medium-Severity Anti-Patterns

### 🟡 Anti-Pattern 5: One-Size-Fits-All Approach
```
❌ BAD: Using same methodology for executive, career changer, and new grad
✅ GOOD: Adapt approach to client's experience level, industry, and situation complexity
```

### 🟡 Anti-Pattern 6: Not Following Up
```
❌ BAD: Session ends with action items, no follow-up on progress
✅ GOOD: Start each session reviewing previous commitments and progress
```

### 🟡 Anti-Pattern 7: Over-Valuing Credentials
```
❌ BAD: "You need another certification to be competitive"
✅ GOOD: Evaluate if credential actually addresses market gap or is just expensive credentialism
```

### 🟡 Anti-Pattern 8: Not Addressing Identity Work
```
❌ BAD: Treating career change as just skill swap
✅ GOOD: Acknowledge that changing careers often means changing how you see yourself
```

## 10.3 Best Practices Checklist

### Before Each Session
- [ ] Review previous session notes and action items
- [ ] Prepare specific questions based on client's current situation
- [ ] Have relevant market data or resources ready

### During Session
- [ ] Ask open-ended questions before giving advice
- [ ] Challenge assumptions gently while validating emotions
- [ ] Ensure client leaves with specific, measurable next steps
- [ ] Document commitments made

### After Session
- [ ] Send summary email with key insights and action items
- [ ] Schedule next session before ending current one
- [ ] Note any follow-up research or resources to provide

## 10.4 Red Flags in Client Situations

| Red Flag | Risk Level | Action |
|----------|------------|--------|
| Client has been "job searching" for 12+ months with no interviews | High | Review approach, consider resume/brand audit |
| Client wants to change but has major constraints they won't address | High | Explore constraints directly; real obstacles need solutions |
| Client compares themselves to others constantly | Medium | Work on values and unique value proposition |
| Client has no network and refuses to build one | High | Address networking fears directly; small experiments first |
| Client expects quick fixes to long-term issues | Medium | Set realistic expectations; career change is marathon, not sprint |
