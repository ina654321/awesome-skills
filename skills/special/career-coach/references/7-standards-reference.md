## § 7 · Standards & Reference

### 7.1 Career Coaching Frameworks

| Framework| When to Use| Key Steps|
|-----------------|----------------------|-------------------|
| **GROW Model** | Goal setting and action planning | 1. Goal → 2. Reality → 3. Options → 4. Will |
| **Career Construction** | Major career transition | 1. Story (past) → 2. Self (identity) → 3. Vision (future) → 4. Plan |
| **Skills Gap Analysis** | Transition to new role | 1. Target role requirements → 2. Current skills → 3. Gap identification → 4. Learning plan |
| **Informational Interview Method** | Network building, market research | 1. Target list → 2. Outreach script → 3. Questions → 4. Follow-up → 5. Next-level connections |

### 7.2 Career Metrics

| Metric| Formula| Target|
|--------------|--------------|---------------|
| **Resume Score** | Keywords matched ÷ job requirements | >70% for initial screening |
| **Network Velocity** | New connections per month during job search | >15/month for active search |
| **Interview Ratio** | Applications sent ÷ interviews received | 10:1 typical; 5:1 strong |
| **Offer Rate** | Interviews ÷ offers received | 3:1 good; 2:1 strong |
| **Salary Benchmark** | Offer vs market median | 90-110% = competitive |
