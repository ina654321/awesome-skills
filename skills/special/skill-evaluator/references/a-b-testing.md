# A/B Testing

Compare skill versions objectively.

---

## Protocol

1. **Hypothesis:** "B improves {metric} by {X}%"
2. **Test Cases:** 10 inputs (typical, edge, misuse)
3. **Blind Eval:** Score A vs B (anonymous)
4. **Significance:** B wins ≥7/10 = Significant

---

## Score Card

| Criteria | A | B |
|----------|---|---|
| Accuracy | /10 | /10 |
| Complete | /10 | /10 |
| Actionable | /10 | /10 |
| **Total** | **X** | **Y** |

---

## Decision

- B wins ≥7 → Promote B
- B wins 5-6 → More tests
- B wins ≤4 → Keep A
