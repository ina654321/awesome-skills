# Standard Tier — Domain Skills

**Scope:** Domain expertise | **Lines:** 150-400 | **Time:** 2 hrs | **Score:** 6.5-8.0

---

## 10-Section Structure

1. **System Prompt** — Role + heuristics
2. **What This Skill Does** — 3-5 capabilities
3. **Core Framework** — Concepts + decision tree
4. **Professional Toolkit** — Tools + templates
5. **Usage Patterns** — 3-5 scenarios
6. **Common Pitfalls** — 3-5 anti-patterns
7. **Risk & Limitations** — Boundaries
8. **Integration** — Cross-skill combos
9. **Quality Checklist** — Self-verify
10. **Metadata** — Full YAML

---

## 2-Hour Flow

```
Min 0-30:   System Prompt + Capabilities + Metadata
Min 30-90:  Framework + Toolkit + Patterns
Min 90-120: Pitfalls + Risks + Integration + Review
```
