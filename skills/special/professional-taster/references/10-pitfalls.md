# Common Pitfalls & Anti-Patterns

## 10.1 High-Severity Anti-Patterns

### 🔴 Anti-Pattern 1: Serving Wine at Wrong Temperature
```
❌ BAD: Serving red wine at room temperature (68°F+) or white wine ice cold
✅ GOOD: Reds: 55-65°F; Whites: 45-50°F; Champagne: 40-45°F
```
**Why it matters**: Temperature dramatically affects perception; too warm amplifies alcohol, too cold suppresses aromas.

### 🔴 Anti-Pattern 2: Not Decanting Properly
```
❌ BAD: Never decanting young tannic reds or over-decanting fragile older wines
✅ GOOD: Young reds: 1-2 hours; older reds: just before serving, limited air exposure
```
**Why it matters**: Proper decanting allows evolution; improper timing ruins the wine.

### 🔴 Anti-Pattern 3: Storing Wine Improperly
```
❌ BAD: Keeping wine in kitchen (temperature swings), upright bottles, direct sunlight
✅ GOOD: 55°F constant, horizontal position (cork wet), darkness, minimal vibration
```
**Why it matters**: Improper storage ruins wine quality and investment potential.

### 🔴 Anti-Pattern 4: Using Wrong Glassware
```
❌ BAD: Drinking fine wine from thick, small glasses; drinking from dirty glasses
✅ GOOD: Thin-rimmed, appropriate-sized glasses; rinse with hot water, no soap
```
**Why it matters**: Glass shape and quality dramatically affect aroma and taste perception.

## 10.2 Medium-Severity Anti-Patterns

### 🟡 Anti-Pattern 5: Swirling Aggressively
```
❌ BAD: Vigorous swirling that splashes and loses wine
✅ GOOD: Gentle swirl to release aromatics; rest between swirls
```

### 🟡 Anti-Pattern 6: Skipping the Spittoon
```
❌ BAD: Drinking continuously during professional tasting sessions
✅ GOOD: Spit-and-pour for professional evaluation; stay sharp and accurate
```

### 🟡 Anti-Pattern 7: Holding Wine Too Long in Mouth
```
❌ BAD: Chewing wine like food; overwhelming palate
✅ GOOD: "Chew" briefly, then suck air through wine to assess aromatics
```

### 🟡 Anti-Pattern 8: Ignoring Personal Palate
```
❌ BAD: Always deferring to expert opinions on what tastes "good"
✅ GOOD: Learn vocabulary; develop your own palate; preferences matter
```

## 10.3 Best Practices Checklist

### Tasting Environment
- [ ] Proper lighting (natural light best)
- [ ] Neutral environment (no competing odors)
- [ ] White surface for color evaluation
- [ ] Appropriate glassware
- [ ] Proper temperature verification

### Personal Preparation
- [ ] Avoid strong perfumes/colognes
- [ ] Don't taste when congested (numbed palate)
- [ ] Avoid strongly flavored foods before tasting
- [ ] Stay hydrated (water between tastings)
- [ ] Start with light wines, progress to bold

### Professional Conduct
- [ ] Use spit-and-pour for multiple tastings
- [ ] Take notes contemporaneously
- [ ] Use structured evaluation form
- [ ] Consider all quality dimensions
- [ ] Be objective — your preference ≠ quality

## 10.4 Red Flags to Watch For

| Red Flag | Risk Level | Action |
|----------|------------|--------|
| Corked wine in restaurant | High | Return it; don't drink flawed wine |
| Wine shows vinegar character | High | Likely spoilage; return or discard |
| "Brett" character (barnyard) | Medium | Controversial — some find it offensive, some consider complexity |
| Oxidation in white wine | High | Too much air exposure; return if just opened |
| Hydrogen sulfide (rotten egg) | Medium | Can often be remedied by decanting or disappears with air |
