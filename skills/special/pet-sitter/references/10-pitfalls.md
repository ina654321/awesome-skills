# Common Pitfalls & Anti-Patterns

## 10.1 High-Severity Anti-Patterns

### 🔴 Anti-Pattern 1: Not Following Feeding Instructions
```
❌ BAD: "She probably needs more food" or giving table scraps
✅ GOOD: Follow exact feeding amounts and schedule; report if pet seems hungry
```
**Why it matters**: Different pets have different needs; overfeeding causes illness; sudden diet changes cause digestive upset.

### 🔴 Anti-Pattern 2: Letting Pet Escape
```
❌ BAD: Leaving door/gate open; not securing collar
✅ GOOD: Double-check all closures; use slip lead if needed; never leave dog unattended
```
**Why it matters**: Escaped pets face traffic, getting lost, injury; owner's trust is shattered.

### 🔴 Anti-Pattern 3: Ignoring Warning Signs
```
❌ BAD: "They seem fine" despite lethargy, not eating, unusual behavior
✅ GOOD: Document and report any changes; contact vet if concerned
```
**Why it matters**: Pets hide illness; changes often indicate problems; early intervention saves lives.

### 🔴 Anti-Pattern 4: Not Disclosing Incidents
```
❌ BAD: "I won't mention the scratch/bite to the owner"
✅ GOOD: Always disclose all incidents honestly and immediately
```
**Why it matters**: Honesty maintains trust; undisclosed incidents create liability; owner needs full information.

## 10.2 Medium-Severity Anti-Patterns

### 🟡 Anti-Pattern 5: Inconsistent Communication
```
❌ BAD: Owner doesn't know what's happening; radio silence for days
✅ GOOD: Send daily updates with photos; report anything notable immediately
```

### 🟡 Anti-Pattern 6: Skipping Daily Logs
```
❌ BAD: "I'll remember what happened" — no written record
✅ GOOD: Keep daily log of feeding, medications, walks, behavior, concerns
```

### 🟡 Anti-Pattern 7: Unauthorized Additions
```
❌ BAD: Giving treats, toys, or items owner didn't approve
✅ GOOD: Stick to approved items; ask before adding anything new
```

### 🟡 Anti-Pattern 8: Inadequate Security
```
❌ BAD: Leaving windows open, doors unlocked, garage accessible
✅ GOOD: Full perimeter check before leaving; know all entry/exit points
```

## 10.3 Best Practices Checklist

### Before Assignment
- [ ] Complete intake consultation
- [ ] Get emergency authorizations in writing
- [ ] Locate all pet supplies
- [ ] Confirm schedule and special needs
- [ ] Get multiple contact methods for owner

### During Assignment
- [ ] Follow feeding/medication schedule exactly
- [ ] Send daily updates with photos
- [ ] Report any incidents immediately
- [ ] Keep detailed daily logs
- [ ] Monitor health daily
- [ ] Secure home on every exit/entry

### After Assignment
- [ ] Clean home as found
- [ ] Complete final health check
- [ ] Provide detailed written report
- [ ] Return all keys/supplies
- [ ] Schedule follow-up if needed

## 10.4 Red Flags to Watch For

| Red Flag | Risk Level | Action |
|----------|------------|--------|
| Pet aggression toward you | High | Stop interaction; contact owner; document |
| Not eating/drinking >24 hours | High | Contact vet; notify owner |
| Signs of pain or distress | High | Immediate vet consultation |
| Property damage | Medium | Document; assess cause; report to owner |
| Unauthorized visitors | High | Secure home; report breach to owner |
