# Common Pitfalls & Anti-Patterns

## 10.1 High-Severity Anti-Patterns

### 🔴 Anti-Pattern 1: Revealing Identity Prematurely
```
❌ BAD: Telling employees you're a mystery shopper when asked casually
✅ GOOD: Stick to your cover story; only reveal if safety is at risk or protocol violated
```
**Why it matters**: Revealing identity compromises current and future shops; client loses valuable data.

### 🔴 Anti-Pattern 2: Letting Personal Bias Score
```
❌ BAD: Rating lower because you didn't like the employee's personality
✅ GOOD: Rate against stated standards, not personal preference; document facts
```
**Why it matters**: Subjective scoring fails the client — they can't act on unfair evaluations.

### 🔴 Anti-Pattern 3: Incomplete Documentation
```
❌ BAD: Submitting report hours later with missing details
✅ GOOD: Document during or immediately after visit; every detail counts
```
**Why it matters**: Memory fades quickly; missing details make reports unreliable.

### 🔴 Anti-Pattern 4: Failing to Complete Assignment
```
❌ BAD: Leaving early because "the experience was bad enough"
✅ GOOD: Complete the full journey as instructed; partial data is worse than none
```
**Why it matters**: Client needs complete picture; partial shops skew results.

## 10.2 Medium-Severity Anti-Patterns

### 🟡 Anti-Pattern 5: Paraphrasing Quotes
```
❌ BAD: "The employee seemed annoyed"
✅ GOOD: Employee stated, "I'm going to need you to wait" (verbatim)
```

### 🟡 Anti-Pattern 6: Forgetting Sensory Details
```
❌ BAD: Only noting employee actions, ignoring environment
✅ GOOD: Include temperature, lighting, sounds, smells — full experience matters
```

### 🟡 Anti-Pattern 7: Inconsistent Timing
```
❌ BAD: Reporting "a few minutes" or "quite a while"
✅ GOOD: Report exact times: "Waited 4 minutes 30 seconds for check"
```

### 🟡 Anti-Pattern 8: Missing Negative Details
```
❌ BAD: Focusing on positives; negatives seem minor to note
✅ GOOD: Report everything; what seems minor may be the pattern client needs
```

## 10.3 Best Practices Checklist

### Before the Shop
- [ ] Brief reviewed and understood
- [ ] Scenario/cover story prepared
- [ ] Documentation materials ready
- [ ] Location research completed
- [ ] Timing requirements confirmed

### During the Shop
- [ ] Arrived on time (or early)
- [ ] Noted exact timestamps
- [ ] Recorded verbatim quotes
- [ ] Captured sensory details
- [ ] Stayed in character throughout
- [ ] Completed full journey

### After the Shop
- [ ] Completed report within 2 hours
- [ ] Included all required elements
- [ ] Scored objectively against standards
- [ ] Verified facts and timings
- [ ] Submitted by deadline

## 10.4 Red Flags to Watch For

| Red Flag | Risk Level | Action |
|----------|------------|--------|
| Employee recognizes you from previous shop | High | Report to client; may need new shopper profile |
| Pressure to reveal identity | High | Follow protocol; report to client immediately |
| Witnessing illegal activity | High | Leave immediately; report to client and authorities |
| Score feels influenced by personal preference | Medium | Review against standards; get second opinion |
| Incomplete visit due to external factors | Medium | Document what you observed; note incomplete |
