# Standard Workflow

## 8.1 Pre-Shop Preparation

### Step 1: Brief Review
- Read assignment details thoroughly
- Note client, location, date/time requirements
- Understand shopper profile to assume
- Identify specific evaluation criteria

### Step 2: Research
- Review client's service standards if provided
- Research competitor benchmarks
- Study the location (website, reviews, maps)
- Understand typical customer journey

### Step 3: Scenario Preparation
- Define your "need" for the visit
- Prepare questions or requests you might make
- Know your backstory if undercover profile needed
- Bring required materials (cash, cards, notepad)

### Step 4: Documentation Setup
- Prepare report template
- Charge phone for photos (if allowed)
- Have stopwatch ready for timing
- Review any specific protocols

## 8.2 During the Shop

### Step 1: Arrival & Environment
- Arrive at scheduled time
- Note appearance: exterior cleanliness, signage, parking
- Note store opening/closing times atmosphere
- Start timing from entry point

### Step 2: Initial Interaction
- Note employee greeting (timing, warmth, eye contact)
- Observe employee appearance and demeanor
- Note any offers of assistance
- Assess store layout and product availability

### Step 3: Transaction/Simulation
- Execute your shopping scenario
- Note product knowledge if applicable
- Time the transaction process
- Observe problem resolution if you raise an issue

### Step 4: Closing Interaction
- Note farewell from employees
- Observe post-transaction behavior
- Final observations before leaving
- Note time of exit

## 8.3 Post-Shop Documentation

### Immediate (Within 2 Hours)
- Complete basic report while memory is fresh
- Note exact quotes heard
- Record specific observations
- Rate each evaluation criterion
- Add any exceptional positive or negative details

### Detailed Report (Within 24 Hours)
- Expand initial notes into full narrative
- Add sensory details (smells, sounds, temperature)
- Include verbatim employee quotes
- Add context for scoring decisions
- Attach receipts or documentation if applicable

### Quality Review
- Check for completeness
- Verify facts and timings
- Ensure objectivity (no interpretation in facts section)
- Confirm scoring aligns with observations

## 8.4 Client Communication

### Standard Reporting
- Submit within required timeframe
- Use client's preferred format/template
- Include all required elements
- Flag urgent issues immediately

### Escalation (If Needed)
- Safety concerns: Report immediately
- Compliance violations: Report immediately to client
- Exceptional service: Note for recognition programs
- Significant failures: Highlight in summary
