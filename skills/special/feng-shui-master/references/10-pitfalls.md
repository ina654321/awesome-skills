# Common Pitfalls & Anti-Patterns

## 10.1 High-Severity Anti-Patterns

### 🔴 Anti-Pattern 1: Adding Remedies Without Clearing
```
❌ BAD: Placing feng shui cures in cluttered, messy spaces
✅ GOOD: Clear clutter FIRST; remedies in cluttered space amplify blocked energy
```
**Why it matters**: Clutter is the primary cause of feng shui problems; remedies can't overcome it.

### 🔴 Anti-Pattern 2: Arbitrary Symbol Placement
```
❌ BAD: Placing dragon here because it's lucky, without considering current Flying Star
✅ GOOD: Match symbols to specific affected stars and elements; timing matters
```
**Why it matters**: Misplaced symbols can create elemental clashes or worsen afflictions.

### 🔴 Anti-Pattern 3: Ignoring Compass Directions
```
❌ BAD: Assuming front of house faces street based on appearance
✅ GOOD: Always use compass to verify true facing; wrong facing = wrong analysis
```
**Why it matters**: All classical feng shui is direction-based; wrong direction leads to wrong remedies.

### 🔴 Anti-Pattern 4: Recommending Based on Modern/BTB Only
```
❌ BAD: Only using Black Sect Tantric (modern) feng shui without understanding limitations
✅ GOOD: Use classical methods for comprehensive analysis; know both traditions
```
**Why it matters**: Classical feng shui provides more precise, time-tested analysis.

## 10.2 Medium-Severity Anti-Patterns

### 🟡 Anti-Pattern 5: Overcomplicating Simple Spaces
```
❌ BAD: Recommending 15 cures for a small apartment
✅ GOOD: Focus on highest-impact changes; start with fundamentals
```

### 🟡 Anti-Pattern 6: Ignoring Occupant's Birth Data
```
❌ BAD: Treating all spaces the same regardless of who's living there
✅ GOOD: Consider occupant's Gua for optimal directions and element needs
```

### 🟡 Anti-Pattern 7: Neglecting External Environment
```
❌ BAD: Only analyzing interior without considering what's outside
✅ GOOD: Check for sha qi, poisoning arrows, and environmental factors
```

### 🟡 Anti-Pattern 8: Assuming All Changes Are Immediate
```
❌ BAD: Expecting instant results from remedies
✅ GOOD: Set realistic expectations; feng shui works with natural cycles
```

## 10.3 Best Practices Checklist

### Before Analysis
- [ ] Obtain accurate floor plan with compass directions
- [ ] Determine building period for Flying Star calculation
- [ ] Gather occupant birth data if available
- [ ] Note external environment factors

### During Analysis
- [ ] Take compass readings in person or verify directions
- [ ] Calculate Flying Star chart for current period
- [ ] Apply Ba Zhai analysis for occupant fit
- [ ] Cross-reference multiple methods

### Recommendations
- [ ] Prioritize by impact and cost
- [ ] Clear clutter first before any other remedy
- [ ] Match remedies to specific afflictions
- [ ] Provide timeline for results

## 10.4 Red Flags to Watch For

| Red Flag | Risk Level | Action |
|----------|------------|--------|
| "Feng shui cure" costing thousands without clearing | High | Always clear before spending on remedies |
| Guarantee of immediate results | Medium | Feng shui works with cycles; no guarantees |
| Recommendations without compass reading | High | Direction-based analysis requires compass |
| One-size-fits-all formulas | Medium | Each space and occupant is unique |
