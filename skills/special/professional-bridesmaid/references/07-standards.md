# Standards & Reference

## 7.1 Wedding Traditions Reference

### Western Wedding Traditions
| Element | Description | Professional Notes |
|---------|-------------|-------------------|
| **Rehearsal** | Practice ceremony day before | Confirm all participants know cues |
| **Processional** | Wedding party walks down aisle | Timing, music cues, spacing |
| **Ceremony** | Vows, rings, kiss | Coordinate with officiant |
| **Recessional** | Newlyweds exit first | Photographer coordination |
| **First Dance** | Couple's debut dance | Timing, song choice |

### Chinese Wedding Traditions
| Element | Description | Key Timing |
|---------|-------------|------------|
| **Jie Yin ( 接亲 )** | Groom's door-opening ceremony | Morning of wedding |
| **Cha Huang ( 查房 )** | Bridal suite visit | Before ceremony |
| **Tea Ceremony** | Tea offering to elders | 1-2 hours before ceremony |
| **San Jian ( 三献 )** | Three bows | During ceremony |
| **Hai Dian ( 闹洞房 )** | Bedroom festivities | Evening after reception |

### Cross-Cultural Considerations
- Ask couple about specific traditions they want included
- Research meaning behind each tradition
- Assign someone to explain to guests unfamiliar with customs
- Plan buffer time for tradition-specific activities

## 7.3 Bridal Party Roles

### Standard Bridal Party
| Role | Primary Duties |
|------|---------------|
| **Maid/Matron of Honor** | Emotional support, bachelorette planning, dress handling |
| **Best Man** | Groom support, rehearsal coordination, rings |
| **Bridesmaids** | Pre-wedding events, ceremony support, moral support |
| **Groomsmen** | Groom preparation, ushers, guest assistance |
