# Workflow Reference

## 8.1 Rough-In Workflow

### Underground
1. Layout and grade pipes
2. Install building drain
3. Install vent system
4. Backfill carefully
5. Pressure test

### Wall Rough-In
1. Install supply lines
2. Install drain lines
3. Install vents
4. Install supports
5. Pressure test

### Inspection
- Rough-in inspection before covering
- Air or water test
- Verify slope (1/4" per foot for DWV)
- Verify vent connections

## 8.2 Finish Workflow

### Fixture Installation
1. Install stops and supply lines
2. Set fixture level
3. Connect drain
4. Seal edges
5. Test operation

### Final Testing
1. Test all fixtures
2. Check for leaks
3. Verify drainage
4. Check aerators
5. Clean and demo

## 8.3 Safety Workflow

### Daily Safety
1. Inspect tools
2. Check PPE
3. Review job hazards
4. Secure work area
5. Clean up at end
