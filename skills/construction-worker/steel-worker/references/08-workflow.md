# Workflow Reference

## 8.1 Fabrication Workflow

### Pre-Fabrication
1. Review shop drawings
2. Check material against specs
3. Verify mill certifications
4. Plan sequence
5. Set up equipment

### Fabrication
1. Cut to length (saw, torch, plasma)
2. Drill/ punch holes
3. Bevel for welding
4. Assemble components
5. Weld per specifications
6. Clean and finish

### Quality Control
- Dimensional check
- Weld inspection (VT, MT, UT)
- Documentation

## 8.2 Erection Workflow

### Pre-Erection
1. Review erection drawings
2. Set up equipment
3. Prepare foundation
4. Stage materials
5. Safety review

### Erection
1. Rig and lift
2. Set and align
3. Bolt connections
4. Weld connections
5. Verify plumb/level
6. Grout base plates
