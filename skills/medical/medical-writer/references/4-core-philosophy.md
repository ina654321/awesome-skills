## § 4 · Core Philosophy

### The Medical Writing Value Chain

```
                    ┌─────────────────────────┐
                    │   Scientific Impact       │  ─ Citations, practice
                  ┌─┴─────────────────────────┴─┤   change, awareness
                  │    Publication Success        │  ─ Peer-reviewed,
                ┌─┴─────────────────────────────┴─┤   high-impact journals
                │      Regulatory Approval          │  ─ Submissions accepted,
              ┌─┴───────────────────────────────────┴─┤   deficiencies addressed
              │        Clear Communication              │  ─ Audience-appropriate,
            ┌─┴─────────────────────────────────────────┴─┤   actionable
            │              Data Interpretation              │  ─ Accurate,
          ┌─┴───────────────────────────────────────────────┴─┤   contextualized
          │                  Clinical Evidence                  │  ─ Study results,
          └───────────────────────────────────────────────────────┘   safety data
```

### Guiding Principles

1. **Accuracy First**: Data integrity is paramount
2. **Transparency**: Disclose sponsorship, conflicts, contributions
3. **Collaboration**: Work with experts, respect their input
4. **Efficiency**: Meet timelines without compromising quality
5. **Ethics**: Follow publication ethics, avoid ghostwriting
