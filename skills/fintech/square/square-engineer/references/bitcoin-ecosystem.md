# Block's Bitcoin Ecosystem

## Overview

Block, Inc. has made Bitcoin a core pillar of its mission to provide economic empowerment. The company operates multiple Bitcoin-focused divisions ranging from open-source development to consumer hardware.

## Spiral

**Purpose**: Bitcoin development and funding
**URL**: spiral.xyz
**Status**: Open Source

### Projects
- **Bitcoin Dev Kit (BDK)**: Library for building Bitcoin applications
- **Lightning Dev Kit (LDK)**: Tools for Lightning Network integration
- **Grants**: Funding Bitcoin developers and projects

### Mission
> "We support Bitcoin developers and projects that advance the open-source Bitcoin ecosystem."

## Proto

**Purpose**: Bitcoin mining hardware and software
**URL**: proto.xyz
**Status**: Partially Open Source (Proto Fleet OS)

### Products
- **Proto Mining Hardware**: Custom ASICs for Bitcoin mining
- **Proto Fleet OS**: Open-source mining software
- **Hosting Services**: Managed mining infrastructure

### Approach
- Decentralized mining
- Focus on accessibility and efficiency
- Open-source software stack

## Bitkey

**Purpose**: Self-custody Bitcoin hardware wallet
**URL**: bitkey.world
**Status**: Commercial product

### Features
- Hardware security module
- Mobile app integration
- Social recovery
- No seed phrase required

### Target Market
- Bitcoin newcomers
- Users wanting self-custody without complexity
- Mobile-first users

## Integration Patterns

### For Merchants
```
Customer → Square POS → Lightning Network → USD Settlement
                              ↓
                        Real-time conversion
```

### For Developers
```
App → Spiral SDK (BDK/LDK) → Bitcoin/Lightning Network
```

## Compliance Notes

- Bitcoin transactions require AML/KYC compliance
- Square handles regulatory requirements for Bitcoin payments
- Self-custody (Bitkey) puts responsibility on user
- Lightning Network reduces fees and settlement time
