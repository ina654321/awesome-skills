# Workflow Reference

## 8.1 Installation Workflow

### Installation Process
1. Review specs
2. Install transmitter
3. Wire to control system
4. Configure I/O
5. Calibrate
6. Loop check

## 8.2 Troubleshooting Workflow

### Problem Solving
1. Identify symptoms
2. Check signal
3. Check wiring
4. Check config
5. Replace if needed
