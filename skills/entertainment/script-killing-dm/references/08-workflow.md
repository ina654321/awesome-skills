## 8. Standard Workflow

### 8.1 Professional 剧本杀 Session Flow

```
Phase 1: Pre-Session Preparation (30-60 mins before)
├── Read script 3+ times: understand plot, identify hidden clues, map all possibilities
├── Prepare NPC voices: develop distinct speech patterns, accents, mannerisms
├── Set up venue: arrange props, test audio, optimize lighting
├── Review player list: note experience levels, special requirements, group dynamics
└── [✓ Done]: All systems tested, all materials ready, all player notes reviewed
    [✗ FAIL]: Missing any element → postpone until complete

Phase 2: Opening & Icebreak (15-20 mins)
├── Welcome players: establish warm, professional tone
├── Distribute characters: dramatic reveal, not mechanical handoff
├── Read 导入语: set scene, establish era, create atmosphere
├── Character introductions: each player presents their role (not secrets yet)
└── [✓ Done]: Players excited, in character, ready to begin
    [✗ FAIL]: Players confused or not engaged → adjust icebreak

Phase 3: Act 1 — Discovery (60-90 mins)
├── First 搜证 round: guide search, ensure all clues found
├── First 集中讨论: facilitate initial theories, note interesting angles
├── DM narration: deliver key story beats, introduce complications
└── [✓ Done]: Core relationships established, mystery partially revealed
    [✗ FAIL]: Players disengaged → inject complication or revelation

Phase 4: Act 2 — Confrontation (60-90 mins)
├── Second 搜证: release more impactful clues
├── Intense debate: manage accusations, maintain civil discourse
├── Personal revelations: trigger character secrets at dramatic moments
├── NPC interactions: perform key scenes with emotional depth
└── [✓ Done]: Core mystery taking shape, player theories forming
    [✗ FAIL]: Wrong person accused → add clue to redirect

Phase 5: Act 3 — Climax & Resolution (30-60 mins)
├── Final 搜证: last chance, make it count
├── 投凶环节: structured voting, collect reasoning
├── Reveal真相: theatrical delivery, honor player contributions
└── [✓ Done]: Mystery solved, all questions answered, players satisfied
    [✗ FAIL]: Unsatisfying solution → enhance 复盘 explanation

Phase 6: Debrief & Closure (15-30 mins)
├── 复盘: explain what really happened, connect all loose ends
├── Character awards: acknowledge standout performances
├── Player reflection: invite sharing of favorite moments
└── [✓ Done]: Emotional closure, players want to return
    [✗ FAIL]: Players confused or dissatisfied → extend debrief
```

### 8.2 Handling Problem Players

```
Step 1: Diagnose the player type
  → 戏霸 (dominate player): hogs airtime, interrupts others
  → 边缘玩家 (peripheral player): quiet, disconnected, unsure
  → 挂机玩家 (disengaged player): on phone, distracted, not trying
  → 抗压玩家 (stressed player): frustrated, taking game too seriously
  → 新手 (beginner): confused about rules, unsure what to do

Step 2: Apply genre-specific technique
  → 戏霸: Assign a负责 role that requires listening to others; use their energy but channel it
  → 边缘玩家: Direct specific questions to them; pair with a supportive player
  → 挂机玩家: Make their character crucial to a key moment; raise stakes
  → 抗压玩家: Check in privately, remind it's for fun, offer simplified version
  → 新手: Pair with veteran, provide extra guidance, celebrate small wins

Step 3: Real-time adjustment
  → If technique fails after 15 mins, try alternative approach
  → If all else fails, have private conversation during break
```
