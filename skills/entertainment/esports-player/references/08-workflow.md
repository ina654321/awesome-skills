# Standard Workflow

## 8.1 Game Analysis Process

### Step 1: Identify the Problem
- Watch gameplay or VOD
- Note specific mistakes
- Categorize: aim, positioning, decision, mechanical
- Prioritize by frequency and impact

### Step 2: Root Cause Analysis
- Ask: Why did X happen?
- Is it habit, knowledge, or skill gap?
- Check: Is this a consistent issue?
- Compare: What should you have done?

### Step 3: Practice Prescription
- Find specific drill/exercise
- Set measurable improvement goal
- Schedule dedicated practice time
- Track progress

### Step 4: Integration
- Practice in isolation first
- Add to matches gradually
- VOD review to confirm improvement
- Repeat if needed

## 8.2 Match Preparation

### Pre-Match (1 hour before)
1. Light warm-up (aim, movement)
2. Review opponent/strategy
3. Communication check with team
4. Set mental goals for match

### During Match
1. Start focused, not aggressive
2. Communicate throughout
3. Adapt based on opponent
4. Take breaks between rounds/games
5. Stay present, don't spiral

### Post-Match
1. Quick mental reset (walk away)
2. Return for VOD review
3. Note 3 key takeaways
4. Plan adjustment for next match

## 8.3 Team Coordination

### Communication Framework
| Call Type | Example | When |
|-----------|---------|------|
| **Info** | "Enemy here" | As it happens |
| **Request** | "Need help" | When support needed |
| **Plan** | "Going A" | Before action |
| **Backup** | "On the way" | Response to request |

### Role Clarity
- Know your role in team comp
- Play to role strengths
- Support teammates appropriately
- Trust others to do their job

## 8.4 Improvement Tracking

### Weekly Review
- Hours played vs. structured practice
- Rank progress
- Death causes breakdown
- Win rate by map/agent/role

### Monthly Assessment
- Goal progress
- Identify plateaus
- Adjust training emphasis
- Set new goals

### Key Metrics by Role
| Role | Focus Metrics |
|------|---------------|
| **DPS/Fragger** | K/D, damage, kills |
| **Support** | Assists, utility usage, lives saved |
| **IGL** | Round wins, calls made |
| **Entry** | First bloods, trades |
