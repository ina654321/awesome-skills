# Common Pitfalls & Anti-Patterns

## 10.1 High-Severity Anti-Patterns

### 🔴 Anti-Pattern 1: Playing Tired
```
❌ BAD: Grinding ranked when exhausted
✅ GOOD: Stop when fatigue affects performance
```
**Why it matters**: Fatigue creates bad habits; you practice mistakes, not improvement.

### 🔴 Anti-Pattern 2: Blaming Teammates
```
❌ BAD: "I'm stuck because of bad teammates"
✅ GOOD: Focus on what YOU can control; carry if possible
```
**Why it matters**: External focus prevents improvement; pros carry bad games.

### 🔴 Anti-Pattern 3: No Warm-Up
```
❌ BAD: Jumping straight into ranked
✅ GOOD: 15-30 min warm-up before competitive
```
**Why it matters**: Cold performance means worse results; increased injury risk.

### 🔴 Anti-Pattern 4: Tilt Queueing
```
❌ BAD: "One more game" after losing streak
✅ GOOD: Stop after 2 losses; reset before continuing
```
**Why it matters**: Tilt leads to worse play; losses compound.

## 10.2 Medium-Severity Anti-Patterns

### 🟡 Anti-Pattern 5: No Review Process
```
❌ BAD: Playing without ever watching own VODs
✅ GOOD: Weekly VOD review of key games
```

### 🟡 Anti-Pattern 6: Practicing Wrong Things
```
❌ BAD: Playing game modes that don't improve skills
✅ GOOD: Focused practice on specific weaknesses
```

### 🟡 Anti-Pattern 7: Copying Pros Without Context
```
❌ BAD: Using pro settings/strategies that don't fit you
✅ GOOD: Adapt to your playstyle; test and iterate
```

### 🟡 Anti-Pattern 8: Ignoring Health
```
❌ BAD: Playing 8+ hours without breaks
✅ GOOD: 5-10 min break every hour; stretch
```

## 10.3 Best Practices Checklist

### Daily Routine
- [ ] Warm-up before competitive play
- [ ] Set session goal before playing
- [ ] Take breaks between games
- [ ] End before fatigue sets in
- [ ] Basic VOD review of key games

### Weekly Routine
- [ ] Review overall performance
- [ ] Identify improvement focus
- [ ] Structured practice schedule
- [ ] Physical exercise
- [ ] Rest and recovery

### Mental Game
- [ ] Manage tilt actively
- [ ] Focus on process, not rank
- [ ] Accept losses as learning
- [ ] Celebrate wins mindfully

## 10.4 Red Flags to Watch For

| Red Flag | Risk Level | Action |
|----------|------------|--------|
| Playing from 3+ hours without break | High | Stop; rest; risk of RSI |
| Blaming same thing on teammates | Medium | Self-review; what could YOU do? |
| Rank declining over week | High | Stop; assess; rest |
| Physical discomfort (wrist, neck) | High | Stop immediately; see doctor |
| Not having fun | Medium | Consider break; reframe goals |
