# Standards & Reference

## 7.1 Competitive Gaming Fundamentals

### Skill Hierarchy
| Skill | Description | Priority |
|-------|-------------|----------|
| **Aim** | Precision targeting, tracking | Critical |
| **Game Sense** | Decision-making, positioning | Critical |
| **Mechanics** | Movement, ability usage | High |
| **Communication** | Team coordination | High |
| **Mental** | Focus, tilt management | High |

### Rank Tiers (General)
| Tier | Description | Typical % |
|------|-------------|-----------|
| **Bronze/Silver** | Learning fundamentals | 0-30% |
| **Gold/Platinum** | Competent play | 30-60% |
| **Diamond/Master** | Strong players | 60-85% |
| **Grandmaster/Challenger** | Elite competitive | 85-99% |
| **Professional** | Top 0.1% | Tournament play |

## 7.2 Training Methodology

### Practice Structure
1. **Warm-up** (15-30 min): Aim training, movement drills
2. **VOD Review** (30 min): Own gameplay analysis
3. **Focused Practice** (1-2 hours): Specific skills
4. **Competitive Play** (2-3 hours): Ranked/scrims
5. **Review** (30 min): Notes and takeaways

### Improvement Focus
| Area | Practice Method | Daily Time |
|------|-----------------|-----------|
| **Aim** | Aim trainers, custom maps | 30-60 min |
| **Positioning** | VOD review, death analysis | 30 min |
| **Decision-making** | Game sense exercises | 30 min |
| **Mechanics** | Movement drills, keybinds | 20 min |

## 7.3 Tournament Preparation

### Pre-Tournament
| Timeline | Activity |
|----------|----------|
| **1 week before** | Reduce ranked play, focus on fundamentals |
| **3 days before** | Light practice, mental prep |
| **1 day before** | Minimal practice, rest, sleep |
| **Day of** | Arrive early, warm up, review strategy |

### During Tournament
- Stay hydrated
- Take breaks between matches
- Communicate clearly with team
- Focus on process, not results
- Reset between games

## 7.4 Mental Performance

### Tilt Management
| Technique | Application |
|-----------|-------------|
| **Micro-breaks** | 30 seconds between games |
| **Breathing** | 4-7-8 technique during stress |
| **Reset ritual** | Physical action to signal new game |
| **Positive self-talk** | Focus on controllable factors |

### Focus Enhancement
- Remove distractions
- Set session goals
- Use Pomodoro technique (25 min focus, 5 min break)
- End sessions before fatigue sets in
