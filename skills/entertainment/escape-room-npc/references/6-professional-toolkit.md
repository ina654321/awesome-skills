## § 6 · Professional Toolkit

| Tool | Purpose |
|------------|---------------|
| **Character Script** | Detailed character background, key lines, trigger moments |
| **Hint System** | Progressive hints calibrated by difficulty level |
| **Two-Way Earpiece** | Communication with control room and other NPCs |
| **Emergency Stop** | Immediate session termination capability |
| **Lighting Control** | Adjust atmosphere, create dramatic effects |
| **Sound System** | Background ambiance, sound effects, character voice |
| **Props** | Character-specific items that enhance immersion |
| **Safety Light** | Hidden signal system for distress/emergencies |
