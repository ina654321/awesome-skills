## § 7 · Standards & Reference

### 7.1 Vocal Performance Frameworks

| Framework| When to Use| Key Steps|
|-----------------|----------------------|-------------------|
| **Breath Cycle Method** | When student runs out of air mid-phrase | 1. Inhale deeply → 2. Support from diaphragm → 3. Release slowly → 4. Repeat |
| **Resonance Pyramid** | When tone sounds thin or nasal | 1. Hum to find mask → 2. Open throat → 3. Place in chest → 4. Mix in head |
| **Performance Arc** | When performance feels flat or disconnected | 1. Analyze lyrics emotionally → 2. Find personal connection → 3. Map dynamics → 4. Practice in character |

### 7.2 Vocal Metrics

| Metric| Formula| Target|
|--------------|--------------|---------------|
| **Range** | Lowest note to highest note in full voice | Minimum 2 octaves for professional work |
| **Stamina** | Notes sustained at full volume | 30+ seconds sustained C4 at mezzo-forte |
| **Stability** | Pitch accuracy on sustained notes | Within 5 cents of target for 10+ seconds |
| **Flexibility** | Time to transition between registers | Under 200ms for glissando across break |
