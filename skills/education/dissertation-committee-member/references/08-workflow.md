# Standard Workflow

## 8.1 Prospectus/Proposal Review

```
Phase 1: Initial Review
├── Read entire prospectus
├── Assess literature coverage
├── Evaluate theoretical framework
├── Review methodology design
├── Check feasibility
└── Provide written feedback

Phase 2: Committee Meeting
├── Schedule meeting
├── Distribute materials in advance
├── Prepare questions
├── Conduct meeting
└── Document feedback
```

## 8.2 Dissertation Progress Monitoring

```
Milestone Tracking:
├── Prospectus defense
├── IRB approval
├── Data collection
├── Preliminary results
├── Writing progress
└── Final defense

Progress Meetings:
├── Regular schedule (monthly)
├── Written progress reports
├── Obstacle identification
├── Resource needs
└── Timeline adjustment
```

## 8.3 Chapter Review Process

```
Review Workflow:
1. Student submits draft
2. Chair reviews (2-3 weeks)
3. Student revises
4. Other committee members review
5. Final approval
6. Proceed to next chapter

Feedback Types:
├── Major concerns (must address)
├── Minor suggestions (should address)
├── Style comments (optional)
└── Clarification requests
```

## 8.4 Defense Preparation

```
Pre-Defense:
├── Complete all revisions
├── Distribute final draft (2-4 weeks before)
├── Schedule defense
├── Arrange room/technology
├── Prepare presentation
└── Anticipate questions

Day of Defense:
├── Arrive early
├── Test technology
├── Distribute handouts
├── Present findings
├── Answer questions
└── Receive decision
```

## 8.5 Post-Defense

```
Final Steps:
├── Address committee revisions
├── Final formatting review
├── Upload to repository
├── Complete survey/evaluation
├── Submit final paperwork
└── Celebrate completion
```
