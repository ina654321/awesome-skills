## § 7 · Standard Workflow

### Phase 1: Observation and Documentation

| Step | Activity | Done Criteria | Fail Criteria |
|------|----------|---------------|---------------|
| 1 | Conduct systematic observations | Anecdotal notes, photos, videos collected | Judgmental notes without objective description |
| 2 | Document learning moments | Learning stories written; panels created | No documentation of children's learning |
| 3 | Analyze developmental progress | Portfolio organized; milestones tracked | Assessment through formal testing only |
| 4 | Identify individual needs | Individual goals set for each child | One-size-fits-all curriculum |
| 5 | Communicate with families | Regular updates; conference preparation | Families unaware of child's progress |

### Phase 2: Planning and Implementation

| Step | Activity | Done Criteria | Fail Criteria |
|------|----------|---------------|---------------|
| 1 | Design emergent curriculum based on observations | Curriculum web created following children's interests | Predetermined curriculum without child input |
| 2 | Prepare environment and materials | Provocations set; centers organized | Worksheets and direct instruction dominate |
| 3 | Facilitate play-based learning | Teacher as co-learner and guide | Teacher-directed activities only |
| 4 | Scaffold learning and language | Open-ended questions; sustained shared thinking | Closed questions; testing children's knowledge |
| 5 | Document and reflect | Documentation displayed; team reflection meetings | No record of learning or program reflection |
