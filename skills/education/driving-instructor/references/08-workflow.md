# Standard Workflow

## 8.1 Beginner Driver Progression

```
Phase 1: Pre-Driving (Classroom)
├── Traffic laws and road signs
├── Vehicle controls and dashboard
├── Blind spot awareness
└── Hazard perception introduction

Phase 2: Closed Course / Parking Lot
├── Starting and stopping
├── Steering and basic maneuvering
├── Parking: straight-in, angled
└── Low-speed hazard responses

Phase 3: Low-Traffic Streets
├── Intersection navigation
├── Lane changing with signals
├── Right-of-way application
└── Speed maintenance at 25-35 mph

Phase 4: Mixed Traffic / Highway
├── Highway entry/exit
├── Merging and weaving
├── Night driving
└── Adverse weather conditions
```

## 8.2 Lesson Planning Workflow

### Standard 60-Minute Lesson Structure

1. **Pre-lesson (10 min)**: Review previous lesson, set goals for this session
2. **Demo/drills (15 min)**: Instructor demonstrates, student observes
3. **Practice (30 min)**: Student drives with corrective coaching
4. **Debrief (5 min)**: Summarize progress, assign home practice

### Remedial Lesson Workflow

1. Identify specific skill deficit (observation during prior lesson)
2. Break skill into micro-components
3. Practice each component in isolation
4. Reintegrate into full driving task
5. Reinforce with positive specific feedback

## 8.3 Assessment Workflow

1. Begin lesson with informal check of prior knowledge
2. Observe student during routine driving tasks
3. Introduce one stress variable at a time
4. Rate performance using standardized rubric
5. Provide written feedback with specific action items
