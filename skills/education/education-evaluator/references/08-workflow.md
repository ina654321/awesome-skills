# Standard Workflow

## 8.1 Standard Evaluation Workflow

```
Phase 1: Scoping (1-2 weeks)
├── Define evaluation purpose and stakeholders
├── Identify evaluation questions (max 5 key questions)
├── Determine evaluation type (formative/summative, process/outcome)
├── Establish baseline metrics and comparison groups
└── Confirm data access and ethics approval

Phase 2: Data Collection Design (2-3 weeks)
├── Select mixed methods (quantitative + qualitative)
├── Design instruments: surveys, rubrics, interview protocols
├── Pilot test instruments with 5-10 pilot users
├── Finalize sampling strategy and recruitment plan
└── Set data collection timeline aligned with intervention milestones

Phase 3: Data Collection (4-12 weeks)
├── Administer baseline surveys/instruments
├── Conduct observations and interviews
├── Collect usage data (platform analytics, LMS logs)
├── Conduct focus groups with key stakeholder groups
└── Monitor data quality throughout

Phase 4: Analysis (3-6 weeks)
├── Quantitative: Descriptive stats, inferential tests, effect sizes
├── Qualitative: Thematic coding, triangulation across sources
├── Integration: Connect quantitative outcomes with qualitative explanations
└── Identify unintended effects (positive and negative)

Phase 5: Reporting (2-4 weeks)
├── Draft findings organized by evaluation question
├── Share with stakeholders for member-checking
├── Finalize report with actionable recommendations
└── Present findings to decision-makers; prepare executive summary
```

## 8.2 Evaluation Question Framework

Good evaluation questions are:
- **Focused**: Address one concept per question
- **Feasible**: Answerable with available time, budget, and data
- **Relevant**: Aligned with stakeholder decision needs
- **Measurable**: Can be operationalized with available instruments

Example: "Does [product] improve student math achievement among Year 3 students in rural schools, compared to business-as-usual instruction over one academic year?"

## 8.3 Data Triangulation Protocol

1. Collect at least 3 data types: quantitative outcomes, qualitative feedback, observational data
2. Cross-check findings across sources — do they converge or diverge?
3. Document discrepancies and investigate root causes
4. Weight evidence quality when synthesizing findings
