# Standards & Reference

## 7.1 Bloom's Taxonomy (Revised)

| Level | Verb Examples | Cognitive Process |
|-------|---------------|-------------------|
| Remember | Define, list, recall | Recall facts |
| Understand | Explain, summarize, classify | Interpret meaning |
| Apply | Use, demonstrate, solve | Execute procedures |
| Analyze | Compare, differentiate, attribute | Break into parts |
| Evaluate | Judge, critique, justify | Make judgments |
| Create | Design, construct, develop | Produce original work |

## 7.2 Assessment Types

### Formative Assessment (Ongoing)
| Method | Purpose | Frequency |
|--------|---------|-----------|
| Exit tickets | Quick comprehension check | Daily |
| Think-pair-share | Active engagement | Weekly |
| Observation | Skill development | Ongoing |
| Quizzes | Knowledge check | Weekly |

### Summative Assessment
| Type | Weight | Purpose |
|------|--------|---------|
| Unit test | 20% | Knowledge retention |
| Project | 30% | Application of skills |
| Final exam | 25% | Comprehensive mastery |
| Presentation | 15% | Communication skills |
| Participation | 10% | Engagement |

## 7.3 Kirkpatrick's Evaluation Model

| Level | Evaluation | Questions |
|-------|------------|-----------|
| 1 | Reaction | Did learners enjoy it? |
| 2 | Learning | Did they learn? |
| 3 | Behavior | Did they apply it? |
| 4 | Results | Did it impact outcomes? |
