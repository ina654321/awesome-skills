# Standard Workflow

## 8.1 Lesson Planning

```
Phase 1: Backward Design (Wiggins & McTighe)
├── Identify desired results
├── Determine acceptable evidence
└── Plan learning experiences

Phase 2: Lesson Structure
├── Opening/Hook (5-10 min)
├── Direct instruction (15-20 min)
├── Guided practice (15-20 min)
├── Independent practice (15 min)
└── Closure (5-10 min)

Phase 3: Materials
├── Supplies prepared
├── Technology tested
├── Handouts ready
└── Room arranged
```

## 8.2 Instruction Delivery

```
Direct Instruction:
├── Clear objectives
├── Modeled examples
├── Think-alouds
├── Guided practice with feedback
└── Gradual release

Small Group:
├── Flexible grouping by data
├── Targeted instruction
├── Rotations or stations
├── Conferring
└── Small group reading
```

## 8.3 Assessment Cycle

```
Pre-Assessment:
├── Prior knowledge check
├── Diagnostic assessment
└── Readiness screening

Formative Assessment:
├── Daily checks for understanding
├── Observation
├── Questioning
├── Exit tickets
└── Quick writes

Post-Assessment:
├── Unit assessment
├── Performance task
├── Rubric scoring
└── Data analysis
```

## 8.4 Intervention Workflow

```
Tiers of Support:
├── Tier 1: Core instruction (all students)
├── Tier 2: Targeted intervention (small groups)
└── Tier 3: Intensive intervention (individual)

Intervention Process:
├── Identify struggling learners
├── Diagnose skill gaps
├── Prescribe intervention
├── Monitor progress
└── Adjust based on data
```
