# Common Pitfalls & Anti-Patterns

## 10.1 Instructional Errors

| Pitfall | Severity | Prevention |
|---------|----------|------------|
| Teaching to the test | 🔴 High | Balanced assessment |
| One-size-fits-all | 🔴 High | Differentiation |
| Moving too fast | 🟡 Medium | Check understanding |
| Skipping foundations | 🟡 Medium | Sequential learning |
| Relying on worksheets | 🟡 Medium | Multimodal learning |

## 10.2 Assessment Errors

| Error | Impact | Prevention |
|-------|--------|------------|
| No pre-assessment | Inappropriate instruction | Always pre-assess |
| Teaching to the test | Poor learning | Balanced approach |
| No formative assessment | No feedback | Daily checks |
| Grading vs assessment | Misunderstanding | Clear purpose |

## 10.3 Differentiation Failures

```
Common Issues:
├── Giving more work
├── Unequal challenge
├── Ignoring learning styles
├── Not using data
├── Inflexible grouping
└── One modality only

Solutions:
├── Tiered assignments
├── Flexible grouping
├── Choice boards
├── Multiple entry points
├── Scaffolding
```

## 10.4 Classroom Management Issues

| Issue | Impact | Prevention |
|-------|--------|------------|
| Inconsistent expectations | Chaos | Clear routines |
| Lack of transitions | Wasted time | Practice transitions |
| No positive reinforcement | Low engagement | Recognition system |
| Power struggles | Conflict | De-escalation |

## 10.5 Parent Communication

| Failure | Impact | Prevention |
|---------|--------|------------|
| Reactive only | Distrust | Proactive communication |
| Blame language | Defensiveness | Partnership language |
| Vague feedback | Confusion | Specific examples |
| Not responding | Frustration | Timely response |
