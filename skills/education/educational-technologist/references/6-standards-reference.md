## § 6 · Standards & Reference

**SAMR Technology Integration Model:**
```
Substitution:     Tech replaces tool with no functional change (e.g., digital worksheet)
Augmentation:     Tech replaces tool with some functional improvement (e.g., digital worksheet with auto-grading)
Modification:     Tech allows for significant task redesign (e.g., collaborative digital document with embedded media)
Redefinition:     Tech enables previously inconceivable tasks (e.g., virtual exchange with global classroom)

Goal: Move toward Modification and Redefinition
```

**Quality Matters Rubric Standards:**
```
1. Course Overview and Introduction
2. Learning Objectives (measurable, aligned)
3. Assessment and Measurement (alignment, variety)
4. Instructional Materials (current, relevant, varied)
5. Learning Activities and Learner Interaction
6. Course Technology (support, accessibility)
7. Learner Support (policies, resources)
8. Accessibility and Usability (WCAG 2.1)
```

**WCAG 2.1 AA Checklist (Key Items):**
```
- Alt text for all images
- Captions/transcripts for video
- Color contrast 4.5:1 minimum
- Keyboard navigable
- Screen reader compatible
- Resizable text up to 200%
- Consistent navigation
```
