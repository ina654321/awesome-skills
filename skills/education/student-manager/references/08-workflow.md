# Standard Workflow

## 8.1 Student Intake & Orientation

```
Orientation Workflow
├── Pre-enrollment
│   ├── Application review and eligibility check
│   ├── Document verification (transcripts, ID, residency)
│   └── Placement assessment administration
├── Enrollment
│   ├── Schedule building with student/parent input
│   ├── Course assignment based on prerequisites
│   └── SEOP/graduation plan creation
├── First-week check-in
│   ├── Attendance monitoring
│   ├── Social-emotional screening
│   └── Intervention referral if needed
└── 30-day progress review
    └── Adjustment of supports as needed
```

## 8.2 Academic Monitoring Workflow

- **Weekly**: Review attendance dashboards, flag patterns of absence
- **Bi-weekly**: Check gradebook for D/F grades, initiate contact
- **Monthly**: Update progress reports, communicate with parents
- **Quarterly**: SLO data review, adjust instructional groupings
- **Semester**: Full transcript audit, credit recovery planning

## 8.3 Intervention Referral Workflow

1. Teacher identifies struggling student (academic/behavioral/social-emotional)
2. MTSS tier determination (Tier 1 core, Tier 2 targeted, Tier 3 intensive)
3. Data collection: grades, behavior logs, assessment scores
4. Intervention selection using evidence-based programs
5. Progress monitoring (bi-weekly data points)
6. Team meeting (SST/IEP/504 review if applicable)
7. Outcome decision: exit, continue, or escalate

## 8.4 Graduation & Transition Workflow

1. Audit graduation requirements against current progress
2. Identify credit deficiencies, plan recovery options
3. Post-secondary planning: college applications, career pathways
4. Final transcript preparation and verification
5. Diploma and records issuance
