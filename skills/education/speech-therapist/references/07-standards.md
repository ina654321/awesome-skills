# Standards & Reference

## 7.1 Official Documentation

- [ASHA Practice Portal](https://www.asha.org/practice-portal/) - American Speech-Language-Hearing Association evidence-based practice guidelines
- [ICF Framework (WHO)](https://www.who.int/standards/classifications/international-classification-of-functioning-disability-and-health) - International Classification of Functioning framework for assessment
- [CASANA Childhood Apraxia of Speech](https://www.apraxia-kids.org/) - Evidence-based standards for apraxia assessment and treatment
- [LDA Learning Disabilities Association](https://ldaamerica.org/) - Multi-disciplinary approach to language-based learning disabilities

## 7.2 Assessment Standards

| Domain | Assessment Tools | Age Range |
|--------|-----------------|:---------:|
| Articulation | Goldman-Fristoe Test of Articulation (GFTA-3) | 2-21+ |
| Phonology | Hodson's Assessment of Phonological Patterns | 3-8 |
| Receptive Language | Peabody Picture Vocabulary Test (PPVT-5), OWLS-II | 2-90+ |
| Expressive Language | Expressive Vocabulary Test (EVT-3), CELF-5 | 5-21 |
| Fluency | Stuttering Severity Instrument (SSI-4) | 2-80+ |
| Voice | Consensus Auditory-Perceptual Evaluation of Voice (CAPE-V) | Pediatric-adult |
| Swallowing (dysphagia) | Videofluoroscopic Swallow Study (VFSS), FEES | All ages |

## 7.3 Clinical Competency Standards

| Skill | Entry-Level Competency | Advanced Competency |
|-------|:---:|:---:|
| Case history review | Independently | Identify cultural/linguistic factors |
| Standardized assessment | Administer with supervision | Interpret with clinical judgment |
| Differential diagnosis | Identify obvious patterns | Distinguish complex co-occurring disorders |
| Treatment planning | Write goals and select approaches | Modify approach mid-treatment based on data |
| Parent/coach counseling | Explain findings in lay terms | Address denial, grief, cultural barriers |

## 7.4 Treatment Fidelity Standards

- Minimum 80% of planned intervention elements implemented per session
- Treatment dose: Typically 2-3 times per week for 30-60 minutes
- Minimum 12 sessions before measuring progress (exceptions: fluency, voice)
- Data collected on every session: probe data, treatment accuracy, subjective rating
- Generalization tracked: skills observed in non-treatment settings
