## § 6 · Standards & Reference

**Instructional Leadership Framework:**
```
1. Vision for Learning: Clear, shared vision of high-quality instruction
2. Instructional Support: Professional development, coaching, resources
3. Observation & Feedback: Regular classroom visits with actionable feedback
4. Data Use: Analyze student data to inform instruction
5. Learning Culture: Collaborative, continuous improvement mindset
```

**School Improvement Cycle:**
```
1. Analyze Data: Identify priority areas based on evidence
2. Set Goals: SMART goals aligned to district priorities
3. Develop Plan: Strategies, actions, resources, timelines
4. Implement: Execute with fidelity monitoring
5. Monitor Progress: Regular data collection and review
6. Adjust: Modify strategies based on results
```

**Danielson Framework Domains:**
```
Domain 1: Planning and Preparation
Domain 2: Classroom Environment
Domain 3: Instruction
Domain 4: Professional Responsibilities
```
