## 7.2 CEFR Descriptors for Language Testing

| CEFR Level | Overall Description | Test Alignment |
|:---:|---------------|---------------|
| A1 | Can understand familiar daily expressions | Beginner standardized tests |
| A2 | Can communicate in simple, routine tasks | Elementary tests (PET, TOEIC R&L 220-465) |
| B1 | Can deal with most travel situations | Intermediate tests (TOEFL iBT 57-86, IELTS 4.0-5.0) |
| B2 | Can interact with native speakers fluently | Upper-intermediate (TOEFL iBT 87-109, IELTS 5.5-6.5) |
| C1 | Can express ideas fluently and spontaneously | Advanced (CPE, TOEFL iBT 110-120, IELTS 7.0-8.0) |
| C2 | Can understand virtually everything heard or read | Near-native (C2 Proficiency, IELTS 8.5-9.0) |

## 7.3 Test Validity and Reliability Standards

| Standard | Requirement |
|----------|-------------|
| Content validity | Test items must represent the construct being measured |
| Face validity | Test should appear valid to test-takers |
| Internal consistency | Cronbach's alpha > 0.80 for high-stakes tests |
| Test-retest reliability | Correlation > 0.85 for score consistency |
| Inter-rater reliability | Cohen's kappa > 0.80 for scored components |
| Standard error of measurement | Published alongside scores for interpretation |

## 7.4 Test Administration Standards

- Testing environment: Distraction-reduced, adequate lighting, appropriate temperature
- Timing: Strict adherence to published timing; breaks at designated intervals
- Accommodations: Extra time, separate rooms, assistive technology per documented need
- Security: Item banks encrypted; test forms randomized; proctoring protocols enforced
- Data protection: Score data retained minimum 2 years; GDPR/PDPA compliance
