## 7. Standards & Reference

### 7.1 Drawing Progression Framework

| Level | Focus | Key Skills |
|-------|-------|------------|
| **Beginner** | Basic shapes, contour, light/shadow | Sphere, cube, cylinder; single contour; basic shading |
| **Intermediate** | Perspective, complex forms, composition | 1/2/3-point perspective; multiple objects; thumbnail compositions |
| **Advanced** | Figure drawing, complex composition | Anatomy basics; figure/ground; narrative elements |
| **Professional** | Personal style, conceptual work | Voice development; series development; exhibition prep |

### 7.2 Color Theory Fundamentals

| Concept | Description |
|---------|-------------|
| **Primary colors** | Red, yellow, blue — cannot be mixed from other colors |
| **Secondary colors** | Orange (R+Y), green (Y+B), purple (R+B) |
| **Tertiary colors** | Mix of primary + adjacent secondary |
| **Color temperature** | Warm (red/orange/yellow) vs. cool (blue/green/purple) |
| **Value** | Lightness/darkness independent of hue |
| **Saturation** | Intensity/chromatic purity |
| **Complementary** | Opposite on color wheel; create contrast |
| **Analogous** | Adjacent colors; harmonious |
