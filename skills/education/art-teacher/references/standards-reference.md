## 7. Standards & Reference

### 7.1 Drawing Progression Framework

| Level | Focus | Skills | Recommended Practice Time |
|-------|-------|--------|---------------------------|
| **Beginner 1** | Basic shapes | Sphere, cube, cylinder; simple objects | 3-6 months daily |
| **Beginner 2** | Contour & outlines | Blind contour, continuous line; still life | 3-6 months |
| **Intermediate 1** | Value & shading | 5-value scale; simple shading; light logic | 6-12 months |
| **Intermediate 2** | Perspective | 1-point, 2-point, 3-point; basic perspective | 6-12 months |
| **Advanced** | Complex subjects | Figure, portrait; personal style development | Ongoing |

### 7.2 Color Theory Basics

| Concept | Description | Application |
|---------|-------------|-------------|
| **Primary Colors** | Red, Yellow, Blue | Cannot be mixed from other colors |
| **Secondary Colors** | Orange, Green, Violet | Mix two primaries |
| **Complementary** | Opposite on color wheel | High contrast; vibrate when adjacent |
| **Analogous** | Adjacent colors | Harmonious; low contrast |
| **Warm Colors** | Red, Orange, Yellow | Advance; energetic; active |
| **Cool Colors** | Blue, Green, Violet | Recede; calm; passive |
| **Value** | Lightness/darkness | Creates depth regardless of hue |

### 7.3 Recommended Materials by Level

| Level | Drawing | Painting |
|-------|---------|----------|
| **Beginner** | 2B, 4B pencils; sketchbook; kneaded eraser | Watercolor set (12); watercolor paper; basic brushes |
| **Intermediate** | Full pencil set (2H-6B); charcoal; Bristol board | Acrylic set (24); canvas; variety of brush sizes |
| **Advanced** | Professional pencils; ink; mixed media | Oil or advanced acrylic; professional-grade surfaces |
