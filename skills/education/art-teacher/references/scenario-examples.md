## 9. Scenario Examples

### 9.1 Drawing: From Shapes to Realistic Drawing

**User:** "我完全不会画画，想学，应该从哪里开始？"

**Art Teacher:**
> **零基础绘画入门方案**
>
> **Step 1: 理解核心真理**
> - 绘画是一项技能，不是天赋
> - 每个人都能学会绘画
> - 秘密：学会观察，不是记忆
>
> **Step 2: 从基础形状开始**
> 所有复杂的物体都可以分解为基础形状：
> ```
> 苹果 = 球体 + 凹陷
> 杯子 = 圆柱体 + 椭圆形开口
> 房子 = 立方体 + 三角形屋顶
> 人头 = 球体 + 下巴方块
> 
> 练习：
> 1. 画10个完美的球体（不同大小，不同视角）
> 2. 画10个完美的立方体
> 3. 画10个完美的圆柱体
> 4. 画10个完美的圆锥体
> 
> 这看起来很无聊，但这是基础！
> ```
>
> **Step 3: 简单静物写生**
> ```
> 第一周：画一个苹果
> 1. 先画轮廓：观察苹果的外形，不是画"苹果符号"
> 2. 找出基本的球体形状
> 3. 标记光影方向（假设光源在左上方）
> 4. 画阴影（背光面）
> 
> 工具：
> - 2B铅笔
> - 白纸
> - 可塑橡皮（轻轻提亮高光）
> ```
>
> **Step 4: 正确的握笔方式**
> ```
> ❌ 写字式握笔 → 紧张，线条不流畅
> 
> ✅ 握铅笔方式（像拿勺子）：
> - 笔放在食指和中指之间
> - 拇指轻轻扶住
> - 手腕放松，不要紧绷
> - 用手臂移动，不是手指
> ```
>
> **Step 5: 每日练习（建议）**
> ```
> 每天15-30分钟：
> 
> 练习1：基础形状（10分钟）
> - 画球体、立方体、圆柱体
> - 不同角度，不同大小
> 
> 练习2：简单物体（15分钟）
> - 画杯子、碗、苹果
> - 用基础形状构建
> 
> 练习3：临摹（5分钟）
> - 找简单插画或素描临摹
> - 学习线条和形状
> 
> 坚持3个月，你会看到明显进步！
> ```
>
> **关键心态：**
> - **画得不好是正常的**：每个伟大的艺术家都从糟糕的画开始
> - **数量比质量重要**：先画100幅，再担心质量
> - **享受过程**：绘画是旅程，不是目的地

### 9.2 Color Theory: Painting a Sunset

**User:** "想画日落，但颜色总是调不对，怎么配色好看？"

**Art Teacher:**
> **日落绘画与色彩运用方案**
>
> **Step 1: 理解日落的色彩层次**
> ```
> 从上到下的典型日落色彩：
> 
> 天空顶部：深蓝紫色 → 表示高空
> 中间：红色、橙色 → 太阳附近
> 接近地平线：黄色、金色 → 最亮
> 地平线：白色/淡黄色 → 太阳位置
> 
> 反射在水面：与天空互补色
> ```
>
> **Step 2: 调色基础**
> ```
> 太阳红：朱红 + 一点点黄色
> 橙色：朱红 + 柠檬黄
> 金黄色：柠檬黄 + 一点点朱红
> 紫色阴影：群青 + 一点点朱红（而不是黑色！）
> 
> 关键：不要用黑色调暗！
> 用互补色来调暗
> ```
>
> **Step 3: 具体画法**
>
> **材料**：
> - 水彩纸
> - 圆形刷子（中号和小号）
> - 调色盘
> - 颜料：朱红、柠檬黄、群青、白色、翠绿
>
> **步骤**：
> ```
> 1. 先刷水：整个纸面刷一遍水（湿画法）
> 
> 2. 从上往下画：
>    - 顶部：群青 + 一点点朱红 → 蓝紫色
>    - 向下过渡到红色
>    - 地平线：黄色
>    - 太阳周围：白色/淡黄色
> 
> 3. 等待干一点后：
>    - 加深紫色区域（蓝紫 + 一点点红）
>    - 加深水面阴影（紫绿色）
> 
> 4. 最后细节：
>    - 太阳倒影在水面上（用短竖线）
>    - 加深轮廓
> ```
>
> **Step 4: 色彩运用技巧**
> ```
> 对比色运用：
> - 橙色的天空 + 紫蓝色的阴影 = 对比强烈
> 
> 渐变技巧：
> - 在两种颜色之间过渡时，中间色很关键
> - 湿画法更容易做出渐变
> 
> 情绪表达：
> - 红色 + 橙色 = 温暖、活力
> - 紫蓝色 + 暗调 = 平静、神秘
> ```
>
> **练习建议**：
> - 先在小纸上试颜色
> - 记住：日落的颜色是变化的，多观察真实的日落

### 9.3 Creative Block

**User:** "画了一段时间，但最近完全没有灵感，不知道画什么，很痛苦，怎么办？"

**Art Teacher:**
> **克服创作瓶颈的方案**
>
> **Step 1: 理解瓶颈是正常的**
> ```
> 创作瓶颈是每个艺术家都会经历的
> - 不是你没有才华
> - 不是你江郎才尽
> - 这是成长的一部分
> ```
>
> **Step 2: 重新激活创意的练习**
> ```
> 练习1：限制激发创意
> - 只用3种颜色画一幅画
> - 只画几何形状
> - 10分钟速写，不许停笔
> 
> 练习2：模仿然后转化
> - 找一幅你喜欢的艺术家作品
> - 用他们的风格画你自己的主题
> - 这不是抄袭，是学习
> 
> 练习3：观察日记
> - 每天画一个小物体（钥匙、咖啡杯、窗外的树）
> - 不追求好看，只是记录
> ```
>
> **Step 3: 灵感来源**
> ```
> 1. 日常生活：画你的房间、你的一天、你的物品
> 2. 自然：画植物、云彩、动物
> 3. 照片：翻看旧照片，画其中有意思的部分
> 4. 音乐：听一首歌，画你听到的"画面"
> 5. 情绪：画"快乐"、"焦虑"、"平静"是什么颜色/形状
> 
> 不要等待灵感，开始画，灵感会来的
> ```
>
> **Step 4: 改变创作方式**
> ```
> - 换一种媒介：一直画铅笔？试试水彩
> - 换一种尺寸：一直画大画？试试明信片大小
> - 换一种地点：一直在房间画？去咖啡馆、公园
> - 换一种时间：早上画 vs. 晚上画
> ```
>
> **Step 5: 心理策略**
> ```
> ❌ "我必须画出伟大的作品"
> ✅ "我只是在练习，进步比完美重要"
> 
> ❌ "别人都画得比我好"
> ✅ "每个人有不同的旅程"
> 
> 记住：
> - 画得不好 = 获得信息 = 知道下次怎么改
> - 空白画布不是敌人，是机会
> ```
>
> **下一步行动**：
> 明天早上花10分钟，不管画什么，画就对了。不需要计划，不需要主题，只是画。
