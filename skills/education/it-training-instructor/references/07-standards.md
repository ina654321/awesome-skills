# Standards & Reference

## 7.1 Official Documentation

- [CompTIA ITF+ and A+ Certification Standards](https://www.comptia.org/certifications) - Industry-recognized IT fundamentals and support technician certifications
- [Microsoft Learn](https://learn.microsoft.com/) - Official Microsoft product and service training paths
- [Google IT Support Professional Certificate](https://www.coursera.org/google-certificates/it-support) - Coursera/Google industry-recognized pathway
- [Cybersecurity and Infrastructure Security Agency (CISA) Workforce Framework](https://www.cisa.gov/topics/training) - National cybersecurity training standards

## 7.2 IT Training Certification Pathways

| Role | Entry Certification | Intermediate | Advanced |
|------|:---:|:---:|:---:|
| Help Desk | CompTIA ITF+ | CompTIA A+ | HDI Desktop Support |
| Network Admin | CompTIA Network+ | CCNA | CCNP |
| Security | CompTIA Security+ | CySA+ | CISSP |
| Cloud | AWS Cloud Practitioner | AWS Solutions Architect | Professional-level |

## 7.3 Training Effectiveness Metrics

| Metric | Description | Benchmark |
|--------|-------------|:---------:|
| Knowledge gain | Pre/post assessment score improvement | >20% improvement |
| Skill transfer | On-the-job application within 30 days | >60% apply new skills |
| Retention | Skill still in use after 90 days | >50% retention |
| Learner satisfaction | Post-training NPS | >40 NPS |
| Time to competency | Days to independently perform skill | Varies by complexity |

## 7.4 Accessibility Standards for Training

- SCORM 2004 4th Edition compliance for LMS compatibility
- WCAG 2.1 AA for all training materials
- Mobile-responsive design for on-the-go learners
- Captions/transcripts required for all video content
- Alternative text for all images in e-learning modules
