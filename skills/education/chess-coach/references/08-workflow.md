# Standard Workflow

## 8.1 Beginner Instruction

```
Phase 1: Chess Basics
├── How pieces move
├── Special moves (castling, en passant, promotion)
├── Check and checkmate
├── Stalemate
├── Basic notation
└── Chess etiquette

Phase 2: Opening Principles
├── Center control
├── Piece development
├── King safety
├── Common opening mistakes
└── First moves repertoire

Phase 3: Tactics Introduction
├── Piece value
├── Checkmate patterns
├── Basic tactical motifs
├── Introduction to puzzles
└── Find the best move
```

## 8.2 Intermediate Training

```
Skill Development:
├── Opening repertoire building
├── Middlegame planning
├── Endgame fundamentals
├── Tactical pattern recognition
├── Positional understanding
└── Time management

Training Methods:
├── Daily puzzles
├── Game analysis
├── Opening study
├── Endgame practice
├── Tournament preparation
└── Blitzen and rapid play
```

## 8.3 Tournament Preparation

```
Pre-Tournament:
├── Tournament selection
├── Time control understanding
├── Material checklist
├── Rest and nutrition
├── Mental preparation
└── Opening preparation

During Tournament:
├── Time management
├── Game recording
├── Result reporting
├── Post-game analysis
└── Sportsmanship
```

## 8.4 Coaching Workflow

```
Individual Lesson:
├── Opening discussion (5 min)
├── Demonstration and explanation (15 min)
├── Guided practice (20 min)
├── Independent play with coaching (15 min)
└── Summary and homework (5 min)

Group Class:
├── Review previous material (10 min)
├── New concept introduction (15 min)
├── Guided practice (20 min)
├── Game play or puzzles (15 min)
└── Closure and preview (5 min)
```
