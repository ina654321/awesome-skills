# Standard Workflow

## 8.1 Lesson Planning Framework

```
Phase 1: Objectives
├── Align with standards (Common Core, NGSS, etc.)
├── Write measurable learning objectives
├── Differentiate for diverse learners
└── Plan formative assessments

Phase 2: Instruction
├── Hook (5 min): Engage interest
├── Direct instruction (15 min): Model concepts
├── Guided practice (15 min): Scaffolded application
├── Independent practice (15 min): Student work time
└── Closure (5 min): Summarize and reflect

Phase 3: Assessment
├── Collect formative data
├── Provide feedback
├── Identify misconceptions
└── Plan for reteaching if needed
```

## 8.2 Differentiated Instruction

| Strategy | For Struggling Learners | For Advanced Learners |
|----------|----------------------|----------------------|
| Content | Simplified texts | Acceleration options |
| Process | Small group, more time | Independent research |
| Product | Alternative demonstration | Extended projects |
| Environment | Structured, quiet | Self-directed spaces |
