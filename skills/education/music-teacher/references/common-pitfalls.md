## 10. Common Pitfalls & Anti-Patterns

### 🔴 High Severity

**Anti-Pattern 1: Learning Pieces, Not Skills

```markdown
❌ BAD: Student learns 20 pieces by reading but can't play a simple melody by ear.
→ Can't create music, only reproduce notation.

✅ GOOD: Balance repertoire with ear training, sight-reading, and improvisation.
A complete musician can both read and create.
```

**Anti-Pattern 2: Practice Mistakes Repeatedly

```markdown
❌ BAD: Practice the same passage the same way 50 times, making the same errors.
→ Muscle memory encodes the error.

✅ GOOD: Slow down 50%, fix the error, play correctly 5×, then gradually speed up.
Quality before quantity.
```

**Anti-Pattern 3: Skipping Scales/Technique / 跳过音阶/技术**

```markdown
❌ BAD: "I want to play songs, not scales." → Technical foundation suffers, limits growth.
Proper technique enables everything else.

✅ GOOD: 10-15 minutes of scales/technique daily. It's not exciting, but it's essential.
Make it musical: scales with dynamics, articulation, expression.
```

### 🟡 Medium Severity

**Anti-Pattern 4: Only Practicing What You Know

```markdown
❌ BAD: Avoiding difficult sections, only playing easy parts.
→ Weaknesses never improve; performance anxiety increases.

✅ GOOD: 70% time on weaknesses, 30% on strengths. Face the hard parts directly.
```

**Anti-Pattern 5: Practice Without Listening

```markdown
❌ BAD: Mechanically practicing without hearing the music, focusing only on notes.
→ Disconnect between technique and musicality.

✅ GOOD: Frequently stop and listen. Ask: "What should this sound like? Does it sound like that?"
Record yourself and compare to professional performances.
```

**Anti-Pattern 6: Comparing to Others

```markdown
❌ BAD: "My friend started at the same time and is already playing harder pieces."
→ Demotivates, creates anxiety, ignores individual learning path.

✅ GOOD: Compare yourself to yourself last month, not to others. Progress is personal.
```
