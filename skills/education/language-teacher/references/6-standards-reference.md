## § 6 · Standards & Reference

**ACTFL Proficiency Levels:**
```
Novice:       Memorized words/phrases (Low/Mid/High)
Intermediate: Create with language, handle simple situations (L/M/H)
Advanced:     Narrate and describe in all time frames (L/M/H)
Superior:     Discuss complex topics, support opinions
Distinguished: Highly articulate, nuanced expression

Each level has specific CAN-DO descriptors for each mode:
- Interpersonal (speaking/listening interaction)
- Interpretive (reading/listening comprehension)
- Presentational (speaking/writing for audience)
```

**Second Language Acquisition Principles (Krashen):**
```
Acquisition-Learning Hypothesis: Acquisition (subconscious) > Learning (conscious rules)
Input Hypothesis: Acquisition occurs with comprehensible input (i+1)
Affective Filter Hypothesis: Low anxiety, high motivation, self-confidence facilitate acquisition
Natural Order Hypothesis: Grammar acquired in predictable sequence regardless of instruction
Monitor Hypothesis: Learned knowledge used only to edit output, not create it
```

**Comprehensible Input Strategies:**
```
- Shelter vocabulary, not grammar
- Use visuals, gestures, realia
- Cognates and loan words
- Repetition and circumlocution
- Check comprehension frequently
- Adjust speech: slower rate, clearer articulation, simpler syntax
```
