## § 7 · Standard Workflow

### Phase 1: Unit Planning

| Step | Activity | Done Criteria | Fail Criteria |
|------|----------|---------------|---------------|
| 1 | Identify unit theme and essential questions | Theme engaging; questions thought-provoking | Theme boring or disconnected from learners |
| 2 | Define proficiency targets by mode | CAN-DO statements written for Novice-Mid, etc. | No clear proficiency targets |
| 3 | Select and create comprehensible input | Texts, videos, stories at appropriate level | Materials too difficult |
| 4 | Design communicative tasks | Tasks mirror real-world language use | Grammar exercises disguised as communication |
| 5 | Plan assessments (IPA format) | Interpretive, interpersonal, presentational aligned | Assessment not matching instruction |

### Phase 2: Instruction and Assessment

| Step | Activity | Done Criteria | Fail Criteria |
|------|----------|---------------|---------------|
| 1 | Establish meaning of key vocabulary | Students demonstrate comprehension | Translating without understanding |
| 2 | Provide rich comprehensible input | Students understand 90%+ of input | Speaking L2 without ensuring comprehension |
| 3 | Check comprehension constantly | Thumbs up/down, exit tickets, quick writes | Assuming comprehension without checking |
| 4 | Support output when ready | Students communicate meaning; accuracy develops | Correcting every error; forcing output |
| 5 | Assess using proficiency rubrics | Feedback focused on communication, not just accuracy | Traditional grammar tests only |
