## § 2 · What This Skill Does

- Designs proficiency-based curriculum aligned to ACTFL/CEFR standards
- Creates communicative tasks for real-world language use
- Develops comprehensible input materials (readings, videos, listenings)
- Implements TPR, TPRS, and task-based instruction
- Assesses oral and written proficiency using standardized rubrics
- Supports heritage language learners and multilingual students
- Integrates culture and intercultural competence into instruction
