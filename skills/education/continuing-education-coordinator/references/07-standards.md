# Standards & Reference

## 7.1 Continuing Education Standards

- [IACET Standards](https://www.iacet.org) - International Association for Continuing Education
- [ANSI/IACET Standards](https://www.iacet.org) - Accreditation standards
- [ SHRM Standards](https://www.shrm.org) - HR certification
- [ACPE Standards](https://www.acpe.org) - Pharmacy education
- [State CE Requirements](https://www.continuingeducation.com) - State licensing boards

## 7.2 CE Credit Types

```
Credit Categories:
├── Contact hours (1 hour = 60 minutes)
├── Continuing Education Units (CEU = 10 contact hours)
├── AMA PRA Category 1 (medical)
├── ANCC Contact Hours (nursing)
├── CPE Credits (pharmacy)
├── PDHs (engineering)
└── CLE Credits (law)

Accreditation:
├── ACCME (physicians)
├── ACPE (pharmacists)
├── ANCC (nurses)
├── APA (psychologists)
└── State licensing boards
```

## 7.3 Adult Learning Principles

```
Knowles' Andragogy:
├── Self-directed learning
├── Experience as resource
├── Readiness related to life roles
├── Problem-centered orientation
├── Internal motivation
└── Need to know

Learning Styles:
├── Visual learners
├── Auditory learners
├── Kinesthetic learners
└── Reading/writing preference
```

## 7.4 Program Design Standards

```
ADDIE Model:
├── Analysis: Needs assessment
├── Design: Learning objectives
├── Development: Materials creation
├── Implementation: Delivery
└── Evaluation: Outcome measurement

Kirkpatrick's Four Levels:
├── Level 1: Reaction
├── Level 2: Learning
├── Level 3: Behavior
└── Level 4: Results
```
