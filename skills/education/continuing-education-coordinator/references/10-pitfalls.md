# Common Pitfalls & Anti-Patterns

## 10.1 Program Design Errors

| Pitfall | Severity | Prevention |
|---------|----------|------------|
| No needs assessment | 🔴 High | Begin with analysis |
| Learning objectives unclear | 🟡 Medium | Use Bloom's verbs |
| Content overload | 🟡 Medium | Chunking, microlearning |
| No practice activities | 🟡 Medium | Active learning |
| Poor accessibility | 🟡 Medium | ADA compliance |

## 10.2 Accreditation Errors

| Error | Impact | Prevention |
|-------|--------|------------|
| Missing elements | Credit denied | Checklist |
| Incomplete documentation | Audit failure | Accreditation guide |
| Wrong credit type | Learner issues | Verify requirements |
| Expired accreditation | Non-compliance | Track renewal |

## 10.3 Delivery Issues

```
Common Problems:
├── Technical difficulties
├── Poor facilitation
├── Low engagement
├── Scheduling conflicts
├── Lack of interaction
└── No real-world application

Solutions:
├── Test all technology
├── Facilitator training
├── Engagement strategies
├── Flexible options
├── Active learning
```

## 10.4 Evaluation Failures

| Failure | Impact | Prevention |
|---------|--------|------------|
| No post-assessment | No proof of learning | Include tests |
| Kirkpatrick level 1 only | No impact data | Multi-level |
| No follow-up | Forgetting | Spaced practice |
| Incomplete data | Poor decisions | Track everything |

## 10.5 Compliance Issues

| Issue | Risk | Mitigation |
|-------|------|------------|
| Missing attendance | Credit denied | Track participation |
| Failed post-test | Non-compliance | Remediation |
| Expired requirements | License issues | Reminder system |
| Fraudulent completions | Legal risk | Verification |

## 10.6 Quality Checklist

- [ ] Needs assessment completed
- [ ] Learning objectives measurable
- [ ] Content evidence-based
- [ ] Accreditation obtained
- [ ] Technology tested
- [ ] Facilitator trained
- [ ] Engagement strategies planned
- [ ] Assessment included
- [ ] Evaluation collected
- [ ] Credits reported
