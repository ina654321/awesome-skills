## § 6 · Standards & Reference

**SAT Structure (Digital as of 2023):**
```
Reading & Writing (64 min):
- 2 modules, 27 questions each
- Topics: Information & Ideas, Craft & Structure, Expression of Ideas, Standard English Conventions

Math (70 min):
- 2 modules, 22 questions each
- Topics: Algebra, Advanced Math, Problem Solving & Data Analysis, Geometry
- Calculator allowed throughout

Score: 400-1600 (200-800 per section)
```

**GRE Structure:**
```
Verbal Reasoning (2 sections, 30 min each):
- Reading Comprehension, Text Completion, Sentence Equivalence

Quantitative Reasoning (2 sections, 35 min each):
- Arithmetic, Algebra, Geometry, Data Analysis

Analytical Writing (1 hour):
- Analyze an Issue, Analyze an Argument

Score: Verbal 130-170, Quant 130-170, Writing 0-6
```

**Test-Taking Strategies:**
```
Time Management:
- Skip difficult questions; return if time permits
- Don't rush easy questions to save time for hard ones
- Practice with strict timing to build pacing

Process of Elimination:
- Eliminate obviously wrong answers first
- Compare remaining choices carefully
- Guess strategically when unsure (no penalty on most tests)

Active Reading:
- Preview questions before reading passage
- Annotate main idea and structure
- Refer back to text for evidence
```
