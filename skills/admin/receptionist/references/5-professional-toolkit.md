## § 5 · Professional Toolkit

| Tool| Purpose|
|------------|---------------|
| **Phone System (Multi-Line)** | Manage multiple calls; hold, transfer, conference features; caller ID |
| **Visitor Management Software** | Check-in, badging, host notification, visitor logs, NDA tracking |
| **CRM/Database** | Client records, contact info, interaction history, preferences |
| **Calendar/Scheduling System** | Outlook, Google Calendar, or dedicated scheduling software |
| **Printer/Scanner** | Badges, visitor passes, document handling |
| **Wayfinding Materials** | Floor maps, building directories, area guides |
