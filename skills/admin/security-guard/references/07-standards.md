# Standards & Reference

## 7.1 Regulatory Requirements

### State Licensing
| Requirement | Description |
|-------------|-------------|
| Guard Card | State-issued security officer registration |
| Background Check | FBI criminal history (no disqualifying offenses) |
| Training Hours | 8-40 hours (varies by state) |
| Continuing Education | Annual refresher required |

### OSHA Safety Standards
- 29 CFR 1910.132: Personal protective equipment
- 29 CFR 1910.134: Respiratory protection (if applicable)
- 29 CFR 1910.120: Hazardous materials awareness

## 7.2 Industry Standards

### ASIS International Guidelines
| Standard | Description |
|----------|-------------|
| ASIS SPA.1 | Security management systems |
| ASIS GOV.1 | Government security officer guidelines |
| ASIS IPC.1 | Intelligence in corporate security |

### Post Orders Components
- Site-specific procedures
- Shift schedules and coverage
- Access control points
- Patrol routes and schedules
- Emergency response protocols
- Communication procedures

## 7.3 Incident Classification

| Level | Type | Response |
|-------|------|----------|
| Level 1 | Minor (noise, trespass) | Monitor, document, report |
| Level 2 | Moderate (theft, vandalism) | Secure scene, notify supervisor, police if needed |
| Level 3 | Serious (assault, weapons) | Immediate police, protect evidence, evacuate if needed |
| Level 4 | Life Safety | Full emergency response, 911, evacuation |

## 7.4 Documentation Standards

### Daily Activity Report (DAR)
- Time and location of all activities
- Incidents observed and actions taken
- Visitors and vehicles logged
- Equipment issues
- Unusual observations

### Incident Report Elements
- Who, what, when, where, how
- Witness statements
- Evidence collected
- Actions taken
- Follow-up recommendations
