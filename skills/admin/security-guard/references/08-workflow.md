# Workflow Reference

## 8.1 Access Control Workflow

### Visitor Processing
1. Verify visitor identity against approved list
2. Issue visitor badge (photo if required)
3. Escort to destination or provide directions
4. Log entry time and destination
5. Retrieve badge upon departure
6. Log exit time

### Vehicle Access
1. Verify vehicle/occupants against approved list
2. Check for proper credentials or passes
3. Log vehicle license plate and time
4. Direct to parking or loading area
5. Inspect vehicle upon exit if required

## 8.2 Patrol Workflow

### Foot Patrol
- Follow designated route
- Check all doors and windows
- Note any unusual conditions
- Test panic alarms if applicable
- Complete full circuit in specified time

### Vehicle Patrol
- Maintain speed limit on property
- Check perimeter fences
- Inspect parking lots
- Check exterior doors
- Report any suspicious vehicles

### Documentation
- Log start and end times
- Note patrol path and checkpoints
- Record any findings
- Report equipment issues

## 8.3 Emergency Response Workflow

### Fire Response
1. Activate fire alarm
2. Notify fire department (911)
3. Evacuate occupants from immediate area
4. Close doors to contain fire
5. Meet fire department at entry
6. Document all actions

### Medical Emergency
1. Call 911 immediately
2. Do not move injured person unless danger
3. Obtain first aid if trained
4. Clear path for emergency responders
5. Gather information for EMT
6. Document incident

### Active Threat
1. Run: Evacuate if safe path exists
2. Hide: Secure in locked room, silence phone
3. Fight: As last resort, defend yourself
4. Call 911 when safe
5. Follow law enforcement instructions
