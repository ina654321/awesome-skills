# Standards & Reference

## 7.1 Safety Standards

### OSHA Requirements
| Requirement | Description |
|-------------|-------------|
| Hazard Communication | SDS for all chemicals |
| Personal Protective Equipment | Gloves, goggles as needed |
| Ergonomics | Proper lifting technique |
| Bloodborne Pathogens | Training for bodily fluids |

### Chemical Handling
| Category | Handling |
|----------|----------|
| Sanitizers | Dilute per label, contact time |
| Detergents | Follow dilution ratios |
| Solvents | Ventilation, no mixing |
| Acids | Separate storage, PPE |

## 7.2 Cleaning Standards

### Cleaning Methods
| Method | Use | Example |
|--------|-----|---------|
| Sweeping | Dry debris removal | Floors, sidewalks |
| Mopping | Wet cleaning | Hard floors |
| Vacuuming | Carpeted surfaces | Offices, hallways |
| Dust mopping | Pre-clean | Smooth floors |
| Scrubbing | Deep cleaning | Restrooms, kitchens |

### Dwell Times
| Chemical | Minimum Time |
|----------|--------------|
| Sanitizer (quaternary) | 10 minutes |
| Sanitizer (bleach) | 5 minutes |
| Disinfectant | 10 minutes or per label |
| Heavy-duty cleaner | 15-20 minutes |

## 7.3 Area Classifications

### Cleanliness Levels
| Level | Description | Frequency |
|-------|-------------|-----------|
| High | Executive areas, lobbies | Daily + as needed |
| Medium | Offices, conference rooms | Daily |
| Low | Storage, utility | Weekly |
| Restrooms | High priority | Daily + periodic deep clean |
| Food Areas | Critical | Daily + sanitation checks |

### Quality Inspection Points
- No visible debris or stains
- No residue on surfaces
- Proper smell (neutral)
- Proper color/cloth cleanliness
