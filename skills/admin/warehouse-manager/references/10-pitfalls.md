# Common Pitfalls

## 10.1 Receiving Pitfalls

### Not Verifying Counts Before Signing
**Risk**: Accepting short/damaged shipments without documentation
**Solution**: Count and inspect every shipment before signing BOL

### SkippingASN Review
**Risk**: Unprepared for shipment volume, dock congestion
**Solution**: Review ASN at least 24 hours before arrival

### Inadequate Damage Documentation
**Risk**: Denied carrier claims, lost reimbursement
**Solution**: Photograph all damage, note specifically on BOL

## 10.2 Inventory Management Pitfalls

### Using Min/Max Without Demand Analysis
**Risk**: Stockouts or overstocking
**Solution**: Review and adjust parameters quarterly using historical data

### Not Conducting Regular Cycle Counts
**Risk**: Inventory inaccuracies, financial statement errors
**Solution**: ABC cycle count schedule (A-items monthly, B-quarterly, C-annually)

### Ignoring Dead Stock
**Risk**: Waste,占用 storage space, obsolete inventory
**Solution**: Quarterly review, liquidation or disposal process

## 10.3 Order Fulfillment Pitfalls

### Wrong Item Selection
**Risk**: Customer complaints, returns, chargebacks
**Solution**: Scanner verification, quality check before packing

### Incorrect Shipping Label
**Risk**: Lost packages, delivery delays
**Solution**: Label verification scan before staging

### Over-packing
**Risk**: Increased material costs, shipping surcharges
**Solution**: Right-size packaging, train packers on carton selection

## 10.4 Safety Pitfalls

### Improper Forklift Operation
**Risk**: Personnel injuries, property damage, OSHA violations
**Solution**: Certified operators only, daily pre-operation checklist

### Blocked Emergency Exits
**Risk**: Life safety violation, OSHA citations
**Solution**: Clear paths, no temporary staging in exit routes

### Inadequate Training
**Risk**: Accidents, non-compliance, liability
**Solution**: Document all training, refresher courses annually
