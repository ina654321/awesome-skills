# Standards & Reference

## 7.1 Regulatory Requirements

### License Requirements
| Type | Requirements |
|------|--------------|
| Standard (Class C) | Age 18+, pass test |
| Commercial (Class A/B) | Age 21+, CDL, medical cert |
| HAZMAT | Background check, TSA approval |
| Passenger (P) | CDL with P endorsement |

### Hours of Service (Commercial)
| Property | Limit |
|----------|-------|
| 11-hour driving | After 8 consecutive hours off |
| 14-hour duty | Cannot extend |
| 30-minute break | After 8 cumulative hours |
| 60/70-hour limit | Based on work week |

## 7.2 Vehicle Standards

### Pre-Trip Inspection (CDL)
| Item | Check |
|------|-------|
| Engine | Oil, coolant, belts |
| Brakes | Service brake, parking brake |
| Lights | Head, tail, brake, turn |
| Tires | Pressure, tread, damage |
| Mirrors | Adjustment, cleanliness |
| Emergency Equipment | Kit, triangles, fire extinguisher |

### DOT Inspection Items
- Brakes
- Lights and signaling
- Tires
- Steering mechanism
- Suspension
- Frame and cargo securement
- Documentation

## 7.3 Safety Standards

### Defensive Driving
| Principle | Action |
|-----------|--------|
| Space cushion | Maintain 4+ second gap |
| Vision | Scan ahead, check mirrors |
| Focus | Avoid distractions |
| Speed | Match conditions |
| Positioning | Avoid blind spots |

### Accident Response
1. Stop immediately
2. Call 911 if injuries
3. Document scene (photos)
4. Exchange info (not admit fault)
5. Report to employer immediately
