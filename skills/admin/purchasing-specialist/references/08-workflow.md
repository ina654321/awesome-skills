# Workflow Reference

## 8.1 Requisition to Pay Process

### Step-by-Step Workflow
1. **Requisition Received**: Review request for completeness
2. **Vendor Selection**: Verify approved vendor list, source quotes
3. **Purchase Order Creation**: Complete PO in ERP system
4. **Approval Routing**: Submit per delegation of authority
5. **PO Issuance**: Send to vendor with terms
6. **Goods Receipt**: Verify delivery against PO
7. **Invoice Processing**: Match PO, receipt, invoice (3-way match)
8. **Payment**: Process per payment terms

### Approval Thresholds
| Amount | Approver |
|--------|-----------|
| < $1,000 | Department Manager |
| $1,000 - $10,000 | Director |
| $10,000 - $50,000 | VP |
| > $50,000 | CFO |

## 8.2 Sourcing Workflow

### Competitive Bidding Process
1. Define specifications and requirements
2. Identify qualified suppliers (minimum 3 for >$10K)
3. Issue RFQ/RFP with clear terms
4. Receive and evaluate bids (score per criteria)
5. Negotiate best terms if needed
6. Award to lowest responsive, responsible bidder
7. Document rationale for selection

### Single Source Justification
Required when only one supplier can meet requirements:
- Unique capability or proprietary item
- Compatibility with existing equipment
- Time constraints (documented emergency)
- OEM warranty requirements

## 8.3 Vendor Management Workflow

### Onboarding Process
1. Collect vendor application and documents
2. Verify business license and insurance
3. Run credit check (for new vendors)
4. Complete vendor setup in ERP
5. Add to approved vendor list
6. Communicate payment terms and expectations

### Performance Review
Quarterly evaluation of key suppliers:
- On-time delivery % (target >95%)
- Quality reject rate (target <2%)
- Invoice accuracy (target >98%)
- Response time to issues
