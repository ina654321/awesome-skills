# Standard Workflow

## 8.1 Skin Care Consultation

```
Phase 1: Client Intake
├── Consultation form (health history, allergies, medications)
├── Skin analysis (type, conditions, concerns)
├── Product review (what are they currently using?)
├── Lifestyle factors (sun exposure, diet, stress, hormones)
├── Contraindications check
└── Document preferences and sensitivities

Phase 2: Service Planning
├── Determine appropriate treatment
├── Select compatible products
├── Patch test if new products
├── Explain procedure and expected results
├── Get informed consent
└── Set realistic expectations

Phase 3: Treatment Execution
├── Double cleanse skin
├── Perform extractions if appropriate
├── Apply treatment products
├── Use appropriate techniques
├── Finish with moisturizer/SPF
└── Recommend home care
```

## 8.2 Facial Treatment Workflow

1. **Sanitize hands** — wash with soap, use sanitizer
2. **Drape client** — protect clothing
3. **Cleansing** — double cleanse (oil-based then water-based)
4. **Skin analysis** — use magnifying lamp
5. **Steam** — 5-10 minutes to soften pores
6. **Extractions** — only if trained, use proper technique
7. **Exfoliation** — chemical or enzyme (not physical if extractions done)
8. **Mask** — appropriate for skin type
9. **Serum/treatment** — active ingredients
10. **Moisturizer + SPF** — protect the skin

## 8.3 Waxing Procedure

```
Safety Protocol:
├── New spatula each pass (no double-dipping)
├── Test wax temperature on inner wrist
├── Pull in direction of hair growth
├── Apply pressure immediately after pull
├── Soothing product after waxing
└── Never wax over broken skin, moles, or irritation
```
