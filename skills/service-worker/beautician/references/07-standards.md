# Standards & Reference

## 7.1 Cosmetology Regulations

### Federal/State Framework
- [21 CFR Part 1100](https://www.accessdata.fda.gov/scripts/cdrh/cfdocs/cfcfr/CFRSearch.cfm?CFRPart=1100): FDA color additive regulations
- [21 CFR Part 740](https://www.accessdata.fda.gov/scripts/cdrh/cfdocs/cfcfr/CFRSearch.cfm?CFRPart=740): Cosmetic labeling requirements
- [State Cosmetology Boards](https://www.nictesting.org/): Each state has licensing requirements
- [Barbering/Cosmetology Acts](https://www.beautylaunchpad.com/): Scope of practice per state

### Licensing Requirements
| Requirement | Typical Standard |
|-------------|-----------------|
| **Training hours** | 1,000-1,600 hours (state varies) |
| **Exam** | State board exam (theory + practical) |
| **Renewal** | 1-2 years with CE |
| **Sanitation** | State board inspection standards |

## 7.2 Infection Control Standards

- [CDC Disinfection Guidelines](https://www.cdc.gov/infection-control/): Salon infection control
- [OSHA Bloodborne Pathogens](https://www.osha.gov/bloodborne-pathogens): Standard for exposure
- [State Sanitation Rules](https://www.beautylaunchpad.com/): Equipment sterilization
- [EPA-Registered Disinfectants](https://www.epa.gov/pesticides): Salon-safe products
