# Standard Workflow

## 8.1 Pre-Move Preparation

```
Phase 1: Walk-Through
├── Walk through origin home with estimator or customer
├── Identify all items to be moved
├── Note heavy items, stairs, tight corners
├── Check for disassembly needs (beds, tables)
├── Identify items requiring special handling
├── Mark or inventory items
└── Confirm delivery address and access

Phase 2: Truck Loading
├── Stage moving pads and equipment
├── Load heaviest items first (appliances, furniture)
├── Load items toward front of truck
├── Fill gaps to prevent shifting
├── Wrap and pad all furniture
├── Stack boxes with heavy on bottom
└── Secure load with straps
```

## 8.2 Loading Sequence

```
Standard Load Order:
1. Appliances (refrigerator, washer/dryer)
2. Large furniture (bed frames, dressers, couches)
3. Medium furniture (chairs, small tables)
4. Mattresses (wrapped in mattress bags)
5. Boxes (heavy on bottom, light on top)
6. Fragile items (last on, first off)
7. Clothing on hangers (in wardrobe boxes)
8. Last items: last-minute picks from closets
```

## 8.3 Heavy Item Handling

```
Weight Limits (NIOSH):
├── Maximum weight for one person: 50 lbs
├── Two-person lift: up to 75 lbs
├── Team lift: 100+ lbs
└── Mechanical aid: required above limits

Proper Lifting Technique:
├── Bend at knees, not waist
├── Keep load close to body
├── Don't twist while lifting
├── Get help for heavy items
├── Use dolly for appliances
├── Clear path before moving
└── Use moving straps for awkward loads
```
