# Standard Workflow

## 8.1 Manicure Service Protocol

```
Phase 1: Client Intake
├── Client consultation (health history)
├── Check for contraindications
├── Review nail condition
├── Discuss desired service
└── Select appropriate products

Phase 2: Preparation
├── Wash hands and sanitize
├── Remove old polish (if applicable)
├── Shape natural nails (file, don't saw)
├── Soften cuticles with cuticle remover
├── Push cuticles gently (don't cut live tissue)
└── Clean under nails

Phase 3: Service
├── Exfoliate hands
├── Hand massage
├── Apply base coat
├── Apply polish (2 thin coats)
├── Apply top coat
└── Air dry

Phase 4: Finish
├── Apply cuticle oil
├── Clean up around nails
├── Final inspection
└── Discuss home care
```

## 8.2 Pedicure Service

```
Standard Pedicure:
1. Sanitize feet, inspect for issues
2. Soak in warm footbath (liner per client)
3. Remove old polish
4. File and shape nails
5. Cuticle treatment (no cutting live skin)
6. Callus removal (pumice stone, not razor)
7. Exfoliate feet and legs
8. Foot/leg massage
9. Apply base coat
10. Polish (2 coats)
11. Top coat
12. Dry thoroughly
```

## 8.3 Artificial Nails

```
Acrylic Application:
1. Sanitize natural nails
2. Apply dehydrator and primer
3. Sculpt nail form
4. Apply acrylic mixture (liquid + powder)
5. Shape and file
6. Buff to desired finish
7. Apply color or design

Gel Application:
1. Prep natural nail
2. Apply base coat (gel)
3. Cure under UV/LED lamp
4. Apply gel color layers
5. Cure between each layer
6. Apply top coat
7. Cure final layer
8. Wipe with cleanser
```
