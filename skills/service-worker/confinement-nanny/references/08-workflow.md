# Standard Workflow

## 8.1 New Mother Care Workflow

```
Phase 1: Physical Recovery Support
├── Monitor vital signs if instructed
├── Assist with personal hygiene
├── Prepare nutritious meals and snacks
├── Manage household tasks (light cleaning, laundry)
├── Prepare ice packs, pads, comfortable clothing
├── Encourage rest — protect postpartum sleep
└── Track medications, appointments

Phase 2: Emotional Support
├── Active listening without judgment
├── Watch for signs of postpartum depression/anxiety
├── Encourage self-care
├── Facilitate family bonding
├── Respect boundaries and privacy
└── Connect to professional support if needed

Phase 3: Newborn Care
├── Feedings (breastfeeding support or bottle prep)
├── Diaper changes
├── Swaddling and soothing
├── Bathing
├── Sleep routines
├── Monitoring for jaundice, fever
└── Document feeding/wet diaper log
```

## 8.2 Breastfeeding Support

```
Latching Checklist:
├── Mouth wide open
├── Lips flanged outward
├── Good portion of areola in mouth
├── Chin touching breast
├── Audible swallowing
└── No pain (pinching sensation = wrong latch)

Pumping Protocol:
1. Wash hands
2. Inspect pump equipment
3. Assemble flanges correctly
4. Start on lower suction, increase gradually
5. 15-20 minutes per session
6. Store milk properly (refrigerate within 1 hour)
7. Clean pump parts after each use
```

## 8.3 Emergency Response

```
Baby Not Breathing / Choking:
├── If choking: 5 back blows, 5 chest thrusts (under 1 yr)
├── If unresponsive: Call 911, start infant CPR
├── AED if available — pediatric pads if under 8

Poisoning:
├── Call Poison Control: 1-800-222-1222
├── Don't induce vomiting unless instructed
├── Save the container/substance for reference

Fever in Newborn:
├── Call pediatrician immediately for temp > 100.4°F under 3 months
├── Don't give medication without doctor approval
```
