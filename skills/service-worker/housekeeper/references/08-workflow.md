# Standard Workflow

## 8.1 Room Turnover Protocol

```
Phase 1: Entry & Assessment
├── Knock twice, announce "housekeeping"
├── Enter with master key
├── Check for guests in room
├── Assess room condition
├── Turn on lights, open curtains
└── Note any damages or special items

Phase 2: Remove Trash & Linens
├── Empty all trash receptacles
├── Remove used linens (do not shake)
├── Remove used towels
├── Clear mini-bar items if applicable
└── Check bathroom trash

Phase 3: Clean & Sanitize
├── Bathroom protocol (see below)
├── Dust all surfaces top to bottom
├── Wipe down furniture
├── Clean mirrors and windows
├── Make bed with fresh linens
├── Vacuum carpet (under bed, furniture)
└── Mop hard floors

Phase 4: Final Inspection
├── Replace amenities (soap, shampoo, etc.)
├── Restock guest supplies
├── Check lighting and electronics
├── Arrange room amenities
├── Remove all cleaning supplies
└── Final walk-through
```

## 8.2 Bathroom Cleaning Protocol

```
Standard Bathroom:
1. Spray toilet with cleaner — let sit
2. Spray shower/tub — let sit
3. Wipe vanity and fixtures
4. Clean toilet (inside, outside, base)
5. Clean shower/tub — rinse well
6. Polish mirrors
7. Mop floor
8. Restock towels and amenities
```

## 8.3 Special Situations

```
Deep Cleaning (Periodic):
├── Behind and under all furniture
├── High dusting (vents, light fixtures)
├── Baseboards
├── Inside closets
├── Window sills
└── Under beds

Checkout vs. Stayover:
├── Checkout: Full turnover, all linens changed
└── Stayover: Replace only requested items, tidy up
```
