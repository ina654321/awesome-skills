# Common Pitfalls & Anti-Patterns

## 10.1 Chemical Safety Errors

| # | Pitfall | Severity | Prevention |
|---|---------|----------|------------|
| 1 | **Skipping patch test** | 🔴 High | Always test 24-48 hours before |
| 2 | **Not mixing per instructions** | 🔴 High | Follow manufacturer ratios exactly |
| 3 | **Overlapping chemicals** | 🔴 High | Never overlap color or bleach |
| 4 | **Improper developer strength** | 🔴 High | Match developer to service |
| 5 | **Leaving chemicals on too long** | 🔴 High | Monitor processing, set timer |

## 10.2 Infection Control Failures

⚠️ **Sanitation Issues:**
- Not disinfecting scissors/combs between clients
- Using dirty tools
- Double-dipping color brushes
- Not washing hands between clients
- Using expired products
- Unclean cape/clothing protection

## 10.3 Consultation Errors

⚠️ **Client Expectation Failures:**
- Not discussing price before starting
- Assuming client's preference without asking
- Not explaining the process
- Hiding potential damage or limitations
- Not doing strand test for major changes

## 10.4 Technical Errors

⚠️ **Common Mistakes:**
- Uneven layers
- Over-processing (chemical burns, breakage)
- Wrong color undertone
- Harsh lines from clippers
- Not checking both sides
- Rushing the finish

## 10.5 Quality Checklist

- [ ] Consultation completed and documented
- [ ] Patch test done for new products
- [ ] Products mixed per instructions
- [ ] Processing time monitored
- [ ] Tools disinfected between clients
- [ ] Hair condition assessed before proceeding
- [ ] Price discussed before starting
- [ ] Aftercare instructions provided
