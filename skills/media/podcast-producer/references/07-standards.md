## 7.2 Audio Quality Standards

| Specification | Minimum | Professional | Broadcast |
|--------------|---------|-------------|-----------|
| Sample rate | 44.1 kHz | 48 kHz | 48 kHz |
| Bit depth | 16-bit | 24-bit | 24-bit |
| Bitrate (MP3) | 96 kbps | 128-192 kbps | 256+ kbps |
| Peak level | -3 dBFS | -3 dBFS | -3 dBFS |
| Noise floor | -50 dBFS | -60 dBFS | -70 dBFS |
| Stereo/Mono | Mono acceptable | Stereo preferred | Stereo |

## 7.4 Episode Structure Standards

| Element | Standard | Duration |
|---------|----------|----------|
| Intro music | Consistent branding | 5-15 seconds |
| Host greeting | Warm, inviting | 15-30 seconds |
| Episode title/topic | Clear, searchable | — |
| Sponsor message | Top (pre-roll) | 60-90 seconds |
| Main content | Core value delivery | 20-50 minutes |
| Mid-roll sponsor | Natural break | 60-90 seconds |
| Outro | CTA, social, subscribe | 30-60 seconds |
| Total length | Audience-driven | 25-55 minutes |

## 7.5 RSS Feed Requirements

| Element | Standard | Notes |
|---------|----------|-------|
| Title | Consistent naming | Include podcast name |
| Description | < 4,000 characters | SEO-optimized |
| Artwork | 3000x3000 px, RGB | Required for all directories |
| Language | RFC 4646 | en-US standard |
| Explicit flag | Accurate | iTunes reviews carefully |
| Category | Primary + secondary | Affects discoverability |
| Author/owner | Contact information | Legal requirement |

## 7.6 Monetization Standards

| Method | Industry Standard Rate | Notes |
|--------|----------------------|-------|
| CPM ads | $15-25 per 1,000 downloads | Dynamic insertion preferred |
| Host-read ads | $20-40 CPM | Premium for host-read |
| Affiliate | 10-30% commission | Disclosure required |
| Premium episodes | Patreon/tiered | 10-20% take |
| Live reads | $25-50 CPM | Highest rates |
| Network deals | Negotiated | 50/50 to 70/30 splits |
