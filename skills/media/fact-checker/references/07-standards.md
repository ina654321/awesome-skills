## 7.2 Claim Classification Standards

| Rating | IFCN Standard | Example |
|--------|---------------|---------|
| True | Completely accurate | "The statement is accurate..." |
| Mostly True | Accurate with minor omissions | "The statement is accurate but..." |
| Half True | Partially accurate, misleading | "The statement is partially accurate..." |
| Mostly False | Contains some truth but misleading | "The statement contains elements..." |
| False | Not accurate | "The statement is not accurate..." |
| Pants on Fire | False and ridiculous | "The statement is not accurate and..." |

## 7.3 Source Hierarchy

| Level | Source Type | Reliability | Verification Needed |
|-------|------------|-------------|-------------------|
| 1 | Peer-reviewed research | Highest | Journal name, author, DOI |
| 2 | Official government data | High | Agency website, direct contact |
| 3 | Established news organizations | High | Cross-reference multiple |
| 4 | Expert testimony | Medium-High | Credentials, potential bias |
| 5 | Press releases/statements | Medium | Independent verification |
| 6 | Social media posts | Low-Medium | Account verification + evidence |
| 7 | Anonymous sources | Requires corroboration | Multiple independent sources |

## 7.5 Verification Checklist

| Item | Required For | Method |
|------|-------------|--------|
| Identity | Named individuals | Government ID, professional records |
| Credentials | Experts | Direct verification, official databases |
| Statistics | Data claims | Primary source, methodology review |
| Quotes | Direct quotes | Audio/video, transcript |
| Images | Photos/videos | Reverse image search, metadata, source verification |
| Dates | Event claims | Calendar, news coverage |
| Locations | Event claims | News, official records |
| Social posts | Viral claims | Account verification, archive |
