## 7.2 Editorial Standards

| Standard | Requirement | Source |
|---------|-------------|--------|
| Attribution | Every claim must be sourced | SPJ |
| Balance | Multiple perspectives for controversial topics | Reuters |
| Correction | Published within 24 hours of error | AP |
| Independence | No conflicts of interest | SPJ |
| Privacy | Minimize harm to subjects | SPJ |

## 7.3 Verification Standards

| Checkpoint | Required For | Standard |
|-----------|-------------|----------|
| Identity | All named individuals | Government ID or corroborating sources |
| Credentials | Experts and officials | Direct confirmation or official records |
| Documents | Statistics and quotes | Primary source required |
| Images | All photographs | Metadata check + reverse image search |
| Social posts | Used as sources | Account verification + archive |

## 7.4 Source Classification

| Category | Use | Requirements |
|----------|-----|-------------|
| On-the-record | Direct publication | Name, title verified |
| On-background | Attribution to type/role | "A senior official..." |
| On-deep-background | No attribution | Information use only |
| On-off-record | Not for publication | Agreement must be explicit |
| Anonymous | Only when necessary | Editor approval, compelling public interest |

## 7.5 Publication Rights

| Right Type | Industry Standard | Notes |
|-----------|------------------|-------|
| First publication | 24-48 hours exclusivity | Negotiated per story |
| Archive rights | Perpetual | Often included in buyout |
| Electronic rights | Usually separate | Increasingly important |
| Syndication | May require separate deal | Creator retains often |
| Translation | Usually separate | Growing value |
| Audio/video adaptation | Emerging | Increasingly negotiated separately |
