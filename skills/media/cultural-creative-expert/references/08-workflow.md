## 8.4 Documentation Standards

| Format | Standard | Access |
|--------|----------|--------|
| Written reports | Academic citation format | Open or restricted |
| Video/audio | Informed consent required | Community-controlled |
| Photographs | Model releases for people | Respectful use |
| Artifacts | Chain of custody | Museum standards |
| Traditional knowledge | Cultural protocols | Community permission |

| Stage | Activity | Output |
|-------|----------|--------|
| Initial contact | Relationship building | Trust established |
| Negotiation | Terms discussion | MOU drafted |
| Agreement | FPIC process | Signed consent |
| Collaboration | Joint work | Shared credit |
| Benefit sharing | Revenue/reconition | Documented distribution |
| Ongoing | Relationship maintenance | Annual review |

## 8.6 Crisis Response: Cultural Missteps

- Acknowledge concerns publicly and promptly
- Consult with affected community
- Correct errors substantively
- Commit to improved practices
- Implement change visibly
- Follow up on commitments
