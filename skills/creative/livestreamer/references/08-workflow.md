# Standard Workflow

## 8.1 Pre-Stream Preparation

```
Preparation Checklist (2-3 hours before)
├── 1. Content Planning
│   ├── Define stream theme and goals
│   ├── Prepare talking points and segments
│   ├── Plan interactive elements (polls, games, giveaways)
│   └── Schedule segments with approximate timing
│
├── 2. Technical Setup
│   ├── Launch OBS/Streamlabs
│   ├── Test camera position and lighting
│   ├── Check microphone levels (peak at -12 dB)
│   ├── Set up alerts and overlays
│   ├── Verify stream key and destination platform
│   └── Test all software (games, browser sources, etc.)
│
├── 3. Community Pre-Warm
│   ├── Post teaser on social media
│   ├── Post in Discord/other communities
│   ├── Pin stream announcement
│   └── Set channel status
│
└── 4. Final Check (15 minutes before)
    ├── Restart computer to clear memory
    ├── Close unnecessary applications
    ├── Fill water bottle, snacks nearby
    ├── Bathroom break
    ├── Go live 5 minutes early
    └── Welcome early viewers personally
```

## 8.2 Stream Structure

```
Standard Stream Format (3-4 hours)
├── FIRST 10 MINUTES: Warm Welcome
│   ├── Greet early viewers by name
│   ├── Preview tonight's content
│   ├── Thank lurkers for lurking
│   └── Community check-in
│
├── MIDDLE: Main Content Loop
│   ├── Play/content block (1-2 hours)
│   ├── 15-min break every 2 hours
│   ├── Mid-stream check-in (how are we feeling?)
│   └── Interactive elements every 30-45 min
│
├── END: Wind Down
│   ├── Thank all viewers, especially regulars
│   ├── Announce next stream and schedule
│   ├── Follow on social media calls-to-action
│   └── Proper goodbye (no abrupt ending)
│
└── POST-STREAM
    ├── VOD edit and upload (if applicable)
    ├── Respond to comments
    └── Review analytics
```

## 8.3 Viewer Engagement Techniques

| Technique | When to Use | Implementation |
|-----------|-------------|----------------|
| **Call and response** | Start of stream | "What's up, [stream name]?!" |
| **Viewer queue** | Gaming, IRL | First-come, first-served order |
| **Polls** | Decision moments | Let chat vote on gameplay choices |
| **Raids** | End of stream | Raid smaller streamers |
| **Sub goals** | At milestones | Celebration and thank you |
| **Giveaways** | Mid-stream boost | Entry requirements (follow, sub, share) |

## 8.4 Emergency Protocols

```
Technical Emergency Response
├── Stream crashes mid-broadcast
│   ├── Pause and attempt restart
│   ├── Alert on social media
│   ├── Return within 5 minutes or end stream
│   └── Post VOD when resolved
│
├── Copyright claim/dmca
│   ├── Stop music immediately
│   ├── Switch to DMCA-safe playlist
│   ├── Do not argue the claim on stream
│   └── Appeal through proper channels after
│
└── Harassment/trolls
    ├── Enable follower-only chat
    ├── Timeout/ban immediately
    ├── Don't engage publicly
    └── Automod keyword filters
```

## 8.5 Post-Stream Checklist

- [ ] Thank and acknowledge all contributors
- [ ] Upload VOD (YouTube) or save locally
- [ ] Edit highlights/clips
- [ ] Respond to comments and DMs
- [ ] Review analytics (watch time, peak viewers, new followers)
- [ ] Schedule next stream
- [ ] Update social media
- [ ] Back up VODs and important files
- [ ] Track revenue (subs, donations, bits)
- [ ] Set aside taxes (25-30%)
