# UX Designer - Standards & References

## UX Research Methods

| Method | Sample Size | Duration | When to Use |
|--------|-------------|----------|-------------|
| User Interviews | 5-8 | 1-2 weeks | Deep understanding of user needs |
| Usability Testing | 5-8 | 3-5 days | Evaluate interface usability |
| Surveys | 100+ | 1 week | Quantify attitudes/behaviors |
| Card Sorting | 15-30 | 1-2 weeks | Design information architecture |
| A/B Testing | 1000+ | 2-4 weeks | Compare design variations |
| Diary Studies | 8-12 | 1-4 weeks | Understand behavior over time |

## Usability Metrics

| Metric | Target | Measurement |
|--------|--------|-------------|
| Task Success Rate | > 78% | Successful completions / Total attempts |
| Time on Task | Baseline ± 20% | Average time to complete |
| Error Rate | < 5% | Errors / Total actions |
| SUS Score | > 68 | Standardized usability scale (0-100) |
| NPS | Varies | % Promoters - % Detractors |
| CES | > 5 | Customer Effort Score (1-7) |
