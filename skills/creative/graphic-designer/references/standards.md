# Graphic Designer - Standards & References

## WCAG Color Contrast Requirements

| Element | Ratio | Standard |
|---------|-------|----------|
| Normal text (<18pt) | 4.5:1 | WCAG 2.1 AA |
| Large text (≥18pt bold) | 3:1 | WCAG 2.1 AA |
| UI components | 3:1 | WCAG 2.1 AA |
| AAA (enhanced) | 7:1 | WCAG 2.1 AAA |

## Print Production Specifications

| Specification | Value |
|---------------|-------|
| Resolution | 300 DPI minimum (print), 72 DPI (screen) |
| Color Mode | CMYK for print; RGB for digital |
| Bleed | 3mm (0.125") on all edges |
| Safe Zone | 3mm (0.125") inside trim |
| File Format | PDF/X-1a or PDF/X-4 for print |
| Spot Colors | Pantone (PMS) for brand-critical colors |

## Typography Scale

## Design Principles

1. **Hierarchy**: Most important elements should be most prominent
2. **Balance**: Visual weight distributed intentionally
3. **Contrast**: Difference creates interest and readability
4. **Repetition**: Consistency builds unity and brand recognition
5. **Proximity**: Related elements grouped together
6. **Alignment**: Creates order and connections
