## 7.2 Rate Standards (US Market)

| Project Type | Union (SAG-AFTRA) | Non-Union |
|-------------|-------------------|-----------|
| TV Commercial (Broadcast) | $750+ (session) | $200-750 |
| Radio Commercial | $500+ (session) | $150-500 |
| Corporate/E-Learning | $350-600 (finished hour) | $100-350 |
| Video Game | $950+ (session) | $300-950 |
| Animation | $1,000+ (session) | $300-1,000 |
| Audiobook (Fiction) | $250-500 (finished hour) | $100-250 |
| Audiobook (Non-Fiction) | $200-400 (finished hour) | $75-200 |
| IVR/Phone System | $250+ (hour) | $75-250 |
| Narration (Documentary) | $500+ (finished hour) | $150-500 |
| Promo/Trailer | $650+ (session) | $200-650 |

### Pay-Play vs. Buy-Out Models

| Model | Description | Typical Range |
|-------|-------------|--------------|
| Pay-Play (PFY) | Paid per finished hour or session | $200-500/hr |
| Buy-Out | One-time payment for all usage rights | $500-5,000+ |
| Royalty | Residuals for broadcast use | Varies by medium |
| Hybrid | Reduced rate + limited buy-out | $150-300 + usage fee |

## 7.3 Technical Standards

| Specification | Standard |
|--------------|----------|
| Bit Depth | 16-bit or 24-bit |
| Sample Rate | 44.1 kHz or 48 kHz |
| File Format | WAV (AIFF acceptable) |
| Noise Floor | Below -60 dBFS |
| Peak Level | -3 dBFS maximum |
| Room Tone | 3-5 seconds at start/end |

## 7.4 Equipment Standards (Minimum Professional)

| Component | Professional Minimum | Recommended |
|-----------|---------------------|-------------|
| Microphone | Large-diaphragm condenser ($300+) | Sennheiser MK4, Neumann U87 |
| Interface | 2-channel USB ($150+) | Focusrite Scarlett 2i2 |
| DAW Software | Audacity (free) or Reaper ($60) | Adobe Audition, Pro Tools |
| Headphones | Closed-back studio ($100+) | Beyerdynamic DT 770 |
| Mic Stand | Adjustable boom | Rode PSA1 boom arm |
| Pop Filter | Essential | Stedman PX-1 |
| Acoustic Treatment | Basic foam panels | Bass traps + reflection filters |

## 7.5 Audition Standards

| Element | Requirement |
|---------|-------------|
| Slate | Name + Agent/Representation (optional) |
| Length | 60-90 seconds total (or per direction) |
| Format | MP3 (128kbps+) or WAV |
| Naming | ProjectName_TakeName_ArtistName |
| Notes | Professional, specific, grateful |
| Timing | Within 24-48 hours of request |
