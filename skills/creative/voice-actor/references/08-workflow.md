# Standard Workflow

## 8.1 Audition Process

```
Audition Workflow
├── 1. Receive Breakdown
│   ├── Review character description thoroughly
│   ├── Identify tone, age range, accent, style
│   ├── Note audition instructions (self-tape, studio session)
│   └── Determine if project aligns with your brand
│
├── 2. Script Analysis
│   ├── Break down the role's emotional arc
│   ├── Identify key moments and beats
│   ├── Note any difficult words/phrases
│   └── Mark emphasis and pacing
│
├── 3. Recording Setup
│   ├── Position microphone (6-12 inches, 45-degree angle)
│   ├── Set levels (-18 to -12 dB average)
│   ├── Do a test recording and listen back
│   └── Minimize background noise
│
├── 4. Take Recording
│   ├── Do multiple takes (3-5 minimum)
│   ├── Vary interpretation slightly between takes
│   ├── Take direction if provided
│   └── If no direction, give your best interpretation
│
└── 5. Edit & Submit
    ├── Edit together best takes (if multiple requested)
    ├── Add slate if required
    ├── Export in requested format
    └── Submit with professional cover note
```

## 8.2 Session Workflow (Directed Session)

```
In-Studio or Remote Directed Session
├── Pre-Session (Day Before)
│   ├── Review full script or sides thoroughly
│   ├── Research the project and character
│   ├── Warm up voice (5-10 minutes)
│   └── Prepare questions for director
│
├── Session Start
│   ├── Arrive early (10-15 minutes)
│   ├── Microphone check with engineer
│   ├── Review sides with director
│   └── Discuss director's vision and approach
│
├── During Session
│   ├── Take direction quickly and professionally
│   ├── Try multiple interpretations when asked
│   ├── Ask for feedback between takes
│   └── Flag if you need adjustment (water, break)
│
└── Session Wrap
    ├── Ask "Any pickups needed?"
    ├── Listen to playback
    ├── Thank director and engineer
    └── Confirm payment and contact info
```

## 8.3 Demo Production Workflow

```
Demo Creation Process (3-6 months to first demo)
├── Phase 1: Identify Direction (1-2 months)
│   ├── List your strongest characters/genres
│   ├── Research demo styles in each category
│   ├── Get feedback from coaches or agents
│   └── Narrow to 3-5 demo clips max
│
├── Phase 2: Secure Material (2-4 weeks)
│   ├── Write or license original scripts
│   ├── Record scratch track for timing
│   ├── Get any required permissions
│   └── Polish scripts to industry quality
│
├── Phase 3: Recording (1-2 days)
│   ├── Book professional studio time
│   ├── Work with experienced producer
│   ├── Record multiple takes and options
│   └── Review and select best takes
│
├── Phase 4: Production (1-2 weeks)
│   ├── Edit to professional pacing
│   ├── Add music and sound design
│   ├── Mix for consistent levels
│   └── Master for streaming (MP3) and CD (WAV)
│
└── Phase 5: Delivery (1 week)
    ├── Review final product critically
    ├── Create 60-second and full versions
    ├── Upload to hosting (SoundCloud, own site)
    └── Distribute to agents/casting directors
```

## 8.4 Voice Care Routine

| Activity | Before Session | During Session | After Session |
|----------|---------------|----------------|---------------|
| Hydration | 32+ oz water | Keep water nearby | Continue hydrating |
| Warm-up | 10-15 min scales | Easy does it first | Gentle cooldowns |
| Vocal rest | 30 min minimum | 5-min breaks hourly | Avoid shouting/singing |
| Diet | Avoid dairy, caffeine, spicy | Avoid mint/citrus | Warm water, honey |

## 8.5 File Delivery Checklist

- [ ] Recorded at correct specifications (44.1kHz/48kHz, 16/24-bit WAV)
- [ ] Noise floor below -60 dBFS
- [ ] Peak levels at -3 dBFS or below
- [ ] Room tone captured (3-5 sec)
- [ ] Named correctly (ProjectName_TakeName_ArtistName)
- [ ] Delivered in requested format
- [ ] Invoice sent (if applicable)
- [ ] Follow-up if no confirmation received
