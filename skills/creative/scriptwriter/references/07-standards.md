## 7.2 Format Standards

### Feature Film Script Format

| Element | Standard |
|---------|----------|
| Font | Courier 12-point |
| Page length | 1 page = ~1 minute of screen time |
| Title page | Title, "Written by," author name, contact info (bottom right) |
| Scene heading | ALL CAPS (INT./EXT. LOCATION - TIME) |
| Action | Uppercase left margin |
| Character name | Centered, uppercase |
| Dialogue | Centered under character name |
| Parenthetical | Under character name, lowercase |
| Transition | Right-aligned, uppercase (CUT TO:, FADE OUT:) |

### TV Script Format

| Element | Standard |
|---------|----------|
| Cold open | First 5-8 pages before Act One |
| Act breaks | Marked (END OF ACT ONE) |
| Act length | 25-35 pages per act in hour drama |
| Teases | 3-5 pages before cold open |
| Cold closes | 1 page before commercial |

### Scene Length Standards

| Type | Target Length | Acceptable Range |
|------|--------------|-----------------|
| Feature | 1-3 pages | 0.5-5 pages |
| TV Scene | 1-2 pages | 0.5-4 pages |
| Webisode | 0.5-1 page | 0.25-2 pages |

## 7.3 Industry Timing & Page Count

| Format | Target Pages | Industry Standard |
|--------|-------------|-----------------|
| Feature (drama) | 110-120 | 90-120 |
| Feature (comedy) | 95-110 | 85-115 |
| Hour TV drama | 52-62 | 50-65 |
| Half-hour comedy | 28-35 | 25-40 |
| Short film | 5-15 | 3-20 |
| Web series ep. | 8-15 | 5-20 |

## 7.4 Coverage & Development Standards

| Stage | Description | Industry Expectation |
|-------|-------------|---------------------|
| Logline | One-sentence story summary | 25-35 words |
| Pitch | Verbal story presentation | 2-5 minutes |
| Treatment | Narrative story document | 10-30 pages |
| Step outline | Scene-by-scene breakdown | 10-40 pages |
| First draft | Initial screenplay | 90-120 pages |
| Production draft | Camera-ready script | Final draft |
