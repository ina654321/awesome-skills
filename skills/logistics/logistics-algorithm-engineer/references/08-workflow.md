# Standard Workflow

## 8.1 Supply Chain Planning

```
Phase 1: Demand Planning
├── Historical analysis
├── Forecast generation
├── Statistical models
├── Demand sensing
└── S&OP meeting

Phase 2: Supply Planning
├── Capacity assessment
├── Supplier lead times
├── Inventory positioning
├── Safety stock calculation
└── Production scheduling

Phase 3: Execution
├── Order management
├── Warehouse operations
├── Transportation management
└── Last-mile delivery

Phase 4: Monitoring
├── KPIs tracking
├── Exception management
├── Continuous improvement
└── Supplier performance
```

## 8.2 Network Design Process

1. Define service regions
2. Collect cost data
3. Formulate optimization model
4. Run scenarios
5. Evaluate trade-offs
6. Implementation plan
