# Standard Workflow

## 8.1 Business Development

```
Lead Generation Flow
├── 1. Lead Identification
│   ├── Research target companies
│   ├── Identify decision-makers (LinkedIn)
│   ├── Warm introduction via network preferred
│   └── Cold outreach (email, LinkedIn)
│
├── 2. Initial Outreach
│   ├── Personalized message addressing their specific challenge
│   ├── Offer value first (insight, not pitch)
│   ├── Request 15-30 minute discovery call
│   └── Include social proof (relevant past work)
│
├── 3. Discovery Call
│   ├── Understand current state, pain points
│   ├── Ask open-ended questions
│   ├── Listen 70%, talk 30%
│   └── Identify if engagement makes sense
│
└── 4. Proposal
    ├── Follow up within 48 hours with proposal
    ├── Address specific issues from discovery
    ├── Propose solution framework
    └── Include pricing and timeline
```

## 8.2 Engagement Lifecycle

```
Project Lifecycle
├── Phase 1: Initiation (Week 1)
│   ├── Contract signed, deposit received
│   ├── Kickoff meeting with stakeholders
│   ├── Initial data request list
│   └── Communication protocols established
│
├── Phase 2: Discovery (Weeks 2-4)
│   ├── Stakeholder interviews
│   ├── Document and process review
│   ├── Gap analysis
│   └── Preliminary findings presentation
│
├── Phase 3: Solution Development (Weeks 4-8)
│   ├── Framework and recommendations
│   ├── Implementation roadmap
│   ├── Risk assessment
│   └── Draft deliverables
│
├── Phase 4: Delivery (Weeks 8-12)
│   ├── Final presentation to leadership
│   ├── Handoff documentation
│   ├── Transition planning
│   └── Project retrospective
│
└── Phase 5: Follow-up (Optional)
    ├── Post-project review (30/60/90 days)
    ├── Additional advisory sessions
    └── Ongoing retainer consideration
```

## 8.3 Client Communication Standards

| Communication Type | Frequency | Format |
|-------------------|-----------|--------|
| Status update | Weekly | Email summary or brief call |
| Major findings | As discovered | Immediate notification |
| Schedule changes | Immediately | Phone/Zoom |
| Scope concerns | Immediately | Written email |
| Invoices | Per contract | PDF via email |

### Email Response Time Standards

| Message Type | Target Response |
|-------------|-----------------|
| Urgent (escalation) | Within 2 hours |
| Client questions | Within 4 hours |
| General inquiries | Within 24 hours |
| Proposals | Within 48 hours |
| Administrative | Within 72 hours |

## 8.4 Deliverable Quality Standards

- **Presentations**: Executive summary first, detail second
- **Reports**: Problem-solution-impact structure
- **Frameworks**: Visual, actionable, adaptable
- **Process documentation**: Step-by-step, with examples
- **Training materials**: Multiple learning styles addressed

## 8.5 Engagement Closure

1. Conduct formal project closeout meeting
2. Document lessons learned
3. Obtain client testimonial (written or LinkedIn)
4. Request referral introductions
5. Transition any ongoing responsibilities
6. Issue final invoice with payment terms
7. Archive project files (7-year retention recommended)
