## 7.2 Pricing Models

| Model | Description | Typical Range | Best For |
|-------|-------------|--------------|----------|
| **Hourly** | Billed per hour | $100-500/hr | Variable scope |
| **Daily** | Fixed per day | $800-4,000/day | Workshops, training |
| **Project-based** | Fixed deliverable | $5,000-100,000+ | Defined outcomes |
| **Retainer** | Monthly commitment | $5,000-50,000/mo | Ongoing advisory |
| **Equity** | Partial ownership | 1-10% | Early-stage startups |
| **Success fee** | % of results | 5-25% | Sales/marketing engagements |

## 7.3 Engagement Structure Standards

### Statement of Work (SOW) Requirements

| Element | Description | Required |
|---------|-------------|----------|
| Objectives | Specific, measurable goals | Yes |
| Scope | What's included and excluded | Yes |
| Deliverables | Tangible outputs | Yes |
| Timeline | Milestones and final date | Yes |
| Pricing | Rate, billing schedule | Yes |
| IP Rights | Work product ownership | Yes |
| Confidentiality | NDA terms | Recommended |
| Termination | Exit clauses | Recommended |

## 7.4 Contract Standards

| Clause | Purpose | Negotiability |
|--------|---------|--------------|
| Scope definition | Prevents creep | Fixed |
| Payment terms | 30-day standard | 15-60 days negotiable |
| IP assignment | Work for hire vs. license | Usually negotiable |
| Non-compete | Market restrictions | Varies by state |
| Non-solicitation | Client/team retention | Usually negotiable |
| Limitation of liability | Risk caps | Always negotiable |
| Confidentiality | NDA both ways | Standard |
