# Common Pitfalls & Anti-Patterns

## 10.1 Anti-Patterns

| # | Anti-Pattern | Severity | Quick Fix |
|---|--------------|----------|-----------|
| 1 | **Working without a contract** | High | Always use written contract (AIGA standard) |
| 2 | **Not collecting deposit upfront** | High | Minimum 25-50% deposit before starting |
| 3 | **Giving away source files for free** | Medium | Charge for source file delivery |
| 4 | **Underpricing to get the job** | High | Quote what you're worth, not desperation pricing |
| 5 | **Unlimited revisions** | Medium | Cap at 2-3 rounds, bill extras |
| 6 | **Scope creep without renegotiation** | High | Document all changes, renegotiate if significant |
| 7 | **Not backing up work** | High | Cloud backup daily, local backup weekly |
| 8 | **Missing deadlines due to overcommitment** | High | Be honest about capacity |
| 9 | **Designing without research** | Medium | Always review brief and do basic research |
| 10 | **Ignoring payment terms** | High | Include late fees, pause work on overdue invoices |

## 10.2 Design Quality Mistakes

- **Designing on low-res screens**: Always design at actual size
- **Ignoring bleeding/trim**: Include 3mm bleed for print
- **Using low-quality images**: Always source 300dpi images minimum
- **Mismatched fonts in deliverables**: Outlines or embed fonts
- **No color profile**: Always specify and convert correctly
- **Over-relying on effects**: Clean vector work ages better

## 10.3 Communication Mistakes

- **Over-emailing**: Consolidate updates into weekly summaries
- **Being unavailable**: Set clear response time expectations
- **Taking feedback personally**: Separate ego from work
- **Not confirming direction in writing**: Always get written approval
- **Dribbling feedback**: Collect all feedback in one review session

## 10.4 Business Mistakes

- **No LLC or business structure**: Personal liability exposure
- **No errors & omissions insurance**: Protects against claims
- **Not tracking time**: You don't know profitability
- **Mixing personal/business finances**: Separate accounts required
- **Not saving for taxes**: 25-30% of income goes to taxes
- **No retirement savings**: Start as early as possible

## 10.5 Quality Checklist

- [ ] Contract signed before any work begins
- [ ] Deposit received before starting work
- [ ] Creative brief completed and approved
- [ ] Competitor analysis done
- [ ] Mood board or visual direction approved
- [ ] All concepts presented professionally
- [ ] Written approval received for each phase
- [ ] Revision count tracked and communicated
- [ ] All deliverables prepared in correct formats
- [ ] Files organized and named properly
- [ ] Source files backed up before handoff
- [ ] Final payment tracked and followed up
- [ ] Work added to portfolio (with permission)
