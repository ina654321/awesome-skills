## 7.2 Pricing Standards

### Hourly Rate Guidelines (US Market)

| Experience Level | Illustration | Graphic Design | UI/UX Design | Motion Graphics |
|-----------------|-------------|----------------|---------------|----------------|
| Entry (0-2 years) | $25-50/hr | $35-60/hr | $40-75/hr | $35-65/hr |
| Mid (2-5 years) | $50-100/hr | $60-125/hr | $75-150/hr | $65-125/hr |
| Senior (5+ years) | $100-200/hr | $125-250/hr | $150-300/hr | $125-250/hr |
| Expert (10+ years) | $200-400/hr | $250-500/hr | $300-500/hr | $250-400/hr |

### Project-Based Pricing Frameworks

| Project Type | Estimated Range | Typical Scope |
|-------------|-----------------|---------------|
| Logo design | $500-5,000 | 3 concepts, 2 revisions |
| Brand identity | $2,000-15,000 | Logo, color palette, typography, guidelines |
| Social media kit | $500-2,500 | 10-20 templates |
| Website landing page | $1,500-5,000 | Design + basic HTML/CSS |
| Packaging design | $1,000-10,000 | Per SKU, structural dielines |
| Print collateral | $300-2,000 | Business cards, letterhead, brochure |

## 7.3 File Format Standards

| Format | Use Case | Resolution | Color Mode |
|--------|----------|-----------|------------|
| AI (Illustrator) | Source files, logos, vectors | Vector | CMYK or Pantone |
| PSD (Photoshop) | Raster edits, photo manipulation | 300 dpi for print | CMYK or RGB |
| EPS | Legacy compatibility, print production | Vector | CMYK |
| PDF | Client presentations, print-ready | Varies | CMYK |
| PNG | Web, transparent backgrounds | 72-150 dpi | RGB |
| JPEG | Web images, previews | 72 dpi | RGB |
| SVG | Web graphics, icons | Vector | RGB |

## 7.4 Color Standards

| Color Space | Use Case | Profile |
|-------------|----------|---------|
| CMYK | Offset printing | Coated: FOGRA39, Uncoated: FOGRA29 |
| RGB | Web, screen display | sRGB (web standard), Display P3 (mobile) |
| Pantone | Spot color printing | Coated C or U |
| Hex | Web design | #RRGGBB |

## 7.5 Typography Standards

| Standard | Specification |
|----------|--------------|
| Screen minimum body text | 16px / 1rem |
| Screen minimum caption | 12px |
| Line height body text | 1.4-1.6 |
| Line height headings | 1.0-1.2 |
| Print minimum body | 9pt |
| Print leading | 120-145% of point size |
