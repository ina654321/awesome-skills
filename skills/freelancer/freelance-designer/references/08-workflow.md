# Standard Workflow

## 8.1 Project Intake

```
Discovery Flow
├── 1. Initial Inquiry
│   ├── Confirm project type and scope
│   ├── Request budget range (not optional - filters serious clients)
│   ├── Ask about deadline
│   └── Determine if client has seen your work before
│
├── 2. Brief Collection
│   ├── Send creative brief questionnaire
│   ├── Collect brand guidelines (if existing brand)
│   ├── Request reference examples (likes/dislikes)
│   └── Schedule discovery call
│
├── 3. Proposal & Contract
│   ├── Draft project scope document
│   ├── Calculate time estimate and pricing
│   ├── Prepare contract (use AIGA or similar standard)
│   └── Include revision limits and payment schedule
│
└── 4. Onboarding
    ├── Contract signed, deposit received
    ├── Set up project folder structure
    ├── Schedule kickoff call
    └── Create shared communication channel
```

## 8.2 Design Process

```
Design Phase
├── Phase 1: Research & Concept (20% of time)
│   ├── Industry and competitor analysis
│   ├── Mood board creation
│   ├── Initial sketching/thumbnailing
│   └── Select 2-3 directions to present
│
├── Phase 2: Design Development (40% of time)
│   ├── Create first-round mockups
│   ├── Internal critique and refinement
│   └── Present to client
│
├── Phase 3: Refinement (30% of time)
│   ├── Incorporate client feedback
│   ├── Conduct revisions (within scope)
│   └── Present refined version
│
└── Phase 4: Finalization (10% of time)
    ├── Final polish and adjustments
    ├── Prepare final deliverables
    └── Handoff package preparation
```

## 8.3 Client Presentation

### First Presentation Guidelines

1. **Limit to 3 concepts** maximum
2. **Present strongest concept first** (anchor)
3. **Explain rationale** for each direction
4. **Do not ask "what do you want?"** - offer expert guidance
5. **Take notes, don't defend** during feedback

### Presentation Format

```
Slide Deck Structure
├── Slide 1: Project recap (brief, deadline, deliverables)
├── Slide 2: Design direction 1 (2-3 variations)
├── Slide 3: Design direction 2 (2-3 variations)
├── Slide 4: Design direction 3 (2-3 variations)
├── Slide 5: Recommendation (your preferred direction)
└── Slide 6: Next steps and timeline
```

## 8.4 Revision Workflow

| Revision Round | Included | Additional Cost |
|---------------|----------|----------------|
| Round 1 | Initial concepts | Included in deposit |
| Round 2 | First revision cycle | Included in project |
| Round 3+ | Additional revisions | $75-150/hr |
| Scope change | Different project type | New estimate required |

## 8.5 File Handoff Checklist

- [ ] Source files (AI/PSD) with all linked assets
- [ ] Print-ready PDF (CMYK, bleed, crop marks)
- [ ] Web-ready files (PNG, JPEG, SVG)
- [ ] Font files (only if client needs to edit)
- [ ] Color swatches (Pantone/CMYK/RGB values)
- [ ] Usage guidelines (if brand work)
