# Standard Workflow

## 8.1 Product Sourcing & Research

```
Research Phase
├── 1. Market Analysis
│   ├── Identify trending product categories (Google Trends, Jungle Scout)
│   ├── Research competitor pricing and reviews
│   ├── Calculate potential margins (cost + fees + shipping + profit)
│   └── Verify market demand (search volume, competition level)
│
├── 2. Supplier Research
│   ├── Source from AliExpress, Alibaba, Faire, or domestic suppliers
│   ├── Request samples before bulk ordering
│   ├── Verify supplier reviews and response time
│   └── Negotiate bulk pricing and minimum order quantities
│
└── 3. Product Validation
    ├── Test product personally (quality assessment)
    ├── Check for patent/trademark conflicts
    ├── Verify age restrictions or import regulations
    └── Confirm packaging and labeling compliance
```

## 8.2 Listing Creation Workflow

### Step 1: Title Optimization

- Front-load with primary keywords (brand + model + key features)
- Maximum 80-200 characters depending on platform
- Include size, color, quantity if applicable
- Avoid keyword stuffing - readable above all

### Step 2: Description Writing

```
Description Structure
├── Opening (1-2 sentences): Who is this for, what's the main benefit
├── Features list (bulleted): 5-8 key features with specific measurements
├── Specifications (if applicable): Dimensions, materials, compatibility
├── Use case (1-2 paragraphs): When/where/how to use
└── Closing: Warranty info, return policy, brand story
```

### Step 3: Image & Video

1. Main image: White background, product centered, no watermarks
2. Additional images: Lifestyle shots, close-ups of details, scale reference
3. Video (if used): Unboxing, demonstration, or comparison

### Step 4: Pricing & Shipping

1. Set competitive price (research similar listings)
2. Configure shipping: flat rate, calculated, or free
3. Enable tracking and insurance for items over $25
4. Set handling time (1-3 days is standard)

## 8.3 Order Fulfillment Workflow

```
Fulfillment Process
├── 1. Order Received
│   ├── Confirm payment cleared
│   ├── Review buyer feedback history
│   └── Note any special requests
│
├── 2. Packaging
│   ├── Use appropriate box size (no excessive empty space)
│   ├── Protect items with bubble wrap/packing paper
│   ├── Include branded thank-you card (optional)
│   └── Apply shipping label securely
│
├── 3. Shipping
│   ├── Drop off at carrier within handling window
│   ├── Enter tracking number on platform
│   └── Notify buyer of shipment
│
└── 4. Post-Sale
    ├── Confirm delivery with buyer
    ├── Request positive feedback (professional, not pushy)
    └── Handle any issues within 24 hours
```

## 8.4 Customer Service Response Times

| Issue Type | Target Response Time | Action |
|------------|---------------------|--------|
| Question | Within 4 hours | Full, friendly answer |
| Problem reported | Within 2 hours | Acknowledge and investigate |
| Return request | Within 4 hours | Approve or negotiate |
| Negative feedback | Within 24 hours | Resolve and request revision |
| Defective/damaged | Immediate | Offer refund or replacement |
