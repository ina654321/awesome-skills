# Standard Workflow

## 8.1 Pre-Livestream Preparation

```
Preparation Flow (2-4 hours before)
├── 1. Product Selection
│   ├── Review inventory and prioritize featured items
│   ├── Select 15-30 products for 2-4 hour stream
│   ├── Arrange products by category/theme
│   └── Confirm stock levels for high-demand items
│
├── 2. Technical Setup
│   ├── Test internet connection (minimum 15 Mbps upload)
│   ├── Position lighting: key light, fill light, backlight
│   ├── Test camera angle and background
│   ├── Test microphone audio levels
│   └── Verify OBS or streaming software settings
│
├── 3. Platform Setup
│   ├── Create/copy product list to platform
│   ├── Enable shopping features and product pins
│   ├── Set up co-host if applicable
│   └── Schedule stream with title, thumbnail, description
│
└── 4. Content Planning
    ├── Write talking points for featured products
    ├── Prepare demo scripts and Q&A responses
    ├── Set up countdown timer and overlays
    └── Prepare exclusive discount codes
```

## 8.2 During Stream Protocol

### First 5 Minutes

1. Welcome viewers by name if returning viewers
2. Quickly introduce yourself and stream theme
3. State any current promotions or giveaways
4. Set expectations: stream length, product categories
5. Prompt viewers to follow/subscribe/turn on notifications

### Product Presentation Flow

```
Per-Product Flow (3-5 minutes each)
├── Introduction (30 sec)
│   ├── Name the product and price
│   └── Create urgency: "Only 20 in stock"
│
├── Demonstration (2-3 min)
│   ├── Show product from multiple angles
│   ├── Demonstrate key features live
│   ├── Use/on-wear/show the product
│   └── Read 1-2 relevant reviews aloud
│
├── Call to Action (30 sec)
│   ├── Direct link to product in bio/shop tab
│   ├── Offer discount code if applicable
│   └── Address common objections
│
└── Transition (30 sec)
    ├── Thank viewers for engagement
    └── Tease next product
```

## 8.3 Post-Stream Workflow

1. **End stream professionally** - no abrupt endings
2. **Fulfill orders within 24 hours** - platforms prioritize active sellers
3. **Reply to comments and DMs** within 2 hours
4. **Review stream analytics**: peak viewers, engagement rate, sales conversion
5. **Document what worked**: best-selling products, peak times, engagement tactics
6. **File taxes on revenue**: set aside 25-30% quarterly

## 8.4 Emergency Protocols

```
Stream Disruption Response
├── Internet drops
│   ├── Have mobile hotspot backup ready
│   ├── Pause and tell viewers "I'll be right back"
│   └── Reconnect within 3-5 minutes or end stream
│
├── Platform outage
│   ├── Switch to backup platform (YouTube, Instagram)
│   ├── Notify viewers across all social channels
│   └── Resume stream on recovery
│
└── Negative comment/scam alert
    ├── Ignore trolling; don't engage publicly
    ├── Remove scam links from chat immediately
    └── Block and report repeat offenders
```
