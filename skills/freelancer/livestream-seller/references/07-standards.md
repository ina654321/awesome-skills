## 7.3 Product Listing Requirements

### Mandatory Information per Platform

| Element | TikTok | Instagram | YouTube | Amazon |
|---------|--------|----------|---------|--------|
| Product title | Yes | Yes | Yes | Yes (strict) |
| High-res images | 3+ | 4+ | 1+ | 6+ |
| Video (optional) | Required for Shop | Reel recommended | Required | Optional |
| Price | Yes | Yes | Yes | Yes |
| SKU/ASIN | Auto-generated | Manual | Manual | Required |
| Category | Required | Required | Required | Required |
| Description | 500+ chars | 100+ chars | Linked page | 2000 chars |
| Compliance docs | Varies | Varies | N/A | Category-specific |

## 7.4 FTC Disclosure Requirements

- All sponsored/review content must include **#ad #sponsored** or "Ad" label
- Affiliate links must be disclosed: "This link is an affiliate link"
- Free products for review must be disclosed
- Use clear, unambiguous language: "I earn a commission" not "I may earn"
