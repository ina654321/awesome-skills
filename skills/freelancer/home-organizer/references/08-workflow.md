# Standard Workflow

## 8.1 Client Onboarding

```
Onboarding Flow
├── 1. Initial Contact
│   ├── Gather basic information (space type, challenges, timeline)
│   ├── Assess client category (organizer vs. hoarding)
│   ├── Determine if specialized services are needed
│   └── Provide intake form and service agreement
│
├── 2. Discovery Call (30-60 min)
│   ├── Understand goals and motivation
│   ├── Discuss previous organizing attempts
│   ├── Identify emotional triggers around possessions
│   └── Determine realistic timeline and budget
│
├── 3. In-Home Assessment (1-2 hours)
│   ├── Walk through all spaces
│   ├── Assess scope of work
│   ├── Identify hazards and safety concerns
│   ├── Document before photos (with permission)
│   └── Create preliminary project plan
│
└── 4. Service Agreement
    ├── Outline scope, schedule, rates
    ├── Set session duration and frequency
    ├── Define decision-making process
    └── Include cancellation policy
```

## 8.2 Organizing Session Workflow

```
Session Structure (4-hour standard)
├── First 15 minutes: Check-in, review session goals
├── Next 30 minutes: Sort phase (keep/maybe/discard)
├── Next 120 minutes: Process categories
├── Next 45 minutes: Implement storage solutions
├── Last 15 minutes: Review progress, set homework
└── After session: Update before/after photos, invoice
```

### Decision-Making Framework

```
Category Sort Protocol
├── KEEP (green bin/bag)
│   ├── Used regularly
│   ├── Has significant sentimental value
│   └── Would cost significant money to replace
│
├── MAYBE (yellow bin/bag)
│   ├── Used seasonally
│   ├── Genuinely unsure
│   └── Needs reflection time
│
└── DISCARD/DONATE (blue bin/bag)
    ├── Broken or unusable
    ├── Multiple duplicates (keep best)
    ├── Outgrown or obsolete
    └── No longer fits lifestyle
```

## 8.3 Category-Specific Workflows

### Closet Organization

1. Empty completely
2. Sort into: keep, donate, discard, relocate elsewhere
3. Clean empty closet
4. Install organizational systems
5. Categorize kept items: by type, then by color
6. Label storage zones
7. Document for maintenance

### Kitchen Organization

1. Remove everything from cabinets
2. Check expiration dates (discard expired)
3. Clean shelves
4. Categorize: cooking, baking, serving, storage, rarely used
5. Determine frequency of use
6. Zone by work triangle (prep, cooking, cleanup)
7. Store items where used

### Paper Management

```
The Four T's System
├── Touch it once: Can it be acted on in 2 minutes? Do it.
├── Trash it: Junk, duplicates, outdated
├── Tend it: Bills, actions needed, calendar items
└── Transfer it: Reference materials, archives
```

## 8.4 Post-Organization Follow-Up

1. Send before/after photos (with permission)
2. Invoice within 24 hours
3. Schedule next session or check-in
4. Provide maintenance tips and storage solutions
5. Follow up on disposal of discard items
6. Debrief on emotional aspects (if applicable)
