# Common Pitfalls & Anti-Patterns

## 10.1 Critical Human Performance Errors

| # | Pitfall | Severity | Consequence | Prevention |
|---|---------|----------|-------------|------------|
| 1 | Bypassing procedure | 🔴 Critical | Safety system disabled | Follow procedure exactly |
| 2 | Failing to verify | 🔴 Critical | Wrong action taken | Independent verification |
| 3 | Assumption without data | 🔴 Critical | Misdiagnosis | Verify before acting |
| 4 | Complacency | 🔴 High | Missed indicators | Vigilant monitoring |
| 5 | Procedural deviation | 🔴 High | Uncontrolled change | Get approval first |

## 10.2 Control Room Errors

| Error | Result | Correct Approach |
|--------|--------|------------------|
| Panic response | Poor decisions | Follow EOPs |
| Delay in action | Escalation | Immediate response |
| Tunnel vision | Missed symptoms | Broad assessment |
| Incomplete handoff | Gaps in coverage | Structured turnover |
| Monitoring error | Late detection | Alert awareness |

## 10.3 Technical Mistakes

| Mistake | Impact | Mitigation |
|--------|--------|------------|
| Wrong instrument reference | Wrong action | Verify indication |
| Misreading trend | Misdiagnosis | Check multiple sources |
| Equipment misidentification | Damage | Positive identification |
| Wrong mode entry | Unplanned trip | Verify mode |
| Incomplete alignment | System failure | Checklist completion |

| Issue | Consequence | Fix |
|-------|-------------|-----|

| Skipped briefing | Missed info | Full briefing |
| Poor documentation | Wrong history | Complete logs |
| Handoff error | Knowledge loss | Structured handoff |

## 10.5 Emergency Procedure Errors

| Violation | Risk | Protocol |
|-----------|------|----------|
| Not following EOP | Uncontrolled response | Use procedure |
| Delaying emergency declaration | Public risk | Early declaration |

## 10.6 Safety System Bypasses

| Error | Long-term Impact | Prevention |
|--------|------------------|------------|
| Inappropriate bypass | Unprotected equipment | Justify each bypass |
| Long bypass duration | Increased risk | Time-limited |
| Not tracking bypasses | Forgotten risks | Track all |
| Bypass without approval | Compliance | Get approval |
| Cascading bypasses | Major outage | Review impacts |

| Mistake | Result | Best Practice |
|---------|--------|---------------|

| Error | Outcome | Correction |
|-------|---------|------------|

## 10.9 Performance Decline

- **Fatigue** - Follow work hour limits
- **Overconfidence** - Peer check process
- **Complacency** - Challenge mindset
- **Distraction** - Control room discipline
- **Stress** - Support available
