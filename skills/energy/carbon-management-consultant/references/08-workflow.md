# Standard Workflow

## 8.1 Carbon Inventory Workflow

```
Emissions Inventory Process
├── Scope 1 - Direct Emissions
│   ├── Stationary combustion
│   ├── Mobile sources
│   ├── Process emissions
│   └── Fugitive emissions
├── Scope 2 - Indirect Energy
│   ├── Purchased electricity
│   ├── Purchased steam
│   ├── Purchased heating
│   └── Purchased cooling
└── Scope 3 - Value Chain
    ├── Purchased goods
    ├── Business travel
    ├── Employee commuting
    ├── Upstream transport
    └── Downstream use
```

## 8.2 Carbon Assessment Workflow

1. **Planning Phase**
   - Define organizational boundaries
   - Identify emission sources
   - Select reporting period
   - Establish baseline

2. **Data Collection**
   - Activity data gathering
   - Emission factor research
   - Data quality assessment
   - Gap filling

3. **Calculation**
   - Apply emission factors
   - Calculate scope by scope
   - Uncertainty assessment
   - Internal verification

4. **Reporting**
   - Prepare inventory report
   - Identify reductions
   - Set targets
   - Third-party verification

## 8.3 Decarbonization Planning

| Phase | Activities | Timeline |
|-------|-----------|----------|
| Assessment | Current baseline, opportunities | 1-3 months |
| Strategy | Target setting, roadmap | 3-6 months |
| Implementation | Projects, investments | Ongoing |
| Monitoring | Tracking, reporting | Continuous |

## 8.4 Target Setting Workflow

1. **Baseline Establishment**
   - Select base year
   - Calculate emissions
   - Identify significant changes
   - Recalculate if needed

2. **Target Development**
   - Absolute vs. intensity
   - Scope 1, 2, 3 coverage
   - Timeframe (2030, 2050)
   - Alignment with 1.5°C

3. **Science-Based Alignment**
   - Select method (SBTi)
   - Calculate budget share
   - Apply reduction factor
   - Submit for validation


## 8.6 Quality Assurance Checklist

- [ ] Organizational boundaries defined
- [ ] Operational control vs. equity share
- [ ] Base year established
- [ ] Emission factors documented
- [ ] Data quality assessed
- [ ] Uncertainty quantified
- [ ] Internal verification complete
- [ ] Third-party review (if needed)
- [ ] Targets aligned with strategy
- [ ] Stakeholder review
