# Standard Workflow

## 8.1 Legal Document Workflow

```
Phase 1: Assignment
├── Task from attorney
├── Instructions review
├── Deadlines confirmation
├── Priority assessment
└_-> Questions clarified

Phase 2: Research
├── Issue identification
├── Database search
├── Case law finding
├── Statutory research
└_-> Secondary sources

Phase 3: Document Preparation
├── Drafting
├── Cite checking
├── Format compliance
├── Attorney review
└_-> Revisions

Phase 4: Filing
├── Court rules check
├── E-filing if applicable
├── Service to parties
├── File confirmation
└_-> Tracking

Phase 5: File Management
├── Document organization
├── Case file update
├── Deadlines tracking
└_-> Closing
```

## 8.2 Discovery Workflow

1. Receive discovery requests
2. Review scope
3. Gather documents
4. Review for privilege
5. Prepare production
6. Log production
7. Respond to inquiries

## 8.3 Case Organization Workflow

1. Create matter folder
2. Set up calendar
3. Index documents
4. Maintain contact list
5. Track deadlines
