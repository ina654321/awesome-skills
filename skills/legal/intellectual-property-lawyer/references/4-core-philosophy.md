## § 4 · Core Philosophy

### 4.1 IP Protection Decision Matrix

```
                    INVENTION/DISCOVERY
                           │
           ┌───────────────┼───────────────┐
           ▼               ▼               ▼
      PATENTABLE     TRADE SECRET      PUBLIC
      + USEFUL       SUITABLE          DOMAIN
           │               │               │
           ▼               ▼               ▼
    ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
    │ • Novel     │ │ • Secret    │ │ • Publish   │
    │ • Useful    │ │ • Economic  │ │ • Defensive │
    │ • Non-obvious│ │   value     │ │   publication│
    │             │ │ • Protectable│ │             │
    │ File patent │ │ Maintain    │ │ No further  │
    │ application │ │ secrecy     │ │ action      │
    └─────────────┘ └─────────────┘ └─────────────┘
```

### 4.2 Guiding Principles

1. **Search Before Filing**
   - Never file patent or trademark without search
   - Cost of bad filing exceeds cost of search

2. **Claims Define the Monopoly**
   - Broad claims are harder to get and easier to invalidate
   - Balance scope with validity strength

3. **Use It or Lose It**
   - Trademarks require continuous use
   - Patent maintenance fees required

4. **Secrecy is Active, Not Passive**
   - Trade secrets need documented protection
   - Once lost, trade secret protection is gone forever

5. **Territorial Strategy**
   - File where you make, sell, or compete
   - PCT and Madrid provide efficiency
