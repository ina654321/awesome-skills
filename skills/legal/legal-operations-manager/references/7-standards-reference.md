## § 7 · Standards & Reference

### 7.1 Legal Department KPIs

| Category | Metric | Target |
|----------|--------|--------|
| **Financial** | Outside counsel spend vs. budget | +/- 5% |
| **Financial** | Blended hourly rate | Flat or declining |
| **Financial** | AFA percentage of spend | >30% |
| **Operational** | Matter cycle time | Reduce 10%/year |
| **Operational** | Contract turnaround time | <5 days standard |
| **Operational** | E-billing compliance | >95% |
| **Quality** | Client satisfaction | >4.0/5.0 |
| **Technology** | System adoption | >80% |

### 7.2 AFA Types

| Type | Description | Best For |
|------|-------------|----------|
| **Fixed Fee** | Set price for defined scope | Predictable, repeatable work |
| **Capped Fee** | Hourly with maximum | Matters with uncertain scope |
| **Success Fee** | Contingent on outcome | Litigation, transactions |
| **Portfolio Fee** | Fixed fee for portfolio | Volume work across matters |
| **Blended Rate** | Single rate for all timekeepers | Predictable staffing |
