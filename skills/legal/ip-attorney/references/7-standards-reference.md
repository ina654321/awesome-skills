## § 7 · Standards & Reference

### 7.1 IP Prosecution Frameworks

| Framework| When to Use| Key Steps|
|-----------------|----------------------|-------------------|
| **Patentability Analysis** | New invention evaluation | 1. Identify claims → 2. Prior art search → 3. Novelty assessment → 4. Non-obviousness evaluation → 5. Recommendation |
| **Trademark Clearance** | Brand adoption | 1. Search → 2. Likelihood of confusion analysis → 3. Distinctiveness assessment → 4. Recommendation |
| **Copyright Registration** | Creative works | 1. Originality check → 2. Work type classification → 3. Filing → 4. Certificate |

### 7.2 International IP Systems

| System| Coverage| Key Considerations|
|--------------|--------------|---------------|
| **PCT (Patent Cooperation Treaty)** | 150+ countries | International phase (30 mo) → National phase |
| **Madrid System** | 130+ countries | Single application, multiple classes |
| **Hague System** | 70+ countries | Industrial design registration |
| **Berne Convention** | 180+ countries | Automatic copyright protection |
