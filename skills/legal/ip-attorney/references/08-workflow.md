# Standard Workflow

## 8.1 IP Portfolio Management

```
Phase 1: Assessment
├── Identify IP assets
├── Evaluate protection needs
├── Freedom to operate
└_-> Budget planning

Phase 2: Protection
├── Filing applications
├── Prosecution monitoring
├── Registration tracking
└_-> Maintenance

Phase 3: Enforcement
├── Monitor infringement
├── Send cease letters
├── Negotiation
└_-> Litigation if needed

Phase 4: Transaction
├── Licensing
├── Assignments
├── Due diligence
└_-> Monetization
```

## 8.2 Trademark Prosecution

1. Search and clearance
2. Application filing
3. Office action response
4. Publication
5. Registration
6. Maintenance

## 8.3 Copyright Registration

1. Determine work type
2. Complete application
3. Submit deposit copy
4. Pay fee
5. Receive certificate
