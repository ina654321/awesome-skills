## § 8 · Standard Workflow

### 8.1 Arbitration Process

```
Phase 1: Initiation
├── Request for arbitration
├── Answer and counterclaims
├── Arbitrator selection
├── Terms of reference (if applicable)
└── Procedural Order No. 1

Phase 2: Written Submissions
├── Statement of Claim
├── Statement of Defense
├── Reply and Rejoinder
└── Document production

Phase 3: Hearing
├── Opening statements
├── Witness testimony
├── Expert testimony
├── Closing arguments
└── Post-hearing briefs (if any)

Phase 4: Award
├── Deliberation
├── Award drafting
└── Final award issuance
```

### 8.2 Mediation Process

```
Step 1: Pre-Mediation
   → Mediator review of submissions
   → Pre-mediation calls with each party
   → Identify decision-makers
   → Set expectations

Step 2: Joint Session
   → Opening statements
   → Agenda setting
   → Issue identification

Step 3: Private Caucuses
   → Confidential discussions with each party
   → Reality testing
   → Option generation
   → Shuttle diplomacy

Step 4: Negotiation
   → Exchange offers
   → Package deals
   → Creative solutions

Step 5: Closure
   → Settlement agreement (if reached)
   → Next steps (if not settled)
   → Confidentiality reminders
```
