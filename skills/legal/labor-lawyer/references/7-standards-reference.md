## § 7 · Standards & Reference

### 7.1 Termination Checklist

| Step | Action | Documentation |
|------|--------|---------------|
| **1** | Confirm at-will status | Employment agreement, handbook |
| **2** | Review performance history | Prior evaluations, warnings, PIP |
| **3** | Verify no protected activity | Recent complaints, leave requests |
| **4** | Check for similar situations | Consistency analysis |
| **5** | Prepare separation package | COBRA, severance, final pay |
| **6** | Plan termination meeting | Script, witness, logistics |
| **7** | Conduct termination meeting | Professional, brief, documented |
| **8** | Secure company property | Laptop, badge, access |
| **9** | Communicate to team | Need-to-know basis |
| **10** | Document file | Complete termination documentation |

### 7.2 Investigation Best Practices

| Element | Standard | Risk if Not Followed |
|---------|----------|---------------------|
| **Promptness** | Begin within 48 hours | Delay suggests lack of seriousness |
| **Impartiality** | No conflicts of interest | Bias claims, incomplete investigation |
| **Confidentiality** | Limited disclosure | Retaliation, defamation |
| **Documentation** | Detailed notes, evidence | Inability to defend decisions |
| **Conclusion** | Supported by evidence | Arbitrary outcome, litigation risk |
