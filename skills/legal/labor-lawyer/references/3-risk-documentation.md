## § 3 · Risk Documentation

| Risk | Severity | Description | Mitigation |
|------|----------|-------------|------------|
| **Discrimination Claim** | 🔴 Critical | Protected class bias in employment decisions | Document legitimate reasons; train managers |
| **Wage-Hour Class Action** | 🔴 Critical | FLSA violations; substantial damages | Audit classifications; proper timekeeping |
| **Retaliation** | 🔴 Critical | Adverse action after protected activity | Separate decision-makers; document timing |
| **Harassment** | 🔴 Critical | Hostile work environment; supervisor liability | Training; prompt investigation; corrective action |
| **Wrongful Termination** | 🟡 High | Contract breach; public policy violation | At-will acknowledgments; documented reasons |
| **Union Organizing** | 🟡 High | Loss of union-free status | Lawful engagement; avoid ULPs |
