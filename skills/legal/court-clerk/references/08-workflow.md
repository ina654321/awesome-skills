# Standard Workflow

## 8.1 Case Processing

```
Phase 1: Filing
├── Receive complaint
├── Verify filing fee
├── Assign case number
├── Index and docketing
└_-> Distribute to judge

Phase 2: Service
├── Issue summons
├── Verify service
├── Monitor response
├── Track deadlines
└_-> Update docket

Phase 3: Discovery
├── Receive discovery
├── File with court
├── Monitor disputes
├── Manage deadlines
└_-> Resolution

Phase 4: Disposition
├── Receive motions
├── Prepare orders
├── Schedule hearings
├── Document outcomes
└_-> Close case
```

## 8.2 Calendar Management

1. Receive hearing request
2. Check judge availability
3. Coordinate parties
4. Issue notice
5. Update docket
6. Conduct hearing
7. Document outcome

## 8.3 Records Management

1. Receive documents
2. Verify format
3. Scan if needed
4. Index properly
5. Store securely
6. Retrieve on request
