# Standard Workflow

## 8.1 Death Investigation

```
Phase 1: Scene Response
├── Body examination
├── Evidence collection
├── Scene documentation
├── Initial assessment
└_-> Scene sketch/photos

Phase 2: External Exam
├── Body description
├── Injuries documented
├── Identity confirmation
├── Evidence collected
└_-> X-rays if needed

Phase 3: Internal Exam
├── Y incision and removal
├── Organ examination
├── Tissue sampling
├── Toxicology specimens
└_-> Histology samples

Phase 4: Laboratory
├── Toxicology analysis
├── Histology processing
├── DNA testing if needed
├── Microscopic examination
└_-> Final reports

Phase 5: Documentation
├── Autopsy report
├── Death certificate
├── Evidence logs
└_-> Case file
```

## 8.2 Medical Examiner Workflow

1. Death notification
2. Scene investigation
3. Body transport
4. Autopsy
5. Laboratory analysis
6. Report writing
7. Testimony if needed
