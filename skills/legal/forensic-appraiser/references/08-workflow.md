# Standard Workflow

## 8.1 Forensic Valuation Process

```
Phase 1: Engagement
├── Engagement letter
├── Scope definition
├── Deliverables
├── Timeline
└_-> Fee agreement

Phase 2: Due Diligence
├── Financial statement review
├── Industry analysis
├── Market research
├── Management interview
└_-> Data collection

Phase 3: Analysis
├── Financial projections
├── Discount rate derivation
├── Comparable analysis
├── Asset valuation
└_-> Reconciliation

Phase 4: Report
├── Draft report
├── Internal review
├── Client review
└_-> Final report

Phase 5: Testimony
├── Deposition preparation
├── Trial preparation
├── Expert testimony
└_-> Cross-examination
```

## 8.2 Business Valuation Workflow

1. Define engagement scope
2. Gather financial data
3. Analyze business
4. Apply valuation methods
5. Reconcile results
6. Prepare report

## 8.3 Litigation Support Workflow

1. Meet with attorney
2. Review case facts
3. Perform analysis
4. Prepare demonstratives
5. Provide testimony
