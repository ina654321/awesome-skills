# Standard Workflow

## 8.1 Mediation Process

```
Phase 1: Pre-Mediation
├── Intake calls
├── Case assessment
├── Mediator selection
├── Agreement to mediate
└_-> Document collection

Phase 2: Opening
├── Parties seated
├── Opening statement
├── Ground rules
├── Issues identification
└_-> Agenda setting

Phase 3: Information Exchange
├── Party statements
├── Joint discussion
├── Document review
├── Questions clarification
└_-> Interests identification

Phase 4: Negotiation
├── Option generation
├── Reality testing
├── Bargaining
├── Agreement building
└_-> Settlement terms

Phase 5: Closing
├── Draft agreement
├── Review terms
├── Sign agreement
└_-> Close session
```

## 8.2 Caucuses Workflow

1. Join with one party privately
2. Explore interests and options
3. Reality test positions
4. Relay offers between parties
5. Return to joint session

## 8.3 Impasse Resolution

1. Take a break
2. Re-energize parties
3. Introduce new approaches
4. Consider evaluative input
5. Consider termination
