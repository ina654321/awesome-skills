# ExxonMobil Workflow Patterns

> **Detailed Execution Workflows for Oil & Gas Operations**

---

## W1: Major Capital Project Development

### Phase 1: Opportunity Identification (Months 0-6)

```
┌─────────────────────────────────────────────────────────────────┐
│            STEP 1.1: STRATEGIC SCREENING                        │
├─────────────────────────────────────────────────────────────────┤
│  Inputs:                                                        │
│  • Corporate strategy and portfolio needs                       │
│  • Market outlook (oil/gas prices, demand)                      │
│  • Competitor activity                                          │
│  • Technology trends                                            │
│                                                                 │
│  Activities:                                                    │
│  □ Screen global opportunities                                  │
│  □ Assess strategic fit                                         │
│  □ Preliminary resource estimation                              │
│  □ High-level economic screening                                │
│                                                                 │
│  Outputs:                                                       │
│  • Opportunity ranking                                          │
│  • Go/no-go for detailed evaluation                             │
│  • Resource allocation for Phase 2                              │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│            STEP 1.2: TECHNICAL EVALUATION                       │
├─────────────────────────────────────────────────────────────────┤
│  Inputs:                                                        │
│  • Seismic data (existing or new acquisition)                   │
│  • Analog performance data                                      │
│  • Geologic models                                              │
│                                                                 │
│  Activities:                                                    │
│  □ Geologic modeling                                            │
│  □ Reservoir characterization                                   │
│  □ Resource estimation (P10/P50/P90)                            │
│  □ Development concept identification                           │
│                                                                 │
│  Outputs:                                                       │
│  • Contingent resource volume                                   │
│  • Recovery factor estimates                                    │
│  • Preliminary development options                              │
│  • Technical risk register                                      │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│            STEP 1.3: ECONOMIC SCREENING                         │
├─────────────────────────────────────────────────────────────────┤
│  Activities:                                                    │
│  □ Preliminary CAPEX estimate (Class 5)                         │
│  □ Operating cost estimates                                     │
│  □ Price scenario analysis                                      │
│  □ Preliminary NPV/IRR calculation                              │
│                                                                 │
│  Gate Criteria:                                                 │
│  • IRR >15% at base case prices                                 │
│  • Positive NPV at $50/bbl (stress case)                        │
│  • Strategic fit confirmed                                      │
│                                                                 │
│  Output: Proceed to Phase 2 (Concept Select)                    │
└─────────────────────────────────────────────────────────────────┘
```

### Phase 2: Concept Select (Months 6-12)

```
┌─────────────────────────────────────────────────────────────────┐
│            STEP 2.1: CONCEPT DEFINITION                         │
├─────────────────────────────────────────────────────────────────┤
│  Activities:                                                    │
│  □ Identify 3-5 development concepts                            │
│  □ Define technical scope for each                              │
│  □ Schedule and resource estimates                              │
│                                                                 │
│  Concept Examples (Offshore):                                   │
│  • Concept A: Fixed platform + pipeline                         │
│  • Concept B: FPSO + subsea                                     │
│  • Concept C: TLP + subsea                                      │
│  • Concept D: Extended reach drilling from existing facility    │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│            STEP 2.2: FEED (SELECT)                              │
├─────────────────────────────────────────────────────────────────┤
│  Activities:                                                    │
│  □ Front-End Engineering Design for selected concept            │
│  □ Class 3 cost estimate (±15%)                                 │
│  □ Schedule development (Level 3)                               │
│  □ Risk analysis and mitigation planning                        │
│  □ Environmental and regulatory assessment                      │
│                                                                 │
│  Key Deliverables:                                              │
│  • Process flow diagrams (PFDs)                                 │
│  • Plot plans and layouts                                       │
│  • Equipment lists (major items)                                │
│  • Utility requirements                                         │
│  • HAZOP/HAZID documentation                                    │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│            STEP 2.3: FID PREPARATION                            │
├─────────────────────────────────────────────────────────────────┤
│  Activities:                                                    │
│  □ Final investment proposal preparation                        │
│  □ Board approval process                                       │
│  □ Partner approvals (if applicable)                            │
│  □ Financing arrangements                                       │
│  □ Contract strategy finalization                               │
│                                                                 │
│  Gate Criteria (FID Checklist):                                 │
│  □ IRR > hurdle rate (typically 10-15%)                         │
│  □ NPV positive at stress prices                                │
│  □ Technical risks mitigated                                    │
│  □ Environmental permits in place or obtainable                 │
│  □ Partner alignment                                            │
│  □ Execution team in place                                      │
└─────────────────────────────────────────────────────────────────┘
```

### Phase 3: Execute & Deliver (Months 12-48)

```
┌─────────────────────────────────────────────────────────────────┐
│            STEP 3.1: DETAILED ENGINEERING                       │
├─────────────────────────────────────────────────────────────────┤
│  Activities:                                                    │
│  □ P&ID development                                             │
│  □ Equipment specifications and procurement                     │
│  □ 3D modeling                                                  │
│  □ Construction planning                                        │
│  □ Regulatory submission and approval                           │
│                                                                 │
│  Key Milestones:                                                │
│  • 30% design review                                            │
│  • 60% design review                                            │
│  • 90% design review                                            │
│  • IFC (Issued for Construction)                                │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│            STEP 3.2: PROCUREMENT                                │
├─────────────────────────────────────────────────────────────────┤
│  Categories:                                                    │
│  • Long-lead items (order first)                                │
│    - Subsea trees and manifolds                                 │
│    - Rotating equipment                                         │
│    - Structural steel                                           │
│                                                                 │
│  • Standard equipment                                           │
│    - Pumps, valves, instruments                                 │
│    - Electrical equipment                                       │
│    - Piping                                                     │
│                                                                 │
│  Strategy:                                                      │
│  • EPC contract or split EPC                                    │
│  • Competitive bidding                                          │
│  • Supplier qualification                                       │
│  • Quality surveillance                                         │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│            STEP 3.3: CONSTRUCTION & INSTALLATION                │
├─────────────────────────────────────────────────────────────────┤
│  Onshore Construction:                                          │
│  □ Site preparation                                             │
│  □ Module fabrication                                           │
│  □ Integration                                                  │
│  □ Pre-commissioning                                            │
│                                                                 │
│  Offshore Installation:                                         │
│  □ Vessel mobilization                                          │
│  □ Platform/ FPSO installation                                  │
│  □ Subsea installation (trees, flowlines)                       │
│  □ Hook-up and commissioning                                    │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│            STEP 3.4: COMMISSIONING & STARTUP                    │
├─────────────────────────────────────────────────────────────────┤
│  Activities:                                                    │
│  □ Mechanical completion                                        │
│  □ Pre-commissioning (loop checks, flushing)                    │
│  □ Commissioning (cold, hot)                                    │
│  □ Performance testing                                          │
│  □ First oil/gas                                                │
│  □ Ramp-up to full production                                   │
│                                                                 │
│  Key Metrics:                                                   │
│  • Schedule adherence                                           │
│  • Budget variance                                              │
│  • Safety record (TRIR)                                         │
│  • Performance vs. design                                       │
└─────────────────────────────────────────────────────────────────┘
```

### Phase 4: Operate & Optimize (Years 5-30+)

```
┌─────────────────────────────────────────────────────────────────┐
│            STEP 4.1: PRODUCTION OPTIMIZATION                    │
├─────────────────────────────────────────────────────────────────┤
│  Activities:                                                    │
│  □ Reservoir management (pressure maintenance)                  │
│  □ Well intervention (workovers, stimulations)                  │
│  □ Artificial lift optimization                                 │
│  □ Facility debottlenecking                                     │
│                                                                 │
│  Key Metrics:                                                   │
│  • Production vs. forecast                                      │
│  • Operating cost per BOE                                       │
│  • Facility uptime                                              │
│  • Ultimate recovery factor                                     │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│            STEP 4.2: RESERVOIR MANAGEMENT                       │
├─────────────────────────────────────────────────────────────────┤
│  Activities:                                                    │
│  □ Production surveillance (pressure, rates)                    │
│  □ 4D seismic monitoring                                        │
│  □ Infill drilling decisions                                    │
│  □ EOR/IOR implementation                                       │
│  □ Reserves booking (annual)                                    │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│            STEP 4.3: P&A (PLUG & ABANDONMENT)                   │
├─────────────────────────────────────────────────────────────────┤
│  Activities:                                                    │
│  □ Well plugging                                                │
│  □ Platform decommissioning (offshore)                          │
│  □ Site restoration                                             │
│  □ Regulatory compliance                                        │
│                                                                 │
│  Timing: End of economic field life                             │
│  Liability: Provisioned throughout field life                   │
└─────────────────────────────────────────────────────────────────┘
```

## W2: Shale Development Program

### Annual Cycle

```
┌─────────────────────────────────────────────────────────────────┐
│              Q1: PLANNING & APPROVALS                           │
├─────────────────────────────────────────────────────────────────┤
│  Activities:                                                    │
│  □ Annual budget development                                    │
│  □ Well count and location selection                            │
│  □ Rig and frac crew contracting                                │
│  □ Infrastructure planning (pads, roads, facilities)            │
│  □ Environmental permits                                        │
│  □ Partner approvals (if non-operated)                          │
│                                                                 │
│  Output: Approved AFE (Authorization for Expenditure)           │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│              Q2-Q4: EXECUTION                                   │
├─────────────────────────────────────────────────────────────────┤
│  Continuous Activities:                                         │
│  • Pad construction                                             │
│  • Drilling operations                                          │
│  • Completions                                                  │
│  • Production tie-in                                            │
│                                                                 │
│  Target: 150-200 wells/year (Permian post-Pioneer)              │
│  Rhythm: 12-18 drilling days per well                           │
│          3-5 frac stages per day                                │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│              CONTINUOUS: OPTIMIZATION                           │
├─────────────────────────────────────────────────────────────────┤
│  Focus Areas:                                                   │
│  □ Well spacing optimization                                    │
│  □ Lateral length extension                                     │
│  □ Completion design (proppant, fluid)                          │
│  □ Drill bit and BHA selection                                  │
│  □ Artificial lift selection and optimization                   │
│  □ Water management and recycling                               │
│                                                                 │
│  Learning Loop:                                                 │
│  1. Collect performance data                                    │
│  2. Analyze EUR vs. design parameters                           │
│  3. Update type curve and economics                             │
│  4. Adjust development plan                                     │
│  5. Implement and measure                                       │
└─────────────────────────────────────────────────────────────────┘
```
