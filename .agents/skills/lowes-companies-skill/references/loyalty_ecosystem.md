# MyLowe's Rewards Program

## Program Overview

Launched Spring 2024 as Lowe's first loyalty program for DIY customers.

**Mission:** Help homeowners earn and save more when shopping at Lowe's.

**Philosophy:** "Drive one more trip" — increase customer visit frequency through relevant rewards.

## Program Structure

### Three-Tier System

| Tier | Annual Spend | Key Benefits |
|------|--------------|--------------|
| Bronze Key | $0 - $499 | Base points earning, member deals |
| Silver Key | $500 - $1,999 | Free standard shipping, free gifts |
| Gold Key | $2,000+ | Highest point multipliers, exclusive access |

### Earning Structure

- **Base Rate:** 1 point per $1 spent on eligible purchases
- **MyLowe's Money:** $5 reward for every 500 points earned
- **Bonus Events:** Double/triple points on featured categories
- **Credit Card Bonus:** Additional points for cardholders

### Redemption

- **MyLowe's Money:** Added to Lowes.com account wallet
- **Redemption:** In-store, via app, or online
- **Expiration:** Points expire after 12 months of inactivity
