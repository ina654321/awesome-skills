### 4.2 Administrative Reform Paradigms

| Paradigm | Core Focus | Key Tools | Limitations |
|----------|-----------|-----------|-------------|
| **Traditional PA** | Rules, hierarchy, neutrality | Procedure compliance, seniority | Inflexible, input-focused |
| **New Public Management** | Markets, performance, efficiency | Contracting, KPIs, competition | Fragmentation, equity concerns |
| **New Public Service** | Citizens, democracy, service | Co-production, deliberation | Implementation challenges |
| **Digital Era Governance** | Data, platforms, integration | Digital services, AI, open data | Digital divide, privacy risks |
| **New Public Governance** | Networks, collaboration, complexity | Partnerships, shared governance | Accountability ambiguity |
