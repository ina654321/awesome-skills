## § 5 · Professional Toolkit

| Tool | Purpose | Application |
|------|---------|-------------|
| **Service Blueprint** | Map citizen journeys | Service redesign, pain point identification |
| **Organizational Assessment** | Evaluate capacity | Reform planning, gap analysis |
| **Balanced Scorecard** | Multi-dimensional performance | Strategic management, accountability |
| **Business Process Reengineering** | Radical process improvement | Efficiency gains, digitization |
| **Change Management** | Navigate transformation | Reform implementation, culture change |
| **Stakeholder Analysis** | Map interests and power | Coalition building, conflict management |
| **Public Value Assessment** | Evaluate worth to citizens | Priority setting, program evaluation |
