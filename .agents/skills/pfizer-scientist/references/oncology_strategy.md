## Therapeutic Area Focus

### Breast Cancer

```
Treatment Paradigm:
┌─────────────────────────────────────────────────────────────────────────┐
│ EARLY STAGE                                                             │
│ ├── HR+/HER2-: Ibrance + endocrine therapy (adjuvant)                   │
│ └── HER2+: Perjeta, Herceptin (trastuzumab)                             │
│                              ↓                                          │
│ METASTATIC                                                            │
│ ├── HR+/HER2-:                                                          │
│ │   ├── 1L: Ibrance + AI                                                │
│ │   └── Post-CDK4/6: Vepdegestrant (Phase 3)                            │
│ ├── HER2+:                                                              │
│ │   └── Tukysa (CNS metastases)                                         │
│ └── Triple-negative: Research programs                                  │
└─────────────────────────────────────────────────────────────────────────┘
```

**Key Trials:**
- **PENELOPE-3**: Ibrance adjuvant for high-risk early breast cancer
- **VERITAC-2**: Vepdegestrant vs standard of care

### Genitourinary Cancer

**Prostate Cancer:**
```
Disease States:
├─ Metastatic Hormone-Sensitive (mHSPC)
│  └── Xtandi + ADT (ARCHES, approved)
│
├─ Non-Metastatic Castration-Resistant (nmCRPC)
│  └── Xtandi (EMBARK, approved)
│
├─ Metastatic Castration-Resistant (mCRPC)
│  ├── Xtandi (standard of care)
│  └── Mevrometostat + Xtandi (Phase 3)
│
└─ Muscle-Invasive Bladder Cancer (MIBC)
   └── Padcev perioperative (EV-304, Phase 3)
```

**Key Data:**
- **ARCHES**: Xtandi + ADT showed 61% reduction in risk of progression/death
- **EMBARK**: Xtandi demonstrated benefit in high-risk nmCRPC

### Lung Cancer

**ALK-Positive NSCLC:**
- **Lorbrena**: First-line approval based on CROWN trial
- **CROWN Update (2024)**: 5-year PFS data showed sustained benefit vs crizotinib

**Novel Targets:**
- **Sigvotatug vedotin**: Integrin beta-6 target in NSCLC
- **PDL1V**: PD-L1 directed ADC

### Hematologic Malignancies

**Multiple Myeloma:**
```
Treatment Sequence:
┌─────────────────────────────────────────────────────────────────────────┐
│ Transplant-Eligible NDMM                                                │
│ └── Darzalex-based quadruplet (VRd-D)                                   │
│                              ↓                                          │
│ Transplant-Ineligible NDMM                                              │
│ ├── MagnetisMM-6: Elrexfio + daratumumab + lenalidomide                 │
│ └── MagnetisMM-5: Elrexfio-based quadruplet                             │
│                              ↓                                          │
│ Relapsed/Refractory                                                     │
│ └── Elrexfio (approved for heavily pretreated)                          │
└─────────────────────────────────────────────────────────────────────────┘
```

**Lymphoma:**
- **Adcetris**: Standard of care for CD30+ lymphomas
- **Combination studies**: Adcetris + checkpoint inhibitors
