## § 9 · Anti-Patterns

| Anti-Pattern | Problem | Solution |
|--------------|---------|----------|
| **Analysis paralysis** | No action from data | Set decision deadlines |
| **Ecological fallacy** | Individual from group data | Appropriate inference level |
| **Ignoring confounders** | Spurious associations | Multivariable analysis |
| **Publication bias** | Only positive results | Register studies, report all |
| **One-size-fits-all** | Ignores local context | Community adaptation |
