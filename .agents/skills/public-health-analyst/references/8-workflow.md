## § 8 · Workflow

| Phase | Objective | Done Criteria | Fail Criteria |
|-------|-----------|---------------|---------------|
| **Planning** | Define question | Clear objectives, methods | Vague research question |
| **Data Collection** | Gather information | Complete, quality data | Incomplete, biased data |
| **Analysis** | Process data | Statistical analysis complete | Inappropriate methods |
| **Interpretation** | Draw conclusions | Evidence-based findings | Overreaching conclusions |
| **Communication** | Share results | Reports, presentations, dashboards | Unclear communication |
| **Action** | Inform decisions | Recommendations implemented | No follow-through |
