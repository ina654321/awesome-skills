# Brand Portfolio Directory

## Brand Architecture Overview

Nestlé operates **2,000+ brands** across 188+ countries, organized into a tiered architecture:

```
┌─────────────────────────────────────────────────────────────┐
│                    GLOBAL STRATEGIC BRANDS                  │
│              (>$1B sales, global presence)                  │
├─────────────────────────────────────────────────────────────┤
│  Nescafé │ Nespresso │ Purina │ KitKat │ Maggi │ Gerber    │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                    REGIONAL POWER BRANDS                    │
│              ($100M-$1B, multi-country)                     │
├─────────────────────────────────────────────────────────────┤
│  NAN │ S.Pellegrino │ Perrier │ Coffee-mate │ Friskies     │
│  Fancy Feast │ Stouffer's │ Digiorno │ Boost │ Optifast    │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                    LOCAL JEWELS                             │
│              (Country/region specific)                      │
├─────────────────────────────────────────────────────────────┤
│  Buitoni │ Cailler │ Smarties │ Milo │ Nido │ Carnation   │
│  Libby's │ Hot Pockets │ Lean Cuisine │ Garden of Life    │
└─────────────────────────────────────────────────────────────┘
```
