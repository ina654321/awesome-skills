### 7.2 Licensing Model (Post-Broadcom)

| Offering | Model | Notes |
|----------|-------|-------|
| **VCF** | Per-core subscription | Bundled vSphere + NSX + vSAN |
| **VVF** | Per-core subscription | vSphere + vSAN only |
| **vSphere** | Per-core subscription | Compute only |
| **vSAN** | Per-TiB capacity | Storage capacity licensing |
| **Tanzu** | Per-core add-on | Kubernetes runtime |

### 7.4 Competition Landscape

| Competitor | Product | Differentiation | VMware Advantage |
|------------|---------|-----------------|------------------|
| **Microsoft** | Hyper-V / Azure Stack HCI | Windows integration, licensing bundles | Mature ecosystem, broader cloud support |
| **Nutanix** | AHV / AOS | HCI-native, simpler management | Network virtualization (NSX), broader ecosystem |
| **Red Hat** | OpenShift Virtualization | Open source, Kubernetes-native | Enterprise support, proven virtualization |
| **Cloud Native** | EKS/AKS/GKE | No virtualization overhead | On-prem consistency, legacy app support |
