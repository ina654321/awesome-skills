# Mercedes-Benz Drive Pilot

## Overview

Drive Pilot is Mercedes-Benz's SAE Level 3 conditional automation system — the first internationally certified Level 3 system in production vehicles.

## Key Facts

| Attribute | Value |
|-----------|-------|
| **SAE Level** | Level 3 (Conditional Automation) |
| **Markets** | Germany, USA (CA, NV) |
| **Max Speed** | 95 km/h (59 mph) — world's fastest certified L3 |
| **Availability** | S-Class, EQS Sedan |
| **Price** | €5,950 (~$6,500) option |

## Operating Conditions

- Clear lane markings on approved highways
- Moderate traffic (<95 km/h)
- Daytime, clear weather
- No construction zones

## Technology

- 35+ sensors (LiDAR, cameras, radar, ultrasonic)
- HD map with centimeter accuracy
- Redundant steering, braking, electrical systems
- Turquoise indicator lights (indicate L3 active)

## Driver Experience

When Drive Pilot is active:
- Driver can take hands OFF wheel
- Driver can take eyes OFF road
- Driver can engage in secondary activities (movies, games, work)
- Driver must remain capable of taking control within 10 seconds
