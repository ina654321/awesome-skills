# Mercedes-Benz Operating System (MB.OS)

## Overview

MB.OS is Mercedes-Benz's proprietary chip-to-cloud operating system, launched in 2025 with the new CLA.

## Architecture

### Four Domains

1. **Infotainment** — Navigation, entertainment, voice
2. **Automated Driving** — Drive Pilot, ADAS
3. **Body & Comfort** — Climate, seats, lighting
4. **Driving & Charging** — Powertrain, battery management

## Key Capabilities

- **Weekly OTA Updates:** Complete vehicle software updates
- **AI Virtual Assistant:** Powered by ChatGPT-4o and Google Gemini
- **Regional Adaptation:** Different navigation providers by market
- **Software-Defined Vehicle:** Hardware flexibility via software

## Development Approach

- Vertical integration of software stack
- In-house development with NVIDIA partnership
- Global R&D hubs: Stuttgart, Berlin, Tel Aviv, Silicon Valley, Beijing, Shanghai

## First Vehicle

New CLA (2025) — first production vehicle with MB.OS
