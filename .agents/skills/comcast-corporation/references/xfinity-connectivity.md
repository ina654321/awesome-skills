## Technology & Infrastructure

### Network Architecture

**HFC (Hybrid Fiber-Coaxial):**
- Fiber to node/neighborhood
- Coaxial cable to home
- Shared bandwidth per node
- Cost-effective for high speeds

**Fiber Deep:**
- Pushing fiber closer to homes
- Reduced node sizes (fewer homes per node)
- Higher capacity, better performance
- Path to DOCSIS 4.0

**Full Fiber (FTTH):**
- Fiber to the home (select areas)
- Symmetrical speeds
- Future-proof infrastructure
- 6 Gbps Gigabit Pro service

### xFi Platform

**xFi Gateway:**
- DOCSIS 3.1 modem
- WiFi 6/6E router
- Advanced security
- Whole-home coverage
- xFi Pods (mesh extenders)

**xFi Mobile App:**
- Network management
- Device monitoring
- Parental controls
- WiFi pause/scheduling
- Troubleshooting

### Competitive Position

**vs. Fiber (Verizon Fios, AT&T Fiber):**
- Pros: Broader footprint, content bundle, WiFi hotspots, wireless
- Cons: Asymmetrical speeds (legacy), shared bandwidth

**vs. Fixed Wireless (Verizon 5G Home, T-Mobile Home Internet):**
- Pros: Reliability, consistency, content bundle, WiFi network
- Cons: Price, simplicity of fixed wireless offers

**vs. Satellite (Starlink):**
- Pros: Lower latency, unlimited data, bundled services
- Cons: Rural coverage (Starlink advantage)
