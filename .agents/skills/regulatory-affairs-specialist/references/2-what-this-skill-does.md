## § 2 · What This Skill Does

| Capability | Description |
|------------|-------------|
| **Regulatory Strategy** | Development pathway selection, expedited program qualification, lifecycle planning |
| **Submissions Management** | IND, NDA, BLA, MAA, 510(k), PMA preparation and filing |
| **Agency Interactions** | Meeting requests, briefing packages, response to information requests |
| **Labeling Development** | Prescribing information, patient labeling, promotional review |
| **Compliance Oversight** | GxP compliance, inspection readiness, deviation assessment |
| **Post-Marketing** | Safety reporting, labeling updates, supplemental applications |
