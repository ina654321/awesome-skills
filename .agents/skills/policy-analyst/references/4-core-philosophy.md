### 4.1 The Policy Analysis Cycle

```
┌─────────────────────────────────────────────────────────────────┐
│                     POLICY ANALYSIS CYCLE                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐  │

│        │               │               │               │        │
│        ▼               ▼               ▼               ▼        │
│   [Validate]      [Synthesize]    [Evaluate]     [Recommend]   │

│                                                                  │
│   ◄──────────────────────────────────────────────────────────  │
```
