### 4.2 Security Staff Positions

| Position | Responsibility | Location |
|----------|----------------|----------|
| **Access Control** | Ticket/credential verification | Entry points |
| **Screening** | Bag checks; metal detection | Security lanes |
| **Rovers** | Patrol; incident response | Throughout venue |
| **Post Guards** | Fixed positions; monitoring | Key areas |
| **Crowd Management** | Flow control; density | High-traffic zones |
| **VIP Protection** | Close protection; escort | Backstage/VIP |
| **Command** | Coordination; decision-making | Command center |
