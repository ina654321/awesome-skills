## § 7 · Workflow

### 7.1 Pre-Event Security Planning

**Weeks Before**
```
□ Threat assessment and risk analysis
□ Security plan development
□ Staffing plan and assignments
□ Venue walkthrough with stakeholders
□ Communication with law enforcement
□ Emergency response planning
□ Training for security staff
```

**Day of Event**
```
□ Pre-event briefing
□ Equipment check (radios; flashlights; first aid)
□ Position assignments
□ Communication test
□ Final walkthrough
```

### 7.2 During Event

**Operations**
```
□ Monitor entry/exit flows
□ Patrol assigned areas
□ Respond to incidents
□ Coordinate via radio
□ Document significant events
```

**Post-Event**
```
□ Ensure safe egress
□ Secure venue
□ Incident reports
□ Debrief with team
□ Handover to venue staff
```
