## § 6 · Domain Knowledge

### 6.1 Prohibited Items (Typical)

| Category | Examples | Action |
|----------|----------|--------|
| **Weapons** | Guns, knives, pepper spray | Confiscate; notify police |
| **Drugs** | Illegal substances | Confiscate; notify police |
| **Alcohol** | Outside beverages | Pour out; allow entry |
| **Professional cameras** | DSLR; detachable lens | Deny entry or check in |
| **Recording devices** | GoPros; selfie sticks | Deny entry |
| **Containers** | Large bags; backpacks | Check or deny |
| **Noisemakers** | Air horns; whistles | Confiscate |

### 6.2 Crowd Density Guidelines

| People/m² | Condition | Action |
|-----------|-----------|--------|
| **<2** | Comfortable | Normal operations |
| **2-4** | Dense | Monitor; manage flow |
| **4-6** | Crowded | Active management; consider restriction |
| **>6** | Dangerous | Immediate intervention; stop entry |

### 6.3 De-escalation Levels

| Level | Behavior | Response |
|-------|----------|----------|
| **1** | Annoyed; questioning | Explanation; options |
| **2** | Argumentative; raised voice | Active listening; empathy |
| **3** | Aggressive; threatening | Firm boundaries; backup |
| **4** | Physical; violent | Restraint; ejection; arrest |
