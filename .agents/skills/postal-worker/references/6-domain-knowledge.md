## § 6 · Domain Knowledge

### 6.1 Mail Classes and Services

| Class | Description | Typical Use |
|-------|-------------|-------------|
| **First-Class Mail** | Letters; postcards; flats | Bills; personal correspondence |
| **Priority Mail** | 1-3 day delivery; tracking | Packages; time-sensitive |
| **Priority Mail Express** | Overnight guarantee | Urgent documents; critical items |
| **Marketing Mail** | Bulk advertising | Flyers; catalogs |
| **Periodicals** | Newspapers; magazines | Subscriptions |
| **Package Services** | Ground delivery; economy | Non-urgent packages |

### 6.2 Prohibited and Restricted Items

| Category | Status | Examples |
|----------|--------|----------|
| **Explosives** | Prohibited | Fireworks; ammunition |
| **Hazardous Materials** | Restricted | Lithium batteries; perfumes |
| **Perishables** | Restricted | Food; live animals |
| **Currency** | Restricted | Cash; coins (insured only) |
| **Intoxicating Liquors** | Restricted | Alcohol (state laws vary) |
| **Drugs** | Prohibited | Illegal substances |

### 6.3 Delivery Safety Protocols

| Hazard | Prevention | Response |
|--------|------------|----------|
| **Dog Attacks** | Watch for signs; carry spray; don't pet | Defend; report; medical attention |
| **Weather** | Proper clothing; traction devices | Delay if dangerous; notify supervisor |
| **Hazardous Conditions** | Report unsafe addresses | Hold mail; customer pickup |
| **Vehicle Safety** | Pre-trip inspection; seatbelts | Defensive driving; report accidents |
| **Suspicious Items** | Recognition training | Isolate; notify; do not handle |
