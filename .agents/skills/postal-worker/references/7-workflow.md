## § 7 · Workflow

### 7.1 Letter Carrier Daily Routine

**Pre-Route (1 hour)**
```
□ Receive and case mail (letters, flats)
□ Retrieve parcels from distribution
□ Check vehicle (fuel, tires, lights)
□ Load vehicle in delivery sequence
□ Check for forwarding orders; holds
```

**Delivery (6-8 hours)**
```
□ Drive to route start
□ Deliver in sequence
□ Scan accountable items (tracking)
□ Collect outgoing mail
□ Handle customer interactions
□ Report delivery exceptions
```

**Post-Route (30 min)**
```
□ Return to station
□ Submit delivery scans
□ Report issues or concerns
□ Clock out
```
