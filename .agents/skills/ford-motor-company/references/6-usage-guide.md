## 6. Usage Guide

### When to Use This Skill

✅ **Use for**:
- Automotive industry strategy and transformation
- EV transition analysis and decision-making
- Commercial fleet and Ford Pro discussions
- Manufacturing and supply chain topics
- UAW/labor relations in automotive context
- Competitive analysis (GM, Stellantis, Tesla, Rivian)

❌ **Don't use for**:
- General financial analysis (non-automotive)
- Non-Ford specific automotive technical questions
- Personal vehicle purchasing advice

### Conversation Starters

```
"How should Ford balance its EV investments with hybrid demand?"

"What's the strategic value of Ford Pro's software ecosystem?"

"How does Ford's UAW agreement impact its competitiveness?"

"What's the path to profitability for Ford Model e?"

"How should Ford position against Tesla in the EV market?"
```

### Output Formats

**Strategic Analysis**: Use frameworks in §1.2 with supporting data from §2.x  
**Product Comparisons**: Reference §2.3 tables with volume and specification data  
**Financial Discussion**: Cite specific segment EBIT and revenue figures  
**Competitive Context**: Position against GM, Stellantis, Tesla using market share data

---

*This skill represents Ford Motor Company as of Q4 2025. Financial data from Ford's official earnings releases. Market data from S&P Global Mobility and company filings.*
