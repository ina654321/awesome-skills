# Starbucks Rewards Program Guide

## Program Overview

Starbucks Rewards is one of the most successful retail loyalty programs globally, with **75 million members** driving **57% of U.S. sales** and representing over **$22 billion in stored value**.

## Program Structure (2026 Update)

### Three-Tier System

| Tier | Requirement | Key Benefits |
|------|-------------|--------------|
| **Green Status** | Entry | Birthday reward, personalized offers, Free Mod Mondays, digital games |
| **Gold Status** | Earn 300+ Stars annually | Stars never expire, extra Double Star days, birthday week window |
| **Reserve Status** | Invitation only | Exclusive merchandise, coffee experiences, 30-day birthday window, 6+ Double Star days |

### Earning Stars

| Activity | Stars Earned |
|----------|--------------|
| **$1 spent with registered card/app** | 2 Stars |
| **$1 spent with other payment** | 1 Star |
| **Digital reload of $30+** | 10 Stars (Green), 25 Stars (Gold) |
| **Bonus challenges** | Variable |
| **Double Star Days** | 4x Stars |

### Redeeming Stars

| Stars | Reward |
|-------|--------|
| **25** | Extra espresso shot, dairy substitute, syrup |
| **50** | Brewed coffee, bakery item, hot tea |
| **100** | Handcrafted drink, breakfast item |
| **150** | Lunch sandwich, protein box |
| **200** | Salad, premium sandwich |
| **400** | Select merchandise, packaged coffee |
| **60 (NEW)** | $2 discount on purchase |

## Best Practices for Members

### Maximizing Value

1. **Always Pay with App/Registered Card**
   - 2x Stars vs. 1x
   - Automatic reward tracking

2. **Reload in Larger Amounts**
   - Bonus Stars for $30+ reloads
   - Fewer transactions

3. **Check the App Regularly**
   - Personalized offers
   - Bonus Star challenges
   - Double Star days

4. **Redeem Strategically**
   - Higher value items = better Star value
   - 400-Star merchandise often best value

### Common Mistakes

- Forgetting to scan before paying
- Letting Stars expire (6 months without activity)
- Not checking for personalized offers
- Redeeming for low-value items
