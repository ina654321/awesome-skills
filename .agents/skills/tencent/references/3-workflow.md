## 3. Workflow

### 3.1 Tencent Product Development Process

```
┌─────────────────────────────────────────────────────────────────┐
│  PHASE 1: USER INSIGHT                                          │
│  ├── WeChat data analysis (with privacy safeguards)            │
│  ├── Mini Program ecosystem trends                             │
│  ├── Gaming player behavior analytics                          │
│  └── Market gap identification                                 │
├─────────────────────────────────────────────────────────────────┤
│  PHASE 2: MVP & ITERATION                                       │
│  ├── Rapid prototyping (internal dogfooding)                   │
│  ├── Limited WeChat release (A/B testing)                      │
│  ├── Data-driven optimization                                  │
│  └── Scale decision ("10-100-1000" growth targets)             │
├─────────────────────────────────────────────────────────────────┤
│  PHASE 3: ECOSYSTEM INTEGRATION                                 │
│  ├── WeChat Mini Program conversion                            │
│  ├── Payment integration (WeChat Pay)                          │
│  ├── Social sharing mechanics                                  │
│  └── Cross-platform sync (QQ, PC, mobile)                      │
├─────────────────────────────────────────────────────────────────┤
│  PHASE 4: MONETIZATION                                          │
│  ├── Freemium model (gaming DNA)                               │
│  ├── Value-add services (not pay-to-win)                       │
│  ├── Advertising (optional, non-intrusive)                     │
│  └── Enterprise/Cloud B2B revenue                              │
├─────────────────────────────────────────────────────────────────┤
│  PHASE 5: LONG-TERM OPERATION                                   │
│  ├── Live-ops content pipeline                                 │
│  ├── Community management                                      │
│  ├── Evergreen game treatment                                  │
│  └── AI-enhanced personalization                               │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 Decision-Making Checklist

**Before Building:**
- [ ] Does this serve user value first?
- [ ] Can this leverage WeChat's distribution?
- [ ] Is the regulatory path clear in China?
- [ ] Does it align with our AI strategy?

**Before Investing:**
- [ ] Is there WeChat ecosystem synergy?
- [ ] Can we provide unfair distribution advantage?
- [ ] Is the founding team world-class?
- [ ] Does this extend our platform capabilities?

**Before Global Expansion:**
- [ ] Is IP/compliance framework secure?
- [ ] Can gaming expertise transfer?
- [ ] Is cloud infrastructure ready?
- [ ] Are geopolitical risks assessed?
