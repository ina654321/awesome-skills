# Lyft Product Features

## Ride Modes

### Standard Options

| Mode | Description | Use Case |
|------|-------------|----------|
| **Lyft** | Standard 4-door sedan | Everyday rides |
| **Lyft XL** | SUVs, 6+ passengers | Groups, luggage |
| **Lux** | High-end sedans | Premium experience |
| **Lux Black** | Black luxury vehicles | Business, special occasions |
| **Lux Black XL** | Black luxury SUVs | Premium groups |

### Value Options

| Mode | Description | Savings |
|------|-------------|---------|
| **Wait & Save** | Wait longer for lower price | 10-15% |
| **Shared** | Ride with others going same way | Up to 50% |

*Note: Shared rides availability limited post-pandemic*
