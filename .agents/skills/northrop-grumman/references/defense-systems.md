## LGM-35A Sentinel (GBSD)

### Program Overview
The Ground-Based Strategic Deterrent (GBSD), now designated LGM-35A Sentinel, is the next-generation intercontinental ballistic missile (ICBM) replacing the aging Minuteman III.

### Program Scope
| Element | Details |
|---------|---------|
| **Prime Contractor** | Northrop Grumman |
| **Contract Award** | 2020 ($13.3B for EMD phase) |
| **Total Program Value** | ~$96 billion (projected over 20 years) |
| **Missiles** | 642 (400 deployed, 242 test/spares) |
| **Launch Facilities** | 450 (refurbished existing + new) |
| **Support Facilities** | 600+ across 40,000 sq miles |
| **States** | Wyoming, Montana, North Dakota, Colorado, Nebraska |
| **IOC Target** | 2029 (restructured) |
| **Full Deployment** | 2036 |

### Technical Features

**Missile Characteristics:**
- **Stages:** Three solid-fuel stages
- **Propulsion:** Modern solid rocket motors
- **Guidance:** Advanced inertial navigation with GPS backup
- **Warhead:** W87-1 in Mk21A reentry vehicle (initially single warhead per missile)
- **Range:** Intercontinental (>5,500 km)
- **CEP:** Significantly improved accuracy vs. Minuteman III

**Key Advancements:**
1. **Open Architecture:** Modular design enabling technology insertion
2. **Digital Infrastructure:** Modern command and control replacing 1970s systems
3. **Enhanced Security:** Cyber-hardened throughout
4. **Improved Maintainability:** Designed for 50+ year service life

### Infrastructure Modernization

**Launch Control Centers:**
- Replace 8-inch floppy disk-era systems
- Modern computers and communications
- Hardened against cyber threats

**Launch Facilities (Silos):**
- Refurbishment of 450 existing silos
- New communications infrastructure
- Improved blast doors and environmental systems

### Program Challenges

**Nunn-McCurdy Breach (2024):**
- Program exceeded cost thresholds triggering review
- Primary driver: Infrastructure complexity underestimated
- Missile development remains on track

**Restructuring (2024-2025):**
- Phased approach to infrastructure work
- Alignment of silo refurbishment with missile delivery
- Continued Air Force commitment to program

### Strategic Importance

**Deterrence Role:**
- "Warhead sponge" forces adversary to target 400+ hardened sites
- Fastest-response leg of nuclear triad
- Visible manifestation of strategic commitment

**Modernization Imperative:**
- Minuteman III first deployed 1970
- Cannot be life-extended indefinitely
- Aging infrastructure requires replacement

## Integrated Air and Missile Defense

### Integrated Battle Command System (IBCS)

**Description:** Revolutionary command and control for air/missile defense

**Innovation:**
- Decouples sensors from shooters
- Enables "any sensor, best shooter" operations
- Network-centric architecture

**Components:**
- Engagement operations centers
- Integrated fire control network
- Plug-and-fight capability

**Deployments:**
- U.S. Army IBCS battalions
- Poland WISŁA program

### Forward Area Air Defense Command and Control (FAAD C2)

**Description:** Command system for short-range air defense

**Users:**
- U.S. Army
- NATO allies (Estonia, Latvia, Lithuania)
- Allied integration for combined operations

### Other Missile Defense

- **Ground-based Interceptor (GBI) boosters** (solid rocket motors)
- **Patriot missile components** (Raytheon partnership)
- **Various classified programs**
