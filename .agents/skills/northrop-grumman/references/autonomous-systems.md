# Autonomous Systems

## Overview

Northrop Grumman is a world leader in autonomous systems, spanning high-altitude long-endurance (HALE) unmanned aircraft, vertical takeoff and landing (VTOL) tactical UAS, and undersea unmanned vehicles.

## RQ-4 Global Hawk

### Program Overview
The Global Hawk is the U.S. Air Force's first operational HALE UAV, providing persistent ISR capabilities worldwide.

### Variants

**RQ-4A Block 10**
- Original production variant
- Retired from USAF service

**RQ-4B Block 20/30/40**
- Current production standard
- Enhanced payload capacity
- Improved sensors

### Specifications (RQ-4B)
| Attribute | Details |
|-----------|---------|
| **Wingspan** | 130.9 ft (39.8 m) |
| **Length** | 47.6 ft (14.5 m) |
| **Height** | 15.3 ft (4.7 m) |
| **Max Takeoff Weight** | 32,250 lb (14,628 kg) |
| **Payload** | 3,000 lb (1,360 kg) |
| **Service Ceiling** | 60,000 ft (18,288 m) |
| **Endurance** | 32+ hours |
| **Range** | 8,700 nautical miles |
| **Engine** | Rolls-Royce AE 3007H turbofan |

### Sensor Payloads

**Integrated Sensor Suite (ISS):**
- **EO/IR:** Electro-optical/infrared camera
- **SAR:** Synthetic Aperture Radar
- **MTI:** Moving Target Indicator
- **SIGINT:** Signals intelligence (certain variants)

### Operational History

**Historic Firsts:**
- First UAV to fly non-stop across Atlantic (1998)
- First UAV to fly non-stop across Pacific (2001)
- First fully autonomous UAV in combat (Afghanistan, 2001)
- First UAV certified by FAA for U.S. national airspace

**Deployments:**
- Iraq, Afghanistan, Libya
- Japan, South Korea, NATO
- Maritime surveillance missions

## MQ-8 Fire Scout

### Program Overview
The MQ-8 Fire Scout is a vertical takeoff and landing tactical unmanned aerial vehicle (VTUAV) designed for maritime operations.

### Variants

**MQ-8A**
- Based on Schweizer 330 helicopter
- Limited production

**MQ-8B**
- Based on Schweizer 333
- Four-blade rotor (vs. three-blade on A model)
- Enhanced payload capacity
- Stub wings for weapons carriage

**MQ-8C**
- Based on Bell 407 helicopter
- Larger airframe (2x payload of B model)
- Longer endurance
- Current production variant

### Specifications (MQ-8C)
| Attribute | Details |
|-----------|---------|
| **Length** | 32.5 ft (9.9 m) |
| **Height** | 10.9 ft (3.3 m) |
| **Rotor Diameter** | 35.0 ft (10.7 m) |
| **Max Takeoff Weight** | 6,000 lb (2,722 kg) |
| **Payload** | 500+ lb (227+ kg) |
| **Endurance** | 12+ hours |
| **Service Ceiling** | 15,000+ ft |
| **Range** | 150+ nm |

### Capabilities

**ISR Mission:**
- Electro-optical/infrared sensors
- Surface search radar (AN/ZPY-8 Osprey)
- Laser designation
- Over-the-horizon targeting

**Weapons (MQ-8B):**
- APKWS laser-guided rockets
- Viper Strike precision munitions
- Hellfire missiles (planned)

### Operational Use

**Platforms:**
- Littoral Combat Ships (LCS)
- Guided missile destroyers
- Amphibious ships

**Deployments:**
- Counter-narcotics operations
- Maritime security
- Surface warfare support

**Historic Events:**
- First autonomous shipboard landings (2006)
- First drug bust by unmanned aircraft (2010)
- Washington D.C. restricted airspace incursion (2010) - led to procedural changes

## X-47B UCAS-D (Historical)

### Overview
Unmanned Combat Air System Demonstration program for U.S. Navy.

### Historic Achievements
- First autonomous aerial refueling by UAV (2015)
- First carrier launches and recoveries by UAV (2013)
- Demonstrated low-observable unmanned combat aircraft potential

### Legacy
Technology fed into:
- MQ-25 Stingray program (Boeing won production)
- Future unmanned combat aircraft programs

## Autonomous Capabilities

### Core Technologies

**Autonomous Navigation:**
- GPS-denied navigation
- Terrain-relative navigation
- Visual-inertial odometry

**Sensor Fusion:**
- Multi-sensor integration
- Onboard target recognition
- Automated cueing

**Human-Machine Interface:**
- Intuitive control stations
- High-level tasking vs. direct control
- Reduced operator workload

### Operational Advantages

1. **Persistence:** Multi-day endurance (HALE platforms)
2. **Reach:** Over-the-horizon operations
3. **Risk Reduction:** No aircrew in harm's way
4. **Cost Efficiency:** Reduced operating costs vs. manned aircraft
5. **Scalability:** Networked operations
