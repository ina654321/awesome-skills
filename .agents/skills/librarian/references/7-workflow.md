## § 7 · Workflow

### 7.1 Collection Development Process

**Phase 1: Assessment (Annual)**
```
□ Analyze circulation data (high/low performers)
□ Review holds ratios (unmet demand)
□ Conduct diversity audit (representation gaps)
□ Assess condition of physical materials
□ Evaluate e-resource usage
□ Gather patron requests and feedback
Deliverable: Collection assessment report
```

**Phase 2: Selection (Ongoing)**
```
□ Review professional reviews (Library Journal, Booklist, etc.)
□ Monitor award lists and bestseller lists
□ Consider patron requests
□ Ensure diversity and representation
□ Balance formats (print, digital, audio)
□ Evaluate cost and licensing terms
Deliverable: Selection orders
```

**Phase 3: Processing (Weekly)**
```
□ Receive and verify shipments
□ Catalog and classify materials
□ Add protective processing
□ Update ILS records
□ Prepare for shelving
Deliverable: Materials ready for circulation
```

**Phase 4: Maintenance (Quarterly)**
```
□ Weed based on MUSTIE criteria
□ Repair damaged items
□ Update catalog records
□ Shift collection for optimal arrangement
□ Analyze collection gaps
Deliverable: Current, relevant, accessible collection
```
