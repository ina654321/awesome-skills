# Applied Materials — Product Lines

## Semiconductor Systems

### 1. Deposition Systems

#### Physical Vapor Deposition (PVD)

**Endura® Platform**
- **Description**: Multi-chamber integrated processing platform
- **Processes**: PVD, CVD, etch, clean in single platform
- **Applications**: Interconnects, barriers, liners, metal gates
- **Key Features**:
  - Integrated metrology
  - Cluster tool architecture
  - High-throughput design
- **Latest Generation**: Endura 5500

**Applications:**
- Copper interconnect barriers (Ta/TaN)
- Metal gate deposition
- Hard masks
- MEMS manufacturing

#### Chemical Vapor Deposition (CVD)

**Centura® Platform**
- **Description**: Single-wafer CVD processing
- **Processes**: Dielectric CVD, tungsten CVD, selective CVD
- **Chamber Types**:
  - DxZ CVD (dielectric films)
  - DCXZ CVD (advanced dielectrics)
  - HDP CVD (high-density plasma)
  - WCVD (tungsten)
- **Latest Generation**: Centura 5200/5300

**Producer® Platform**
- **Description**: High-throughput CVD for memory applications
- **Specialty**: 3D NAND deposition
- **Key Feature**: High-productivity batch processing

**Film Types:**
- Silicon dioxide (SiO₂)
- Silicon nitride (SiN)
- Low-k dielectrics
- Tungsten (W)
- Polysilicon

#### Atomic Layer Deposition (ALD)

**Applications:**
- High-k gate dielectrics (HfO₂)
- Gate-All-Around (GAA) channel formation
- 3D NAND conformal liners
- Advanced memory cell structures

**Key Capabilities:**
- Sub-nanometer thickness control
- 100% conformality
- Atomic-scale precision

#### Epitaxy (Epi)

**Centura Epi**
- **Process**: Silicon, SiGe, SiGeC epitaxial growth
- **Applications**:
  - GAA nanosheet formation
  - Strained silicon for mobility enhancement
  - Advanced channel materials
- **Temperature Range**: 500-1200°C

### 2. Etch Systems

#### Dielectric Etch

**Centura DPS® (Decoupled Plasma Source)**
- **Description**: High-density plasma dielectric etch
- **Applications**:
  - Contact etch
  - Via etch
  - Trench etch
  - Hard mask open
- **Features**: Independent ion energy and density control

**Sym3® Series**
- **Latest**: Sym3 Magnum (>$1.2B since Feb 2024 launch)
- **Description**: Advanced dielectric etch for 3D structures
- **Applications**:
  - High-aspect ratio (HAR) etch
  - 3D NAND channel etch
  - DRAM capacitor etch
- **Key Capability**: Etch profiles >100:1 aspect ratio

#### Metal Etch

**Applications:**
- Aluminum interconnect etch
- Metal hard mask patterning
- Backside etch processes

#### Silicon Etch

**Applications:**
- TSV (Through-Silicon Via) formation
- MEMS release etch
- Nanosheet channel release

### 3. Patterning Solutions

#### EUV Lithography Enhancement

**Sculpta®**
- **Description**: Pattern-shaping system for EUV lithography
- **Function**: Corrects line edge roughness, improves CD uniformity
- **Benefit**: Extends EUV capability, reduces mask complexity
- **Technology**: Deposition and etch pattern-shaping

**Patterning Strategy:**
- Complement ASML EUV scanners
- Reduce lithography cost per layer
- Enable continued scaling beyond 3nm

### 4. Metrology & Inspection

#### Optical Metrology

**VeritySE®**
- **Description**: Spectroscopic ellipsometry
- **Applications**:
  - Film thickness measurement
  - Optical property characterization
  - Process monitoring

#### e-Beam Inspection

**PROvision®**
- **Description**: High-resolution e-beam inspection
- **Applications**:
  - Critical dimension measurement
  - Defect review
  - 3D structure inspection
- **Latest**: PROvision 4 for sub-10nm nodes

#### Overlay Metrology

- Advanced imaging for pattern overlay
- Critical for multi-patterning schemes
- Machine learning-enabled analysis

### 5. Ion Implantation

#### VIISta® Systems

**Product Range:**
- **VIISta 900**: Medium current
- **VIISta 3000**: High current
- **VIISta Trident**: High energy
- **VIISta PLAD**: Plasma doping

**Applications:**
- Source/drain doping
- Well formation
- Threshold voltage adjustment
- Advanced annealing

**Technology:**
- Precision dose control
- Angle accuracy for 3D structures
- Uniformity optimization

### 6. Chemical Mechanical Planarization (CMP)

#### Reflexion® Platform

**Description:**
- Integrated CMP and cleaning
- Multi-head polishing architecture
- In-situ endpoint detection

**Applications:**
- Interconnect planarization
- STI (shallow trench isolation)
- Tungsten CMP
- Advanced node dielectrics

**Key Features:**
- High removal rate with excellent uniformity
- Low defectivity
- Real-time process control

### 7. Thermal Processing

#### Rapid Thermal Processing (RTP)

**Radiance® Series**
- **Technology**: Lamp-based rapid heating
- **Applications**:
  - Spike anneal
  - Laser anneal
  - Oxidation
  - Nitridation
- **Temperature Control**: ±1°C uniformity

**Vantage® Series**
- Advanced RTP for leading-edge nodes
- Millisecond annealing capability
- Defect reduction technology
