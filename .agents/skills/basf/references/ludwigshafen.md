# Ludwigshafen Verbund Site Reference

## Overview

The **Ludwigshafen site** is BASF's headquarters and the world's largest integrated chemical complex owned by a single company. It is the birthplace of the Verbund concept.

| Attribute | Detail |
|-----------|--------|
| Area | ~10 km² |
| Employees | ~39,000 (BASF + partner companies) |
| Founded | 1865 (Friedrich Engelhorn) |
| Production plants | 200+ |
| Pipeline network | 2,850 km |
| Rail network | 230 km |
| Annual production | 8.5 million tonnes |
| Products | 8,000+ |

**Significance:**
- World's largest chemical complex
- Cradle of the Verbund concept
- Global R&D headquarters
- Model for sustainable production

- Waste heat from one process powers another
