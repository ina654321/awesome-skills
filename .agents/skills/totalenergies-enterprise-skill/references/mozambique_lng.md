# Mozambique LNG Project

## Project Overview

**Mozambique LNG** is one of the largest LNG projects in Africa and TotalEnergies' most significant development in the region. The project represents a $20+ billion investment and is the largest private investment in Africa.

| Attribute | Details |
|-----------|---------|
| **Project Name** | Mozambique LNG |
| **Location** | Offshore Cabo Delgado, northern Mozambique |
| **Operator** | TotalEnergies (26.5% stake) |
| **Total Investment** | $20+ billion |
| **Liquefaction Capacity** | 12.9 million tonnes per annum (Mtpa) |
| **Gas Reserves** | Golfinho and Atum fields (75 Tcf recoverable) |
| **First LNG Expected** | 2028-2029 (revised timeline) |

## Force Majeure and Restart

### Timeline of Events

| Date | Event |
|------|-------|
| 2019 | Final Investment Decision (FID) |
| 2020 | Construction begins |
| March 2021 | Force majeure declared |
| March 2021 - Oct 2025 | Construction suspended |
| October 2025 | Force majeure lifted |
| Late 2025 | Full activities resumed |
| 2028-2029 | Expected first LNG |

### Force Majeure Background

**Trigger Events (2021):**
- Islamic State-linked insurgency in northern Cabo Delgado
- Attacks on Palma town (near project site)
- Security deterioration in region
- Evacuation of personnel

**Impact:**
- All construction activities suspended
- Personnel withdrawn
- Equipment secured
- Timeline extended by 4+ years

### Security Situation

**Regional Stabilization (2023-2025):**
- Rwandan military deployment (supported by Mozambique government)
- SADC military presence
- Palma town recaptured and stabilized
- Security improved sufficiently for restart

**Current Status:**
- Force majeure lifted (October 2025)
- Full construction activities resumed
- Security protocols enhanced
- Continued military presence in region

### Offshore Facilities
- **Drilling**: Subsea wells, manifolds
- **Flowlines**: High-pressure gas export pipelines
- **Platform**: Floating production facility

| Component | Specification |
|-----------|--------------|
| **Liquefaction Trains** | 2 trains |
| **Capacity per Train** | ~6.45 Mtpa |
