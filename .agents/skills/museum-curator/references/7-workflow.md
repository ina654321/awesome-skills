## § 7 · Workflow

### 7.1 Exhibition Development Process

**Phase 1: Concept Development (6-12 months)**
```
□ Identify exhibition theme and thesis
□ Assess feasibility (budget, lenders, timeline)
□ Secure preliminary funding
□ Develop preliminary object list
□ Write exhibition proposal
Deliverable: Approved exhibition proposal
```

**Phase 2: Research & Planning (12-24 months)**
```
□ Deep research on objects and themes
□ Contact lenders (museums, private collectors)
□ Negotiate loan agreements
□ Write interpretive plan
□ Develop educational programming
□ Finalize checklist
Deliverable: Confirmed object list; interpretive plan
```

**Phase 3: Design & Production (6-12 months)**
```
□ Gallery layout and design
□ Case and mount design
□ Label writing and editing
□ Audio/multimedia production
□ Fabrication
□ Conservation treatment as needed
Deliverable: Complete exhibition package
```

**Phase 4: Installation (2-4 weeks)**
```
□ Gallery preparation
□ Object arrival and condition checking
□ Installation
□ Lighting adjustment
□ Final proofreading
□ Soft opening
Deliverable: Installed exhibition
```

**Phase 5: Opening & Programming (3-6 months)**
```
□ Member preview
□ Public opening
□ Public programs and tours
□ School group visits
□ Special events
Deliverable: Successful exhibition run
```
