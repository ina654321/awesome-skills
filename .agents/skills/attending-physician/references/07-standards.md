# Standards & Reference

## 7.1 Clinical Practice Guidelines

- [ACP Clinical Practice Guidelines](https://www.acponline.org/clinical-information/guidelines) - American College of Physicians evidence-based recommendations
- [NICE Guidelines](https://www.nice.org.uk/guidance) - UK National Institute for Health and Care Excellence
- [UpToDate](https://www.uptodate.com) - Evidence-based clinical decision support resource
- [CDC Clinical Practice Guidelines](https://www.cdc.gov/clinical-guidance/index.html) - Disease-specific guidance
- [WHO Guidelines](https://www.who.int/clinical-guidelines) - International health standards

## 7.3 Medical Decision-Making Frameworks

```
Safety/Risk Assessment:
├── Probability-based diagnosis (pre-test/post-test probability)
├── Risk-benefit analysis framework
├── Shared decision-making model
└── Clinical prediction rules (Wells, PERC, HEART)

Diagnostic Reasoning:
├── System 1 (intuitive): Pattern recognition
├── System 2 (analytical): Hypothetico-deductive reasoning
└── Dual-process theory application
```

## 7.4 Evidence-Based Medicine

| Level | Evidence Type | Recommendation Strength |
|-------|--------------|------------------------|
| Ia | RCT systematic review | Strong recommendation |
| Ib | Individual RCT | Strong recommendation |
| IIa | Cohort study | Moderate recommendation |
| IIb | Case-control study | Moderate recommendation |
| III | Case series | Weak recommendation |
| IV | Expert opinion | Expert guidance only |
