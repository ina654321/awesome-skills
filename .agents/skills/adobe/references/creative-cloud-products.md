# Creative Cloud Product Matrix

## Core Applications

### Photoshop

**Version**: 2025 (v26.x)
**Category**: Image Editing & Compositing

**Core Capabilities**:
- Photo editing and retouching
- Digital painting and illustration
- 3D design capabilities
- Motion graphics preparation

**AI-Powered Features (Firefly)**:
| Feature | Description | Availability |
|---------|-------------|--------------|
| Generative Fill | Add/remove/replace image areas | GA |
| Generative Expand | Extend canvas with AI content | GA |
| Distraction Removal | Auto-detect people/wires | GA |
| Generate Similar | Create variations of selections | GA |
| Generative Workspace | Multi-prompt brainstorming | Beta |
| Selection Brush | AI-powered subject selection | GA |

**File Formats**:
- Native: PSD, PSB
- Export: JPEG, PNG, TIFF, GIF, WebP, HEIF
- Video: MP4, MOV (timeline export)

**System Requirements**:
- GPU: DirectX 12 or OpenGL 4.x
- RAM: 8GB minimum, 16GB recommended
- Storage: 4GB available space

## Creative Cloud Services

### Adobe Stock

**Library Size**: 300M+ assets

| Asset Type | Count | License Types |
|------------|-------|---------------|
| Photos | 200M+ | Standard, Extended |
| Videos | 20M+ | Standard, Extended |
| Templates | 100K+ | Standard |
| Vectors | 50M+ | Standard |
| Music | 100K+ | Standard, Premium |

**Integration**:
- In-app search across Creative Cloud
- License directly within apps
- Visual search (upload image to find similar)

### Adobe Fonts

**Library**: 20,000+ fonts

**Features**:
- Desktop sync (unlimited activations)
- Web font hosting
- Font activation in Creative Cloud

### CC Libraries

**Purpose**: Cross-app asset sharing

**Supported Elements**:
- Colors (ASE, Hex, RGB)
- Character Styles
- Layer Styles
- Graphics (AI, PSD, PNG)
- Video LUTs

### Frame.io

**Capabilities**:
- Cloud-based video review
- Time-stamped comments
- Version comparison
- Camera to Cloud (C2C)

**Integration**:
- Premiere Pro: Native panel
- After Effects: Import comments
- iOS/Android: Mobile review
