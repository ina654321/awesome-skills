# Standards & Reference

## 7.1 Magic Categories

### Close-Up Magic
| Category | Description | Venue |
|----------|-------------|-------|
| **Card Magic** | Cards, flourishes, sleights | Tables, walking around |
| **Coin Magic** | Palms, vanishes, production | Close-up, strolling |
| **Mentalism** | Mind reading, predictions | Intimate settings |
| **Bizarre Magic** | Storytelling, mystical | Halloween events |

### Stage Magic
| Category | Description | Venue |
|----------|-------------|-------|
| **Illusions** | Large-scale illusions | Theater, stage |
| **Mentalism** | Mind magic on stage | Cabaret, theater |
| **Escape Arts** | Houdini-style escapes | Stage, close-up |

### Specialty
| Category | Description |
|----------|-------------|
| **Street Magic** | Bold, direct, camera-friendly |
| **Corporate Magic** | Business themes, icebreakers |
| **Children's Magic** | Colorful, interactive, silly |
| **Comedy Magic** | Humor integrated with magic |

## 7.2 Essential Sleight of Hand

### Card Sleights (Difficulty Rating)
| Sleight | Rating | Use |
|---------|--------|-----|
| **Classic Force** | Medium | Make spectator choose specific card |
| **Double Lift** | Medium | Show top card as chosen |
| **Pass** | Hard | Change order secretly |
| **Tenkai Palm** | Hard | Palm card at back of deck |
| **Top Change** | Medium | Change top two cards |
| **Hofzinser** | Expert | Multiple selection process |

### Coins (Basic)
| Sleight | Description |
|---------|-------------|
| **Classic Palm** | Hide coin in palm |
| **French Drop** | Vanish coin with hand |
| ** Shuttle Pass** | Pass coin between hands |
| **Downs Palm** | Palm multiple coins |

## 7.3 Performance Standards

### The Three Ps of Magic
1. **Preparation**: Practice until you can't get it wrong
2. **Presentation**: Story, emotion, connection with audience
3. **Performance**: Execution under pressure

### Professional Conduct
| Standard | Requirement |
|----------|-------------|
| **Punctuality** | Arrive 30+ minutes early |
| **Appearance** | Professional attire; clean and neat |
| **Setup** | Everything ready before audience arrives |
| **Cleanup** | Leave venue as you found it |
| **Payment** | Clear terms before booking |

### Audience Management
- Select helpers fairly (avoid always choosing same type)
- Build up helpers before asking them to participate
- Never embarrass or single out inappropriately
- Give helpers a moment to shine
