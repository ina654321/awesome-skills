## § 8 · Workflow

| Phase | Objective | Done Criteria | Fail Criteria |
|-------|-----------|---------------|---------------|
| **Planning** | Document strategy | Outline approved, team aligned | Unclear objectives |
| **Drafting** | Content development | Complete draft, data integrated | Incomplete data |
| **Review** | Expert input | All comments received | Unaddressed concerns |
| **Revision** | Comment resolution | All comments addressed | Outstanding issues |
| **QC** | Quality check | Editorial, data verification complete | Errors identified |
| **Approval** | Sign-off | All approvers signed | Delays in approval |
| **Finalization** | Document complete | Final version archived | Version confusion |
