# Khanmigo: AI Tutor Features Reference

## Overview

Khanmigo is Khan Academy's GPT-4 powered AI tutor and teaching assistant, launched in 2023 and expanded to 2M+ users by 2025.

## Core Philosophy

> "Nobody learns anything by being given the answer. Instead, Khanmigo asks questions. Questions are thought-provoking. Questions are open-ended. And most importantly, questions lead to learning."
> — Sal Khan

## Student Features

### Socratic Tutoring
- Never gives direct answers
- Asks guiding questions
- Encourages productive struggle
- Builds critical thinking skills

### Subject Coverage
- Mathematics (K-12 through Calculus)
- Science (Biology, Chemistry, Physics)
- Humanities (History, Literature)
- Computer Science (Python, JavaScript)
- Test Prep (SAT, LSAT)

### Interactive Activities
- **Chat with Historical Figures:** Converse with Cleopatra, Jane Austen, etc.
- **Debate Mode:** Argue topics like "Should schools ban AI?"
- **Writing Coach:** Real-time essay feedback
- **Code Tutor:** Programming guidance without writing code

## Teacher Features

### Lesson Planning Tools
| Tool | Function | Time Saved |
|------|----------|------------|
| Lesson Plan Generator | Creates structured plans with objectives, activities, exit tickets | ~45 min |
| Rubric Generator | Standards-aligned assessment rubrics | ~60 min |
| Discussion Prompts | Generates engaging conversation starters | ~20 min |
| Exit Ticket Creator | Formative assessment questions | ~15 min |

### Differentiation Support
- **Leveler:** Adjusts reading complexity of texts
- **Make it Relevant!:** Connects topics to student interests
- **IEP Assistant:** Streamlines Individualized Education Plan creation
- **Alternative Explanations:** Multiple ways to explain concepts

### Classroom Management
- Student chat history summaries (anonymized)
- Assignment recommendations based on data
- Progress insights dashboard
- Intervention identification
